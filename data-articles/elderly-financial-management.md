# お金の管理ができなくなった親に家族ができること

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  在宅訪問で独居の患者様を担当していたときのことです。定期的に薬をお届けしていましたが、ある日、居間に未開封の薬の袋が山積みにされているのを見つけました。<br><br>
  詳しくお話を伺うと、患者様は「お金の計算が合わなくて不安で、何も手につかない」と仰るのです。詳しく聞くと、支払い忘れが重なり、公共料金の催促状が届いてパニックになっていました。薬を飲むどころか、生活そのものが破綻しかけていたのです。その時、私は「お薬の管理」だけでなく「生活の土台」が揺らいでいることに気づかされました。
</div>

お薬の管理はできていても、生活の根幹である「お金の管理」が崩れると、治療の継続は困難になります。今回は、親のお金の管理に不安を感じ始めた家族に対し、私たち薬剤師がどのようなアドバイスや連携ができるかを整理しました。

---

## 💰 1. 親のお金の管理、どこからが「危険信号」？

加齢や認知機能の低下により、これまでできていたことが急にできなくなることがあります。家族が見逃してはならない「サイン」は以下の通りです。

| チェックポイント | 具体的な兆候 |
| :--- | :--- |
| <strong>支払いの滞納</strong> | 公共料金や家賃の督促状が放置されている |
| <strong>収支の混乱</strong> | 大量の同じもの（食品や日用品）を買い込んでいる |
| <strong>資産の不自然な減少</strong> | 通帳や現金の残高が理由なく大幅に減っている |
| <strong>金銭感覚の変化</strong> | 高額な訪問販売や怪しい投資話に乗りやすい |
| <strong>管理能力の欠如</strong> | 請求書や領収書が部屋中に散乱し、整理できない |

---

## ⚖️ 2. 家族ができる法的な対応策

お金の管理が困難になった場合、早めの準備が将来のトラブルを防ぎます。特に「成年後見制度」や「家族信託」というキーワードは、家族から相談された際に提示できる選択肢として重要です。

### ① 成年後見制度（法定後見）
認知症などが進行し、自分で判断することが難しくなった場合に利用します。家庭裁判所を通じて支援者（後見人）を選任してもらい、財産管理を行います。
- <strong>メリット</strong>: 専門家が管理するため不正が起きにくい。
- <strong>デメリット</strong>: 一度開始すると、本人が亡くなるまで原則やめられない。

### ② 家族信託（民事信託）
判断能力があるうちに、信頼できる家族に財産の管理を任せる契約を結びます。
- <strong>メリット</strong>: 柔軟な管理が可能。本人の意思を反映しやすい。
- <strong>デメリット</strong>: 専門家（弁護士や司法書士）による設計が必要で、費用がかかる。

## 💊 3. 薬剤師が現場でできる「支援」の形

私たち薬剤師に、金銭管理の代行はできません。しかし、<strong>「生活が立ち行かなくなる予兆」をいち早くキャッチし、適切な窓口へ繋ぐこと</strong>は可能です。

### ケアマネジャーとの情報共有
服薬が疎かになっている背景に金銭管理の問題がある場合、早急にケアマネジャーへ報告しましょう。訪問看護やヘルパーの介入強化、あるいは「日常生活自立支援事業（社会福祉協議会）」への橋渡しが必要かもしれません。

### 「お薬代」の支払いをスムーズにする工夫
金銭管理が不安定な患者様には、以下の工夫が有効です。
* <strong>口座振替・カード決済の確認</strong>: 現金のやり取りを減らし、自動引き落としに移行するよう家族に提案する。
* <strong>お薬代の精算方法の整理</strong>: 薬局での支払いが負担であれば、月単位の請求書払いや、家族による代理支払いへの変更を検討する。

## 🌿 4. まとめ：孤立させないための連携

親のお金の管理は、非常にデリケートで家族内でも話しにくい問題です。しかし、そこを無視して「服薬管理」だけを徹底することは不可能です。

1. <strong>早期発見</strong>: 滞納や無駄遣いのサインを見逃さない。
2. <strong>専門職への相談</strong>: 地域包括支援センターやケアマネジャーに相談を促す。
3. <strong>制度の利用</strong>: 成年後見制度などの法的手段を検討する。

薬剤師として「薬」というツールを通じて患者様の生活を見守り、必要に応じて専門家へとバトンを渡すこと。それが、患者様とご家族の生活を守るための大切なミッションだと考えます。
