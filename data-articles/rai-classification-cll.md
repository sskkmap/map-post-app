# Rai分類とは？慢性リンパ性白血病の進行度評価法

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ノート：CLL患者さんとの対話】</strong><br>

  血液がんという響きに不安を感じつつも、治療の優先順位がすぐには高くないことへの戸惑いも感じられました。その際、病期分類の意味を臨床検査値（リンパ球数、脾腫、貧血など）と結びつけて説明したところ、患者さんの表情が少し和らぎました。がん薬物療法の進歩は早いですが、まずは「今、患者さんがどの段階にいるのか」を正しく理解し、寄り添うことの重要性を再認識した出来事でした。
</div>

慢性リンパ性白血病（CLL）は進行が緩やかなケースも多いため、すべての患者さんに即座に強力な治療が必要なわけではありません。そこで重要になるのが、<strong>「Rai分類（ライ分類）」</strong>を用いた進行度評価です。今回は、薬剤師として把握しておくべきRai分類の基礎知識と、治療方針の決定に関わるポイントを解説します。

---

## 📊 1. Rai分類による進行度評価（リスク層別化）

Rai分類は、リンパ球増加、リンパ節腫脹、肝脾腫、貧血、血小板減少という5つの項目を組み合わせ、リスクをLow, Intermediate, Highの3段階に分類します。

| リスク群 | 病期 | 評価項目（特徴） | 治療方針の目安 |
| :--- | :--- | :--- | :--- |
| <strong>Low</strong> | Stage 0 | リンパ球増加のみ | 経過観察（Watch and Wait） |
| <strong>Intermediate</strong> | Stage I | Stage 0 ＋ リンパ節腫脹 | 経過観察 または 治療検討 |
| | Stage II | Stage 0 ＋ 肝脾腫（±リンパ節腫脹） | 経過観察 または 治療検討 |
| <strong>High</strong> | Stage III | Stage 0 ＋ 貧血（Hb < 11.0g/dL） | 治療適応 |
| | Stage IV | Stage 0 ＋ 血小板減少（Plt < 100×10⁹/L） | 治療適応 |

※貧血や血小板減少の原因が他の疾患（自己免疫性溶血性貧血など）に起因する場合を除外して評価します。

---

## 🔍 2. 薬剤師が押さえておくべきチェックポイント

Rai分類は単なる病期分けではなく、<strong>「今、治療を開始すべきか」</strong>を判断する重要な指標です。特に以下の点に注目して患者さんのデータを確認しましょう。

### 🚨 貧血・血小板減少のサイン（Stage III・IV）
Stage IIIやIVは治療介入の明確なサインです。血液検査値の推移を確認し、Hb値や血小板数の急激な低下がないかチェックします。また、貧血に伴う倦怠感や血小板減少による出血傾向（紫斑、歯肉出血など）の聞き取りも欠かせません。

### ⚖️ 経過観察（Watch and Wait）の重要性
LowやIntermediateのリスク群では、無理に治療を開始せず経過観察となることが一般的です。「治療しない＝放置」ではなく、<strong>「無益な毒性を避けるための慎重な管理」</strong>であることを、患者さんに正しく伝えることも薬剤師の役割です。

## 📝 まとめ：データと患者さんのQOLをつなぐ

Rai分類は、CLLの治療において非常に実用的なガイドラインです。

*   <strong>Low/Intermediate</strong>: 患者さんの日常の様子、感染症リスクに注意したモニタリングを。
*   <strong>High</strong>: 治療介入のタイミングを逃さず、副作用管理（特に分子標的薬特有の心血管系リスクや感染症）に注力する。

「Rai分類のステージが上がった」という事実は、患者さんにとって大きなストレスです。数値だけでなく、患者さんの生活の質（QOL）をどう守っていくか、私たち薬剤師は常に多職種と連携しながら考えていきましょう。
