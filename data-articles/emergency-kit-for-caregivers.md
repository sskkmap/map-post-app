# 介護中の家族が備えるべき非常用品とは？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験記】</strong><br>
  数年前の台風による停電時、訪問先のお宅で非常に困った経験があります。呼吸器を使用している患者さんや、インスリンが必要な患者さんが、「いつもの薬」がどこにあるか分からず、暗闇の中で混乱されていたのです。<br><br>
  「薬はここにあるけれど、名前が読みづらくて…」「予備のオムツはどこだっけ？」というご家族の不安な声を聞き、薬局としてもっと事前の準備をサポートしておくべきだったと痛感しました。災害は「いつもの日常」を瞬時に奪います。だからこそ、介護者が迷わず動けるための「事前の準備」が不可欠なのです。
</div>

今回は、介護されているご家族を守るために、薬剤師の視点から「これだけは備えておきたい」というポイントを整理しました。

## 🎒 1. 災害時に「まず持ち出すもの」チェックリスト

避難が必要になった際、介護者はご本人の移動介助だけで手一杯になります。<strong>「これさえ持てば最低限なんとかなる」という専用の避難セット</strong>を玄関に置いておくことが鉄則です。

| 項目 | 具体的な内容 | 薬剤師のアドバイス |
| :--- | :--- | :--- |
| <strong>お薬情報</strong> | お薬手帳・薬剤情報提供書 | コピーを取って防水袋に。処方医の連絡先も記載を。 |
| <strong>常用薬</strong> | 7日分程度の予備薬 | 薬局で「災害備蓄用」として相談し、重複期間を調整。 |
| <strong>衛生用品</strong> | オムツ、清拭剤、手袋 | 嵩張るため、圧縮袋を活用して最小限に。 |
| <strong>医療機器</strong> | 予備の電池・電源 | 呼吸器等を使用している場合、充電器の場所を明確に。 |
| <strong>本人情報</strong> | 介護保険証・身分証 | 避難先で医療を受ける際に必須です。 |

## 💊 2. 「お薬」の備蓄はどうすればいい？

「災害用だからといって、勝手に薬を多めに処方してもらうことはできるの？」と聞かれることがあります。結論から言うと、<strong>「災害時対応の枠組み」がある場合は可能です。</strong>

<div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe;">
  <strong>💡 薬剤師からのワンポイント：災害備蓄薬の考え方</strong><br>
  普段から「災害時持ち出し用として、1週間分だけ予備を作っておきたい」と主治医や薬剤師に相談しておきましょう。特に、<strong>インスリンなどの要冷蔵製剤や、急な断薬が命に関わる精神薬</strong>などは、優先的に備蓄を検討すべきです。
</div>

### 災害時に役立つ情報源リスト
- <strong>お薬手帳のデータ</strong>: 常に最新の状態に更新し、アプリ版と紙の両方で備えておく。
- <strong>お薬のコピー</strong>: 避難袋の中に、現在服用中の「薬の名称・用法用量・注意点」を紙で入れておく（被災地で薬が切れた際、薬剤師が代用薬を判断する重要な材料になります）。

---

## 🚰 3. 介護現場ならではの「隠れた必需品」

意外と忘れがちですが、避難所や停電時には「水」が貴重になります。以下のアイテムがあると、介護者の負担が大きく減ります。

1. <strong>清拭用ウェットティッシュ（大判）</strong>: お風呂に入れない日が続いた際、体を拭くための大きなウェットティッシュは必須です。
2. <strong>ドライシャンプー</strong>: 頭皮の清潔はご本人のQOL維持に直結します。
3. <strong>高エネルギーの経腸栄養剤・ゼリー</strong>: 食欲がない時でも摂取できる、メイバランス等の栄養補助食品を数日分ストックしましょう。
4. <strong>ゴミ袋（消臭タイプ）</strong>: オムツの廃棄問題は避難所では切実です。消臭袋は介護の必需品です。

## 🛠️ 4. 薬剤師に今すぐ相談してほしいこと

「もしもの時にどうすればいいか分からない」という不安は、専門家と共有することで具体策が見えてきます。ぜひ、次回の訪問や来局時に薬剤師へ以下の質問を投げかけてみてください。

* 「今の薬は、もし飲み忘れてもすぐに危険な状態にならないものですか？」
* 「今の処方薬、災害時に手に入りにくそうなものはありますか？」
* 「服用中の薬と併用してはいけない市販薬はありますか？」
