# Child-Pugh分類とは？肝硬変の重症度判定に用いられる指標を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場あるある：その用量、肝臓は耐えられますか？】</strong><br>
  ある日の外来調剤で、肝硬変の既往がある患者さんに「NSAIDs」が処方されていました。「先生、肝硬変の方は代謝や蛋白結合率の変化があるから慎重投与ですよね…」と冷や汗をかきながら疑義照会をした経験はありませんか？<br><br>
  検査値だけ見て「肝機能が悪そう」と判断しがちですが、肝硬変の重症度を客観的に評価するツールが「Child-Pugh（チャイルド・ピュー）分類」です。このスコアを知っているだけで、処方提案の説得力は格段に上がります。「なんとなく怖い」から「根拠を持って慎重に」へシフトするための知識を共有します。

## 📊 1. Child-Pugh分類の算出項目とスコアの読み方

Child-Pugh分類は、肝硬変患者の予後予測や、薬物投与量の調整の指標として広く用いられています。臨床検査値と身体所見からなる「5つの項目」で構成されています。

<div style="overflow-x: auto;">
| 判定項目 | 1点 | 2点 | 3点 |
| :--- | :--- | :--- | :--- |
| <strong>肝性脳症</strong> | なし | Ⅰ～Ⅱ度 | Ⅲ～Ⅳ度 |
| <strong>腹水</strong> | なし | 軽度（コントロール可） | 中等度～高度 |
| <strong>総ビリルビン (mg/dL)</strong> | 2.0未満 | 2.0～3.0 | 3.0超 |
| <strong>アルブミン (g/dL)</strong> | 3.5超 | 2.8～3.5 | 2.8未満 |
| <strong>プロトロンビン活性 (%)</strong> | 70超 | 40～70 | 40未満 |
</div>

### 重症度判定基準
合計点数によって、以下のように分類されます。

*   <strong>Child-Pugh A (5～6点)</strong>：軽症（代償期）
*   <strong>Child-Pugh B (7～9点)</strong>：中等症
*   <strong>Child-Pugh C (10～15点)</strong>：重症（非代償期）

## ⚠️ 3. 疑義照会・処方提案のヒント

Child-Pugh分類を確認し、以下のような処方箋が来た場合は注意が必要です。

### ① 代謝経路を確認する
肝代謝型薬剤（特に初回通過効果を受けるもの）は、Child-Pugh B〜Cの患者さんでは<strong>常用量の半分以下から開始</strong>、あるいは慎重に用量調節することが推奨されるケースが多いです。添付文書の「肝機能障害患者への投与」欄を必ずチェックしましょう。

### ② 禁忌薬を再確認する
肝硬変患者において禁忌、あるいは慎重投与とされる代表的な薬剤には以下のようなものがあります。
*   <strong>NSAIDs</strong>：腎血流量の低下や凝固能低下（プロトロンビン時間延長）を助長し、消化管出血リスクが高まるため避けるべきです。
*   <strong>ベンゾジアゼピン系睡眠薬</strong>：肝性脳症を誘発・悪化させる恐れがあるため、肝代謝を受けにくい（抱合体で代謝される）オキサゼパム等への変更を検討します。

### ③ 「腹水」や「浮腫」の有無を確認する
腹水のコントロール不良はChild-Pughスコアに直結します。利尿薬が処方されている場合、低カリウム血症による「肝性脳症」リスクを考慮し、スピロノラクトンの併用や、カリウム値のモニタリング提案を行うと非常に実用的です。

## 🌿 4. まとめ：検査値を見て、臨床症状を想像する

Child-Pugh分類は、単なる「点数」ではありません。
*   <strong>総ビリルビン</strong>の上昇 ＝ 黄疸の危険
*   <strong>アルブミン</strong>の低下 ＝ 低栄養と浮腫の危険
*   <strong>プロトロンビン時間</strong>の延長 ＝ 出血リスク
といった<strong>患者さんの全身状態を映し出す鏡</strong>です。

処方箋を見たときに、「この患者さんのChild-Pugh分類は何点くらいだろう？」と検査値に目を配り、もしBやCに該当しそうな場合は、薬剤の減量や代替薬の検討を医師へ提案してみてください。

その一声が、肝硬変患者さんの副作用リスクを大きく減らす、薬剤師にしかできない「攻めの薬学管理」になります！
