# 🌡️ 熱傷指数とは？熱傷重症度の評価方法をわかりやすく解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験談】</strong><br>
  ある日、当直中に救急外来から「広範囲の熱傷患者さんに軟膏を処方したい」という連絡が入りました。慌てて軟膏の必要量を計算しようとした際、医師から「熱傷指数（BI）は○点だから、重症度も考慮して計算してくれ」と言われ、一瞬フリーズしてしまった経験があります。<br><br>
  学生時代に習ったはずの「9の法則」や「熱傷指数」ですが、現場で即座にアウトプットできる準備ができている薬剤師は案外少ないものです。特に、熱傷指数は輸液の必要量や軟膏の選択だけでなく、予後予測の指標にもなります。今回は、あの日冷や汗をかいた自分への戒めも込めて、熱傷評価の基本を解説します。
</div>

熱傷患者さんの対応において、私たちがまず知っておくべきは<strong>「どの程度の広さで、どの深さまで到達しているか」</strong>という客観的な指標です。

---

## 🔢 1. 熱傷の広さと深さを測る「ものさし」

熱傷の重症度は、<strong>「熱傷面積（％）」</strong>と<strong>「熱傷の深さ（I〜III度）」</strong>の組み合わせで決まります。ここで重要になるのが<strong>「熱傷指数（Burn Index: BI）」</strong>です。

### 熱傷指数の計算式
<div style="background-color: #fefce8; padding: 1rem; border-radius: 8px; border: 1px solid #fde68a; text-align: center; font-weight: bold; font-size: 1.2rem; color: #854d0e;">
  熱傷指数（BI）＝ Ⅱ度熱傷面積（％）× 1/2 ＋ Ⅲ度熱傷面積（％）
</div>

※計算式からもわかる通り、II度熱傷よりもIII度熱傷の方が重症度への寄与が大きくなります。

### 熱傷の深さによる特徴（薬剤師の観察ポイント）
軟膏選択において「壊死組織の除去（デブリードマン）」が必要か、「保湿」が優先かを見極める指標です。

| 熱傷の深さ | 特徴 | 薬剤師の視点（軟膏・外用薬） |
| :--- | :--- | :--- |
| <strong>I度</strong> | 発赤のみ、痛みあり | 対症療法。保湿剤などで十分な場合が多い |
| <strong>浅達性Ⅱ度(SDB)</strong> | 水疱形成、強い痛み | 感染対策と治癒促進。ゲーベン、アズノールなど |
| <strong>深達性Ⅱ度(DDB)</strong> | 水疱、痛み鈍い、白〜赤 | 感染リスク高。抗菌薬含有軟膏の検討が必要 |
| <strong>Ⅲ度</strong> | 蒼白・炭化、痛みなし | 神経まで損傷。デブリードマン必須、外科的対応 |

## 📏 2. 面積の評価方法：「9の法則」を再確認

頭から足先までを9％単位で割り振る<strong>「9の法則（Rule of Nines）」</strong>が一般的です。

<div style="overflow-x: auto;">
| 部位 | 面積（％） |
| :--- | :--- |
| 頭頸部 | 9% |
| 右上肢 | 9% |
| 左上肢 | 9% |
| 体幹前面 | 18% |
| 体幹後面 | 18% |
| 右下肢 | 18% |
| 左下肢 | 18% |
| 会陰部 | 1% |
</div>

*※掌（手のひら）を患者さんの手掌面積とすると、だいたい1％相当です。不規則な熱傷範囲を推測する際の参考にしてください。*

## 🏥 3. 熱傷指数と重症度の目安

BIは輸液量や転院の判断基準になります。以下の表を頭の片隅に入れておきましょう。

<div style="overflow-x: auto;">
| 重症度 | 熱傷指数（BI）の目安 | 薬剤師として気をつけるポイント |
| :--- | :--- | :--- |
| <strong>軽症</strong> | 10未満 | 外来通院での処方が中心。軟膏の塗布量指導が重要。 |
| <strong>中等症</strong> | 10〜20 | 入院治療の検討。鎮痛剤の併用や感染対策が必要。 |
| <strong>重症</strong> | 20以上 | 全身管理。輸液による蘇生が必要で、代謝変動にも注意。 |
</div>

<div style="background-color: #eff6ff; border: 1px solid #bfdbfe; padding: 1rem; border-radius: 8px; margin-top: 1rem;">
  <strong>💡 薬剤師の実務アドバイス</strong><br>
  重症熱傷では、代謝が亢進し、薬剤のクリアランスが大きく変化します。また、広範囲に外用薬を使用する場合、薬剤の<strong>経皮吸収</strong>による副作用や、全身への影響を考慮しなければなりません。特に小児や高齢者の場合は注意が必要です。
</div>

## 💊 4. まとめ：数値から治療プランを想像する

熱傷指数を算出することで、私たち薬剤師は以下の情報を医師や看護師と共有できるようになります。

1.  <strong>必要な外用薬の量</strong>: 面積から総量を予測し、不足がないか確認する。
2.  <strong>感染リスクの推測</strong>: 深さと面積から、抗菌薬入りの外用薬が必要か、全身性の抗菌薬投与が開始されるかを想定する。
3.  <strong>全身状態のモニタリング</strong>: BIが高い場合、輸液管理や栄養サポート（NST）の重要性が増す。

「熱傷指数」は単なる数字ではなく、<strong>患者さんが今、どのような状態にあり、今後どのようなリスクが待ち受けているかを知るための「羅針盤」</strong>です。

次に救急外来から処方箋や相談が来た際は、ぜひこの計算を思い出してみてください。知識を持っているだけで、チーム医療における薬剤師としての説得力がグッと高まりますよ！
