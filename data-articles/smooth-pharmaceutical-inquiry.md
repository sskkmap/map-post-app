# 疑義照会をスムーズに行うための準備とは

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  新人薬剤師だった頃、処方箋の記載内容に違和感があり、「先生に電話しなきゃ…」と冷や汗をかきながら受話器を握りしめていました。いざ繋がると、しどろもどろな説明をしてしまい、医師から「で、結局何が言いたいの？」と冷たい一言。自信を失い、その後しばらく疑義照会が怖くてたまらなくなりました。<br><br>
  しかし、先輩から<strong>「疑義照会は『質問』ではなく『提案』の場である」</strong>と教わり、事前準備の重要性を痛感しました。今では、準備を整えることで、短時間で感謝される疑義照会ができるようになりました。
</div>

疑義照会は、患者さんの安全を守るための薬剤師の大切な仕事です。しかし、忙しい医師とのやり取りに苦手意識を持つ方は少なくありません。今回は、明日からの業務が少し楽になる「疑義照会をスムーズに行うための準備」について解説します。

---

## 🔍 1. 疑義照会をスムーズにする「3つの準備」

電話をかける前に、以下の3点を整理するだけで、会話の質が劇的に変わります。

| 準備項目 | 内容 | 目的 |
| :--- | :--- | :--- |
| <strong>根拠の明確化</strong> | ガイドライン、添付文書、インタビューフォーム | 自分の推測ではなく「客観的事実」を用意する |
| <strong>患者背景の確認</strong> | 腎機能（eGFR）、併用薬、アレルギー歴 | 医師が見落としている情報を補足する |
| <strong>提案の具体化</strong> | 代替薬の提案、休薬、用量調節 | 「どうしたいか」を明確にして選択肢を提示する |

### 🛠 準備の手順
1. <strong>問題点の抽出</strong>: 「なぜ確認が必要か」を1文で言えるようにする。
2. <strong>情報の収集</strong>: 添付文書の「禁忌」や「用法・用量」の項目をすぐ開けるようにしておく。
3. <strong>代案の用意</strong>: 「Aという問題があるので、Bへの変更は可能でしょうか？」という形に落とし込む。

---

## ⚡ 2. 医師に伝わる「話し方」の鉄則

医師との電話は「手短に、かつ正確に」が基本です。以下の構成で話すと、相手も状況を即座に理解しやすくなります。

<div style="background-color: #eff6ff; padding: 1.5rem; border-radius: 8px; border: 1px solid #bfdbfe;">
  <h4 style="margin-top: 0; color: #1e3a8a;">提案型・疑義照会テンプレート</h4>
  <ol style="margin-bottom: 0; color: #1e40af;">
    <li><strong>「◯◯先生の処方で、××という点について確認（または提案）がございます」</strong>（結論）</li>
    <li><strong>「現在、患者様の腎機能が低下しており、添付文書上では△△の減量が必要とされています」</strong>（根拠）</li>
    <li><strong>「今回のお薬、〇〇mgへの減量、もしくは中止をご検討いただけますでしょうか？」</strong>（代案）</li>
  </ol>
</div>

このように、<strong>「事実 → 理由 → 提案」</strong>の順で話すことで、プロフェッショナルな印象を与え、結果として疑義照会がスムーズに進みます。

---

## 📈 3. 効率を上げる「疑義照会メモ」の活用

頻繁に生じる疑義照会は、パターン化してメモに残しておきましょう。

- <strong>腎機能調節</strong>: 年齢や血清クレアチニン値からeGFRを算出し、減量が必要な薬リストを作成しておく。
- <strong>薬物相互作用</strong>: よくある「併用注意」の組み合わせに対する医師の過去の回答（「この組み合わせはOK」「変更して」等）を記録しておく。

<div style="background-color: #fef2f2; border: 1px solid #fecdd3; padding: 1rem; border-radius: 8px; color: #991b1b;">
  <strong>💡 薬剤師のTips：情報の引き出しを増やそう</strong><br>
  「先生、以前同じような症例で〇〇という対応をしていただきましたが、今回はどうされますか？」というフレーズは、医師の過去の判断を尊重しつつ、円滑な合意形成を図る際に非常に有効です。
</div>

---

## 🌿 4. まとめ：準備は「患者さんのため」だけでなく「自分を守るため」

疑義照会の準備をしっかり行うことは、以下のようなメリットがあります。

* <strong>医師からの信頼向上</strong>: 的確な根拠に基づく提案は、医師にとって「頼れるパートナー」という印象を与えます。
* <strong>ストレスの軽減</strong>: 準備ができていれば、電話への恐怖心が大幅に減ります。
* <strong>患者さんの安全</strong>: 最終的に、患者さんに最も安全な薬物療法を提供することに繋がります。
