# IADLとは？手段的日常生活動作評価の基本

<div style="background-color: #f8fafc; border-left: 4px solid #6366f1; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験談】</strong><br>
  在宅訪問を始めたばかりの頃、ある一人暮らしの患者さんの「薬の飲み忘れ」が頻発していました。私は「朝夕食後に飲むだけですよ」と何度も説明し、カレンダーや薬ケースを提案しました。しかし、一向に改善しません。<br><br>
  後からケアマネジャーさんに報告した際、「あの患者さん、実は最近買い出しにも行けなくなっていて、献立を考えるのも難しい状態なんです」と言われてハッとしました。単に「薬の管理能力」の問題だと思い込んでいましたが、その背景には「IADL（手段的日常生活動作）の低下」という大きな壁があったのです。
</div>

高齢者の薬学管理において、バイタルや検査値だけでなく、生活背景を理解することは不可欠です。「薬が飲めていない」という結果に対して、どのような介入が必要か。そのヒントとなる「IADL」について深掘りしていきましょう。

---

## 🔍 1. IADL（手段的日常生活動作）とは？

<strong>IADL（Instrumental Activities of Daily Living）</strong>は、日本語で<strong>「手段的日常生活動作」</strong>と呼ばれます。
食事・着替えといった基本的な動作（ADL）よりも、より複雑で、社会生活を送るために必要な能力を指します。

### ADLとIADLの違い
*   <strong>ADL（基本的日常生活動作）</strong>: 生きていくための最小限の動作（食事、移動、入浴、排泄など）
*   <strong>IADL（手段的日常生活動作）</strong>: 自立した生活を維持するための応用的な能力

薬剤師にとって特に重要なのは、<strong>「薬の自己管理能力」もこのIADLの一部</strong>であるという点です。

---

## 📊 2. IADLを評価する代表的な項目（Lawtonのスケール）

IADLは主に以下の項目で評価されます。これらの能力が低下すると、服薬コンプライアンス（アドヒアランス）に直結します。

<div style="overflow-x: auto;">
| 項目 | 内容 | 薬剤師の視点 |
| :--- | :--- | :--- |
| <strong>電話の使用</strong> | 電話をかける、受ける | 電話での服薬確認が成立するか？ |
| <strong>買い物</strong> | 必要なものを自分で購入 | 薬局への来局や通院は可能か？ |
| <strong>食事の準備</strong> | 献立の作成と調理 | 食前・食後など用法を守れるか？ |
| <strong>家事</strong> | 掃除や整理整頓 | 薬の保管・管理は適切か？ |
| <strong>洗濯</strong> | 衣類の洗濯 | 一包化等の工夫が必要か？ |
| <strong>移動手段</strong> | 公共交通機関の利用 | 通院・薬の受け取り能力 |
| <strong>服薬管理</strong> | <strong>薬の準備・服用</strong> | <strong>自己管理能力の直接的評価</strong> |
| <strong>金銭管理</strong> | 買い物、支払い | 医療費の支払い、自己負担金の把握 |
</div>

## 💊 3. IADL低下が疑われる時の「服薬支援」アプローチ

IADLが低下している患者さんに対し、単に「カレンダーを使ってください」と伝えるだけでは根本解決になりません。IADLのどの項目が低下しているかに合わせ、処方提案や工夫が必要です。

<div style="display: grid; grid-template-columns: repeat(1, 1fr); gap: 1rem; margin: 1.5rem 0;">
  <div style="background-color: #f0fdf4; padding: 1rem; border-radius: 8px; border: 1px solid #bbf7d0;">
    <div style="font-weight: bold; color: #166534; margin-bottom: 0.5rem;">① 食事の準備が難しい →「用法を簡略化」</div>
    <p style="font-size: 0.9em; color: #14532d;">食前・食後といった細かな用法は、食事の準備能力が低下すると負担になります。医師に相談し、1日1回投与や食直前・食直後への変更を提案しましょう。</p>
  </div>
  <div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe;">
    <div style="font-weight: bold; color: #1e3a8a; margin-bottom: 0.5rem;">② 家事・整理が難しい →「一包化と管理方法の見直し」</div>
    <p style="font-size: 0.9em; color: #1e40af;">薬が散らばってしまうなら、カレンダー型の一包化が効果的です。また、訪問薬剤管理指導を活用し、整理を一緒に行う環境作りも検討してください。</p>
  </div>
  <div style="background-color: #fffbeb; padding: 1rem; border-radius: 8px; border: 1px solid #fde68a;">
    <div style="font-weight: bold; color: #92400e; margin-bottom: 0.5rem;">③ 金銭管理が難しい →「支払いの工夫・残薬調整」</div>
    <p style="font-size: 0.9em; color: #b45309;">「薬代が高いから」と勝手に休薬しているケースも。残薬確認を徹底し、不要な処方を減らす提案（ポリファーマシー対策）が有効です。</p>
  </div>
</div>

## 🎯 4. まとめ：IADLを把握して「生活者としての患者」を支える

薬剤師にとって、IADLを評価することは<strong>「患者さんが自宅で自立して薬を飲み続けられるか」を予測するスキル</strong>です。

1.  <strong>残薬が発生している時</strong>、まずは「どの動作（買い物、管理、金銭など）が困難なのか」を観察する。
2.  「飲み忘れ」を個人の責任にせず、<strong>IADLの低下を補うためのサポート（一包化、訪問指導など）</strong>をチームで考える。
3.  ケアマネジャーや訪問看護師と<strong>「IADLの情報」を共有</strong>し、多職種連携の輪を広げる。

「薬を渡して終わり」ではなく、患者さんの「生活の手段」まで考えた介入を行うことこそが、私たち薬剤師の価値になります。次回の訪問や投薬の際には、ぜひ少しだけ患者さんの「生活動作」に目を向けてみてくださいね！
