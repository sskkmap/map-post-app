# 白血球数（WBC）ってなに？高い・低いときに考えられる原因

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の外来調剤で、新薬の抗精神病薬が処方された患者さんの検査値を確認していたときのことです。前回値と比較して白血球数が少しずつ下がっていることに気づきました。「まだ基準値内だし大丈夫かな」と軽く考えていましたが、念のため主治医に共有したところ、「その薬の副作用（無顆粒球症のリスク）を警戒していたんだ、指摘ありがとう」と感謝されました。<br><br>
  単に「異常値かどうか」を見るだけでなく、「薬の影響で低下傾向にないか」という<strong>「推移」</strong>を見ることの重要性を痛感した瞬間でした。
</div>

検査値の中でも、白血球数（WBC）は感染症や炎症の指標として非常にポピュラーですが、同時に「薬剤による副作用」を見極めるための重要なサインでもあります。今回は、白血球数の基礎知識と、実務で遭遇しやすい変動原因について整理していきます。

---

## 🔬 1. 白血球（WBC）の基本的な役割

白血球は、血管内をパトロールし、細菌やウイルスなどの異物から体を守る「免疫の主力部隊」です。主に以下の5種類で構成されています。

| 種類 | 特徴 | 主な役割 |
| :--- | :--- | :--- |
| <strong>好中球</strong> | 最も数が多い（50-70%） | 細菌感染に対する防御 |
| <strong>リンパ球</strong> | 2番目に多い（20-40%） | ウイルス感染、免疫記憶 |
| <strong>単球</strong> | 大型で食作用が強い | 老廃物の除去、抗原提示 |
| <strong>好酸球</strong> | 寄生虫やアレルギー | アレルギー反応に関与 |
| <strong>好塩基球</strong> | 最も少ない（1%以下） | アレルギーや炎症の調整 |

通常、成人では <strong>3,300〜8,600/μL</strong> が基準値とされますが、喫煙やストレス、日内変動でも変化するため、一過性の数値だけで判断せず、<strong>「以前と比較してどう変化しているか」</strong>を追うことが重要です。

---

## 📈 2. 白血球数が増加しているとき（白血球増多症）

白血球数が多いということは、何らかの理由で免疫系が活性化している状態です。

<div style="display: grid; grid-template-columns: repeat(1, 1fr); gap: 1rem; margin: 1.5rem 0;">
  <div style="background-color: #fff1f2; padding: 1rem; border-radius: 8px; border: 1px solid #fecdd3;">
    <div style="font-weight: bold; color: #9f1239; margin-bottom: 0.5rem; border-bottom: 2px solid #fecdd3; padding-bottom: 0.25rem;">主な原因</div>
    <ul style="color: #881337; font-size: 0.95em;">
      <li><strong>細菌感染症</strong>：肺炎、膀胱炎、虫垂炎など（好中球増多が主体）</li>
      <li><strong>ストレス・炎症</strong>：手術後、組織壊死（心筋梗塞など）</li>
      <li><strong>ステロイド投与</strong>：好中球の血管内遊走を抑制し、循環血液中の数が増える</li>
      <li><strong>血液疾患</strong>：慢性骨髄性白血病など</li>
    </ul>
  </div>
</div>

<strong>薬剤師の視点：</strong>
ステロイドを使用中の患者さんでは、感染がなくても白血球数が上昇することがあります。「白血球が高いから感染しているはず！」と決めつけず、発熱やCRP値などの他の所見とセットで評価しましょう。

---

## 📉 3. 白血球数が減少しているとき（白血球減少症）

特に注意が必要なのが、白血球数の減少です。免疫力が低下するため、日和見感染症のリスクが高まります。

### 🚨 薬物性の白血球減少症に注意！
薬剤師として最も警戒すべきは、<strong>「薬による骨髄抑制」</strong>です。

| 薬剤の種類 | 臨床的な警戒ポイント |
| :--- | :--- |
| <strong>抗がん剤</strong> | 治療サイクルに伴う必ず起こる副作用。感染予防が必須。 |
| <strong>抗甲状腺薬</strong> | 飲み始めの初期に無顆粒球症を起こすリスクがある。 |
| <strong>抗精神病薬</strong> | クロザピンなど、定期的な採血モニタリングが義務付け。 |
| <strong>抗菌薬・NSAIDs</strong> | 長期投与や多剤併用時に稀に引き起こす可能性。 |

  <strong>💡 薬剤師が聞くべき患者への質問</strong><br>
  数値が下がってきた患者さんには、「最近、喉が痛いことはありませんか？」「原因不明の発熱はありましたか？」と、<strong>感染症のサインがないか</strong>を必ず確認しましょう。
</div>

## 💡 4. まとめ：薬剤師が検査値を見る際の鉄則

白血球数の増減を見る際、以下のステップを習慣にしてみてください。

1. <strong>基準値との乖離を見る</strong>：まずは極端な異常値（高すぎる・低すぎる）がないか。
2. <strong>推移を追う</strong>：前回値と比較して、右肩上がりに増えているか、急激に落ちているか。
3. <strong>他薬の影響を確認する</strong>：ステロイドや免疫抑制剤、最近開始した新薬はないか。
4. <strong>自覚症状をセットにする</strong>：数値と、患者さんの発熱や感染症状（咽頭痛、口内炎など）が一致しているか。

白血球の検査値は、単なる数字ではなく<strong>「患者さんの生体防御能力の現在地」</strong>です。次回の処方監査時に、ぜひ検査値の推移に目を向けてみてくださいね！
