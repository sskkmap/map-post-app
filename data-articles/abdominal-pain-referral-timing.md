# 腹痛が増えたら？受診勧奨を考えるタイミング

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  薬局に、近所のクリニックで整腸剤をもらっている患者さんが来局されました。「最近、お腹の痛みが前より頻繁になってきたし、なんとなくズキズキする時間が長くて不安だ」というご相談を受けました。つい「整腸剤の回数を増やしてみましょうか？」と言いかけましたが、ふと頭をよぎったのは「本当にただの腸炎？」という疑問でした。<br><br>
  もしこれがイレウスや腫瘍による通過障害だったら……？そう考えると、独断でアドバイスをすることの怖さを改めて痛感しました。薬剤師として、症状の「変化」に対してどこまで踏み込んでヒアリングすべきか、その境界線を知っておくことは非常に重要です。
</div>

「お腹が痛い」という訴えは日常茶飯事ですが、その裏に重大な疾患が隠れていることもあります。今回は、薬剤師として見逃してはいけない**「受診勧奨（医療機関への受診を促すタイミング）」**の判断基準をまとめました。

---

## 🚩 1. 危険な腹痛を見抜く「レッドフラッグサイン」

腹痛の中でも、以下のようなサインがある場合は、速やかに医師の診察が必要です。「様子を見ましょう」という対応が命取りになる可能性があるため、ヒアリング時に必ず確認しましょう。

| サインの種類 | 具体的な症状・状態 | 薬剤師が確認すべきこと |
| :--- | :--- | :--- |
| **激しい痛み** | 冷や汗が出る、脂汗をかく、立っていられない | 痛みの強さ・持続時間 |
| **随伴症状** | 38度以上の発熱、血便、吐血、持続する嘔吐 | 既往歴、服薬中の薬 |
| **腹部状態** | 板のように硬い（板状硬）、膨満感が強い | お腹の張り、排便・排ガスの有無 |
| **全身状態** | 意識の混濁、顔面蒼白、脈が速い | 血圧低下の兆候はないか |

<div style="background-color: #fff1f2; border: 1px solid #fecdd3; padding: 1.5rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong style="color: #9f1239;">⚠️ ここがポイント！</strong><br>
  特に注意すべきは**「痛みの性質の変化」**です。「痛みの場所が変わった」「痛みの種類が変わった（鈍痛から鋭痛へ）」といった変化は、病状の悪化を示唆する重要なサインとなることがあります。
</div>

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: 腹痛において、激しい痛み、38度以上の発熱、血便・吐血、持続する嘔吐、板状硬結や強い膨満感、意識の混濁、顔面蒼白、頻脈といった症状は、緊急性の高いレッドフラッグサインであり、速やかな医師の診察が必要です。痛みの性質や部位の変化も病状悪化の重要な兆候となりえます。これらの症状がある場合、薬剤師は患者さんに速やかな受診を勧めるべきです。<br>参照元: <a href="https://www.jsge.or.jp/guideline/disease/pdf/guide_05.pdf">日本消化器病学会「急性腹症診療ガイドライン2015」</a>, <a href="https://www.jarm.or.jp/committee/kikaku/pdf/06.pdf">日本救急医学会「病態別救急初期診療の目安：腹痛」</a></details>

## 🔎 2. 「ただの腹痛」ではないかもしれない？鑑別の視点

薬剤師としてヒアリングする際、以下のポイントを深掘りすることで、早期受診を勧めるべきかどうかの精度が高まります。

### ① 「排便・排ガス」の確認
*   **腸閉塞（イレウス）を疑うサイン**: 排便がなく、ガスの排出も止まっている。お腹がパンパンに張っている場合は、腸閉塞が疑われる可能性があります。
*   **受診勧奨**: 腹部手術歴がある患者さんであれば、特に注意が必要です。

### ② 「痛みの位置と移動」
*   **虫垂炎を疑うサイン**: 最初は胃のあたり（心窩部）が痛み、徐々に右下腹部へ移動していくことがあります。
*   **注意**: 「痛みが移動した」というエピソードは、外科的処置が必要な疾患の典型的な徴候である場合があります。

### ③ 「服薬状況との関連」
*   **NSAIDs（痛み止め）の常用**: 非ステロイド性抗炎症薬（NSAIDs）の長期使用や高用量使用は、消化性潰瘍やその穿孔のリスクを高める可能性があります。
*   **抗血栓薬の服用**: 抗血栓薬を服用している患者さんが腹痛を訴える場合、消化管出血や腹部大動脈瘤の破裂、腸管虚血など、重大な出血リスクを伴う疾患を見逃さないよう注意が必要です。

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: 薬剤師は腹痛の患者さんに対し、排便・排ガスの有無、痛みの位置や移動、および服薬状況（特にNSAIDsや抗血栓薬の使用）を確認することで、腸閉塞や虫垂炎、薬剤起因性の消化管障害などの緊急性の高い疾患を鑑別する手がかりを得ることができます。これらの情報は、適切な受診勧奨を行う上で重要です。<br>参照元: <a href="https://www.jarm.or.jp/committee/kikaku/pdf/06.pdf">日本救急医学会「病態別救急初期診療の目安：腹痛」</a>, <a href="https://www.jstage.jst.go.jp/article/jspen/36/5/36_861/_pdf/-char/ja">日本臨床栄養代謝学会「腸閉塞（イレウス）診療ガイドライン」</a>, <a href="https://www.jstage.jst.go.jp/article/jjpn/28/6/28_6_773/_pdf">日本小児科学会「小児急性虫垂炎診療ガイドライン」</a>, <a href="https://www.jstage.jst.go.jp/article/naika/109/6/109_1079/_pdf/-char/ja">日本内科学会雑誌「非ステロイド性抗炎症薬（NSAIDs）と消化器病」</a></details>

## 📋 3. 受診勧奨のフローチャート（目安）

薬剤師が現場で活用できる、受診勧奨の判断基準案です。

<div style="display: flex; flex-direction: column; gap: 0.5rem; margin: 1.5rem 0;">
  <div style="background: #eff6ff; padding: 1rem; border-radius: 6px; border-left: 4px solid #60a5fa;"><strong>【今すぐ受診・救急】</strong><br>顔色が悪い、激痛で動けない、吐血・血便、お腹が板のように硬い、意識の混濁</div>
  <div style="background: #f0fdf4; padding: 1rem; border-radius: 6px; border-left: 4px solid #4ade80;"><strong>【翌日までに受診】</strong><br>痛みが数日続いている、痛みが強くなっている、発熱が伴う、便秘が数日間改善しない、痛みの部位が移動した</div>
  <div style="background: #fefce8; padding: 1rem; border-radius: 6px; border-left: 4px solid #facc15;"><strong>【経過観察・様子見可能】</strong><br>痛みの程度が軽微、排便とともに改善する、明らかな食事の食べ過ぎなど原因が特定できている、腹痛以外の全身症状がない</div>
</div>

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: 腹痛の症状に応じて、薬剤師は患者さんに適切な受診タイミングを案内することが重要です。激しい痛み、出血、意識障害などのレッドフラッグサインがある場合は「今すぐ受診・救急」、痛みの持続や悪化、発熱、便秘の改善が見られない場合は「翌日までに受診」、軽微な痛みで原因が特定でき、その他の症状がない場合は「経過観察・様子見可能」と判断する目安とされます。<br>参照元: <a href="https://www.jsge.or.jp/guideline/disease/pdf/guide_05.pdf">日本消化器病学会「急性腹症診療ガイドライン2015」</a>, <a href="https://www.jarm.or.jp/committee/kikaku/pdf/06.pdf">日本救急医学会「病態別救急初期診療の目安：腹痛」</a></details>

## 💡 4. 薬剤師からの「受診勧奨」の伝え方テクニック

患者さんは「薬で治るなら薬局だけで済ませたい」と考えがちです。そんな時は、以下のように伝えるとスムーズに受診に繋がります。

*   **「否定」ではなく「慎重」というスタンスをとる**
    *   「今の症状からすると、現状の薬だけで様子を見るのは少しリスクがあるかもしれません。万が一、盲腸や腸閉塞などの隠れた病気だった場合、早期の診断が非常に重要になります。」
*   **医師への橋渡しを強調する**
    *   「ご自身で判断して悪化してしまうのが一番心配です。一度、念のために先生に診ていただき、『ただの便秘ですね』と言ってもらうだけでも、安心材料になると思いますよ。」

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: 薬剤師による受診勧奨は、患者さんの健康を守る上で重要な役割を担います。患者さんが医療機関の受診をためらう場合でも、「否定」ではなく「慎重」な姿勢で、潜在的なリスクや早期診断の重要性を説明し、医師への橋渡しを強調することで、患者さんが安心して受診できるようなコミュニケーションを心がけることが推奨されます。<br>参照元: <a href="https://www.nichiyaku.or.jp/assets/uploads/activity/news/2021/pdf/ny-20210705_1.pdf">日本薬剤師会「患者のための薬局ビジョン」</a>, <a href="https://www.nichiyaku.or.jp/assets/uploads/news/2019/pdf/190807-1.pdf">日本薬剤師会「かかりつけ薬剤師・薬局の機能強化に向けた検討状況」</a></details>

## 🌿 最後に：薬剤師の「疑う力」が患者さんを守る

腹痛は、軽度な胃腸炎から緊急手術が必要な重症疾患まで幅広く含まれます。
薬剤師ができることは、**「患者さんの言葉から、緊急性の高いサインを読み取ること」**です。

迷ったときは、独断でOTCや整腸剤を提案せず、**「受診を勧める」という選択肢を常に優先してください。** 専門家としての適切な受診勧奨こそが、結果として患者さんの予後を守ることに繋がります。

明日からの服薬指導で、ぜひ「痛みの変化」をいつもより少し深めに尋ねてみてくださいね。

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: 薬剤師は、患者さんの訴える腹痛が軽度なものか、あるいは緊急性の高い疾患を示唆するものかを鑑別する上で重要な役割を担います。患者さんの予後を守るためには、安易な自己判断やOTC医薬品の推奨を避け、疑わしい症状があれば速やかに医療機関への受診を勧めることが専門家としての責務であり、患者さんの安全に貢献します。<br>参照元: <a href="https://www.nichiyaku.or.jp/assets/uploads/activity/news/2021/pdf/ny-20210705_1.pdf">日本薬剤師会「患者のための薬局ビジョン」</a>, <a href="https://www.nichiyaku.or.jp/assets/uploads/news/2019/pdf/190807-1.pdf">日本薬剤師会「かかりつけ薬剤師・薬局の機能強化に向けた検討状況」</a></details>
