# 🚑 高齢者の救急受診で持参したい情報一覧

<div style="background-color: #f0fdf4; border-left: 4px solid #22c55e; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場の薬剤師からのメッセージ】</strong><br>
  先週、担当している高齢の患者さんが夜間に救急搬送されました。翌朝、救急病院から服薬照会がありましたが、ご家族は慌てていたため「いつもの薬」しか持ち出しておらず、最近追加されたばかりの頓服薬や、他科で処方されているサプリメントの把握ができていませんでした。<br><br>
  「お薬手帳は持っています」とご家族は仰いましたが、実は中身が最新の状態に更新されていなかったのです。現場で情報の齟齬が起き、治療方針の決定に迷いが生じるリスクを目の当たりにしました。私たち薬剤師が平時から「何を、どう準備すべきか」を患者さんに伝えておく重要性を痛感した出来事でした。
</div>

高齢者の緊急搬送時、限られた時間の中で適切な医療を提供するために、正確な服薬・身体情報は不可欠です。本記事では、薬剤師が患者さんやご家族に指導すべき「持参すべき情報リスト」をまとめました。

## 📋 1. なぜ「お薬手帳」だけでは不十分なのか？

多くの患者さんは「お薬手帳があるから大丈夫」と考えがちです。しかし、救急の現場では以下の情報が抜け落ちることで、医療ミスや治療の遅れにつながるケースがあります。

* <strong>最新の処方変更が反映されていない</strong>（手帳の貼り忘れ）
* <strong>市販薬やサプリメントの併用</strong>が記録されていない
* <strong>アレルギー・副作用歴</strong>が記載されていない

これらを補うために、患者さん・ご家族には「お薬手帳＋α」の準備を促す必要があります。

## 📊 2. 救急受診時に持参すべき情報一覧

以下の表を参考に、患者さんへの指導にお役立てください。

| 項目 | 具体的な内容 | なぜ必要なのか？ |
| :--- | :--- | :--- |
| <strong>お薬手帳</strong> | 最新の全記録（直近の変更含む） | 常用薬の確認、相互作用のチェック |
| <strong>服薬状況のメモ</strong> | 飲み忘れ、残薬の有無、自己調節の有無 | アドヒアランスの把握、毒性・欠乏リスク評価 |
| <strong>他科受診・サプリ</strong> | 他科処方、市販薬、健康食品のリスト | 重複投薬や成分重複、禁忌の回避 |
| <strong>副作用・アレルギー</strong> | 過去の薬疹や副作用歴、アナフィラキシー | 緊急時の薬剤選択における禁忌判断 |
| <strong>緊急連絡先・情報</strong> | かかりつけ医、薬局、ケアマネの連絡先 | 情報の補完や方針確認のため |

## 💡 3. 患者さんへの「伝え方」のポイント

高齢の患者さんやご家族に対し、不安を煽らずに準備を促すには、具体的な「仕組み化」のアドバイスが有効です。

### 📌 ① 「緊急時セット」を玄関先に置く
お薬手帳の表紙に、「緊急時用」として以下をクリップで止めておくよう指導しましょう。
- <strong>直近の処方明細書のコピー</strong>
- <strong>「使用中の市販薬・サプリメントリスト」</strong>（手書きメモでOK）
- <strong>「アレルギー・副作用歴カード」</strong>

### 📌 ② 「かかりつけ薬局」の連絡先を明記する
病院の救急外来から薬局へ直接問い合わせができるよう、手帳の分かりやすい位置に薬局の連絡先シールや名刺を貼っておくことが、迅速な情報提供に直結します。

---

## ⚠️ 4. 薬剤師が見落としてはいけないチェック項目

救急照会を受けた際、以下の点は特に重点的に確認してください。

1. <strong>抗血栓薬の使用状況</strong>: ワーファリンやDOACの服用有無（出血リスク評価）
2. <strong>血糖降下薬の使用状況</strong>: 低血糖リスクと、食事制限の有無
3. <strong>ベンゾジアゼピン系薬剤の有無</strong>: 転倒やせん妄のリスク評価
4. <strong>腎機能低下時の用量調整</strong>: 常用薬が腎排泄型の場合の注意喚起

  <strong>【薬剤師のアドバイス】</strong><br>
  「もしもの時」の話は、患者さんも少し身構えてしまいます。「最近はお薬の種類が増えてご家族も把握が大変ですよね。もしもの時に安心できるよう、情報を一つにまとめておきませんか？」といった、<strong>サポートの姿勢</strong>で提案することが、患者さんの行動変容を促す鍵となります。
</div>

## 🌿 まとめ

高齢者の救急受診において、情報は「命綱」です。
私たち薬剤師は、単に調剤するだけでなく、患者さんが万が一の時にも安全な医療を受けられるよう、平時から<strong>「情報の一元管理」を促すアドバイザー</strong>であるべきです。

次回の服薬指導時に、「お薬手帳にアレルギー歴は書いてありますか？」「サプリメントを併用していませんか？」と一言添えることから始めてみませんか？その一言が、いつか患者さんの命を守ることにつながります。
