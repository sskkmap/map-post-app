# 高齢者の低栄養を見逃さないためのチェックポイント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験：見過ごされた「体重減少」】</strong><br>
  数年前、定期的に来局されていた一人暮らしの高齢患者さんの処方箋を確認していた時のことです。血圧の薬と脂質異常症の薬を漫然と服用されていましたが、お薬手帳の記録を見ると、ここ半年で体重が5kgほど減少していました。<br><br>
  「最近、少し食欲がなくて……」と仰るその患者さんに、私は薬の副作用ばかりを疑ってしまいました。しかし、深く話を聞いてみると、実は「買い物に行くのが億劫になり、食事を作る気力がない（＝低栄養）」という深刻な問題が隠れていたのです。薬を調整する前に、まず栄養状態を評価すべきだったと痛感した瞬間でした。
</div>

高齢者の低栄養は、フレイル（虚弱）の入り口であり、要介護状態へと直結する重要なサインです。今回は、忙しい調剤業務の合間でもパッと確認できる「高齢者の低栄養リスク」の評価方法をまとめました。

---

## 🔍 1. 低栄養を疑うための「3つのサイン」

問診や服薬指導の際に、意識してチェックしたい項目です。以下の項目が当てはまる場合、低栄養リスクが高いと考えられます。

| チェック項目 | 具体的な確認ポイント |
| :--- | :--- |
| <strong>体重減少</strong> | 過去6ヶ月で3%〜5%以上の意図しない体重減少はないか |
| <strong>食事量・内容</strong> | 1日2食以下になっていないか、肉や魚などのタンパク質が減っていないか |
| <strong>身体機能</strong> | 握力低下や、以前より歩く速度が遅くなったと感じるか |

特に「急激な体重減少」は危険信号です。お薬手帳の備考欄に、患者さんの体重をメモしておく癖をつけておくと、変化にいち早く気づくことができます。

## 🤝 4. 薬剤師からの具体的なアプローチ

低栄養のリスクを感じたら、まずは<strong>「多職種連携」</strong>のスイッチを入れましょう。

1. <strong>お薬調整の提案</strong>: 食欲不振の原因と思われる薬剤（特に多剤併用）がないかを見直し、医師へ減薬を提案します。
2. <strong>口腔環境の確認</strong>: 「噛みにくい」という訴えがある場合は、歯科受診を勧めることも重要です。
3. <strong>サプリメント・栄養補助食品の紹介</strong>: 薬局で取り扱いがある場合は、高タンパク・高エネルギーの栄養食品を提案するのも一つの手段です。
4. <strong>ケアマネジャーへの情報提供</strong>: 生活状況に課題がある場合は、担当のケアマネジャーへ「最近体重が減っており、栄養状態が心配です」と情報を共有しましょう。

  「栄養を摂ってください」と伝えるよりも、<strong>「お薬を飲んでいると体力が落ちやすいので、タンパク質を少し意識して摂るといいですよ」</strong>というように、薬の管理とセットで伝えると、患者さんも受け入れやすくなります。

## 🌿 まとめ：小さな変化を見逃さない「目」を持つ

高齢者の低栄養は、日々の生活の中での「ちょっとした変化」の積み重ねです。

*   <strong>「最近、少し痩せましたか？」</strong>という問いかけをルーチンにする。
*   <strong>お薬手帳の備考欄</strong>に体重や食欲の推移を記録しておく。
*   <strong>副作用のサイン</strong>を疑いつつ、生活背景（食事状況）にまで耳を傾ける。

私たち薬剤師の専門知識は、薬だけにとどまりません。患者さんの「食べる力」を守ることは、結果として「飲み続ける力」を守ることにつながります。日々の業務の中で、ぜひ意識してみてくださいね。
