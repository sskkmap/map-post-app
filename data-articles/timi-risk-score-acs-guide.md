# TIMIリスクスコアとは？急性冠症候群のリスク評価を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日、循環器内科の門前薬局で、心筋梗塞で入院・退院直後の患者さんの処方箋を受け取りました。当時はまだ経験も浅く、「抗血小板薬2剤併用療法（DAPT）中なんだな」くらいにしか考えていませんでした。<br><br>
  医師との疑義照会で「この患者さんはTIMIスコアが高値なので、出血リスクと虚血リスクのバランスを考慮して〇〇の期間を少し長めに設定します」と言われ、ハッとさせられました。単にガイドライン通りの期間を設定するのではなく、患者個別のリスクスコアに基づいた治療設計がなされていることを痛感した瞬間でした。
</div>

循環器領域の処方に関わる薬剤師にとって、患者さんの「リスク評価」を理解することは、適切な服薬指導や副作用モニタリングを行うための必須スキルです。今回は、急性冠症候群（ACS）診療で頻用される<strong>TIMIリスクスコア</strong>について、その中身を紐解いていきましょう。

---

## 🔬 1. TIMIリスクスコアとは？なぜ重要なのか

TIMI（Thrombolysis In Myocardial Infarction）リスクスコアは、<strong>急性冠症候群（ACS）患者さんの死亡や心筋梗塞の再発リスクを予測するための指標</strong>です。

心電図変化やバイタル、血液検査データなどの簡単な項目を点数化することで、誰でも短時間でリスクを層別化できるのが最大の特徴です。薬剤師としては、このスコアが高い患者さんほど<strong>「より積極的な抗血栓療法が必要な一方、出血リスクも高まる」</strong>という臨床的背景を理解しておくことが重要です。

---

## 📊 2. TIMIリスクスコアの算出項目（UA/NSTEMIの場合）

不安定狭心症（UA）や非ST上昇型心筋梗塞（NSTEMI）の患者さんでは、以下の7項目をチェックします。1項目該当につき1点として計算されます。

| 評価項目 | 内容 |
| :--- | :--- |
| <strong>年齢</strong> | 65歳以上 |
| <strong>冠危険因子</strong> | 冠動脈疾患の既往、家族歴、高血圧、高コレステロール血症、糖尿病の3つ以上 |
| <strong>冠動脈狭窄</strong> | 既往の冠動脈狭窄（50%以上） |
| <strong>ST変化</strong> | 心電図上の明らかなST偏位（0.05mV以上） |
| <strong>心筋マーカー</strong> | 心筋壊死マーカー（トロポニン等）の上昇 |
| <strong>頻拍・低血圧</strong> | 直近24時間以内の重症な虚血イベントが2回以上 |
| <strong>アスピリン使用</strong> | 直近7日以内のアスピリン服用歴 |

<div style="background-color: #fef2f2; border: 1px solid #fecaca; padding: 1rem; border-radius: 8px; color: #991b1b; margin-top: 1rem;">
  <strong>💡 薬剤師の視点：</strong><br>
  「アスピリン服用歴があること」がリスクスコアに含まれるのは、既に抗血小板療法を受けていても虚血イベントが起きた＝リスクが高い、という判断になるからです。薬歴管理の際、直近のアスピリン服薬状況を確認する重要性がここにも表れています。
</div>

## 🩺 4. 薬剤師が現場で活かすためのポイント

TIMIリスクスコアを知っておくことで、日々の業務にどのような「深み」が出るのでしょうか。

### ① 副作用モニタリングの強化
スコアが高い患者さんは、当然ながら<strong>抗血小板薬や抗凝固薬の多剤併用（DAPT、トリプルセラピー）</strong>が処方される確率が高くなります。高リスク患者さんには、特に消化管出血や皮下出血などの徴候について、重点的に質問するようにしましょう。

### ② 服薬アドヒアランスの重要性指導
「なぜこの薬を長く飲まなければならないのか？」という患者さんの疑問に対し、「あなたのリスク評価に基づくと、再発を予防するためにこの期間の継続が非常に重要です」という根拠のある説明が可能になります。

### ③ 併用薬チェックの徹底
高血圧、糖尿病、脂質異常症を合併している患者さんが多いのがACSの特徴です。リスクスコアで「危険因子あり」と判断された患者さんの場合、服薬状況が悪化するとイベント再発リスクが急上昇します。重複投与や飲み忘れがないか、残薬調整も含めて丁寧なフォローが求められます。

## 🌿 5. まとめ

TIMIリスクスコアは、医師が治療方針を決めるための「ものさし」ですが、私たち薬剤師にとっても、<strong>患者さんの状態をより深く理解し、先回りしたケアを行うための最強のツール</strong>となります。

「この人はなぜこれほど強力な抗血栓療法をしているのか？」と疑問に思ったときは、ぜひTIMIスコアを意識してみてください。スコアを知ることは、薬の処方意図を深く読み解くことにつながります。循環器内科領域の薬学的管理をさらにレベルアップさせていきましょう！
