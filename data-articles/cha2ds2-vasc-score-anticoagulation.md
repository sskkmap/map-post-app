# CHA2DS2-VAScスコアとは？抗凝固療法の判断に役立つ指標

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ノート】</strong><br>
  「先生、この患者さん、CHADS2スコアは1点ですが、CHA2DS2-VAScスコアで評価するとリスクが高くないでしょうか？」<br><br>
  先日、外来薬局で心房細動の患者さんの処方箋を確認した際のことです。従来のCHADS2スコアだけでは見落とされがちな「血管疾患」や「女性であること」などの因子が含まれるCHA2DS2-VAScスコアを算出したところ、リスクの解釈が変わることに気づきました。処方医に確認したところ、「気づかなかった、ありがとう」と感謝された経験があります。循環器領域において、このスコアを正しく理解することは、適切な抗凝固療法を提案する「薬剤師の武器」になると痛感しました。
</div>

心房細動（AF）患者における脳卒中リスクを評価する「CHA2DS2-VAScスコア」は、DOAC（直接作用型経口抗凝固薬）の導入や継続を判断する上で不可欠です。本記事では、スコアの算定方法と、臨床現場での活用ポイントを整理します。

---

## 🔢 1. CHA2DS2-VAScスコアの算出方法

CHA2DS2-VAScスコアは、各項目のアルファベット（C, H, A, D, S, V, A, Sc）の頭文字から来ています。以下の表を用いて、患者さんの背景を確認してみましょう。

| 項目 | 因子 | 点数 |
| :--- | :--- | :---: |
| <strong>C</strong> | Congestive heart failure（心不全） | 1点 |
| <strong>H</strong> | Hypertension（高血圧） | 1点 |
| <strong>A2</strong> | Age ≧ 75歳（75歳以上） | 2点 |
| <strong>D</strong> | Diabetes mellitus（糖尿病） | 1点 |
| <strong>S2</strong> | Stroke / TIA / 血栓塞栓症の既往 | 2点 |
| <strong>V</strong> | Vascular disease（血管疾患：心筋梗塞、末梢動脈疾患など） | 1点 |
| <strong>A</strong> | Age 65-74歳 | 1点 |
| <strong>Sc</strong> | Sex category（女性） | 1点 |
| <strong>合計</strong> | 最大スコア | 9点 |

### 💡 薬剤師の確認ポイント
- <strong>「年齢」の扱い:</strong> 65歳未満は0点、65〜74歳は1点、75歳以上は2点と、段階的にリスクが上がります。
- <strong>「女性」の因子:</strong> 女性というだけで＋1点になります。これは女性の心房細動患者が男性よりも脳梗塞リスクが高い傾向にあることを反映しています。

---

## 📉 2. リスクと推奨される治療（目安）

スコアを算出した後、その数値が何を意味するのかを理解しておく必要があります。

<div style="background-color: #eff6ff; padding: 1.5rem; border-radius: 8px; border: 1px solid #bfdbfe; margin: 1.5rem 0;">
  <div style="font-size: 1.1em; font-weight: bold; margin-bottom: 0.5rem; color: #1e3a8a;">臨床的な解釈の目安</div>
  <ul style="color: #1e40af; line-height: 1.8;">
    <li><strong>0点（男性）/ 1点（女性）:</strong> 低リスク。原則として抗凝固療法は行わない。</li>
    <li><strong>1点（男性）/ 2点（女性）:</strong> 中等度リスク。抗凝固療法を「考慮」する（患者の背景や出血リスクを慎重に判断）。</li>
    <li><strong>2点以上（男性）/ 3点以上（女性）:</strong> 高リスク。<strong>抗凝固療法が強く推奨</strong>される。</li>
  </ul>
</div>

※上記は一般的なガイドラインの基準ですが、最終的な判断は患者さんの出血リスク（HAS-BLEDスコアなど）と総合的に判断されます。

---

## 💊 3. 薬剤師が知っておくべき「抗凝固療法」の選択

DOACを選択する際、CHA2DS2-VAScスコアが高い＝「脳卒中予防効果」を重視したい、という視点は非常に重要です。

### 臨床でよく使われるDOACの比較

| 薬剤名 | 主な特徴（薬剤師の視点） |
| :--- | :--- |
| <strong>アピキサバン</strong> | 出血リスクが低い患者から高い患者まで幅広く対応しやすい。 |
| <strong>リバーロキサバン</strong> | 1日1回投与のためアドヒアランスが良い。重症高血圧患者には注意。 |
| <strong>エドキサバン</strong> | 低体重や腎機能に応じた用量調節が明確。 |
| <strong>ダビガトラン</strong> | 消化器症状（胃部不快感）に注意。腎機能低下時は禁忌となる場合も。 |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; color: #92400e; margin-top: 1rem;">
  <strong>⚠️ 注意すべき実務上の視点</strong><br>
  DOACの選択においては、CHA2DS2-VAScスコアで「脳卒中リスク」を評価すると同時に、<strong>腎機能（eGFR/CrCl）</strong>と<strong>出血リスク</strong>を必ずセットで確認してください。特に高齢者では、eGFR低下に伴う用量調節が必要な薬剤が多く、血栓予防と出血抑制のバランスが肝となります。
</div>

## 🌿 4. まとめ：スコアを活用して「提案できる薬剤師」へ

CHA2DS2-VAScスコアは、医師が治療方針を決定するための重要な指標ですが、私たち薬剤師にとっても、患者さんのリスク背景を理解するための非常に有効なツールです。

- <strong>薬歴入力時にスコアを意識する:</strong> 患者さんの既往歴を見て、簡易的に脳梗塞リスクを予測しておく。
- <strong>疑義照会の根拠にする:</strong> 高リスクなのになぜ抗凝固薬が出ていないのか、あるいは低リスクなのに長期投与されている場合の確認材料として活用する。

「ただ薬を渡すだけ」ではなく、「なぜこの薬が必要なのか」をスコアという客観的根拠に基づいて理解することで、より質の高い服薬指導や処方提案につながります。ぜひ明日の服薬指導から、患者さんの背景因子をチェックしてみてくださいね！
