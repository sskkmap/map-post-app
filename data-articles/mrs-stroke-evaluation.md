# mRSとは？脳卒中後のADL評価に用いられる指標

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード：その「介助」の必要性は？】</strong><br>

  その時、私はmRSという言葉を医療ニュースで聞いたことはあっても、実際の生活レベルと結びつけて正確に理解できていなかったことに気づき、冷や汗をかきました。mRSの段階を知ることは、単なる数値の確認ではなく、患者さんの生活の自立度を理解し、最適な服薬支援方法を提案するための重要な手がかりだったのです。

脳卒中に関わる現場では必ずと言っていいほど目にする<strong>「mRS」</strong>。今回は、この評価指標が具体的に何を意味し、私たち薬剤師の服薬指導にどう役立つのかを紐解いていきます。

---

## 📋 1. mRS（修正ランキンスケール）の基礎知識

mRS（Modified Rankin Scale）は、脳卒中後の患者さんが、どの程度「自立」して生活できているかを評価するための指標です。0〜6の7段階で評価され、数値が小さいほど自立度が高いことを示します。

| グレード | 状態の定義 | 薬剤師がイメージすべき状況 |
| :--- | :--- | :--- |
| <strong>0</strong> | 症状なし | 健常者と変わらない生活。自己管理は完璧。 |
| <strong>1</strong> | 症状あるが障害なし | 後遺症はあるが、日常の業務・生活に支障なし。 |
| <strong>2</strong> | 軽度障害 | 発症前の活動は一部制限されるが、自立可能。 |
| <strong>3</strong> | 中等度障害 | <strong>歩行や身の回りの世話に介助が必要。</strong> |
| <strong>4</strong> | 中等度重度障害 | 食事やトイレ等に介助が必要。寝たきりではない。 |
| <strong>5</strong> | 重度障害 | 寝たきり。排泄・食事すべてに介助が必要。 |
| <strong>6</strong> | 死亡 | - |

---

## 🔍 2. 薬剤師がmRSを意識すべき理由

「評価は医師や看護師がするもの」と思っていませんか？実は、mRSのステージを把握することは、<strong>「服薬アドヒアランス」を左右する重要な情報</strong>になります。

### 服薬管理の提案に直結する
*   <strong>mRS 0〜2</strong>: 基本的にセルフメディケーションや自己管理が可能です。一包化の必要性は低いかもしれません。
*   <strong>mRS 3〜4</strong>: 認知機能の低下や麻痺がある場合、一包化（日付印字）や、服薬カレンダーの導入が必須となるケースが多いです。
*   <strong>mRS 5</strong>: 介護者への指導がメインとなります。介護者が管理しやすい薬の剤形変更（散剤への変更や、頓服の工夫）を提案するチャンスです。

---

## 💊 3. 脳卒中後のフォローアップで押さえるべきポイント

mRSの数値は、患者さんのリハビリの進捗によって変動します。次回来局時、前回より改善しているか、悪化しているかを確認することは、病状の悪化（再発の予兆）を見抜くヒントにもなります。

### 薬剤師の視点でのチェックリスト
1.  <strong>嚥下機能の変化</strong>: mRSが悪化している場合、嚥下機能が低下している可能性が高いです。錠剤の粉砕や、ゼリー剤への変更を検討しましょう。
2.  <strong>副作用の早期発見</strong>: 特に降圧剤による「ふらつき（めまい）」は、mRSの数値（転倒リスク）を直接悪化させます。「最近、歩きにくくなった気がしませんか？」という問いかけが大切です。
3.  <strong>多剤併用の見直し</strong>: リハビリ中の患者さんは多くの薬を服用しがちです。不要な胃薬や便秘薬がないか、mRSを指標にしながら処方提案を行いましょう。

---

## ✨ まとめ：数値の先にある生活を見る

mRSは単なる評価シート上の数字ではありません。それは、患者さんの<strong>「今の生活の景色」</strong>を映し出す鏡です。

*   <strong>mRS 3以上の患者さん</strong>には、服薬支援ツール（カレンダー、お薬ボックス）を積極的に活用する。
*   <strong>mRSが変動している患者さん</strong>には、嚥下障害や転倒リスクといった「隠れた副作用」に気を配る。

私たちがmRSという共通言語を正しく理解し、多職種連携の中に活かしていくことで、脳卒中後の患者さんの生活の質（QOL）は大きく向上します。次回の患者さんの来局時には、ぜひ「この方のmRSはどのあたりかな？」と想像しながら、薬の管理方法を見直してみてくださいね！
