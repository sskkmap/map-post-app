# ALS重症度分類とは？筋萎縮性側索硬化症の評価法

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験談】</strong><br>
  ある日の外来受付で、進行性のALS（筋萎縮性側索硬化症）と診断された患者さんの処方箋を受け取りました。そこには「ALS重症度分類：4」との記載が。当時の私は「とりあえずリルゾールとエダラボンの確認を…」と薬のことばかり考えていました。<br><br>
  しかし、投薬カウンターで患者さんの呼吸音の浅さや、飲み込みにくそうな様子を目の当たりにし、「お薬を渡すこと」よりも「現在のADL（日常生活動作）や呼吸機能の状態を正しく把握し、服薬を継続するための工夫を提案すること」こそが薬剤師の役割だと痛感しました。重症度分類は単なる事務的な数字ではなく、患者さんの「今」を示す大切なバロメーターなのです。
</div>

ALS患者さんの支援において、お薬の相互作用や副作用チェックはもちろん重要ですが、<strong>「今、患者さんはどの程度の状態なのか」</strong>を客観的に把握することも、適切な指導には不可欠です。今回は、臨床でよく目にする「ALS重症度分類」について整理します。

---

## 📊 1. ALS重症度分類とは？

ALS重症度分類は、厚生労働省の指定難病要件にも関わる指標です。運動障害の進行度によって<strong>「1度から5度」</strong>まで分類されます。

| 分類 | 状態の目安 | 薬剤師が注意すべきポイント |
| :--- | :--- | :--- |
| <strong>1度</strong> | 独歩可能・軽度筋力低下 | ADL低下の兆候・副作用の早期発見 |
| <strong>2度</strong> | 独歩可能・中等度筋力低下 | 転倒リスク・服薬管理能力の確認 |
| <strong>3度</strong> | 歩行困難・食事/排泄に介助必要 | 嚥下機能の低下・薬の剤形変更検討 |
| <strong>4度</strong> | 車椅子使用・食事/排泄に介助必要 | 呼吸機能の低下・夜間不眠の訴え |
| <strong>5度</strong> | 寝たきり・呼吸補助必要 | 経管栄養への移行・薬の投与経路確認 |

### 🔍 なぜ薬剤師が把握しておくべきなのか？
分類の数字が上がるにつれ、経口薬の飲み込みが困難になったり、食事の時間と薬の時間の調整が必要になったりと、<strong>「剤形」や「服薬方法」の再考</strong>が求められます。

---

## 💊 2. 重症度別・薬剤師が提案できる「服薬支援」

重症度が進むにつれて、単純な「処方通りに渡す」という作業だけでは服薬が困難になるケースが増えます。

### 嚥下機能が低下してきたら（3度〜）
- <strong>剤形の検討</strong>: 錠剤が大きくて飲み込みにくい場合は、医師と相談の上で<strong>「細粒・散剤への変更」</strong>や<strong>「錠剤の粉砕」</strong>を検討します。
- <strong>服薬ゼリーの活用</strong>: 薬を包み込み、喉越しを良くする服薬ゼリーの提案は非常に有効です。

### 経管栄養（PEG）へ移行したら（4度〜5度）
- <strong>懸濁の可否</strong>: 錠剤のまま注入するのは閉塞のリスクがあります。粉砕後の懸濁安定性や、チューブを通過できるかを確認する必要があります。

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; color: #92400e;">
  <strong>💡 実務ワンポイントアドバイス</strong><br>
  粉砕する際は、その薬が「腸溶錠」や「徐放錠（SR錠など）」ではないか必ず確認してください！これらを粉砕すると血中濃度が急上昇し、危険な場合があります。また、吸湿性が高く粉砕に適さない薬もありますので、事前に添付文書や製薬メーカーへの確認を徹底しましょう。
</div>

## 🔄 3. よく使われるALS治療薬と評価法

ALSの進行を抑制する薬として、以下の2剤が代表的です。重症度分類と照らし合わせながら効果を評価します。

| 薬剤名 | 作用機序 | 薬剤師の確認ポイント |
| :--- | :--- | :--- |
| <strong>リルゾール</strong> | グルタミン酸遊離抑制 | 肝機能障害（定期的な血液検査結果のチェック） |
| <strong>エダラボン</strong> | フリーラジカル消去 | 投与時のアレルギー反応・腎機能 |

## 🎯 4. まとめ：数字の向こうにある患者さんを見よう

ALS重症度分類は、以下のような視点で活用してください。

1. <strong>現在の状態把握</strong>: 処方箋の数字から、患者さんのADLを想像する。
2. <strong>早期の提案</strong>: 嚥下機能が落ちてからではなく、落ちそうだなと感じた時点で剤形の変更を医師に提案する。
3. <strong>生活の質（QOL）を守る</strong>: 薬の調整が患者さんの負担を減らし、家族の介護負担軽減にも繋がります。

ALSという非常に過酷な疾患と向き合う患者さんにとって、薬剤師が提供できる最大の価値は、<strong>「安心して飲み続けられるお薬のサポート」</strong>です。

これからも「ALS重症度分類」という指標を一つのヒントとして、患者さんの生活に寄り添った服薬指導を心がけていきましょう！
