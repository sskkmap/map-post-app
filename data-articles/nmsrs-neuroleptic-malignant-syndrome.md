# NMSRSとは？悪性症候群の重症度評価尺度を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験談：見逃してはいけないサイン】</strong><br>
  ある日、精神科病棟から「急に患者さんの体温が上がり、筋肉が硬直している気がする」と相談を受けました。その患者さんは、抗精神病薬の増量を開始したばかりでした。<br><br>
  私が真っ先に思い浮かべたのは「悪性症候群（NMS）」です。しかし、どれくらい緊急度が高いのかを医師にどう報告すべきか、少し迷ってしまいました。「なんとなく熱が高い」という主観的な報告では、治療の判断を遅らせる恐れがあります。その時、チームで活用していたのが「NMSRS（Neuroleptic Malignant Syndrome Rating Scale）」でした。客観的な指標を持つことで、緊急時の連携が圧倒的にスムーズになった経験があります。
</div>

悪性症候群は、稀ですが致死的な重篤副作用です。今回は、その評価尺度であるNMSRSについて、薬剤師が知っておくべき知識をまとめます。

---

## 🧐 NMSRS（悪性症候群重症度評価尺度）とは？

NMSRSは、悪性症候群の臨床症状を数値化して重症度を評価するための尺度です。
単に「熱がある・硬い」という観察だけでなく、以下の主要な項目をスコアリングすることで、<strong>早期発見と治療介入のタイミングを逃さない</strong>ためのツールとして用いられます。

### 主要な評価項目（概略）
NMSRSでは、主に以下の項目をチェックし、重症度を段階的に評価します。

| 評価項目 | 観察のポイント |
| :--- | :--- |
| <strong>筋硬直</strong> | 歯車様硬直、鉛管様硬直の有無と程度 |
| <strong>高熱</strong> | 体温の上昇幅（38℃以上が目安） |
| <strong>意識障害</strong> | 意識レベルの変化（JCS/GCSでの評価） |
| <strong>自律神経症状</strong> | 発汗、頻脈、血圧変動 |
| <strong>検査値</strong> | CK（CPK）上昇、白血球増多など |

---

## ⚠️ 薬剤師がチェックすべき悪性症候群のサイン

悪性症候群のリスクは、精神科領域だけでなく、制吐薬や抗ドパミン作用を持つ薬を使用するすべての科で考慮する必要があります。

### 🚨 特に注意が必要な「ハイリスク状況」
以下の条件下では、NMSRSを意識した厳重なモニタリングが必要です。

*   <strong>抗精神病薬の開始・増量時</strong>（服用開始2週間以内が特に注意）
*   <strong>薬剤の急激な減量・中止時</strong>（ドパミン遮断からの離脱）
*   <strong>脱水状態や感染症があるとき</strong>
*   <strong>複数の抗精神病薬の併用</strong>（多剤大量処方）

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 薬剤師の介入アドバイス</strong><br>
  患者さんやご家族への服薬指導時には、<strong>「急激な高熱」</strong>と<strong>「体のこわばり」</strong>が出た場合は、すぐに服用を中止し受診するよう指導してください。また、介護者がいる場合は「いつもと様子が違う（ぼんやりしている、体が動かしにくそう）」という主観的な変化も見逃さないよう伝えることが重要です。
</div>

## 🏥 悪性症候群の治療・対応におけるポイント

悪性症候群が疑われる場合、薬剤師として関与すべき緊急対応は以下の通りです。

1.  <strong>原因薬剤の中止</strong>: 原因と考えられる薬剤を即座に特定・中止します。
2.  <strong>対症療法</strong>: 体温管理（冷却）、輸液による脱水改善。
3.  <strong>薬物療法</strong>: ダントロレン（筋弛緩薬）やブロモクリプチン（ドパミン受容体作動薬）の準備・確認。

### 比較表：似た症状を呈する疾患との鑑別

| 疾患 | 主な特徴 | 薬剤師の視点 |
| :--- | :--- | :--- |
| <strong>悪性症候群</strong> | 筋硬直、高熱、CK上昇 | 抗精神病薬服用歴、ドパミン遮断 |
| <strong>セロトニン症候群</strong> | ミオクローヌス、振戦、興奮 | SSRI/SNRI、トリプタン等の服用歴 |
| <strong>悪性高熱症</strong> | 麻酔薬による筋硬直・高熱 | 手術直後の発生が典型的 |

## 📝 まとめ：客観的な評価が救命につながる

NMSRSは、悪性症候群という「重篤になりうるサイン」を客観的に可視化する指標です。

*   <strong>精神科薬物療法</strong>の安全管理において、数値による客観的評価は不可欠です。
*   薬剤師は、処方内容から悪性症候群のリスクを予測し、<strong>「いつもと違う」というサインを拾い上げる役割</strong>を担っています。

「何かおかしい？」と感じたときに、NMSRSのような尺度を頭の片隅に置いておくことで、冷静かつ迅速な疑義照会や医師への報告が可能になります。患者さんの命を守る最後の砦として、副作用の徴候に敏感であり続けましょう！
