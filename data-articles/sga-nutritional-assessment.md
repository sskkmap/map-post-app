# SGAとは？主観的包括的栄養評価の進め方

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の臨床現場エピソード】</strong><br>
  ある日の病棟カンファレンスでの出来事です。アルブミン値が3.5g/dLと「正常範囲内」の患者さんに対して、医師から「この患者さん、少し栄養状態が悪そうだから栄養介入を考えてみて」と相談がありました。当時の私は「検査値は悪くないのに、なぜ？」と戸惑いました。<br><br>
  しかし、実際にベッドサイドで身体を観察し、病歴を詳しく聴取すると、明らかに筋肉の萎縮があり、食事摂取量も低下していることが判明。検査値はあくまで「過去の結果」であり、「今の状態」を反映していないことを痛感しました。これが、私がSGA（主観的包括的栄養評価）を本格的に学ぶきっかけでした。
</div>

臨床現場において、私たちの栄養評価は「血液検査データ」に頼りがちです。しかし、検査値は脱水や炎症の影響を大きく受けます。そこで重要になるのが、<strong>「SGA（Subjective Global Assessment）」</strong>です。

## 🔍 1. SGA（主観的包括的栄養評価）の基本概念

SGAとは、体重の変化や食事内容、身体的特徴などを総合的に判断し、栄養状態を「良好」「中等度（あるいは疑い）」「重度」の3段階で評価する手法です。

SGAが優れている点は、<strong>「検査データという『指標』」ではなく、「患者の身体と生活という『現実』」を直接評価できること</strong>にあります。特にNST（栄養サポートチーム）活動を行う薬剤師にとって、検査値を待たずに介入の緊急度を判断する強力な武器となります。

---

## 📝 2. SGAの評価項目とチェックポイント

SGAでは、主に「病歴（History）」と「身体所見（Physical Exam）」の2つの側面から評価を行います。

### 📊 SGA評価項目一覧

| 評価項目 | 重点チェックポイント |
| :--- | :--- |
| <strong>体重変化</strong> | 直近6ヶ月以内の意図しない減少率（5%以上は要注意） |
| <strong>食事摂取量</strong> | 摂取量の変化、期間、食事形態（ソフト食への移行など） |
| <strong>消化器症状</strong> | 食欲不振、嘔気、嘔吐、下痢、便秘の持続期間 |
| <strong>機能的低下</strong> | 日常生活動作（ADL）の低下、寝たきり時間の増加 |
| <strong>代謝ストレス</strong> | 感染症、手術、外傷などの現在の侵襲度 |
| <strong>身体所見</strong> | 筋肉の萎縮（側頭部、肩、鎖骨下）、皮下脂肪の減少、浮腫・腹水の有無 |

## 🛠️ 3. 実践！現場で使えるSGAの進め方

SGAを臨床で活用する際の手順を整理します。

### ステップ1：病歴聴取（ヒアリング）
まずは患者さんやご家族から情報を集めます。「最近、服が緩くなったと感じませんか？」「食事の好みが変わったり、固いものが食べにくくなったりしていませんか？」といった具体的な質問を投げかけましょう。

### ステップ2：視診・触診（身体所見）
薬剤師であっても、服の上からでも確認できる部位をチェックします。
- <strong>側頭部・鎖骨下</strong>: 筋肉の萎縮が最も現れやすい部位です。
- <strong>手背</strong>: 皮下脂肪の減少がないか確認します。
- <strong>浮腫</strong>: 脛骨前面の圧痕の有無を確認します。※浮腫があると体重が減っていても隠れてしまうため注意が必要です。

### ステップ3：総合評価（ランク付け）

## ⚠️ 4. 薬剤師が注意すべき「栄養の死角」

SGAを行う際、薬剤師として特に注意したいポイントが<strong>「薬物療法の副作用」と「栄養」の関連</strong>です。

1. <strong>薬剤による消化器症状</strong>: 悪心・嘔吐を引き起こしやすい薬剤（抗がん剤、一部の抗菌薬など）を服用している場合、それが栄養状態の悪化原因となります。
2. <strong>多剤併用（ポリファーマシー）</strong>: 薬の影響で眠気が強くなり、食事の時間が起きられない。これも立派な「栄養阻害要因」です。
3. <strong>検査値との解離</strong>: 肝機能障害や腎機能障害がある場合、アルブミン値はあてになりません。この時こそ、SGAのような身体評価が真価を発揮します。

## 🌿 5. まとめ：検査データ＋αの評価を

SGAは特別な医療機器を必要とせず、薬剤師の「眼」と「対話」だけで実施できる素晴らしい評価法です。

検査データはあくまで「過去の積み重ね」です。目の前の患者さんの鎖骨や手背の痩せ具合、日々の食事の様子に目を向けることで、検査値が出る前に「栄養改善のチャンス」を掴むことができます。

皆さんの病棟でも、ぜひ次回の回診から「SGA」の視点を取り入れてみてください。「あ、この患者さん、検査値は正常だけど栄養介入が必要だ！」という気づきが、チーム医療における薬剤師の価値をさらに高めてくれるはずです。
