# EPDSとは？産後うつ病のスクリーニングに用いられる質問票

<div style="background-color: #f0fdf4; border-left: 4px solid #22c55e; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  ある日の午後、乳幼児を連れた母親が薬局にいらっしゃいました。いつもの薬の受け渡しの際、ふと「夜、眠れてますか？」と何気なく声をかけると、「実は…赤ちゃんが泣き止まないと、自分がダメな母親のような気がして、涙が止まらなくて」と打ち明けられました。<br><br>
  その時、私は「産後うつ」の可能性が頭をよぎりましたが、どう対応すべきか言葉に詰まってしまった経験があります。単なる「育児疲れ」と「うつ状態」の境界線は非常に曖昧です。そんな時、客観的な指標となるのが<strong>EPDS（エジンバラ産後うつ病質問票）</strong>です。
</div>

薬剤師は、患者さんの心身の変化に最も近い距離で触れられる医療職の一人です。今回は、産後うつ病のサインを逃さないためのツール「EPDS」について、実務的な視点で深掘りしていきます。

---

## 🔍 1. EPDS（エジンバラ産後うつ病質問票）とは？

EPDS（Edinburgh Postnatal Depression Scale）は、産後うつ病を早期に発見するために開発された、<strong>自己記入式の質問票</strong>です。世界中で広く用いられており、その簡便さと精度の高さから、妊産婦健診でもスタンダードになっています。

### なぜ薬剤師が知っておくべきなのか？
産後うつは、ホルモンバランスの急激な変化や育児のストレスが重なり、誰にでも起こり得る疾患です。患者さんが直接「うつっぽい」と訴えることは少なく、<strong>「眠れない」「頭痛がする」「母乳が出にくい」といった身体症状</strong>を主訴に来局されるケースが非常に多いためです。

---

## 📝 2. EPDSの仕組みと判定基準

EPDSは、過去7日間の気持ちの状態を尋ねる全10項目から構成されています。

| 項目数 | 内容の要点 |
| :--- | :--- |
| 1〜3 | 喜びや楽しみの喪失（アニヘドニア） |
| 4〜6 | 不安やパニック状態 |
| 7〜8 | 罪悪感や自責の念 |
| 9〜10 | 自傷や抑うつ気分 |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 判定の目安</strong><br>
  ・<strong>合計9点以上</strong>：うつ状態の可能性が高い。<br>
  ・<strong>13点以上</strong>：うつ病の可能性が強く、専門医への受診勧奨が推奨される。<br>
  <br>
  ※あくまでスクリーニングツールであり、確定診断には医師の診察が必要です。
</div>

## 💊 3. 産後うつ治療と薬剤師の関わり

産後うつと診断された場合、SSRIなどの抗うつ薬が選択されることがありますが、授乳中であれば<strong>「母乳への移行」</strong>を懸念される患者さんは非常に多いです。

### 授乳中の薬物治療に関する比較

| 薬剤の種類 | 授乳中の考え方 | 薬剤師の助言ポイント |
| :--- | :--- | :--- |
| <strong>セルトラリン</strong> | 比較的安全性が高いとされる | 母乳への移行率が低いことを伝え、服薬継続を支援 |
| <strong>パロキセチン</strong> | セルトラリンに準じ検討 | 離脱症状に注意が必要であることを情報提供 |
| <strong>その他抗不安薬</strong> | 注意が必要 | 眠気等の副作用について服薬指導で確認 |

<div style="border: 2px dashed #94a3b8; padding: 1.5rem; border-radius: 8px; margin-top: 1.5rem;">
  <strong>【薬剤師からのアドバイス】</strong><br>
  薬を渡す際、<strong>「赤ちゃんへの影響を心配して自己判断で中断してしまう」</strong>患者さんが後を絶ちません。<br>
  「このお薬は、お母さんが元気でいることが赤ちゃんにとっても一番の栄養になるという考え方で処方されていますよ」というように、<strong>「お母さんのケア＝赤ちゃんのケア」</strong>であるという視点を伝えて安心感を醸成しましょう。
</div>

## 🎯 4. まとめ：薬局から始める「こころのケア」

EPDSのスコアが高い患者さんや、日々の会話でサインを感じた患者さんに対して、薬剤師ができることは以下の通りです。

1. <strong>話を否定せず、共感する</strong>: 「辛いですね」「よくここまで頑張りましたね」という言葉が最も大切です。
2. <strong>専門機関への架け橋になる</strong>: 精神科や心療内科だけでなく、産婦人科のフォローアップ外来も有効です。
3. <strong>継続的なモニタリング</strong>: お薬をお渡しする際に「その後、お気持ちは落ち着いていますか？」と一言添えるだけで、孤立を防ぐことができます。

EPDSというツールを頭の片隅に置いておくことで、日々の服薬指導の質は大きく変わります。患者さんが「ここに来れば、話を聞いてもらえる」と思えるような、温かい薬局作りを目指していきましょう。
