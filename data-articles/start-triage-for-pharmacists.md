# START法とは？災害医療におけるトリアージの基本を解説

  <strong>【薬剤師の現場体験：災害研修会での出来事】</strong><br>

  正直なところ、私は「重症度が高い人から」と漠然と考えていましたが、現場では瞬時に判断を下し、タグ付けをしていく必要があります。その際、医師ではない私たちが指標として知っておくべきなのが<strong>「START法」</strong>という救急現場の国際的なルールでした。災害時、薬剤師が薬の提供だけでなく「トリアージのサポート」を求められる場面は必ずあります。その時にパニックにならないための基礎知識をまとめました。

## 🚑 1. トリアージとは？なぜSTART法が必要なのか

災害現場では、限られた医療資源で「最大多数の最大幸福」を実現しなければなりません。そのため、負傷者の緊急度と重症度を素早く判別する「トリアージ」が行われます。

<strong>START法（Simple Triage and Rapid Treatment）</strong>は、訓練を受けたスタッフであれば、誰でも短時間（1人あたり30秒〜1分）で判定できるように設計された、世界的に最も標準的なトリアージ手法です。

---

## 🔍 2. START法の4ステップ判定手順

START法では、呼吸・循環・意識の3点を確認し、以下の4つの色に分類します。

<div style="overflow-x: auto;">
| 判定色 | 優先度 | 対象の状態 |
| :--- | :--- | :--- |
| <strong>赤（RED）</strong> | 最優先 | 直ちに処置が必要（生命の危機がある） |
| <strong>黄（YELLOW）</strong> | 待機可能 | 処置は必要だが緊急性は低い（生命の危険は差し迫っていない） |
| <strong>緑（GREEN）</strong> | 軽症 | 自力歩行が可能（「歩ける人」はまずここに分類） |
| <strong>黒（BLACK）</strong> | 死亡・救命不可 | 死亡、または救命の見込みがない |
</div>

### 【判定の流れ】
1. <strong>まず叫ぶ:</strong> 「歩ける人はこちらへ！」 → <strong>緑（軽症）</strong>
2. <strong>呼吸確認:</strong> 呼吸がない場合、気道確保をして呼吸が戻れば <strong>赤</strong>、戻らなければ <strong>黒</strong>。
3. <strong>呼吸数:</strong> 30回/分以上であれば <strong>赤</strong>。
4. <strong>循環確認:</strong> 橈骨動脈（手首の脈）が触れない、または毛細血管再充満時間（爪を押して色が戻る時間）が2秒以上 → <strong>赤</strong>。
5. <strong>意識状態:</strong> 簡単な命令（「手を握って」など）に従えない → <strong>赤</strong>。

※これらに当てはまらない、または上記以外の中等症 → <strong>黄</strong>

## 💊 3. 災害医療における薬剤師の役割とは？

トリアージそのものは医師や救急隊が行うのが基本ですが、薬剤師が現場にいる場合、以下の役割が重要になります。

### 現場で役立つアドバイス
* <strong>薬の持参確認:</strong> 患者さんの「お薬手帳」の有無は災害現場で最強の武器になります。トリアージの際に手帳を持っていることがわかれば、その後の治療方針が劇的に立てやすくなります。
* <strong>トリアージ後の継続管理:</strong> トリアージは一度やったら終わりではありません。容態の変化を見逃さないためのモニタリングが必要です。

## 🌿 4. 最後に：備えあれば憂いなし

災害医療は「非日常」ですが、そこに備えることは「薬剤師としての専門性」を再確認することでもあります。

START法を覚えるコツは、とにかく<strong>「呼吸・脈拍・意識」の順で確認する癖をつけること</strong>です。災害が発生した際、私たちが「薬を出すだけの存在」ではなく、医療チームの一員としてトリアージの概念を理解していれば、救える命は確実に増えます。
