# LASMIとは？精神障害者の生活機能評価尺度を解説

そんな反省を活かし、精神科領域で活用されている<strong>「LASMI（ラスミ）」</strong>という指標について整理しました。服薬指導の質を一段高めたいと考えている方は、ぜひ参考にしてください。

## 🔍 1. LASMI（生活機能評価尺度）とは？

LASMI（Life Assessment Scale for the Mentally Ill）は、精神障害者の<strong>「生活機能（日常生活を営む能力）」を客観的に評価するための尺度</strong>です。

精神疾患の治療目標は、単なる症状の改善だけではありません。「いかに社会生活を送り、QOLを高めるか」が重要です。LASMIは、患者さんが社会でどのように機能できているかを以下の5つの領域で数値化し、評価します。

### LASMIの5つの評価領域
| 評価領域 | 内容のポイント |
| :--- | :--- |
| <strong>I. 精神・行動</strong> | 幻覚、妄想、興奮などの症状が生活に与える影響 |
| <strong>II. 日常生活</strong> | 食事、入浴、清潔保持などのセルフケア能力 |
| <strong>III. 対人関係</strong> | 家族、友人、スタッフとのコミュニケーション能力 |
| <strong>IV. 労働・活動</strong> | 仕事や作業所など、日中の活動への参加状況 |
| <strong>V. 知識・能力</strong> | 金銭管理、薬の管理、買い出しなどの実用的なスキル |

## 💡 2. なぜ薬剤師がLASMIを知っておくべきか？

「LASMIの評価は主治医や精神保健福祉士がやることでは？」と思うかもしれません。しかし、薬剤師がこの視点を持つことで、<strong>服薬指導の内容が劇的に変わります。</strong>

## 📊 3. LASMIに基づく服薬支援の具体策

LASMIの評価項目を意識すると、患者さん一人ひとりに合わせた「オーダーメイドの提案」が可能になります。

### ケース別：薬剤師の介入例
- <strong>日常生活（II）が低下している場合</strong>
  - <strong>提案</strong>：服薬を習慣化するために、食卓に薬を置くよう指導する、あるいは服薬をスマホアラームと連動させる等の工夫を検討。
- <strong>労働・活動（IV）が安定してきた場合</strong>
  - <strong>提案</strong>：日中の活動量が増えることで、夕方の眠気やふらつきがリスクになる可能性を説明し、服薬タイミングの見直しを主治医に提案。
- <strong>知識・能力（V）に懸念がある場合</strong>
  - <strong>提案</strong>：多剤併用や複雑な用法を避け、主治医と相談の上で配合剤への変更や回数減を検討する。

## ⚠️ 4. 注意点：LASMIはあくまで「指標」

LASMIは非常に有用なツールですが、以下の点には注意が必要です。

1. <strong>患者さんへの「診断」ではない</strong>：LASMIは患者さんの能力をラベリングするためのものではなく、支援の必要性を明確にするためのツールです。
2. <strong>情報の共有範囲</strong>：患者さん本人のプライバシーやプライドにも関わる部分です。評価結果を指導の中で伝える際は、患者さんの意欲を削がないよう、常に「支援のための情報」として扱うことが重要です。

## 🌿 5. まとめ：生活を見つめる服薬指導を

精神科領域の薬物療法において、お薬はあくまで生活を支える一つのツールです。

LASMIの視点を持つということは、<strong>「薬を飲ませること」ではなく「薬を使って、患者さんがより良い生活を送れるようサポートすること」</strong>に意識を向けることに他なりません。

明日からの服薬指導で、少しだけ患者さんの「生活機能」に目を向けてみてください。その変化が、患者さんとの信頼関係を築く大きな一歩になるはずです。
