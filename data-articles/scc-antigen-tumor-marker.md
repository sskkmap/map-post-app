# SCC抗原とは？扁平上皮がんの腫瘍マーカーを解説

<div style="background-color: #f8fafc; border-left: 4px solid #64748b; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  先日、外来患者さんの検査値データを確認していた際、SCC抗原が基準値を大きく上回っていることに気がつきました。私は慌てて「がんの再発か？」と思い込み、疑義照会の準備をしてしまったのですが、念のため他の検査値を確認すると、その患者さんは重度のアトピー性皮膚炎で、最近皮膚症状が悪化していたのです。<br><br>
  結局、皮膚の炎症による偽陽性であったことが判明し、冷や汗をかいた経験があります。SCC抗原は「扁平上皮がん」の特異的なマーカーである一方、皮膚疾患などの良性疾患でも容易に上昇するという罠を忘れてはならない、と痛感した出来事でした。
</div>

今回は、臨床現場で遭遇することの多い「SCC抗原」について、薬剤師が知っておくべき知識を整理していきます。

---

## 🔍 1. SCC抗原とは？：扁平上皮がんの指標

SCC抗原（Squamous Cell Carcinoma related antigen：扁平上皮がん関連抗原）は、子宮頸部などの扁平上皮から発見された腫瘍マーカーです。

基本的には、<strong>「扁平上皮がん」</strong>が発生すると、がん細胞から血中に放出されるため、数値が上昇します。

### SCC抗原が注目される主な疾患
- <strong>肺がん（扁平上皮がん）</strong>：肺がんの中でもSCC抗原が有効な指標となります。
- <strong>子宮頸がん</strong>：婦人科領域でのスクリーニングや経過観察に用いられます。
- <strong>食道がん、頭頸部がん</strong>：これらも扁平上皮由来のため、マーカーとして利用されます。

---

## ⚠️ 2. 【重要】なぜ数値が上がる？「偽陽性」の注意点

薬剤師が最も注意すべきは、<strong>「SCC抗原＝悪性腫瘍」と即断してはいけない</strong>という点です。SCC抗原は、皮膚や粘膜の炎症によっても血中濃度が上昇するため、以下の疾患や状況で「偽陽性」を示すことがよくあります。

| 状況・疾患 | 上昇のメカニズム |
| :--- | :--- |
| <strong>アトピー性皮膚炎</strong> | 皮膚のバリア機能障害と慢性炎症により血中に漏出 |
| <strong>湿疹・乾癬</strong> | 広範囲の皮膚炎症により数値が跳ね上がることがある |
| <strong>腎不全（CKD）</strong> | SCC抗原は主に腎臓で代謝・排泄されるため、腎機能低下で蓄積する |
| <strong>肺疾患（気管支炎など）</strong> | 肺の炎症によっても上昇する可能性がある |

## 💊 4. まとめ：検査値の裏側を読むために

SCC抗原は非常に有用な腫瘍マーカーですが、その特異性には限界があります。

1. <strong>がん診断の確定ではない</strong>: あくまで「疑い」や「経過観察」の補助指標です。
2. <strong>皮膚疾患・腎機能を確認</strong>: 高値を見たときには、検査データだけを見ず、服薬指導時の問診で皮膚の状態や腎機能の推移を確認しましょう。
3. <strong>チーム医療での活用</strong>: 数値の変動が激しい場合、皮膚疾患の増悪なのか、腫瘍の増大なのか、医師の判断を仰ぐための情報収集（既往歴の確認など）を薬剤師が行うことで、より安全な薬物療法につながります。

検査値は「物語」です。その数字がなぜ動いているのか、患者さんの生活や背景から読み解くことで、私たちの薬剤師としての価値はより高まるはずです！
