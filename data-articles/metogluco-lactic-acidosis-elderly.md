# 高齢者にメトグルコは危険？乳酸アシドーシスとは

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  先日、70代後半の患者さんから「最近、胃腸の調子が悪いし、足もだるくて力が入らない」という相談を受けました。お薬手帳を確認すると、糖尿病治療薬としてメトグルコ錠が処方されています。<br><br>
  「メトホルミンだし、消化器症状なら副作用の範囲内かな」と軽く考えていたのですが、ふと添付文書の警告欄を再確認してゾッとしました。脱水気味であることや、最近の検査値で腎機能が低下している可能性を見落としていたのです。もしこれが「乳酸アシドーシスの初期症状」だったらと思うと、背筋が凍る思いをしました。
</div>

メトホルミン（製品名：メトグルコなど）は、糖尿病治療の第一選択薬として非常に有用ですが、高齢者への使用には「乳酸アシドーシス」という重大なリスクが付きまといます。今回は、薬剤師として最低限押さえておくべきポイントを整理します。

---

## 🧪 1. 乳酸アシドーシスってどんな病態？

乳酸アシドーシスとは、血液中に乳酸が蓄積し、体液が酸性に傾くことで、急激な全身倦怠感や意識障害などを引き起こす緊急性の高い病態です。

メトホルミンは「糖新生を抑制する」という優れた作用を持ちますが、それと同時に<strong>糖の嫌気的代謝を促進し、乳酸の産生を増やす</strong>という側面があります。通常は肝臓で乳酸が代謝されますが、腎機能が低下していると、排泄が追いつかずに蓄積し、発症リスクが急上昇します。

### 🚨 乳酸アシドーシスの危険因子
以下に該当する患者さんへの服薬指導は、特に注意が必要です。

| リスク因子 | 理由・注意点 |
| :--- | :--- |
| <strong>腎機能障害</strong> | eGFRの低下に伴い、メトホルミンの排泄が遅延する |
| <strong>高齢者（75歳以上）</strong> | 生理的な腎機能低下に加え、脱水に気づきにくい |
| <strong>脱水状態</strong> | 嘔吐・下痢・発熱・発汗時。腎血流量が減少し、排泄能が低下する |
| <strong>過度のアルコール</strong> | 乳酸代謝を阻害し、乳酸アシドーシスを誘発しやすい |
| <strong>造影剤検査</strong> | 検査前後は腎機能が一時的に低下するため休薬が必要 |

## 🛑 2. 薬剤師がチェックすべき「休薬基準」

添付文書では、腎機能に応じた投与量の調整や、脱水時の休薬が義務付けられています。特に<strong>「シックデイ（体調不良時）」</strong>の対応は、患者さんへの説明の要です。

<div style="background-color: #fff1f2; border: 1px solid #fecdd3; padding: 1rem; border-radius: 8px; color: #9f1239;">
  <strong>💡 ポイント：シックデイ・ルールを具体的に指導する</strong><br>
  「熱が出たとき」「激しい下痢・嘔吐があるとき」「食事があまり摂れないとき」は、メトグルコの服用を一時的に中止するように伝えてください。
</div>

### eGFR値による目安（参考）
*   <strong>eGFR 60以上</strong>: 通常通り使用可能（高齢者は慎重投与）
*   <strong>eGFR 30〜60</strong>: 慎重に投与、または減量を検討
*   <strong>eGFR 30未満</strong>: <strong>原則禁忌</strong>

※施設やガイドラインにより基準が異なる場合があるため、常に最新の添付文書を参照してください。

## ⚠️ 3. 初期症状を見逃さないための聞き取り

乳酸アシドーシスの初期症状は非特異的で、風邪や胃腸炎と区別がつきにくいのが特徴です。以下のサインがないか、定期的なフォローアップで確認しましょう。

1.  <strong>消化器症状</strong>: 胃部不快感、腹痛、吐き気、嘔吐、下痢
2.  <strong>全身症状</strong>: 倦怠感、脱力感、筋肉痛
3.  <strong>呼吸・循環</strong>: 過呼吸（クスマウル呼吸：深く速い呼吸）、低血圧、徐脈

<strong>「最近、食欲がない」「なんとなく体がだるい」「お腹の調子が悪い」</strong>という訴えがあった際は、安易に整腸剤や胃薬を出す前に、「水分は摂れているか？」「腎機能の推移はどうなっているか？」を確認し、必要であれば主治医へ情報提供を行いましょう。

## 🌿 4. まとめ：薬剤師の役割

メトグルコは決して「危険な薬」ではありません。適切に管理すれば、高齢者の糖尿病治療において非常に強力な味方になります。

私たちが果たすべき役割は、<strong>「薬そのものを怖がる」のではなく、「リスクが高い状態（脱水や腎機能低下）をいち早く察知する」</strong>ことです。

- <strong>処方監査時</strong>: eGFRの値をチェックし、年齢に見合っているか確認する。
- <strong>服薬指導時</strong>: 「体調が悪いときは飲まないで、まず連絡をください」と伝える。
- <strong>薬歴管理</strong>: 血液検査値（特にクレアチニン、eGFR）を時系列で把握しておく。

患者さんの「小さな違和感」を「乳酸アシドーシスのサインかもしれない」と疑えるかどうかが、重大な副作用を防ぐ最後の一線になります。明日からの外来対応で、ぜひ意識してみてくださいね。
