# 🩺 サルコイドーシスの重症度分類と薬剤師が知っておくべき評価ポイント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の臨床エピソード】</strong><br>
  「ずっと経過観察だったのに、急に強いステロイドが始まったんです……」<br>
  薬局のカウンターで不安そうに話すサルコイドーシスの患者さん。これまでは半年に一度の検査だけだったのに、プレドニゾロン30mg/日の処方が出ました。<br><br>
  「肺の影は変わっていないのになぜ？」と疑問に思い、お薬手帳を確認すると、最近「眼科」と「循環器内科」を相次いで受診していることがわかりました。サルコイドーシスは「どの臓器に」「どの程度」影響が出ているかで治療方針がガラリと変わる疾患だと、改めて痛感した瞬間でした。
</div>

サルコイドーシスは、肺、眼、心臓など全身の至る所に「非乾酪性上皮細胞性肉芽腫」ができる疾患です。薬剤師として、患者さんが<strong>「なぜ治療が必要になったのか（重症度はどうなのか）」</strong>を把握することは、副作用モニタリングや服薬アドヒアランス向上のために非常に重要です。

今回は、臨床で使われる重症度分類と、臓器障害評価のポイントを解説します。

---

## 📊 1. サルコイドーシスの重症度分類（厚生労働省基準）

サルコイドーシスは指定難病であり、医療費助成の対象となる「重症度」の基準が定められています。主に<strong>「臨床症状」</strong>と<strong>「臓器障害」</strong>の2軸で評価されます。

### ① 総合的重症度判定

一般的に、以下の「重症（Ⅲ度以上）」に該当する場合、ステロイド全身投与などの積極的な治療が検討されます。

| 重症度 | 判定基準の概要 | 治療の必要性 |
| :--- | :--- | :--- |
| <strong>Ⅰ度（軽症）</strong> | 臓器障害が軽微で、日常生活に支障がない | 原則、経過観察 |
| <strong>Ⅱ度（中等症）</strong> | 中等度の臓器障害を認める | 症例により加療 |
| <strong>Ⅲ度（重症）</strong> | <strong>生命予後や機能的予後に影響する障害</strong>がある | <strong>治療開始を検討</strong> |
| <strong>Ⅳ度（最重症）</strong> | 重篤な臓器不全（心不全、呼吸不全など）を伴う | <strong>強力な治療が必須</strong> |

### ② 胸部X線Stage分類（肺サルコイドーシスの進行度）

呼吸器系の評価で必ず登場するのが、Scaddingの分類に基づいたStage判定です。これは重症度と必ずしも一致しませんが、病態の把握には欠かせません。

## 👁️ 2. 臓器障害評価のチェックポイント：注意すべき「3大臓器」

サルコイドーシスの予後を左右するのは、<strong>「肺・心臓・眼」</strong>の障害です。これらに重大な障害がある場合、難病の重症度判定で「重症」以上となります。

### 🩺 心臓サルコイドーシス（最重要！）
最も注意すべきは心臓への浸潤です。<strong>心不全や致死的不整脈</strong>の原因となるため、この兆候がある場合は即治療対象となります。
- <strong>評価ポイント</strong>: 心電図異常（完全右脚ブロック、房室ブロック）、心エコーでの壁運動低下、心臓MRI/FDG-PETでの集積。
- <strong>薬剤師の視点</strong>: 「動悸や息切れ、めまいはありませんか？」という確認が、潜在的な心病変の発見に繋がります。

### 👁️ 眼サルコイドーシス
視力低下や失明のリスクがあるため、QOLへの影響が非常に大きいです。
- <strong>評価ポイント</strong>: ぶどう膜炎、硝子体混濁、霧視（かすみ目）。
- <strong>薬剤師の視点</strong>: 点眼薬（ステロイドなど）が併用されている場合、眼の炎症が強いことを示唆します。

### 🫁 肺サルコイドーシス
多くの患者で見られますが、進行して肺線維症になると呼吸不全に至ります。
- <strong>評価ポイント</strong>: 肺機能検査（%VCや%DLCOの低下）、労作時息切れ。
- <strong>薬剤師の視点</strong>: KL-6などの間質性肺炎マーカーの上昇にも注目です。

## 💊 3. 治療薬の選択と薬剤師の役割

サルコイドーシスの第一選択薬は<strong>副腎皮質ステロイド（プレドニゾロン）</strong>です。

### ステロイド治療の標準的な流れ

| ステップ | 薬剤・用量の目安 | ポイント |
| :--- | :--- | :--- |
| <strong>初期投与量</strong> | PSL 0.5mg/kg/日（約20〜30mg） | 4〜8週間継続し、効果を判定 |
| <strong>維持量</strong> | PSL 5〜10mg/日 | 1〜2年かけて漸減し、維持する |
| <strong>難治性の場合</strong> | メトトレキサート（MTX）等 | ステロイド減量が困難な場合に併用 |

<div style="background-color: #fff1f2; border: 1px solid #fecdd3; padding: 1rem; border-radius: 8px; color: #9f1239; margin: 1.5rem 0;">
  <strong>⚠️ ステロイド開始時の注意点</strong><br>
  ステロイドの開始により、高血糖、脂質異常、骨粗鬆症、感染症のリスクが高まります。特に中等量以上で開始する場合は、<strong>ST合剤（バクタ等）によるニューモシスチス肺炎の予防</strong>や、骨粗鬆症予防薬（ビスホスホネート等）の併用が適切になされているか確認しましょう。
</div>

## 🔍 4. 薬剤師が確認すべき検査値まとめ

処方箋から重症度や病態を推測するために、以下の数値は要チェックです。

- <strong>ACE（アンジオテンシン変換酵素）</strong>: 肉芽腫の活動性を反映。治療効果の指標になります。
- <strong>sIL-2R（可溶性インターロイキン2受容体）</strong>: 全身の炎症・活動性を反映。
- <strong>血清カルシウム</strong>: 肉芽腫がビタミンDを活性化させるため、<strong>高カルシウム血症</strong>を来すことがあります。尿路結石の原因にもなるため、高値の場合は注意が必要です。
