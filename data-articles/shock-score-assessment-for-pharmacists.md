# ショックスコアとは？ショック状態の評価法を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の病棟カンファレンスでの出来事です。担当患者さんの血圧が「110/70 mmHg」と、いわゆる正常範囲内であったため、私は特に異常を感じていませんでした。しかし、後にその患者さんは消化管出血によるショック状態で緊急搬送されることとなりました。<br><br>
  後から確認すると、心拍数が110回/分と上昇しており、脈拍と収縮期血圧の比率をとると「1.0」を超えていたのです。当時の私は「血圧が保たれているから大丈夫」という思い込みから、代償機転（体がショックに耐えようとする反応）を見逃していました。血圧低下はショックの「末期サイン」であることを、身をもって学んだ瞬間でした。
</div>

バイタルサインをパッと見ただけで、「この患者さんは今、ギリギリで耐えているのかも？」と気づけるスキルがあれば、早期介入の鍵を握る薬剤師になれます。今回は、そんな現場の視点から<strong>「ショックスコア（Shock Index: SI）」</strong>について解説します。

---

## 📊 1. ショックスコア（SI）とは何か？

ショックスコア（Shock Index: SI）とは、ショック状態の重症度や早期兆候を評価するための簡便な指標です。

<div style="text-align: center; font-size: 1.2rem; font-weight: bold; padding: 1rem; border: 2px solid #3b82f6; border-radius: 8px; margin: 1rem 0;">
  ショックスコア（SI） ＝ 心拍数（bpm） ÷ 収縮期血圧（mmHg）
</div>

血圧は、体が極限まで頑張って維持していることが多く、<strong>「血圧が下がった時点ですでに重症」</strong>と言えます。一方、心拍数は血圧を維持するために代償的に上昇するため、ショックの極めて初期段階で変化が現れます。

### SIによる重症度評価の目安

| SI値 | ショックの程度 | 臨床的意義 |
| :--- | :--- | :--- |
| <strong>0.5 〜 0.7</strong> | 正常 | 代償機能が十分に働いている状態 |
| <strong>0.7 〜 1.0</strong> | 軽度ショック疑い | 要観察。早期発見の目安となる数値 |
| <strong>1.0 〜 1.5</strong> | 中等度ショック | 循環不全の可能性大。介入が必要 |
| <strong>1.5 以上</strong> | 重症ショック | 緊急事態。循環動態の崩壊を示唆 |

## ⚡ 2. 薬剤師が知っておくべき「SIの落とし穴」

SIは便利ですが、万能ではありません。以下のケースでは指標が当てはまらない可能性があるため注意が必要です。

### ⚠️ 高齢者やβ遮断薬服用中の患者
高齢者やβ遮断薬を使用中の患者さんでは、<strong>「心拍数が上がらない（代償できない）」</strong>ことが多くあります。
- 本来なら心拍数が120になるべき状況でも、薬の影響で70程度にとどまる。
- 結果としてSI値が低く算出され、ショックを見逃すリスク（偽陰性）があります。

### ⚠️ 薬剤によるバイタルへの影響
薬剤師の視点として、患者が服用しているお薬がバイタルサインにどう影響しているかを常に考える必要があります。

| 薬剤の種類 | バイタルへの影響 | 注意すべき点 |
| :--- | :--- | :--- |
| <strong>β遮断薬</strong> | 徐脈傾向 | SI値を低く見積もってしまう |
| <strong>利尿薬</strong> | 循環血液量減少 | 脱水によるショックの増悪 |
| <strong>降圧薬</strong> | 血圧低下 | 低血圧とショックの判別が困難 |

## 🩺 3. 現場で「ショック？」と疑うべきポイント

薬剤師が処方検討や病棟訪問時に、以下のような状況があれば、数値化するまでもなくショックを疑うべきです。

1. <strong>冷汗・顔面蒼白</strong>: 交感神経が活性化しているサインです。
2. <strong>尿量減少</strong>: 腎血流が低下している可能性が高いです。
3. <strong>意識状態の変化</strong>: 軽度のぼんやり感や不安感は、脳血流低下の初期兆候です。

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; color: #92400e;">
  <strong>💡 薬剤師の実務アドバイス</strong><br>
  「血圧が高いからショックではない」と思い込むのは危険です。特に「数日前まで高血圧だったのに、今日は血圧が110/70まで下がっている」といった<strong>変化の過程（トレンド）</strong>に注目してください。SIを計算することで、「あれ？脈拍数とのバランスが悪いかも？」という違和感を言葉にできます。
</div>

## 🏁 4. まとめ：数値＋「違和感」を武器にしよう

ショックスコアはあくまで一つのツールです。しかし、この計算を行う習慣を持つことで、<strong>「バイタルサインの『違和感』を客観的なデータとして医師に伝える」</strong>ことができます。

もし病棟で「心拍数と血圧のバランスがおかしいな？」と感じたら、ぜひSIを計算してみてください。

- 収縮期血圧 ÷ 心拍数 ではなく、<strong>心拍数 ÷ 収縮期血圧</strong> です！
- 1.0を超えていたら要注意。
- 基礎疾患や常用薬によるバイタルの変動を加味して評価する。
