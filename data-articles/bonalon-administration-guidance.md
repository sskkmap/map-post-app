# ボナロンはなぜコップ1杯の水で服用するのか？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場でヒヤリとした服薬指導】</strong><br>
  新人薬剤師の頃、「ボナロン経口ゼリー」を処方された患者さんから「ゼリータイプだから水なしで飲めるんでしょ？」と聞かれ、「そうですね、ゼリーなので大丈夫ですよ」と安易に答えてしまったことがあります。<br><br>
  後から先輩にこっぴどく怒られました。「ゼリー剤であっても、食道への停滞を防ぐために『十分な水』が必要なことに変わりはないんだよ」。それ以来、ボナロンの服薬指導においては、剤形に関わらず「お水で流し込む」ことの重要性を強調するようになりました。
</div>

ボナロン（一般名：アレンドロン酸ナトリウム水和物）をはじめとするビスホスホネート製剤は、骨粗鬆症治療の要となるお薬です。しかし、患者さんにとって「起床時の服用」「30分間横になってはいけない」といった制限は、なかなか遵守しにくいのが実情です。

今回は、なぜこれほどまでに厳しい服用条件が課されているのか、その背景にある「食道粘膜へのリスク」を中心に整理していきます。

---

## 🧪 1. なぜ「コップ1杯の水」が必要なのか？

ビスホスホネート製剤が「コップ1杯の水」での服用を強く求められる最大の理由は、<strong>食道粘膜に対する直接的な刺激性</strong>です。

ビスホスホネート製剤は、食道内で溶け出すと粘膜を直接刺激し、炎症や潰瘍を引き起こす可能性があります。そのため、<strong>「素早く胃まで届けること」</strong>と<strong>「食道に留まらせないこと」</strong>が最優先事項となります。

| 項目 | 理由 |
| :--- | :--- |
| <strong>食道停滞の防止</strong> | 錠剤が食道に引っかかると粘膜に長時間接し、炎症・潰瘍の原因となる |
| <strong>胃内への早期到達</strong> | 水で一気に胃まで押し流すことで、粘膜との接触時間を最小限にする |
| <strong>濃度上昇の抑制</strong> | 十分な水で希釈し、粘膜への刺激濃度を下げる |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 薬剤師の指導ポイント</strong><br>
  「コップ1杯」とは大体180〜200mLです。「お薬を飲み込むための少しの水」ではなく、<strong>「お薬を胃まで確実に送り届けるための水」</strong>であることを患者さんに具体的にイメージしてもらうことが大切です。
</div>

## ⚠️ 2. 起こりうる重大な副作用「食道潰瘍」

食道や胃の粘膜障害を防ぐため、添付文書には厳しい服用制限が記載されています。これを軽視すると、以下のようなリスクが高まります。

*   <strong>食道炎・食道潰瘍</strong>: 胸の痛み、飲み込みにくさ、逆流感などの症状が現れます。
*   <strong>咽頭潰瘍</strong>: 高齢者で飲み込みの力が弱っている場合、食道以前の喉に張り付いてしまうことがあります。

### 守るべき「3つの鉄則」
1.  <strong>起き抜けに服用する</strong>（立位または坐位を保つ）
2.  <strong>コップ1杯の水で飲む</strong>（お茶やジュースは避ける）
3.  <strong>服用後30分は横にならない</strong>（逆流を防ぐ）

---

## 💊 3. 製剤による違いと指導の切り口

ボナロンには錠剤だけでなく、飲み込みが難しい患者さんのための「経口ゼリー」や「内用液」もあります。剤形別の指導のヒントをまとめました。

| 剤形 | 指導のポイント |
| :--- | :--- |
| <strong>錠剤</strong> | 砕いたり噛んだりせず、多めの水で必ず飲み込むように伝える |
| <strong>経口ゼリー</strong> | 「ゼリーだから水なしでOK」と誤解されやすい。<strong>必ず水で流し込む</strong>ことを強調 |
| <strong>内用液</strong> | 飲みやすい分、食道に残りやすいため注意。必ずコップ1杯の水と共に摂取 |

## 🎯 まとめ：患者さんの「生活」に落とし込んだ服薬指導を

ボナロンの服薬指導で大切なのは、ただ「説明書通りに説明する」ことではなく、<strong>「なぜその動作が必要なのか」を患者さんに納得してもらうこと</strong>です。

「食道が荒れてしまうと、毎日の食事が辛くなりますよね。それを防ぐために、コップ1杯のお水で胃までしっかり流し込んであげましょう」

このように伝えることで、患者さんの服薬アドヒアランスはぐっと向上します。日々の業務では「コップ1杯のお水」という決まり文句の裏にあるリスクを想像し、患者さんの生活に寄り添った丁寧な指導を心がけていきましょう！
