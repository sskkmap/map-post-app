# DIEPSSとは？薬原性錐体外路症状評価尺度の見方と薬剤師が知っておきたいポイント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  精神科門前で働いていた頃、抗精神病薬を増量した患者さんが「最近、イライラして落ち着かない」と訴えてこられました。私はてっきり「精神症状の悪化」だと考え、医師に「精神症状が不安定のようです」と報告しました。<br><br>
  しかし、後から分かったのは、それが<strong>アカシジア（静坐不能症）という薬の副作用</strong>だったのです。当時の私には、精神症状の変化と副作用を客観的に見分けるための「物差し」が欠けていました。もしその時、DIEPSSの視点を持っていれば、もっと早く気づいて介入できたはずだと痛感しました。
</div>

抗精神病薬を扱う上で避けて通れない「錐体外路症状（EPS）」。今回は、副作用を客観的に評価するための重要なツール<strong>「DIEPSS（薬原性錐体外路症状評価尺度）」</strong>について、現場ですぐに使える視点で解説します。

---

## 🔍 1. DIEPSS（ディーイープス）ってなに？

DIEPSS（Drug-Induced Extrapyramidal Symptoms Scale）は、抗精神病薬などによって引き起こされる錐体外路症状を、<strong>「5つの項目」で客観的に評価するための尺度</strong>です。

医師の診断だけでなく、薬剤師が定期的にチェックすることで、<strong>「服薬開始後や増量後の早期発見」</strong>や、<strong>「用量調節の根拠」</strong>として非常に役立ちます。

### 評価する5つの主な症状
| 項目 | 特徴的なサイン |
| :--- | :--- |
| <strong>1. 歩行</strong> | 小股歩行、前傾姿勢、腕の振りの減少 |
| <strong>2. 姿勢</strong> | 体幹の屈曲、筋固縮（歯車様強剛） |
| <strong>3. 振戦</strong> | 手指の震え（丸薬を丸めるような動き） |
| <strong>4. 流涎（よだれ）</strong> | 嚥下障害による口角からの流出 |
| <strong>5. アカシジア</strong> | じっとしていられない、足の貧乏ゆすり |

※DIEPSSでは、0（正常）〜4（重症）の5段階でスコア化します。

---

## 🧠 2. 薬剤師が押さえておきたい！評価のポイント

DIEPSSは単なるチェックリストではなく、患者さんの「QOL（生活の質）」を守るためのツールです。薬局で患者さんと接する際、以下のポイントを意識してください。

### ① 「主観」と「客観」のギャップを確認する
患者さんが「なんとなく調子が悪い」と言うとき、それが精神的なものなのか、アカシジアによる焦燥感なのかを区別することが重要です。「足がムズムズしませんか？」「座り続けているのがつらくないですか？」と具体的に問いかけることで、見えない副作用を拾い上げます。

### ② 服薬状況との相関を記録に残す
<div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe;">
  <strong>💡 現場でのアドバイス</strong><br>
  薬歴に「EPSスコア：0→1へ変化あり」と記録を残すだけで、次回の疑義照会や医師への提案が格段にスムーズになります。「〇〇の増量後から振戦が出現しているため、減量または抗コリン薬の併用を検討してはどうか」という提案は、薬剤師としての信頼を大きく高めます。
</div>

## 💊 4. 薬剤師としての「介入」ステップ

副作用の疑いがある場合、どのように動くべきでしょうか？

1.  <strong>服薬遵守状況の確認</strong>: 飲み忘れがないか、あるいは自己判断で増量していないか。
2.  <strong>DIEPSSを用いた問診</strong>: 上記の5項目に沿って、患者さんの行動を観察。
3.  <strong>医師への疑義照会・フィードバック</strong>: 
    *   「現在のアカシジアの評価をDIEPSSで確認したところ、以前より上昇しています」
    *   「抗コリン薬の追加、あるいは抗精神病薬の減量について先生のご判断を仰ぎたいです」
4.  <strong>患者教育</strong>: 「この症状は薬の副作用である可能性が高く、調整できること」を伝え、不安を和らげる。

## 🌿 まとめ：DIEPSSを「患者さんとの共通言語」に

DIEPSSは専門的な評価尺度ですが、薬剤師がその視点を持つことで、患者さんの服薬アドヒアランスは大きく改善します。

「精神科の薬だから難しい」と敬遠せず、まずは<strong>「歩き方」「表情」「落ち着きのなさ」</strong>の3点から観察を始めてみてください。それが、患者さんのQOLを守る第一歩になります。

今日からあなたの薬局でも、患者さんの「表情」や「足の動き」に、ぜひ少しだけ注目してみてくださいね！
