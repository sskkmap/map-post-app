# SASとは？Simpson-Angus Scaleによる錐体外路症状評価

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  精神科の外来調剤を行っていた時のことです。長年リスペリドンを服用されている患者さんの服薬指導中、ふと「なんとなく歩き方がぎこちないかな？」と気になりました。ご本人に尋ねても「特に変わりないよ」とのこと。しかし、後日医師に相談したところ、軽い筋強剛（パーキンソン症状）が出現していることが発覚しました。<br><br>
  「なんとなく」の違和感を、どのように数値化して医師へフィードバックすべきか悩んでいた時、教えてもらったのが「SAS（Simpson-Angus Scale）」でした。感覚的な評価を、客観的な指標に落とし込む重要性を痛感した瞬間でした。
</div>

抗精神病薬を使用する上で、<strong>錐体外路症状（EPS）</strong>のモニタリングは薬剤師にとって非常に重要な業務です。今回は、EPS評価ツールの代表格であるSASについて、明日からの実務に役立つ知識を整理します。

---

## 🧠 1. なぜ「SAS」を使うのか？

抗精神病薬による副作用には、パーキンソン症状（筋強剛、振戦、流涎など）が含まれます。これらの症状は患者さんが自覚していないことも多く、我々医療者が早期に発見する必要があります。

<strong>SAS（Simpson-Angus Scale）</strong>は、以下の特徴を持つ評価尺度です。
- <strong>目的</strong>: 薬剤誘発性パーキンソン症状の重症度評価
- <strong>対象</strong>: 抗精神病薬服用中の患者
- <strong>メリット</strong>: 専門的な神経学的検査が難しい現場でも、簡単な動作確認でスコア化できる

---

## 📊 2. SASの評価項目と点数化のポイント

SASは主に10項目からなり、各項目を0〜4点で評価します。合計点数が高いほど、錐体外路症状が重いと判断されます。

| 項目 | 評価のポイント |
| :--- | :--- |
| <strong>歩行</strong> | 足を引きずっていないか、歩幅が狭くないか |
| <strong>振戦（ふるえ）</strong> | 手指や四肢の静止時振戦の有無 |
| <strong>唾液分泌（流涎）</strong> | 口角からのよだれ、頻繁な嚥下の有無 |
| <strong>筋強剛（頸部・四肢）</strong> | 関節を他動的に動かした時の抵抗感 |
| <strong>姿勢（前屈姿勢）</strong> | 直立時の姿勢の崩れ |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 薬剤師のワンポイントアドバイス</strong><br>
  すべてを完璧に評価しようとすると時間がかかります。まずは<strong>「歩行」と「表情（仮面様顔貌）」</strong>の観察から始めましょう。特に、服薬指導で患者さんが薬局に来るまでの「歩き方」は、診察室では見られない貴重な情報です。
</div>

---

## 🏥 3. 他の評価尺度との使い分け

臨床現場では、SAS以外にもいくつかの評価尺度が使われます。違いを理解して使い分けましょう。

| 尺度名 | 評価対象となる症状 | 主な用途 |
| :--- | :--- | :--- |
| <strong>SAS</strong> | パーキンソン症状（筋強剛など） | 錐体外路症状の全体像把握 |
| <strong>AIMS</strong> | 遅発性ジスキネジア（不随意運動） | 口部・舌の異常運動のチェック |
| <strong>DIEPSS</strong> | 錐体外路症状全般 | 総合的なEPS重症度評価 |

※<strong>AIMS（異常不随意運動尺度）</strong>は、特に長期間の抗精神病薬服用による「遅発性ジスキネジア」を早期発見するために重要です。

---

## 🛡️ 4. 薬剤師ができる「副作用モニタリング」の実践

SASを使って「何かおかしい」と感じたら、次のようなアクションをとるのが理想的です。

1. <strong>服薬状況の確認</strong>: 服用タイミングや飲み忘れがないか（血中濃度が不安定になっていないか）。
2. <strong>随伴症状の問診</strong>: 「最近、文字が書きにくくなった」「食事がこぼれやすい」など、患者さん本人が気づいていないサインを探る。
3. <strong>医師への疑義照会・情報提供</strong>: 「SASスコアで〇点相当の筋強剛が見受けられるため、減量や抗コリン薬の併用を検討いただけないでしょうか」と<strong>数値を用いて</strong>提案する。

  「なんとなくの違和感」を「SAS」というツールを使って言語化・数値化することで、医師との連携は格段にスムーズになります。患者さんのQOLを守るために、ぜひ日々の外来や在宅医療の現場でこの「視点」を取り入れてみてください。
