# 💊 Sokalスコアとは？慢性骨髄性白血病の予後予測指標

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の臨床現場エピソード】</strong><br>
  ある日、慢性骨髄性白血病（CML）の診断を受けたばかりの患者さんの外来薬指導を担当しました。処方されたのは分子標的薬のイマチニブ。「先生からリスクスコアの話を聞いたけど、何のことかよくわからなくて…」と不安そうな表情を浮かべる患者さん。<br><br>
  当時、CMLの分子標的薬は種類も増え、患者さんのリスクに応じて治療薬の選択肢が変わる時代になっていました。私は患者さんの検査データを確認し、「Sokalスコア」という指標が治療戦略のベースになっていることを丁寧に説明しました。患者さんは「自分の病状と治療の目的がクリアになった」と安心され、服薬アドヒアランスの向上に繋がったことを今でも鮮明に覚えています。
</div>

CML（慢性骨髄性白血病）の治療現場において、患者さん一人ひとりのリスクを評価し、適切な治療薬を選択することは極めて重要です。本記事では、薬剤師が知っておくべき「Sokalスコア」の概要と、実務での活用法について解説します。

---

## 🔬 1. Sokalスコアが果たす役割とは？

Sokalスコアは、診断時の<strong>「年齢」「脾臓の腫大」「血小板数」「芽球の割合」</strong>の4因子を用いて計算される、CMLの予後予測指標です。

以前はインターフェロン療法が主流でしたが、現在のTKI（チロシンキナーゼ阻害薬）治療時代においても、リスク分類に基づいた治療選択は非常に重要です。

### 📊 スコア算出に用いられる因子
計算式は複雑ですが、主な評価項目は以下の通りです。

| 因子 | 予後に与える影響 |
| :--- | :--- |
| <strong>年齢</strong> | 高齢であるほどスコアが高くなる（リスク増） |
| <strong>脾腫（脾臓の大きさ）</strong> | 腫大が大きいほどスコアが高くなる |
| <strong>血小板数</strong> | 値が高いほどスコアが高くなる |
| <strong>芽球数（末梢血）</strong> | 割合が高いほどスコアが高くなる |

※算出にはオンラインの計算ツール（Sokal score calculator）を用いるのが一般的です。

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 薬剤師のポイント</strong><br>
  リスク分類は「低リスク」「中間リスク」「高リスク」の3群に分かれます。スコアが高い（＝高リスク）ほど、より強力な治療効果が期待される第二世代以降のTKI（ダサチニブ、ニロチニブ、ボスチニブなど）を初期治療から検討する場合があります。
</div>

---

## 🔄 2. CML治療薬（TKI）の特徴と比較

リスク分類を確認した上で、現在主流となっているTKIの主な特徴を整理しました。

| 薬剤名 | 特徴 | 注意すべき副作用 |
| :--- | :--- | :--- |
| <strong>イマチニブ</strong> | 第一世代。長期成績が豊富 | 浮腫、筋肉痛、皮疹 |
| <strong>ダサチニブ</strong> | 第二世代。高い抗腫瘍効果 | 胸水、血小板減少 |
| <strong>ニロチニブ</strong> | 第二世代。イマチニブより強力 | 高血糖、脂質異常症、動脈閉塞性疾患 |
| <strong>ボスチニブ</strong> | 第二世代。副作用プロファイルが異なる | 下痢、肝機能障害 |

---

## 🎯 3. 服薬指導で薬剤師ができること

Sokalスコアによるリスク分類が高い患者さんは、特に治療の継続性が予後を左右します。以下のポイントに注力しましょう。

### 1. 副作用の早期発見
特にリスクの高い群に投与される第二世代TKIは、副作用の種類が多岐にわたります。
* <strong>ダサチニブなら</strong>：息切れや体重増加（胸水の兆候）の確認。
* <strong>ニロチニブなら</strong>：空腹時服用の徹底と、代謝系への影響モニタリング。
* <strong>ボスチニブなら</strong>：服薬初期の下痢への対処（整腸剤の併用提案など）。

### 2. アドヒアランスの維持
TKI治療は「飲み忘れ」が分子学的寛解（MR）の達成を妨げる最大の要因です。スコアに基づいた「なぜこの薬が必要なのか」という説明は、患者さんのモチベーション維持に直結します。

## ⚠️ 4. 注意点：Sokalスコア以外の評価指標

最近ではSokalスコアに加え、<strong>ELTS（EUTOS Long-Term Survival）スコア</strong>も普及しています。これはTKI治療時代の予後をより正確に反映する指標として、Sokalスコアと並んで臨床現場で参照されるようになっています。

## 🌿 5. まとめ：データから個別化医療へ

Sokalスコアは単なる数字ではなく、<strong>「患者さんがどの程度の治療強度を必要としているか」</strong>を示す羅針盤です。

薬剤師として、検査値の変化だけでなく、リスク分類という背景情報を理解しておくことで、医師への提案内容や患者さんへの説明の質が大きく変わります。CML治療における「寛解」を目指して、チーム医療の一員として専門性を発揮していきましょう！
