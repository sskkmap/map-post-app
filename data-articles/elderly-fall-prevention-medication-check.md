# 🚶 親が転びやすくなったと感じたら確認したいポイント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ヒヤリハット】</strong><br>
  ある日、外来で「最近、父が自宅でよく転ぶようになったんです。足腰が弱ってきたのかと心配で…」と相談されました。処方内容を確認すると、不眠を訴える患者さんに対して、新しく「ベンゾジアゼピン系睡眠薬」が追加されていました。<br><br>
  「足元がふらつくのは薬のせいかもしれません」と医師に疑義照会し、減量・変更を提案したところ、数週間後には「ふらつきが減った」と報告が。単なる加齢のせいだと見過ごしていたら、骨折という大きな事故につながっていたかもしれない……薬剤師として「薬が生活を制限していないか」を考える重要性を痛感した出来事でした。
</div>

高齢者の転倒は、骨折による要介護状態への入り口です。薬剤師として、患者さんやご家族から「最近転びやすくなった」という言葉を聞いたとき、私たちは「筋力」だけでなく「薬」にも目を向ける必要があります。

## ⚠️ 1. なぜ高齢者は薬で「転ぶ」のか？

高齢者は、若年層に比べて薬の代謝・排泄能が低下しており、<strong>「ふらつき」「ふらつき・めまい・眠気」</strong>といった副作用が出やすい状態にあります。特に注意が必要な薬剤を以下にまとめました。

| 薬剤の分類 | 転倒リスクを高める主な要因 | 注意が必要な薬剤例 |
| :--- | :--- | :--- |
| <strong>睡眠薬・抗不安薬</strong> | 筋弛緩作用、鎮静作用による反応の遅れ | ベンゾジアゼピン系、Z-drug |
| <strong>降圧薬</strong> | 起立性低血圧による脳血流の低下 | α遮断薬、利尿薬、過度な降圧 |
| <strong>抗ヒスタミン薬</strong> | 鎮静作用、抗コリン作用による注意力の低下 | 市販のかぜ薬、鼻炎薬、一部の眠り薬 |
| <strong>抗精神病薬</strong> | パーキンソニズム、錐体外路症状による歩行障害 | 定型・非定型抗精神病薬 |

## 🔍 2. 実務で確認すべき「3つの視点」

診察や服薬指導の際、以下のポイントを意識して会話を進めてみましょう。

### ① 「いつ」転びやすいのか？
- <strong>夜間・早朝</strong>: 睡眠薬の作用が残っていないか、利尿薬の影響で夜間の排尿回数が増えていないかを確認します。
- <strong>立ち上がった瞬間</strong>: 降圧薬による起立性低血圧が疑われます。血圧手帳の記録があれば、早朝・日中の差を確認しましょう。

### ② 自己判断での「市販薬」使用はないか？
高齢者の方は、病院の薬以外に「眠れないから市販の睡眠改善薬を飲んでいる」「鼻炎がひどいから市販の鼻炎薬を併用している」というケースが多くあります。これらが処方薬と重複し、鎮静作用を増強させているリスクがあります。

### ③ 薬の「飲み忘れ」と「まとめ飲み」
飲み忘れた分を一度に飲んでしまい、一時的に血中濃度が急上昇してふらつきが生じている場合もあります。

## 🚀 4. 薬剤師からの「次の一手」

「転びやすくなった」という主訴に対し、薬剤師がアクションを起こす手順を整理します。

1. <strong>お薬手帳の網羅的確認</strong>: 処方薬だけでなく、重複投与がないか徹底確認する。
2. <strong>医師への処方提案</strong>: 転倒リスクが高いと思われる場合、以下の提案を検討します。
   - 作用時間の短い薬剤への切り替え
   - 減量（特に睡眠薬や降圧薬）
   - 服用タイミングの調整（例：夜間頻尿があるなら利尿薬の服用を午前中へ）
3. <strong>多職種連携</strong>: 介護サービスを利用している場合は、ケアマネジャーへ「服薬上のリスクがある」旨を情報提供し、生活環境での注意喚起を依頼する。

## 🌿 まとめ：薬の整理は「転倒予防」になる

高齢者の転倒を減らすことは、単に薬を減らすことではなく、<strong>「その人らしい生活を守ること」</strong>に直結します。「最近、ふらつく気がする」という一言を見逃さず、薬を見直すことで、患者さんの活動的な毎日を支えていきましょう。

些細なことでも、相談してもらえる関係性を築くことが、転倒を防ぐ最強のツールになります。ぜひ、次回の外来から「最近の歩き心地」について、一言添えてみてくださいね。
