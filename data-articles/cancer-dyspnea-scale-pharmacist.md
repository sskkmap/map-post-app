# Cancer Dyspnea Scale（CDS）とは？がん患者の呼吸困難評価尺度

  医師へ報告する際、「息苦しそうです」としか伝えられず、結局「レスキュー薬を追加しましょう」という提案にとどまってしまいました。後日、緩和ケアに詳しい先輩から<strong>「CDS（Cancer Dyspnea Scale）を使って客観的に評価してみたら？」</strong>と助言され、呼吸困難の「質」と「強さ」を分けて評価できるようになってから、呼吸苦に対する提案がぐっと的確になった経験があります。

がん患者さんにとって「呼吸困難」は非常に苦痛な症状ですが、主観的な訴えのため評価が難しい側面があります。今回は、臨床現場で役立つ評価尺度「CDS」について深掘りしていきましょう。

---

## 🔍 1. CDS（Cancer Dyspnea Scale）とは？

<strong>Cancer Dyspnea Scale（CDS）</strong>は、がん患者さんの呼吸困難感を、12項目の質問から「身体的苦痛」「不快感」「恐怖感」の3つの因子に分類して評価するツールです。

「単に苦しい」だけでなく、<strong>「具体的に何がどのように辛いのか」</strong>を可視化することで、薬物療法（オピオイドや抗不安薬、ステロイド等）の選択や、非薬物療法のアプローチが明確になります。

### 📊 CDSの構成因子
| 因子 | 評価内容 |
| :--- | :--- |
| <strong>身体的苦痛 (Physical)</strong> | 呼吸の努力や身体的な息切れ感（例：息が切れる、重苦しい） |
| <strong>不快感 (Discomfort)</strong> | 呼吸のしにくさに対する嫌悪感（例：空気が足りない、苦しい） |
| <strong>恐怖感 (Fear)</strong> | 呼吸困難に対する心理的な不安（例：死ぬのではないか、怖い） |

「苦しいですか？」と聞くのではなく、CDSの要素を念頭に<strong>「呼吸は重たい感じですか？それとも、息が吸えなくて怖い感じがしますか？」</strong>と具体的にヒアリングすることで、患者さんの「真の苦痛」に到達しやすくなります。

## 💊 3. 呼吸困難に対する主な薬物療法一覧

CDSで評価した苦痛の質に合わせて、医師と相談の上で治療方針を調整します。

| 薬剤の種類 | 主な適応（CDSとの関連） | 備考 |
| :--- | :--- | :--- |
| <strong>医療用麻薬</strong> | 身体的苦痛・呼吸の重苦しさ | モルヒネ等が第一選択。呼吸中枢の感受性低下に作用。 |
| <strong>抗不安薬</strong> | 呼吸困難に伴う「恐怖感・不安」 | ベンゾジアゼピン系を使用。過度な鎮静に注意。 |
| <strong>ステロイド</strong> | 炎症による気道閉塞・浮腫 | 肺がんのリンパ管症や上大静脈症候群等に有効。 |
| <strong>気管支拡張薬</strong> | 気道狭窄による呼吸苦 | COPD併存例や喘鳴がある場合に有効。 |

## ⚠️ 4. 薬剤師として注意すべきポイント

CDSはあくまで指標の一つです。以下の点に留意して活用しましょう。

1. <strong>鎮静剤の副作用</strong>: 呼吸困難が「恐怖感」によるものだとしても、安易な抗不安薬の増量は「過鎮静」や「せん妄」を招くリスクがあります。
2. <strong>非薬物療法の併用</strong>: 呼吸が苦しいときには、扇風機で顔に風を送る、姿勢を整える（起坐位）、リラクゼーション法などの非薬物的介入も非常に重要です。薬だけの提案にならず、包括的にケアを考えましょう。
3. <strong>継続的な評価</strong>: がんの病態は刻一刻と変化します。CDSを用いて定期的（あるいは増悪時）にスコアを追い、薬の処方意図が患者さんのニーズと合致しているか確認してください。
