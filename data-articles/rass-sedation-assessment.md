# RASSとは？ICUでの鎮静・興奮評価尺度を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場あるある：鎮静の評価は難しい？】</strong><br>
  集中治療室（ICU）で鎮静薬の調整をサポートしている時のことです。医師から「鎮静を浅くしたい」と言われ、漫然とプロポフォールを減量したところ、患者さんが急激に不穏になり、人工呼吸器との同調が悪化してしまったことがありました。<br><br>
  その際、先輩から「今、患者さんはRASSいくつなの？」と聞かれ、答えられなかった苦い経験があります。感覚だけで「落ち着いているかな？」と判断して薬剤量を決めるのではなく、<strong>客観的なスケール（RASS）に基づいて評価することの重要性</strong>を痛感した瞬間でした。
</div>

今回は、ICUや救急現場で働く薬剤師なら必ず押さえておきたい「RASS（Richmond Agitation-Sedation Scale）」について、その評価基準と薬剤調整への応用を解説します。

---

## 📊 RASS（リッチモンド興奮鎮静尺度）の評価基準

RASSは、患者さんの鎮静レベルを「＋4（闘争的）」から「－5（昏睡）」までの10段階で評価する指標です。<strong>「0」が意識清明で落ち着いた状態</strong>を指します。

| スコア | 用語 | 評価基準 |
| :--- | :--- | :--- |
| <strong>+4</strong> | 闘争的 | 暴力的、スタッフに危険 |
| <strong>+3</strong> | 非常に興奮 | 呼吸器回路を引っ張る |
| <strong>+2</strong> | 興奮 | 呼吸器に頻繁に抵抗 |
| <strong>+1</strong> | 不穏 | 不安そうだが動きは過激でない |
| <strong>0</strong> | 意識清明 | 自発的に落ち着いている |
| <strong>-1</strong> | 傾眠 | 10秒以上覚醒・アイコンタクト |
| <strong>-2</strong> | 軽度鎮静 | 10秒未満の覚醒・アイコンタクト |
| <strong>-3</strong> | 中等度鎮静 | 声かけに応じるがアイコンタクトなし |
| <strong>-4</strong> | 深い鎮静 | 声かけに応じないが刺激で反応 |
| <strong>-5</strong> | 昏睡 | 声かけ・物理的刺激に反応なし |

---

## 💊 RASSを意識した薬剤師の介入ポイント

RASSを理解すると、薬剤師の提案の幅がぐっと広がります。

### 1. 鎮静の目標設定（Sedation Target）
ガイドライン（PADISガイドライン等）では、原則として<strong>「軽度鎮静（RASS －1 ～ 0）」</strong>を目標とすることが推奨されています。
*   <strong>薬剤師の視点:</strong> 常に深鎮静（－4以下）になっていないか、日々のバイタルや鎮静薬の投与速度をチェックしましょう。深い鎮静はせん妄やICU滞在期間の延長リスクを高めます。

### 2. 鎮静薬の減量・オフの提案
RASSが目標値よりも低い（深すぎる）場合、医師へ以下の提案を検討します。
*   <strong>鎮静休止（SAT: Spontaneous Awakening Trial）:</strong> 毎日決まった時間に鎮静を切り、RASSを評価する。
*   <strong>薬剤の切り替え:</strong> プロポフォール等の持続投与から、鎮痛薬（フェンタニル等）単独へのシフトや、デクスメデトミジンへの変更検討など。

### 3. せん妄リスクのモニタリング
RASSが＋1〜＋2の状態は、せん妄の前兆や過鎮静からの離脱症状である可能性があります。
*   <strong>薬剤師の視点:</strong> 抗精神病薬の追加が必要か、あるいは鎮痛が不十分で不穏になっているのか（鎮痛の評価ツールであるBPSやCPOTと併用）をアセスメントしましょう。

## ⚠️ 臨床で注意すべき「薬剤師のチェックリスト」

RASSを評価する前に、以下の項目をクリアできているか確認することも重要です。

<div style="background-color: #fff7ed; padding: 1.5rem; border-radius: 8px; border: 1px solid #fed7aa;">
  <ul style="margin: 0; padding-left: 1.2rem;">
    <li style="margin-bottom: 0.5rem;"><strong>疼痛の除外:</strong> 痛みが原因で「興奮（＋）」になっていませんか？まずは鎮痛の最適化（Pain first）です。</li>
    <li style="margin-bottom: 0.5rem;"><strong>せん妄評価の併用:</strong> RASSで興奮（＋）の場合、CAM-ICU等を用いたせん妄評価が不可欠です。</li>
    <li style="margin-bottom: 0.5rem;"><strong>薬剤の蓄積性:</strong> プロポフォール等の持続投与中、RASSが変動しない理由に薬剤の蓄積はありませんか？</li>
  </ul>
</div>

## 🌿 まとめ：RASSは「チームの共通言語」

RASSは単なる評価ツールではなく、<strong>医師・看護師・薬剤師が「患者さんの状態」を共有するための共通言語</strong>です。

「プロポフォールを何ml/hr減らしましょう」と提案する際、「RASSが現在－4なので、－1を目指して減量しませんか？」と根拠を添えるだけで、チームからの信頼度は大きく変わります。

ぜひ、毎日のICUラウンドでは、鎮静薬の投与量だけでなく、患者さんのRASSスコアにも注目してみてください。客観的な指標を用いた介入こそが、患者さんの早期回復を支える力になります！
