## 1. 🏥 薬局の機能（調剤基本料・地域支援体制加算など、薬局の算定要件）
**「この薬局だから算定できる」**

### 基本となる評価
<details style="background-color: #f0f9ff; border: 1px solid #bae6fd; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
<summary style="font-weight: bold; font-size: 1.1em; cursor: pointer; color: #0284c7;">調剤基本料１～３（薬局の基本的な運営体制や地域における役割を評価する点数）</summary>

**💡 ざっくり言うと？**
「どのくらい処方箋が来るか」「特定の病院に依存していないか」によって決まる、薬局のベースとなる点数（基本給のようなもの）です。
色んな病院からバランス良く処方箋を受けている薬局ほど高い点数（基本料１）になり、特定の病院からばかり処方箋を受けている大型薬局は点数が下がります。

**📊 点数と条件のまとめ**

<div style="overflow-x: auto;">

| 区分 | <span style="white-space: nowrap;">点数</span> | 主な対象・条件 |
| :--- | :--- | :--- |
| **調剤基本料１** | <span style="white-space: nowrap;">**47点**</span> | 下記の２〜３に当てはまらない、一般的な薬局 |
| **調剤基本料２** | <span style="white-space: nowrap;">**30点**</span> | ・月4,000回超 ＆ 集中率70％超<br>・月600回超〜4,000回以下 ＆ 集中率85％超（※都市部の新規薬局）<br>・特定の病院の処方箋が月4,000回超 |
| **調剤基本料３** | <span style="white-space: nowrap;">**イ）25点**<br>**ロ）20点**<br>**ハ）37点**</span> | グループ展開している大型薬局や、特定の病院への集中率が極めて高い薬局（※受付回数と集中率でイ〜ハに分類） |

</div>

⚠️ **減算されるケース（要注意ルール）**
- **妥結率が50％以下など**: ベースの点数が **半分（▲50％）** になってしまいます。
- **複数処方箋の同時受付**: 同じ病院からの2枚目以降の処方箋は **2割引き（▲20％）** で計算されます。
- **処方箋受付回数のカウント**: 同一建物内の複数病院は合算され、グループ薬局内で集中率が最も高い病院が同一の場合は受付回数に含まれます。
<small>
妥結率（だけつりつ）とは、医療機関や薬局が医薬品卸売業者から仕入れる医薬品について、「あらかじめ購入価格（取引価格）が決定している医薬品の割合」のことです。
</small>
</details>

<details style="background-color: #f0f9ff; border: 1px solid #bae6fd; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
<summary style="font-weight: bold; font-size: 1.1em; cursor: pointer; color: #0284c7;">特別調剤基本料A・B（「通常の調剤基本料の対象にならない薬局」に適用される基本料）</summary>

**💡 ざっくり言うと？**
「病院の敷地内にある薬局」や「そもそも国に届出を出していない薬局」など、通常のルールから外れる薬局に対して適用される、非常に低い基本料です。
ペナルティ的な意味合いが強く、他の加算も軒並み取れなくなります。

**📊 点数と条件のまとめ**

<div style="overflow-x: auto;">

| 区分 | <span style="white-space: nowrap;">点数</span> | 主な対象・条件 |
| :--- | :--- | :--- |
| **特別調剤基本料A** | <span style="white-space: nowrap;">**5点**</span> | 病院と特別な関係（同一敷地内にある等） ＆ 集中率50％超の薬局 |
| **特別調剤基本料B** | <span style="white-space: nowrap;">**3点**</span> | 調剤基本料の届出を行っていない薬局 |

</div>

⚠️ **減算されるケース（要注意ルール）**
- **特別Aの制限**: 地域支援体制加算・後発医薬品調剤体制加算等が **▲90%** に減らされます。また、一部を除き薬学管理料が算定できません。
- **特別Bの制限**: 調剤基本料の各種加算および薬学管理料に属する項目は一切算定できません。
- **多剤投与の制限**: A・Bともに、1処方につき7種類以上の内服薬が出た場合、薬の値段（薬剤料）が **▲10％減算** されます。

</details>

<details style="background-color: #f0f9ff; border: 1px solid #bae6fd; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
<summary style="font-weight: bold; font-size: 1.1em; cursor: pointer; color: #0284c7;">分割調剤（1枚の処方箋の薬を「2回以上に分けて」渡した場合の評価）</summary>

**💡 ざっくり言うと？**
薬が長期保存できない場合や、ジェネリック医薬品のお試しなどの患者さんの事情により、1枚の処方箋の薬を「2回以上に分けて」渡したときの評価です。

**📊 点数と条件のまとめ**

<div style="overflow-x: auto;">

| 区分 | <span style="white-space: nowrap;">点数</span> | 主な対象・条件 |
| :--- | :--- | :--- |
| **分割調剤（2回目以降）** | <span style="white-space: nowrap;">**5点**</span> | 1枚の処方箋で、2回目以降に薬を渡すたびに算定 |
| **分割調剤（2回目のみ）** | <span style="white-space: nowrap;">**5点**</span> | 1枚の処方箋で、2回目のみ算定可能 |

</div>
</details>

<details style="background-color: #f0f9ff; border: 1px solid #bae6fd; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
<summary style="font-weight: bold; font-size: 1.1em; cursor: pointer; color: #0284c7;">門前薬局等立地依存減算（特定医療機関への依存度が高い薬局は減算）</summary>

**💡 ざっくり言うと？**
都市部など「薬局がたくさんある地域」や「医療モール」の中にあり、特定の病院に依存しきっている薬局に対する減点ルールです。

**📊 点数と条件のまとめ**

<div style="overflow-x: auto;">

| :--- | :--- | :--- |
| **門前薬局等立地依存減算** | <span style="white-space: nowrap;">**▲15点**</span> | 都市部の薬局多数地域や、医療モールに所在する特定の薬局（※既存の薬局は除く） |

</div>
</details>

## 2. 💊 薬の調製（調剤技術料・薬剤料などの算定要件）
**「薬を適切な形で患者へ渡すための技術」**

### 基本的な調剤

<details style="background-color: #f0f9ff; border: 1px solid #bae6fd; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
<summary style="font-weight: bold; font-size: 1.1em; cursor: pointer; color: #0284c7;">薬剤調製料（薬を棚から集めて揃える基本料金）</summary>

**💡 ざっくり言うと？**
処方箋の通りに薬を棚から集めてきて、患者さんに渡す準備をする（ピッキングする）ための、一番ベースとなる作業代です。薬の種類（内服、外用、水薬など）によって点数が異なります。

**📊 点数と条件のまとめ**

<div style="overflow-x: auto;">

| 薬の種類 | <span style="white-space: nowrap;">点数</span> | 条件・上限 |
| :--- | :--- | :--- |
| **内服薬**（飲み薬） | <span style="white-space: nowrap;">**24点**</span> | 1剤につき。3剤分まで算定可能 |
| **屯服薬**（頓服薬） | <span style="white-space: nowrap;">**21点**</span> | 1調剤につき算定 |
| **外用薬**（塗り薬や湿布） | <span style="white-space: nowrap;">**10点**</span> | 1調剤につき。3調剤分まで算定可能 |
| **注射薬** | <span style="white-space: nowrap;">**26点**</span> | 1調剤につき算定 |
| **内服用滴剤** | <span style="white-space: nowrap;">**10点**</span> | 1調剤につき算定 |
| **浸煎薬**（煎じて飲む薬） | <span style="white-space: nowrap;">**190〜400点**</span> | 7日分以下:190点、8日分以上:1日+10点、29日分以上:400点。3調剤まで |
| **湯薬** | <span style="white-space: nowrap;">（規定点数）</span> | 1調剤につき。3調剤分まで算定可能 |

</div>
</details>

### 薬に加工・工夫をすることで点数アップ

<details style="background-color: #f0f9ff; border: 1px solid #bae6fd; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
<summary style="font-weight: bold; font-size: 1.1em; cursor: pointer; color: #0284c7;">自家製剤加算（市販されていない形に薬を加工する）</summary>

**💡 ざっくり言うと？**
錠剤をすり潰して粉薬にしたり、薬局内で特別な薬を作ったりと、患者さんが飲みやすいように「市販されていない形」に薬を加工したときの技術料です。飲み込みにくい患者さんのために錠剤を砕く工夫もここで評価されます。

**📊 点数と条件のまとめ**

<div style="overflow-x: auto;">

| 薬の種類 | <span style="white-space: nowrap;">点数</span> | 備考・条件 |

| **調剤物価対応料** | <span style="white-space: nowrap;">**1点**</span> | <span style="white-space: nowrap;">処方箋受付時に算定（3ヶ月に1回まで。※令和9年6月から2点にアップ）</span> |

</div>
</details>

### 介護報酬（令和6年6月1日施行分）
<details style="background-color: #f0f9ff; border: 1px solid #bae6fd; padding: 15px; border-radius: 8px; margin-bottom: 15px;">
<summary style="font-weight: bold; font-size: 1.1em; cursor: pointer; color: #0284c7;">居宅療養管理指導費（介護保険を使って家を訪問する）</summary>

**💡 ざっくり言うと？**
患者さんが「要介護認定」を受けている場合、医療保険の「在宅患者訪問薬剤管理指導料」ではなく、こちらの「介護保険」の枠組みを使って家を訪問し、指導を行います。（※やっている業務内容は医療保険の訪問とほぼ同じです）

**📊 訪問指導の基本単位数（※医療保険の「点」ではなく「単位」で計算します）**

<div style="overflow-x: auto;">

| 区分 | <span style="white-space: nowrap;">単位数</span> | 主な対象・条件 |
| :--- | :--- | :--- |
| **単一建物居住者 1人** | <span style="white-space: nowrap;">**518単位**</span> | 一軒家や、マンション・施設内で1人だけの患者さんを訪問した場合 |
| **単一建物居住者 2〜9人** | <span style="white-space: nowrap;">**379単位**</span> | 同じ施設内で2〜9人の患者さんをまとめて訪問した場合 |
| **単一建物居住者 10人以上** | <span style="white-space: nowrap;">**342単位**</span> | 同じ施設内で10人以上の患者さんをまとめて訪問した場合 |
| **オンラインを用いた指導** | <span style="white-space: nowrap;">**46単位**</span> | スマホのビデオ通話などで遠隔で指導した場合 |

</div>

**📊 訪問時にさらに追加できる「付随加算」**

<div style="overflow-x: auto;">

| 加算名 | <span style="white-space: nowrap;">単位数（割合）</span> | 条件 |
| :--- | :--- | :--- |
| **麻薬管理指導加算** | <span style="white-space: nowrap;">**100単位**</span> | 在宅で麻薬を使用している患者さんの痛みの確認や保管指導をした場合 |
| **医療用麻薬持続注射療法加算** | <span style="white-space: nowrap;">**250単位**</span> | 麻薬の持続注射ポンプなどを使用している患者さんの管理をした場合（※オンライン不可） |
| **在宅中心静脈栄養法加算** | <span style="white-space: nowrap;">**150単位**</span> | 在宅で中心静脈栄養をしている患者さんを管理した場合（※オンライン不可） |
| **特別地域加算** | <span style="white-space: nowrap;">基本単位の **15％増**</span> | 離島など、国が定める特別な地域へ訪問した場合 |
| **中山間地域等小規模事業所加算** | <span style="white-space: nowrap;">基本単位の **10％増**</span> | 中山間地域（山間部など）の小規模な事業所が訪問した場合 |
| **中山間地域等居住者サービス提供加算** | <span style="white-space: nowrap;">基本単位の **5％増**</span> | 中山間地域に住んでいる患者さんの家へ訪問した場合 |

</div>
</details>
