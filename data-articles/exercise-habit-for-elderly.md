# 🏃 運動習慣が続かない高齢者へのおすすめ対策

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場の薬剤師のリアルな悩み】</strong><br>
  外来で「運動はしていますか？」と尋ねると、「医者にも言われているけど、足腰が痛くてね」「何から始めたらいいかわからなくて」と返されることがよくあります。高血圧や糖尿病の患者さんに「運動不足ですね」と伝えても、それが行動変容に繋がることは稀です。<br><br>
  ある日、独居の高齢患者さんに強く運動を勧めたところ、翌月に「無理して歩いて転んでしまった」と打ち明けられました。良かれと思ったアドバイスがリスクになることもあると痛感した経験です。それ以来、私は「頑張る運動」ではなく「生活に溶け込む活動」を提案するように意識を変えました。
</div>

運動習慣の継続は、サルコペニアやフレイル予防の鍵ですが、高齢者にとって「運動」という言葉自体がハードルになっていることも多いものです。この記事では、<strong>高齢者の運動習慣</strong>をいかにスムーズに生活へ組み込むか、薬剤師の視点から具体的なアプローチを紹介します。

---

## 💡 1. 運動を阻害する「心理的・身体的バリア」を知る

運動が続かない理由は「やる気の問題」ではなく、環境や身体的制約にあることがほとんどです。まずは患者さんの現状を紐解くことが大切です。

| バリアの種類 | 患者さんの言葉 | 薬剤師の視点（対応のヒント） |
| :--- | :--- | :--- |
| <strong>身体的</strong> | 「膝や腰が痛くて歩けない」 | 有酸素運動よりも、まずは椅子に座ったままの筋トレを提案 |
| <strong>心理的</strong> | 「わざわざ着替えて外に出るのが面倒」 | 「特別な準備」を不要にする工夫が必要 |
| <strong>認知面</strong> | 「何をすればいいのかわからない」 | 選択肢を1つに絞り、具体的な動作を提示する |
| <strong>孤立感</strong> | 「一人でやっていてもつまらない」 | 通所リハビリや地域サロンの紹介を検討 |

---

## 🧠 2. 薬剤師が提案する「ついで運動（ピギーバッキング）」

心理学の「ピギーバッキング（抱き合わせ）」という手法を応用し、すでに定着している習慣（服薬など）に新しい習慣をくっつける方法が有効です。

### 💊 ① お薬カレンダー・薬のセットと連動させる
「朝食後の薬を飲む前に、椅子から5回立ち上がる」「薬をセットする時に、足首を回す」など、<strong>お薬と連動した小さな目標</strong>を提案してみてください。

### 📺 ② 「テレビCM中だけ」のながら運動
テレビを見ている時間は、高齢者の貴重な余暇時間です。CMが流れている間だけ、膝の屈伸や足踏みをするなど、時間を限定することで「長くやらなきゃ」というプレッシャーを取り除きます。

## 🛠 4. 指導のポイント：「運動」という言葉を捨てる

患者さんに対して「運動してください」と言うと、相手は「わざわざ歩かなければならない」と解釈しがちです。以下のような言い換えを試してみてください。

*   <strong>「散歩しましょう」</strong> → <strong>「ゴミ出しのついでに、もう一軒先まで歩いてみませんか？」</strong>
*   <strong>「筋トレしてください」</strong> → <strong>「テレビを見ながら、足首を動かすだけでいいですよ」</strong>
*   <strong>「毎日頑張ってください」</strong> → <strong>「週に2〜3回、思い出した時だけで十分ですよ」</strong>

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; color: #92400e;">
  <strong>💡 薬剤師としての一言アドバイス</strong><br>
  「運動ができた日」を記録する簡単なチェックシートを渡すだけでも、自己効力感（自分にもできるという自信）が高まります。次回の来局時に「先週はどうでしたか？」と確認するだけで、患者さんの意識はぐっと高まります。

## 🌿 まとめ：小さな習慣が、大きな治療の一歩になる

運動習慣が続かないのは、本人の努力不足ではなく、提案の方法が生活に馴染んでいないだけかもしれません。「薬を飲む」という日々のルーチンに、「体を動かす」という小さな動作を紐付けてあげること。それが、薬剤師にしかできない<strong>「生活の処方箋」</strong>です。

まずは、次の服薬指導の際に「最近、椅子から立ち上がる時に足の力を使ってますか？」と、日常の動作から会話を広げてみてはいかがでしょうか。
