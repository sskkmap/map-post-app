# 🩺 バージャー病の重症度分類とは？診断と治療のポイント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  薬局に、足の痛みを訴える患者さんが来局されました。お薬手帳を確認すると、血管拡張薬と抗血小板薬が処方されており、病名は「バージャー病」。当時は知識が浅く、単に「血流を良くするお薬ですね」とだけ伝えてしまいました。<br><br>
  後日、この患者さんが「禁煙外来」にも通っていることを知り、バージャー病にとって禁煙が「最強の治療薬」であることを痛感しました。薬剤師として、ただ薬を渡すだけでなく、生活習慣への介入がいかに重要かを思い知らされた忘れられない経験です。
</div>

バージャー病（閉塞性血栓血管炎）は、四肢の末梢動脈に閉塞性の炎症が起こる疾患です。希少疾患ではありますが、遭遇した際に適切な服薬指導ができるよう、その分類と治療薬のポイントを押さえておきましょう。

## 🔍 1. バージャー病（閉塞性血栓血管炎）の診断のポイント
バージャー病の診断には「Shionoyaの診断基準」などが用いられますが、薬剤師として注目すべきは以下の特徴です。

*   <strong>若年男性（特に20〜40代）に多い</strong>
*   <strong>喫煙者で発症率が極めて高い</strong>（禁煙なしでの改善は困難）
*   <strong>四肢の末梢（手足の指先など）から症状が出る</strong>（間欠性跛行や安静時疼痛）

## 📈 2. 重症度分類（Fontaine分類）
バージャー病をはじめとする末梢動脈疾患（PAD）の重症度は、一般的に<strong>Fontaine（フォンテイン）分類</strong>を用いて判断されます。

| 分類 | 症状 | 薬剤師が注意すべきポイント |
| :--- | :--- | :--- |
| <strong>I度</strong> | 無症状 | 禁煙指導の徹底、危険因子の管理。 |
| <strong>II度</strong> | 間欠性跛行（歩行時の痛み） | 運動療法の継続と血流改善薬の効果確認。 |
| <strong>III度</strong> | 安静時疼痛 | 鎮痛薬（NSAIDs/オピオイド）の併用検討。 |
| <strong>IV度</strong> | 潰瘍・壊死 | 感染予防、疼痛管理、精神的サポート。 |

  <strong>💡 薬剤師の実務アドバイス</strong><br>
  III度・IV度の患者さんは非常に強い痛みを抱えています。処方されている鎮痛薬が適切か、痛みで眠れているかなど、QOLに関わる問診を丁寧に行うことが重要です。
</div>

## 💊 3. 薬物治療の主役：血管拡張薬と抗血小板薬
治療の中心は<strong>「禁煙」</strong>と<strong>「血流改善」</strong>です。

### 血管拡張薬の特徴比較
| 薬剤名 | 特徴 | 薬剤師の確認ポイント |
| :--- | :--- | :--- |
| <strong>アルプロスタジル</strong> | PGE1製剤。強力な血管拡張作用と血小板凝集抑制。 | 点滴静注や内服など。副作用の頭痛・ほてりに注意。 |
| <strong>ベラプロスト</strong> | プロスタサイクリン誘導体。内服薬として汎用。 | 血小板凝集抑制作用があるため、出血傾向に注意。 |
| <strong>シロスタゾール</strong> | PDE3阻害薬。血管拡張と血小板抑制。 | 頻脈や動悸の有無を確認。他薬との併用確認。 |

## ⚠️ 4. 薬剤師が必ず行うべき服薬指導

### ① 禁煙の重要性を再認識させる
「どんなに良い薬を飲んでも、タバコを吸い続けたら効果は相殺されてしまう」という厳しい現実を、患者さんの心に寄り添いながら伝える必要があります。禁煙外来への橋渡しや、心理的なサポートも薬剤師の役割です。

### ② 血小板凝集抑制薬による出血リスク
多くの患者さんが抗血小板薬を併用しています。
*   歯茎からの出血が止まりにくい
*   アザができやすくなった
*   鼻血が出る
これらのサインがあれば、速やかに医師へ情報提供しましょう。

### ③ 潰瘍がある場合の感染予防
IV度の患者さんでは、皮膚に潰瘍ができていることが多いです。外用薬の塗布方法や、清潔保持について指導を行うことで、創部の悪化を防ぐことができます。

## 📝 まとめ：チーム医療の中での役割

バージャー病の治療は、医師、看護師、そして私たち薬剤師が連携するチーム医療が不可欠です。

1.  <strong>禁煙継続の応援</strong>: 治療の土台。
2.  <strong>副作用モニタリング</strong>: 血管拡張薬による頭痛や動悸、出血傾向のチェック。
3.  <strong>痛みへの配慮</strong>: 鎮痛薬の調整によるQOLの維持。

この疾患は患者さんのライフスタイルに大きく依存します。専門家として、薬の知識だけでなく「患者さんの生活」にまで目を向けた服薬指導を心がけていきましょう！
