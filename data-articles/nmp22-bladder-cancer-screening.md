# 🛡️ NMP22とは？尿路上皮がん診断補助に用いられる検査

  その時、私は検査値の読み方やNMP22の性質を即座に説明できず、医師の診察を待つようにお伝えすることしかできませんでした。その後調べてみると、この検査は感度・特異度の面で非常に重要である反面、実は「血尿」や「結石」などの影響も受けやすいことを知りました。単なる数値の結果だけでなく、患者さんの背景まで踏まえた説明ができれば、もっと安心していただけたはずだと後悔した出来事でした。

今回は、臨床現場で遭遇する機会が増えている<strong>「NMP22」</strong>について、薬剤師が知っておくべき知識を整理していきます。

---

## 🔍 1. NMP22ってなに？どんな検査？

NMP22（Nuclear Matrix Protein 22：核マトリックス蛋白22）は、膀胱がんをはじめとする<strong>尿路上皮がんの診断補助</strong>に用いられる腫瘍マーカーです。

尿路上皮がんの細胞は、増殖の過程で核が崩壊し、核マトリックス蛋白が尿中に放出されます。NMP22検査は、この蛋白を尿中で測定することで、腫瘍の存在を推測するものです。

### 📊 NMP22検査の主な特徴
| 特徴 | 内容 |
| :--- | :--- |
| <strong>主な用途</strong> | 尿路上皮がんの診断補助、術後の再発モニタリング |
| <strong>検体</strong> | 尿（随時尿） |
| <strong>測定対象</strong> | 尿中の核マトリックス蛋白22 |
| <strong>臨床上の意義</strong> | 細胞診（尿検査）では見つけにくい早期の腫瘍発見を補助する |

---

## ⚠️ 2. 陽性＝がん、ではない？知っておくべき注意点

薬剤師が患者さんから相談を受けた際に注意すべき点は、<strong>「NMP22は特異的な検査だが、偽陽性も存在する」</strong>という点です。尿路上皮がん以外でも、以下の状態で陽性を示すことがあります。

「陽性」と結果が出ても、直ちに癌と確定するものではありません。<strong>「まずは炎症や血尿の原因を探り、必要に応じて画像診断（CTや膀胱鏡検査）へ進むためのステップである」</strong>という視点でアドバイスすると、患者さんの不安を過度に煽らずに済みます。

## 💊 3. 薬剤師が現場で活用できる「伝え方」のポイント

患者さんが「NMP22が陽性だった」と薬局に相談に来た際、私たちは診断を下すことはできません。しかし、次のようなサポートは可能です。

1. <strong>結果の解釈を補足する</strong>
   「この検査は、尿中のタンパク質を調べて膀胱の病気の可能性をスクリーニングするものです。癌だけでなく、炎症や結石でも反応することがあるため、医師が総合的に判断しますよ」と、冷静な視点を提供しましょう。
2. <strong>血尿の有無を確認する</strong>
   「最近、おしっこに血が混じったり、排尿痛があったりしませんでしたか？」とヒアリングし、その情報を医師への申し送りとして整理しておくのも重要です。
3. <strong>受診勧奨を促す</strong>
   陽性結果を放置せず、必ず泌尿器科を受診して確定診断を受けるよう、丁寧にお伝えしてください。

## 🌿 4. まとめ

NMP22検査は、尿路上皮がんの早期診断を支える非常に有用なツールです。しかし、あくまで「診断を補助するもの」であることを理解しておくことが大切です。

* <strong>NMP22は尿中の核タンパクを測定する検査である</strong>
* <strong>偽陽性（結石や炎症、操作の影響）があることを知っておく</strong>
* <strong>患者さんの不安に対しては、検査の目的と専門医による総合診断の重要性を説明する</strong>

現場で「数値」だけを見てパニックになる患者さんがいたら、ぜひ今回の知識を参考に、落ち着いて寄り添える対応を心がけてみてください。正しい知識を武器にすることで、私たち薬剤師もチーム医療の中で大きな安心感を提供できるはずです。
