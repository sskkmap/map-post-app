# SHRSとは？錐体外路症候群評価尺度の特徴を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場あるある】</strong><br>
  ある日、外来で「最近、なんだか手足が震えるし、体がこわばるような気がする」と相談された患者さん。その方は統合失調症治療中で、抗精神病薬を服薬中でした。カルテを見ると、抗パーキンソン病薬が追加されたばかり……。<br><br>
  「とりあえず薬が増えたから様子見かな」と考えそうになりますが、実はその震えが「たまたまの不安感」なのか、それとも薬剤性の「錐体外路症状（EPS）」が進行しているのかを客観的に判断するのは難しいものです。そんなとき、評価指標としての「SHRS」を知っているかどうかで、医師へのフィードバックの精度が格段に変わります。
</div>

精神科領域の服薬指導において、副作用の早期発見は薬剤師の重要な責務です。今回は、客観的な評価指標である「SHRS（Simpson-Angus Rating Scale）」について解説します。

---

## 🔍 1. SHRS（Simpson-Angus Rating Scale）とは？

SHRSは、抗精神病薬の服用によって生じる<strong>「薬剤性パーキンソニズム（錐体外路症状）」</strong>を評価するための尺度です。

特に、筋肉の強剛（こわばり）や振戦（震え）、流涎（よだれ）などの症状を数値化することで、<strong>「どの程度の強さで副作用が出ているのか」「治療によって改善しているのか」</strong>を客観的に把握するために用いられます。

### SHRSで評価する主な項目
SHRSは以下の10項目から構成されており、それぞれの重症度を0〜4点で評価します。

| 評価項目 | 主なチェックポイント |
| :--- | :--- |
| <strong>歩行</strong> | 小刻み歩行、腕の振りの減少 |
| <strong>腕の振りの減少</strong> | 歩行時の腕の動きが左右で差がないか |
| <strong>剛強（硬直）</strong> | 首、腕、脚の関節を動かした際の抵抗感 |
| <strong>震え（振戦）</strong> | 静止時の手足の震え |
| <strong>流涎（よだれ）</strong> | 口角からの唾液の漏出 |
| <strong>顔の表情（仮面様顔貌）</strong> | 表情の乏しさ、まばたきの減少 |
| <strong>その他</strong> | 姿勢、反射などの身体所見 |

---

## ⚖️ 2. 他の評価尺度との違い（DIEPSSとの比較）

現場ではSHRS以外にも評価尺度が使われることがあります。違いを把握して使い分けましょう。

| 尺度名 | 特徴と評価対象 | 薬剤師の活用シーン |
| :--- | :--- | :--- |
| <strong>SHRS</strong> | 薬剤性パーキンソニズム（こわばり等）に特化 | 抗パーキンソン薬の調整時、パーキンソン症状の重症度把握 |
| <strong>DIEPSS</strong> | 錐体外路症状全般（パーキンソニズム、ジスキネジア等）を評価 | 総合的な抗精神病薬の副作用モニタリング |
| <strong>AIMS</strong> | 遅発性ジスキネジア（不随意運動）の評価に特化 | 長期服薬中の患者さんの口周囲の動きのチェック |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 薬剤師の視点：なぜ使い分けるの？</strong><br>
  SHRSは、特に<strong>筋肉の「こわばり」や「動きの鈍さ」</strong>に焦点を当てています。例えば、抗パーキンソン薬を減量する際に、SHRSスコアが悪化しないかを確認することで、減量の是非を論理的に判断する材料になります。
</div>

## 💊 3. 実務で役立つ！SHRS活用術

SHRSを完璧に使いこなす必要はありませんが、以下の視点を持つだけで服薬指導の質が変わります。

### ① 「歩行」と「腕の振り」を確認する
診察室に呼ばれるまでの待合室での様子や、薬局内での歩き方をさりげなく観察してみてください。
*   「歩幅が以前より狭くなっていないか？」
*   「歩くときに左右の腕がしっかり振れているか？」
これらはSHRSの評価項目でもあり、患者さん自身が自覚しにくいサインです。

### ② 言語化して医師へ情報共有する
医師に疑義照会をする際、「患者さんが震えていると言っています」だけでは、処方変更の根拠として弱いです。
*   「歩行時に上肢の振りが減少し、首筋の筋肉に緊張が認められます。SHRSの評価でも〇点相当の所見があり、薬剤性パーキンソニズムが疑われます」
このように具体的に伝えることで、<strong>「薬の専門家」としての信頼度</strong>が高まります。

---

## ⚠️ 4. 注意点：SHRSスコアはあくまで目安

SHRSは非常に有用なツールですが、以下の点には注意が必要です。

*   <strong>高齢者の場合は注意</strong>: 高齢者はもともと身体機能が低下しているため、薬の副作用なのか、加齢によるものかの判別が難しい場合があります。
*   <strong>心理的要因</strong>: 不安が強い場合、精神的な緊張から震えが出ることがあります。薬の副作用と決めつけず、精神状態もしっかりヒアリングしましょう。

---

## 🌿 5. まとめ：客観的な評価が安心を作る

SHRSのような評価尺度を活用することは、単に数字を記録することではありません。
<strong>「患者さんの苦痛を可視化し、適切な薬物療法をサポートすること」</strong>が真の目的です。

明日からの服薬指導では、
1. 患者さんの歩き方、表情を観察する
2. 震えやこわばりがないか、SHRSの項目を頭の片隅に置いて質問する
3. 疑わしい場合は、記録とともに医師へ「客観的な所見」を伝える

これらを意識してみてください。皆さんの日々の観察が、患者さんのQOLを守る大きな盾になりますよ！
