# SIRS診断基準とは？全身性炎症反応症候群の判断ポイント

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の臨床現場エピソード】</strong><br>
  ある日、当直中の私のもとに病棟の看護師さんから連絡が入りました。「入院中のAさん、なんとなく食欲がなくて、体温が38.2度まで上がっているんです。でもCRPはそこまで高くないし、様子見でいいかな…？」<br><br>
  カルテを見ると、呼吸数が増えており、心拍数も上昇傾向。これを見て私は「CRPだけで判断せず、SIRSの項目を確認しましょう」と伝えました。結果、SIRSの基準を複数満たしており、敗血症の初期段階であることが判明。迅速な抗菌薬介入が行われ、事なきを得ました。CRPなどの数値だけでなく、バイタルサインから「全身の火事」を見抜く重要性を痛感した瞬間でした。
</div>

薬剤師にとって、検査値（特にCRPや白血球数）は重要な指標です。しかし、それ以上に「患者さんの体全体で起きている異常」をいち早く察知する力は、臨床で非常に重要です。今回は、重症感染症の入り口とも言える「SIRS（全身性炎症反応症候群）」について整理します。

---

## 🔍 1. SIRS（全身性炎症反応症候群）とは？

SIRSは「Systemic Inflammatory Response Syndrome」の略称です。感染症に限らず、火傷、外傷、手術など、<strong>何らかの侵襲によって全身性に激しい炎症が起きている状態</strong>を指します。

敗血症（Sepsis）の概念が「感染による制御不能な宿主反応」へと変化し、現在はqSOFAスコアなどが注目されていますが、依然として現場の「アラート指標」としてSIRSの考え方は非常に重要です。

### 📊 SIRS診断基準（以下の4項目のうち2項目以上を満たす）

| 項目 | 判断のポイント |
| :--- | :--- |
| <strong>体温</strong> | 38℃以上 または 36℃未満 |
| <strong>心拍数</strong> | 90回/分以上 |
| <strong>呼吸数</strong> | 20回/分以上（または PaCO2 32mmHg未満） |
| <strong>白血球数</strong> | 12,000/μL以上 または 4,000/μL以下（または幼若球10%超） |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 薬剤師のチェックポイント</strong><br>
  検査室から送られてくるデータだけでなく、バイタルサインに注目してください。特に<strong>「呼吸数」</strong>はカルテ記載が見落とされがちですが、重症化を予測する極めて鋭敏な指標です。
</div>

## 🚨 2. SIRSから敗血症へ：薬剤師が見るべき予兆

SIRSの基準を満たしている場合、その裏には「感染症」が隠れている可能性があります。特に薬剤師は、以下の点に注目して疑義照会や医師への提案を行いましょう。

### 介入のポイント
1. <strong>抗菌薬の適正化</strong>: SIRS基準を満たす急変時、原因菌が不明な場合は「広域スペクトラム」の抗菌薬が開始されます。培養結果が出次第、速やかにde-escalation（デエスカレーション）を提案する準備をしましょう。
2. <strong>輸液バランス</strong>: SIRS時は血管透過性が亢進し、サードスペースへの水分漏出が起こりやすくなります。利尿薬や輸液量の調整が適切か、腎機能データと併せて確認が必要です。
3. <strong>免疫抑制薬の確認</strong>: ステロイドや免疫抑制剤を使用中の患者さんは、感染症でも「発熱しにくい」ことがあります。SIRS基準を満たさないからといって安心せず、バイタル変動を見逃さないことが重要です。

## 🌿 4. まとめ：薬剤師も「全身」を診る意識を

SIRSはあくまで診断基準の一つですが、<strong>「患者さんが今、どれほどの侵襲にさらされているか」</strong>を可視化する強力なツールです。

「お薬を渡して終わり」ではなく、処方変更の際に「なぜこのタイミングで抗菌薬が強化されたのか？」「この患者さんはSIRSの兆候があるのではないか？」と一歩踏み込んでカルテを見ることで、薬剤師としての臨床的貢献度は大きく高まります。

これからも、「データ」だけでなく「患者さんの状態」を総合的に捉える視点を持ち続け、安全な薬物療法を提供していきましょう。
