# Fontaine分類とは？閉塞性動脈硬化症の重症度分類

<div style="background-color: #f0f9ff; border-left: 4px solid #0284c7; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード：その「足の痛み」を見逃さないで】</strong><br>
  ある日、薬局で高齢の患者さんに「最近、少し歩くとふくらはぎが痛くなって、立ち止まらないと歩けないんだよね」と相談されました。私は反射的に「筋肉疲労かもしれませんね、湿布を出しておきましょうか？」と言いかけました。<br><br>
  しかし、その患者さんは高血圧と脂質異常症で治療中。「もしや…」と思い、立ち止まれば痛みが引くかどうかを確認すると「そう、休めばすぐ良くなる」とのこと。これはただの疲れではなく、血管に問題がある可能性（ASO）を示唆しています。この時、Fontaine分類の知識があったおかげで、受診勧奨という適切な判断ができました。
</div>

閉塞性動脈硬化症（ASO）の患者さんは、血圧管理や抗血小板薬の服用で薬局にも頻繁にいらっしゃいます。患者さんの「歩くと足が痛い」という言葉の裏にある重症度を読み解くツールが、<strong>Fontaine（フォンテイン）分類</strong>です。

---

## 🔬 1. Fontaine分類によるASO重症度評価

Fontaine分類は、閉塞性動脈硬化症の臨床症状を4つのステージに分けたものです。患者さんがどのステージにいるかを知ることは、治療目標や薬剤管理において非常に重要です。

| 分類 | ステージ | 主な症状 |
| :--- | :--- | :--- |
| <strong>I度</strong> | 無症状 | 冷感、しびれ感のみ |
| <strong>II度</strong> | 間欠性跛行 | 一定距離を歩くと足が痛む（休息で改善） |
| <strong>III度</strong> | 安静時疼痛 | 夜間や安静時にも足が痛い |
| <strong>IV度</strong> | 潰瘍・壊死 | 足の指や踵（かかと）の潰瘍、壊死 |

### 各ステージの臨床的解釈
*   <strong>II度（間欠性跛行）</strong>: AS Oの典型的な初期症状です。「何メートル歩くと痛むか」を把握することで、病態の変化を追うことができます。
*   <strong>III度・IV度（重症虚血肢）</strong>: ここまで進むと緊急性が高く、血管外科での専門的な治療が必要となります。薬局で潰瘍や黒ずみを見かけたら、すぐさま医師への報告が必要です。

## 💊 2. 薬剤師が押さえるべきASOの薬物療法

Fontaine分類ステージII以降では、血行再建術や薬物療法がメインとなります。我々が服薬指導で注意すべき薬剤の特徴をまとめました。

<div style="overflow-x: auto;">

| 薬剤の種類 | 代表的な薬名 | 薬剤師の確認ポイント |
| :--- | :--- | :--- |
| <strong>抗血小板薬</strong> | クロピドグレル、シロスタゾール | 出血傾向、胃腸障害、シロスタゾールの頭痛 |
| <strong>プロスタグランジン製剤</strong> | ベラプロスト、イロプロスト | 血管拡張によるほてり、頭痛、血圧低下 |
| <strong>スタチン・脂質改善薬</strong> | アトルバスタチン等 | 動脈硬化の進行抑制。肝機能値の推移 |

</div>

### 💡 特に注意したい「シロスタゾール」
ASOの第一選択薬としてよく使われる「シロスタゾール」は、PDE3阻害作用により血管を広げ、血流を改善します。この薬は副作用として<strong>頭痛や動悸、頻脈</strong>が出やすいため、服薬初期の患者さんには「頭が痛くなることがありますが、飲み続けると慣れることが多いです」とフォローを入れてあげましょう。

## 🩺 3. 現場で役立つチェックポイント：受診勧奨のサイン

患者さんの訴えから「これはFontaine分類のステージIII以上かも？」と疑うべき、重要なサインがあります。

1.  <strong>夜間疼痛の有無</strong>: 「寝ているときに足が痛くて目が覚めることはないですか？」
2.  <strong>皮膚の観察</strong>: 診察時に足を見せてもらう際、指先の黒ずみや治りにくい傷がないかを確認。
3.  <strong>脈の触知</strong>: 脈が触れにくい、足が異常に冷たいといった訴え。

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin-top: 1.5rem;">
  <strong>【薬剤師のアクション】</strong><br>
  Fontaine分類III度以上（安静時疼痛・壊死）が疑われる場合は、<strong>「速やかな専門医（循環器内科・血管外科）への相談」</strong>を促してください。この段階ではNSAIDsで痛みを誤魔化すことが遅延の原因になります。
</div>

## 🌿 4. まとめ：患者さんの「歩き」に注目しよう

Fontaine分類は、閉塞性動脈硬化症の患者さんのQOL（生活の質）と生命予後を評価する大切な尺度です。

*   <strong>I度</strong>: 冷感・しびれを放置しない
*   <strong>II度</strong>: 「どれくらい歩けるか」を聞き出し、治療効果を評価
*   <strong>III・IV度</strong>: 潰瘍や安静時痛の有無をチェックし、緊急性の高いサインを見逃さない

私たちが日頃の服薬指導で「最近、歩くのは楽になりましたか？」と一言声をかけるだけで、ASOの重症化を未然に防ぐ重要なスクリーニングになるはずです。専門的な知識を、明日からのカウンター業務にぜひ活かしてみてくださいね！
