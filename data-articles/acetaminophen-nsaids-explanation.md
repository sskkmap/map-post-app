## 💊 1. アセトアミノフェンとNSAIDs：それぞれの「顔」を知る

まずは、それぞれの薬がどのような特徴を持っているのかを理解しましょう。これが、適切な<strong>鎮痛薬の使い分け</strong>の第一歩です。

### 🧠 アセトアミノフェンの特徴：脳に働きかける優しい鎮痛剤

<div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe; margin-bottom: 1.5rem;">
  <div style="font-weight: bold; color: #1e3a8a; margin-bottom: 0.5rem; border-bottom: 2px solid #bfdbfe; padding-bottom: 0.25rem;">作用機序と特徴</div>
  <ul style="margin: 0; padding-left: 1.2rem; color: #1e40af; font-size: 0.9em;">
    <li style="margin-bottom: 0.25rem;"><b>中枢性作用：</b>主に脳の痛みの感覚を抑える中枢作用が主。プロスタグランジンの合成を阻害することで解熱・鎮痛作用を発揮します。</li>
    <li style="margin-bottom: 0.25rem;"><b>抗炎症作用は弱い：</b>炎症を抑える作用はほとんどありません。</li>
    <li style="margin-bottom: 0.25rem;"><b>胃腸への負担が少ない：</b>消化管への直接的な刺激や、プロスタグランジンによる胃粘膜保護作用の抑制が少ないため、胃に負担が少ないとされています。</li>
    <li style="margin-bottom: 0.25rem;"><b>腎臓への影響が比較的少ない：</b>腎血流量への影響も少ないため、腎機能が低下している患者さんにも比較的使いやすいです。</li>
    <li style="margin-bottom: 0.25rem;"><b>小児や妊娠・授乳中にも選択肢：</b>多くの解熱鎮痛剤の中で、使用できる年齢層や状況が広いです。</li>
    <li><b>肝機能への注意：</b>過量服用すると肝臓に負担がかかるため、用法・用量厳守が重要です。</li>
  </ul>
</div>

<strong>代表的な製品名</strong>: カロナール、タイレノールなど

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: アセトアミノフェンは、主に中枢神経系に作用し、プロスタグランジンの合成を阻害することで解熱鎮痛作用を発揮します。抗炎症作用は弱く、胃腸への直接的な刺激や、プロスタグランジンによる胃粘膜保護作用の抑制が少ないため、胃への負担が少ないとされています。また、腎血流量への影響も比較的少ないため、腎機能が低下している患者さんにも慎重に用いれば比較的使いやすいとされています。小児や妊娠・授乳中の患者にも選択肢となることが多いですが、過量服用により肝機能障害のリスクがあるため、定められた用法・用量を守ることが重要です。<br>参照元: <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310005_1149001F1020_1_05">カロナール錠200 添付文書</a>, <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310005_1149007F1022_1_04">タイレノールA錠 添付文書</a>, <a href="https://www.med.or.jp/people/knw_06.html">アセトアミノフェンの正しい使い方</a></details>

### 💪 NSAIDs（非ステロイド性消炎鎮痛剤）の特徴：炎症と痛みに強く！

<div style="background-color: #fef2f2; padding: 1rem; border-radius: 8px; border: 1px solid #fca5a5; margin-bottom: 1.5rem;">
  <div style="font-weight: bold; color: #9f1239; margin-bottom: 0.5rem; border-bottom: 2px solid #fecdd3; padding-bottom: 0.25rem;">作用機序と特徴</div>
  <ul style="margin: 0; padding-left: 1.2rem; color: #be123c; font-size: 0.9em;">
    <li style="margin-bottom: 0.25rem;"><b>末梢性作用：</b>炎症や痛みの原因物質であるプロスタグランジンの生成を、酵素（COX-1、COX-2）を阻害することで強力に抑制します。</li>
    <li style="margin-bottom: 0.25rem;"><b>抗炎症作用が強い：</b>炎症を伴う痛み（関節炎、外傷、喉の腫れなど）には特に効果が期待できます。</li>
    <li style="margin-bottom: 0.25rem;"><b>胃腸障害のリスク：</b>炎症や痛みに関連するのは主にCOX-2ですが、COX-1も阻害することにより胃粘膜保護作用のあるプロスタグランジンも抑えるため、胃痛、胃潰瘍などの<strong>副作用</strong>を起こす可能性があります。</li>
    <li style="margin-bottom: 0.25rem;"><b>腎機能低下のリスク：</b>腎血流量を調節するプロスタグランジンも抑えるため、腎機能低下を招くことがあります。特に高齢者や腎機能障害患者では注意が必要です。</li>
    <li style="margin-bottom: 0.25rem;"><b>循環器系への影響：</b>心臓病や高血圧の患者さんでは、血圧上昇や体液貯留、血栓リスクの増大など、注意すべき点があります。</li>
    <li><b>喘息発作誘発の可能性：</b>アスピリン喘息の既往がある患者には禁忌です。</li>
  </ul>
</div>

<strong>代表的な製品名</strong>: ロキソニン、ボルタレン、イブプロフェン、アスピリンなど

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: NSAIDsは、炎症や痛みの原因物質であるプロスタグランジンの生成をシクロオキシゲナーゼ（COX-1、COX-2）を阻害することで抑制し、解熱・鎮痛・抗炎症作用を発揮します。COX-1阻害により胃粘膜保護作用のあるプロスタグランジンも抑制されるため、胃腸障害（胃痛、胃潰瘍など）の副作用を起こす可能性があります。また、腎血流量を調節するプロスタグランジンも抑制するため、腎機能低下を招くことがあり、特に高齢者や腎機能障害患者では注意が必要です。心血管系への影響や、アスピリン喘息の既往がある患者への禁忌も重要な注意点です。<br>参照元: <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310006_1149002F1026_1_05">ロキソニン錠60mg 添付文書</a>, <a href="https://www.jstage.jst.go.jp/article/jjpn/73/4/73_272/_pdf/-char/ja">NSAIDsの作用と副作用</a>, <a href="https://www.mhlw.go.jp/stf/seisakunitsuite/bunya/kenkou_iryou/iyakuhin/topics/tp130612-01.html">NSAIDsによる副作用にご注意ください</a></details>

## 🧐 3. 「この患者さんにはどっち？」使い分けの具体的なポイント

ここからは、実際の現場で<strong>解熱鎮痛剤</strong>を選ぶ際に、薬剤師がどのような情報を聞き取り、どう判断していくかのポイントを解説します。これが<strong>薬の使い分け</strong>における核心です。

### ① 胃腸への負担を考えたいとき
*   <strong>患者像</strong>: 消化性潰瘍や十二指腸潰瘍の既往がある、胃が弱い、高齢者、ステロイドや抗血栓薬を併用している。
*   <strong>選択</strong>: <strong>アセトアミノフェン</strong>を優先します。NSAIDsは胃粘膜保護作用を低下させるため、胃腸障害のリスクが高いです。やむを得ずNSAIDsを使用する場合は、胃粘膜保護薬の併用を検討します。

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: 消化性潰瘍の既往がある患者や、胃が弱い患者、高齢者、ステロイドや抗血栓薬を併用している患者には、胃腸への負担が少ないアセトアミノフェンが推奨されます。NSAIDsは胃粘膜保護作用を低下させるため、胃腸障害のリスクが高いことから、これらの患者への使用は慎重に行うか、胃粘粘膜保護薬の併用を検討する必要があります。<br>参照元: <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310005_1149001F1020_1_05">カロナール錠200 添付文書</a>, <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310006_1149002F1026_1_05">ロキソニン錠60mg 添付文書</a></details>

### ② 腎臓への影響を考慮するとき
*   <strong>患者像</strong>: 慢性腎臓病（CKD）患者、高齢者、脱水状態、利尿薬・ARB/ACE阻害薬を併用している。
*   <strong>選択</strong>: <strong>アセトアミノフェン</strong>を選択します。NSAIDsは腎血流量を低下させることで、急性腎不全を誘発するリスクがあるため、原則避けるべきです。

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: 慢性腎臓病（CKD）患者、高齢者、脱水状態の患者、利尿薬やARB/ACE阻害薬を併用している患者には、腎臓への影響が比較的少ないアセトアミノフェンが選択されます。NSAIDsは腎血流量を低下させることにより急性腎不全を誘発するリスクがあるため、これらの患者への使用は原則として避けるべきとされています。<br>参照元: <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310005_1149001F1020_1_05">カロナール錠200 添付文書</a>, <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310006_1149002F1026_1_05">ロキソニン錠60mg 添付文書</a></details>

### ③ 喘息やアレルギーがあるとき
*   <strong>患者像</strong>: アスピリン喘息の既往がある、NSAIDsでじんましんなどのアレルギー症状が出たことがある。
*   <strong>選択</strong>: <strong>アセトアミノフェン</strong>が第一選択です。NSAIDsはアスピリン喘息を誘発する可能性があるため禁忌です。アセトアミノフェンでも稀に喘息が悪化することがあるため、注意深く観察します。

<details><summary>根拠となる情報元の詳細を見る</summary>引用元文章: アスピリン喘息の既往がある患者やNSAIDsでアレルギー症状が出たことのある患者には、アセトアミノフェンが第一選択となります。NSAIDsはアスピリン喘息を誘発する可能性があり、これらの患者には禁忌とされています。アセトアミノフェンもごく稀に喘息の悪化を招くことがあるため、服用中は患者の状態を注意深く観察することが重要です。<br>参照元: <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310005_1149001F1020_1_05">カロナール錠200 添付文書</a>, <a href="https://www.pmda.go.jp/PmdaSearch/iyakuDetail/ResultDataSetPDF/310006_1149002F1026_1_05">ロキソニン錠60mg 添付文書</a></details>

### ④ 妊娠中・授乳中の患者さん
*   <strong>患者像</strong>: 妊娠初期～中期、妊娠後期、授乳中。
*   <strong>選択</strong>:
    *   <strong>妊娠初期～中期</strong>: <strong>アセトアミノフェン</strong>が推奨されます。NSAIDsは催奇形性リスクが完全に否定できないため、使用を避けるべきです。
    *   <strong>妊娠後期（特に28週以降）</strong>: NSAIDsは胎児の動脈管早期閉鎖や腎機能障害のリスクがあるため<strong>禁忌</strong>です。<strong>アセトアミノフェン</strong>が唯一の選択肢となります。
    *   <strong>授乳中</strong>: <strong>アセトアミノフェン</strong>が推奨されます。NSAIDsの一部は母乳中に移行するため、添付文書を確認し、可能であればアセトアミノフェンを選びます。

### ⑤ 小児や高齢者
*   <strong>患者像</strong>: 小児（特に乳幼児）、高齢者。
*   <strong>選択</strong>:
    *   <strong>小児</strong>: 発熱や痛みに第一選択となるのは<strong>アセトアミノフェン</strong>です。インフルエンザなど特定のウイルス感染症では、NSAIDs（特にアスピリン）はライ症候群との関連が指摘されているため、原則避けます。
    *   <strong>高齢者</strong>: 臓器機能の低下、多剤併用、脱水傾向などから、副作用のリスクが高まります。胃腸障害や腎機能障害のリスクが低い<strong>アセトアミノフェン</strong>を優先します。NSAIDsを使用する際は、最小限の用量と期間で、副作用のモニタリングを徹底します。

### ⑥ 併用薬との相互作用
*   <strong>患者像</strong>: ワーファリンなどの抗凝固薬、メトトレキサート、リチウム、降圧剤などを服用している。
*   <strong>選択</strong>:
    *   NSAIDsは抗凝固薬の効果を増強させ、出血リスクを高めることがあります。また、メトトレキサートの排泄を遅延させ毒性を増強したり、降圧剤の効果を減弱させたりすることがあります。
    *   これらの併用薬がある場合は、<strong>アセトアミノフェン</strong>を選択するのがより安全です。

### ⑦ 手術前後や出血傾向があるとき
*   <strong>患者像</strong>: 手術を控えている、血小板減少症などの出血傾向がある。
*   <strong>選択</strong>: NSAIDsは血小板凝集能を抑制するため、出血時間を延長させる可能性があります。手術前後の患者や出血傾向のある患者には、<strong>アセトアミノフェン</strong>が適しています。

## 💡 5. 市販薬（OTC）を選ぶ際の注意点と指導ポイント

薬剤師として、患者さんが薬局で購入する<strong>市販の解熱鎮痛剤</strong>についても適切なアドバイスができる必要があります。市販薬の中には、アセトアミノフェン単剤、NSAIDs単剤、あるいはカフェインや鎮静成分などとの配合剤が多数あります。

*   <strong>成分確認の重要性</strong>: 患者さんが「いつもの痛み止め」として持参した市販薬が、処方薬と同じ成分、または重複する成分を含んでいることがあります。必ず成分を確認し、重複服用や相互作用のリスクがないか確認しましょう。
*   <strong>配合剤への注意</strong>: SG配合顆粒（アセトアミノフェン含有）などの複数の成分が配合されている場合、特に見落としが多い例です。各配合剤に含まれる成分を把握することは、重複処方を防ぐ上で不可欠です。
*   <strong>症状と選択</strong>:
    *   <strong>単なる発熱や頭痛で、胃腸への負担を抑えたい方、お子さん</strong>: アセトアミノフェン単剤（例: タイレノールA）
    *   <strong>炎症を伴う強い痛み（生理痛、歯痛、肩こり痛）</strong>: NSAIDs単剤（例: ロキソニンS、イブA錠）
*   <strong>購入時の問診</strong>: 薬剤師は、患者の既往歴、アレルギー、現在服用中の薬、妊娠・授乳の有無を丁寧に問診し、最も安全で効果が期待できる<strong>鎮痛薬</strong>を提案する責任があります。
