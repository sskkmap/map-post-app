# SDSSとは？主観欠損症候群評価尺度で何がわかるのか

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ヒヤリハット：その「飲みたくない」の裏側】</strong><br>
  ある精神科の外来で、統合失調症の患者さんが「私はどこも悪くないのに、なぜこんな高い薬を飲まないといけないんだ」と激昂されたことがありました。私はマニュアル通りに「お薬を飲まないと再発しますから」と伝えたのですが、患者さんの不信感は募るばかり。<br><br>
  後で担当医に相談したところ、「彼は今、自分の病状を正しく認識できていない『主観欠損』の状態なんだよ」と教えてもらいました。薬の効能を並べる前に、患者さんの「主観的な実感」に寄り添う必要があると痛感した経験です。

精神科領域の薬物療法において、患者さんの「病識」は服薬アドヒアランスを左右する極めて重要な要素です。今回は、患者さんの「自分は病気ではない」という主観的な感覚を評価する<strong>SDSS（Subjective Deficit Syndrome Scale：主観欠損症候群評価尺度）</strong>について、臨床現場でどのように活用できるか整理します。

---

## 🔍 1. SDSS（主観欠損症候群評価尺度）とは？

SDSSは、主に統合失調症患者さんが抱く<strong>「病気に対する主観的な認識の欠如」</strong>を評価するために開発された尺度です。

「病識（病気であるという認識）」は客観的に判断されることが多いですが、患者さん自身の「自分は健康である」という実感は、客観的な診察所見とは乖離することが多々あります。SDSSを用いることで、以下のポイントを可視化します。

| 評価項目 | どのような状態を評価するか |
| :--- | :--- |
| <strong>病気の実感</strong> | 「自分は統合失調症である」という自覚の有無 |
| <strong>症状の帰属</strong> | 幻聴や妄想を「自分の病気」と捉えているか、あるいは「外的な攻撃」と考えているか |
| <strong>治療の必要性</strong> | 服薬や通院が自分の回復に必要であると理解しているか |
| <strong>主観的な不快感</strong> | 薬の副作用が、病気の治療という大義名分を上回っていないか |

---

## 🧠 2. SDSSでわかること：服薬アドヒアランスとの関係

SDSSのスコアが高い（＝主観的な病識が低い）患者さんは、以下のリスクを抱えやすいことが臨床データでも示唆されています。

### ① 「薬＝毒」という認識の強さ
SDSSが高い患者さんは、症状を「病気」と認識していないため、抗精神病薬を「健康な体に毒を入れる行為」と誤解してしまう傾向があります。

### ② 副作用への過敏性
「病気の治療」という目的意識が薄いため、わずかな眠気や錐体外路症状（EPS）に対しても、「不必要な薬による苦痛」と判断し、自己中断を引き起こす確率が高まります。

  服薬指導の際、「なぜこの薬を飲んでいるか」と尋ねたときに、「先生が飲めと言うから」「家族に言われるから」という回答ばかりで、<strong>「自分の調子を整えるため」という言葉が出てこない場合</strong>、SDSS的な意味での病識欠如が疑われます。

## 🛠️ 3. 実践！薬剤師ができる「SDSS的アプローチ」

SDSSを厳密にスコア化するのは医師の診断業務ですが、薬剤師もこの視点を持つことで、服薬指導の質が大きく変わります。

### ① 「症状」ではなく「生活の困りごと」からアプローチ
「統合失調症の陽性症状」という言葉は、病識が低い患者さんには拒絶されがちです。
* <strong>×</strong> 「幻聴を抑えるために薬を飲みましょう」
* <strong>○</strong> 「最近、夜ぐっすり眠れていないようですが、お薬を調整して睡眠の質を整えてみませんか？」

### ② 服薬のゴールを共有する
患者さんの主観的な苦痛（眠れない、イライラする、集中できない）を「薬の目的」と結びつけることで、主観的な病識のギャップを埋めることができます。

### ③ 家族の視点を活用する
主観的な欠損が強い場合、患者さん本人の認識は変わりにくいものです。ご家族に対して「今のお薬が、生活のどのような部分を支えているか（睡眠、集中力、気分の安定など）」を具体的に説明し、家庭内でのサポートをお願いすることも重要です。

## 📝 4. まとめ：主観を否定せず、共存する

SDSSのスコアが改善しないからといって、決して焦る必要はありません。患者さんの「病気ではない」という主観は、彼ら自身の防衛機制かもしれません。

薬剤師としては、<strong>「病識の欠如＝治療拒否」と決めつけるのではなく、患者さんが感じている「現在の苦痛」に対して薬剤的な解決策を提示し続けること</strong>が、結果として服薬アドヒアランスの向上に繋がります。

明日からの投薬カウンターで、「お薬の飲み心地はどうですか？」という質問から、患者さんの主観的な世界に少しだけ踏み込んでみてください。それが、治療の第一歩になるはずです。
