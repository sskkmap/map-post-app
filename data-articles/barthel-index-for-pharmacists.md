# Barthel Indexとは？日常生活動作評価の基本

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場のリアル：その薬管理、本当に大丈夫？】</strong><br>
  ある日の在宅訪問。独居の患者さん宅で、一包化された薬が1週間分も残っているのを見つけました。「飲み忘れてないですか？」と尋ねると、「いや、飲んでるつもりなんだけど、袋が開けられなくてね…」という返答。その時、私は患者さんのADL（日常生活動作）を深く考えずに、薬の量ばかりを気にしていました。<br><br>
  「手が不自由なことは分かっていたのに、なぜ薬の開封という動作まで想像できなかったのか」。この経験が、私がADL評価指標である「Barthel Index」を学び直すきっかけとなりました。
</div>

在宅医療や高齢者の多い外来調剤において、薬の効果を最大化させるには「薬学的管理」だけでなく「身体的な背景の理解」が欠かせません。今回は、薬剤師が知っておくべき<strong>「Barthel Index（バーセルインデックス）」</strong>について解説します。

---

## 📋 1. Barthel Index（バーセルインデックス）とは？

Barthel Indexとは、日常生活動作（ADL）を評価するための国際的な指標です。食事、移動、整容、トイレ動作など、生活における10の項目を「自立・部分介助・全介助」の段階でスコア化します。

薬剤師がこの指標を知っておく最大のメリットは、<strong>「服薬支援のニーズ」を客観的に推測できること</strong>です。

### 評価項目と点数一覧

| 評価項目 | 自立(10/15点) | 部分介助(5点) | 全介助(0点) |
| :--- | :--- | :--- | :--- |
| <strong>食事</strong> | 10点 | 5点 | 0点 |
| <strong>車椅子→ベッド移乗</strong> | 15点 | 10/5点 | 0点 |
| <strong>整容</strong> | 5点 | 0点 | - |
| <strong>トイレ動作</strong> | 10点 | 5点 | 0点 |
| <strong>入浴</strong> | 5点 | 0点 | - |
| <strong>歩行</strong> | 15点 | 10/5点 | 0点 |
| <strong>階段昇降</strong> | 10点 | 5点 | 0点 |
| <strong>更衣</strong> | 10点 | 5点 | 0点 |
| <strong>排便管理</strong> | 10点 | 5点 | 0点 |
| <strong>排尿管理</strong> | 10点 | 5点 | 0点 |

<div style="font-size: 0.85em; color: #64748b; margin-top: 0.5rem;">※合計100点で満点。点数が低いほど介助が必要な度合いが高いことを示します。</div>

---

## 💊 2. 薬剤師がADL評価を活用するポイント

Barthel Indexの項目は、そのまま「薬を飲むための動作」に直結します。

### 「食事」の項目を服薬に変換する
食事の評価が「部分介助」の患者さんは、食事の時間は誰かのサポートがあるかもしれません。しかし、食後の薬はどうでしょうか？もし食事の動作に制限があるなら、<strong>「袋を開ける」「コップを持ち上げる」動作にも困難が生じている可能性</strong>が高いです。

### 「整容」の項目は「薬の管理能力」と連想する
顔を洗ったり髪を整えたりする整容の動作が困難な方は、手指の細かな動作（巧緻性）やバランス感覚が低下していることが多く、薬をカレンダーにセットしたり、錠剤をシートから押し出したりする動作が苦痛である可能性を疑います。

<div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe;">
  <strong>💡 実践的アドバイス：薬剤師の介入例</strong><br>
  ADLスコアが低い患者さんに対して、単に「お薬カレンダーを使いましょう」と提案するだけでは不十分です。「カレンダーのポケットに入れる動作」すら難しい場合があるからです。この場合は、<b>「一包化」や「服用しやすさを考慮した剤形変更」</b>といった、動作自体を減らす提案が優先されます。
</div>

## 🔄 3. 他の評価指標との違い：なぜBarthel Indexなのか？

介護現場では「FIM（機能的自立度評価表）」なども使われますが、Barthel Indexは<strong>「実際の生活の様子」を把握しやすい</strong>という特徴があります。

*   <strong>Barthel Index</strong>: 実際に「できているか」を評価（生活の質に近い）
*   <strong>FIM</strong>: 何ができるかという「能力」を細かく評価（リハビリテーションに近い）

薬剤師にとって重要なのは、患者さんがリハビリ室でできることよりも、<strong>自宅で実際に「薬を取り出して口に運べているか」</strong>です。そのため、ADLの簡易指標としてBarthel Indexを知っておくことが、ケアマネジャーや訪問看護師と共通言語で話すための近道になります。

---

## 🚀 4. まとめ：明日からの調剤室でできること

Barthel Indexを完璧に暗記する必要はありません。大切なのは、患者さんの動作を見て、その背景を想像する力です。

<div style="border: 2px dashed #94a3b8; padding: 1.5rem; border-radius: 8px; background-color: #ffffff;">
  <strong>薬剤師のための「チェックリスト」</strong>
  <ul>
    <li>□ 薬局に来店した際の歩行状態や、座る動作はスムーズか？</li>
    <li>□ 処方箋の袋を開けたり、お薬手帳を出したりする動作に時間はかかっていないか？</li>
    <li>□ 「残薬」がある場合、それは「忘れ」なのか「開封できない」のかを疑ってみたか？</li>
  </ul>
</div>

患者さんのADLを少しでも意識することで、処方提案の引き出しは大きく増えます。「この患者さん、もしかして階段昇降が難しいなら、朝夕の薬を回数減らしたほうが生活しやすいのでは？」そんな視点を持って、日々の調剤に取り組んでみてください！
