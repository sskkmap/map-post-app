# RIFLE基準とは？AKI評価の基本となる分類

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場の薬剤師が直面する疑問】</strong><br>
  ある日、当直明けの申し送りで「〇〇さんの腎機能が急激に悪化してAKIが疑われます」という話を聞きました。私は「クレアチニン（Cr）がどのくらい上がったの？」と尋ねましたが、先輩から「Crの値だけでなく、尿量も合わせてRIFLE基準で評価してあるよ」と言われました。<br><br>
  当時の私は「RIFLE基準って、何かの省略形だったっけ？」と頭が真っ白に。検査値だけを見て「腎機能低下」と一括りにしていた自分を深く反省しました。薬剤師として、なぜCrだけでなく「尿量」まで考慮してAKIのステージを分けるのかを知ることは、適正な薬物投与設計を行う上で避けて通れない知識でした。
</div>

今回は、急性腎障害（AKI）の評価において世界的なスタンダードとなった<strong>「RIFLE基準」</strong>について、薬剤師が実務で押さえるべきポイントを整理して解説します。

---

## 🔬 1. RIFLE基準の成り立ちと目的

AKI（Acute Kidney Injury）は、かつて「急性腎不全」と呼ばれていましたが、現在はより広い概念として「急性腎障害」と表現されます。このAKIを統一した尺度で評価するために作られたのがRIFLE基準です。

RIFLEという名前は、各ステージの頭文字をとったものです。

*   <strong>R</strong>isk（リスク）
*   <strong>I</strong>njury（損傷）
*   <strong>F</strong>ailure（不全）
*   <strong>L</strong>oss（喪失）
*   <strong>E</strong>nd-stage kidney disease（末期腎不全）

この基準の素晴らしい点は、<strong>「血清クレアチニン値（腎機能の指標）」</strong>と<strong>「尿量（腎臓の排泄機能の直接的な結果）」</strong>という、臨床現場ですぐに確認できる2つの指標を組み合わせて重症度を判定している点にあります。

---

## 📊 2. RIFLE基準の分類一覧

RIFLE基準では、ベースライン（入院時や過去の正常時）と比較して、クレアチニンが何倍になったか、あるいは尿量がどの程度低下したかによって重症度を分類します。

| ステージ | 血清クレアチニン基準 (Cr) | 尿量基準 |
| :--- | :--- | :--- |
| <strong>Risk</strong> | 1.5倍に上昇 | 0.5 mL/kg/時 未満 (6時間以上) |
| <strong>Injury</strong> | 2.0倍に上昇 | 0.5 mL/kg/時 未満 (12時間以上) |
| <strong>Failure</strong> | 3.0倍に上昇 (または 4.0mg/dL以上) | 0.3 mL/kg/時 未満 (24時間以上) または無尿(12時間以上) |
| <strong>Loss</strong> | 持続的な腎機能喪失 (4週間以上) | - |
| <strong>End-stage</strong> | 末期腎不全 (3ヶ月以上) | - |

※ベースラインのCr値が不明な場合は、腎機能正常（GFR 75mL/min）を仮定して計算することもあります。

## 💊 3. 薬剤師がRIFLE基準を実務でどう活かすか？

「基準を知っていること」と「実務で使えること」には大きな差があります。薬剤師として以下の3点を意識してみてください。

### ① 「薬物血中濃度」への影響を予測する
AKIのステージが進行するほど、腎排泄型の薬剤（抗菌薬、抗がん剤、抗凝固薬など）の蓄積リスクが高まります。
*   <strong>Risk段階</strong>: 投与量や投与間隔の見直し検討（TDMの早期実施）。
*   <strong>Failure段階</strong>: 薬剤の中止・変更の疑義照会準備。

### ② 尿量減少のサインを見逃さない
クレアチニンの上昇は、腎臓の損傷から24〜48時間遅れて現れることがよくあります。一方で、<strong>「尿量の減少」は最も早い「アラート」</strong>です。カルテの入力画面で「水分出納（IN/OUT）」を確認し、尿量が減少傾向にあれば、Crが正常であっても腎機能悪化を想定して処方提案を行うのがプロの薬剤師です。

### ③ 「腎毒性薬剤」の適正使用の根拠にする
例えば、NSAIDSやACE阻害薬、アミノグリコシド系抗菌薬など、腎毒性が懸念される薬剤を投与している患者さんに対し、「現在RIFLE分類でRiskに該当するため、用量調整を推奨します」とカルテに記載したり医師に提案したりすることで、重症化を未然に防ぐことができます。

## 🩺 4. まとめ：変化に気づく「最初のキーマン」になろう

RIFLE基準を理解していると、患者さんの検査値や尿量データを見たときに「今はAKIのどの段階にあるのか」「どの薬剤を減量すべきか」を論理的に判断できるようになります。
