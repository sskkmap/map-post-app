# 修正ボルグスケールとは？主観的運動強度の評価法を解説

<div style="background-color: #f8fafc; border-left: 4px solid #64748b; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード：運動指導の壁】</strong><br>
  ある日の外来で、心不全の既往がある患者さんから「散歩を始めたら息切れがするようになった」と相談を受けました。「無理はしないでくださいね」と伝えましたが、患者さんの「無理」の基準は主観的で、どこまでなら頑張って良いのか悩むことはありませんか？<br><br>
  実は、心臓リハビリテーションの現場では、客観的な数値（心拍数など）だけでなく、患者さん自身の「どれくらいキツイか」という感覚を数値化する<strong>「修正ボルグスケール」</strong>が非常に重要視されています。これを導入してからは、患者さんと「今日は何レベルだった？」と会話ができるようになり、指導の精度が格段に上がりました。
</div>

運動指導が必要な患者さんに対して、私たちは「どの程度の強度までなら安全か」を共有しておく必要があります。今回は、現場で即使える「修正ボルグスケール」の評価法について解説します。

---

## 🏃‍♂️ 1. 修正ボルグスケール（Modified Borg Scale）とは？

修正ボルグスケールは、運動中の「息切れ」や「疲労感」といった<strong>主観的な運動強度（RPE：Rate of Perceived Exertion）</strong>を数値化したものです。

心臓に疾患がある患者さんは、心拍数だけで強度を判断するのが難しい（β遮断薬の影響や不整脈など）ケースが多いため、患者さんの「主観的な感覚」を数値で把握することが安全管理において非常に役立ちます。

### 修正ボルグスケールの評価表

運動中は、以下の数値を指標に「どのレベルを目指すべきか」を患者さんと共有します。

| スケール | 感覚の目安 | 状態の判断 |
| :--- | :--- | :--- |
| <strong>0</strong> | 何もしない（安静） | 安静状態 |
| <strong>1</strong> | 非常に楽 | 余裕がある |
| <strong>2</strong> | 楽 | 準備運動レベル |
| <strong>3</strong> | やや楽 | 快適なペース |
| <strong>4</strong> | ややきつい | <strong>適度な運動レベル</strong> |
| <strong>5</strong> | きつい | 注意が必要 |
| <strong>6</strong> | かなりきつい | オーバーペースの可能性 |
| <strong>7-8</strong> | 非常にきつい | すぐに中断を検討 |
| <strong>9</strong> | 非常に非常にきつい | 危険なレベル |
| <strong>10</strong> | 限界 | 直ちに運動中止 |

---

## 🎯 2. 目標設定と薬剤師ができる指導のポイント

心臓リハビリテーションのガイドライン等では、<strong>「やや楽〜ややきつい（3〜4）」</strong>程度の強度が安全で効果的とされています。

<div style="background-color: #f0fdf4; padding: 1.5rem; border-radius: 8px; border: 1px solid #bbf7d0;">
  <strong>💡 薬剤師からのワンポイント・アドバイス</strong><br>
  患者さんに指導する際は、ただ数値を見せるだけでなく、以下のように日常会話に落とし込んであげましょう。<br>
  ・<strong>レベル3「やや楽」</strong>：「お友達と会話しながら笑顔で歩けるペースですね」<br>
  ・<strong>レベル4「ややきつい」</strong>：「息は弾むけれど、短い文章なら会話ができる限界のラインです」<br>
  ・<strong>レベル5「きつい」</strong>：「息切れが強くなり、会話が苦しくなるので、ここで一度休憩しましょう」
</div>

### 患者さんへ伝える「注意すべきサイン」
もし運動中にレベル5（きつい）を超えたり、以下のサインが出た場合は、たとえボルグスケールが低くても運動を即座に中止するよう指導してください。

*   <strong>胸痛、圧迫感</strong>
*   <strong>めまい、ふらつき</strong>
*   <strong>冷や汗</strong>
*   <strong>過度な動悸</strong>

---

## 💊 3. 服薬状況との関連性を考える

薬剤師として特に注目したいのは、患者さんが服用している薬剤がボルグスケールの評価に与える影響です。

1.  <strong>β遮断薬（メインテート、アーチスト等）</strong>: 心拍数が上がりにくいため、心拍数だけを指標にすると「まだ余裕がある」と勘違いし、過度な運動をしてしまうリスクがあります。ボルグスケールのような主観的評価がより重要になります。
2.  <strong>利尿薬（ラシックス、サムスカ等）</strong>: 脱水傾向になりやすく、ボルグスケールが急速に悪化（上昇）することがあります。天候や発汗状況とセットでスケールを評価してもらうよう促しましょう。

---

## 🌿 4. まとめ：患者さんの「感覚」を数値化して安心を届ける

修正ボルグスケールは、特別な道具も必要なく、日々の服薬指導の会話の中で簡単に取り入れられる優れたツールです。

*   <strong>「やや楽〜ややきつい（3〜4）」</strong>が運動の適正範囲。
*   <strong>客観的数値（心拍数）に頼りすぎない</strong>（薬剤の影響を考慮）。
*   <strong>会話が続くかどうか</strong>を基準に、患者さんと共通言語を持つ。

「運動しなきゃ」と焦る患者さんに対して、ボルグスケールを使って「このレベルなら安全に続けられますよ」と具体的なガイドラインを提示することで、患者さんの不安を軽減し、長期的な服薬・運動アドヒアランスの向上に繋げることができます。

ぜひ次回の外来から、「散歩のときは『ややきつい』と感じたら休むようにしてみてくださいね」と声をかけてみてください。あなたの何気ない一言が、患者さんの安全な生活を守る大きな力になります。
