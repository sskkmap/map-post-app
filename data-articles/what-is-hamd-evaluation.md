# HAM-Dとは？ハミルトンうつ病評価尺度の判定方法を解説

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験記】</strong><br>
  精神科の外来調剤を担当し始めた頃、医師のカルテに「HAM-D 18点→12点」と記載されているのを見ました。「点数が下がったから良くなっているんだな」とは理解できましたが、具体的にどの項目が改善したのか、服薬指導でどこを重点的に聞くべきかまでは見えていませんでした。<br><br>
  ある患者さんに「睡眠はどうですか？」と聞くと「眠れるようになった」との回答。後でカルテを確認すると、HAM-Dの睡眠関連項目が改善していたことが分かりました。点数だけでなく、その背景にある具体的な症状の変化を知ることで、患者さんとの会話の深みが変わることを実感した瞬間でした。
</div>

精神科領域の薬剤師として、症状の「見える化」を理解しておくことは、適切なモニタリングに直結します。今回は、うつ病の評価尺度として最もポピュラーな<strong>HAM-D（ハミルトンうつ病評価尺度）</strong>について深掘りしていきましょう。

---

## 📋 1. HAM-D（ハミルトンうつ病評価尺度）とは？

HAM-Dは、うつ病の重症度を評価するために開発された構造化面接形式の尺度です。世界中で幅広く使われており、抗うつ薬の治験や臨床現場での経過観察によく用いられます。

- <strong>評価項目</strong>: 17〜24項目（一般的には17項目版が標準として多く用いられます）
- <strong>判定方法</strong>: 医師が面接を通して評価（0〜4点の5段階評価など）
- <strong>合計点の目安</strong>:
  - 0〜7点：正常範囲
  - 8〜13点：軽症
  - 14〜18点：中等症
  - 19〜22点：重症
  - 23点以上：最重症

---

## 📊 2. HAM-Dの主要な評価項目一覧

HAM-Dは、身体症状から精神症状まで多岐にわたります。薬剤師が日々のモニタリングで特に注目すべき項目を整理しました。

| 項目分類 | 主な評価内容 | 薬剤師の観察ポイント |
| :--- | :--- | :--- |
| <strong>睡眠</strong> | 入眠障害、中途覚醒、早朝覚醒 | 抗うつ薬の鎮静作用や不眠への影響 |
| <strong>抑うつ気分</strong> | 悲しみ、絶望感、無価値感 | 薬効の現れ方、抑うつ症状の軽減度 |
| <strong>不安</strong> | 精神的不安、身体的不安 | SSRI等導入初期の焦燥感の有無 |
| <strong>精神運動</strong> | 落ち着きのなさ、遅延 | 賦活症候群や錐体外路症状の確認 |
| <strong>身体症状</strong> | 全身症状、胃腸症状、性機能低下 | 薬剤の副作用（便秘、口渇、性機能障害） |
| <strong>体重減少</strong> | 食欲不振による減少 | 食欲の変化、消化器副作用の有無 |

## 🔍 3. 薬剤師が注目すべき「注意点」

HAM-Dは有用なツールですが、以下の点には注意が必要です。

1. <strong>副作用を症状と勘違いしない</strong>
   - HAM-Dの項目には「胃腸症状」や「全身倦怠感」が含まれます。これらはうつ病そのものの症状であることもあれば、服用中の抗うつ薬の副作用である可能性もあります。点数が高い＝悪化と即断せず、副作用の可能性を常に頭に置くことが大切です。
2. <strong>睡眠障害の解釈</strong>
   - 睡眠項目は非常に重視されますが、抗うつ薬の影響で日中の眠気が強くなっている場合もあります。「夜の睡眠」だけでなく「日中の活動性」とセットで確認しましょう。
3. <strong>主観的訴えとの乖離</strong>
   - HAM-Dは医師による評価（客観評価）がベースです。患者さん本人の「辛い」という主観的な訴えとスコアが乖離している場合、自己評価尺度（SDSやBDIなど）と併せて多角的に状態を捉える必要があります。

## 🌿 4. まとめ：数値の裏側にある「患者さんの物語」を読み解こう

HAM-Dは、治療の進捗を客観的に測るための非常に強力な羅針盤です。しかし、薬剤師にとっての真の価値は、<strong>「数値が下がったのは、薬がどの症状に効いたおかげなのか？」</strong>を考察し、次の服薬指導に活かすことにあります。

- 眠れるようになったのか？
- 不安が減って外出できるようになったのか？
- 食事がとれるようになったのか？

こうした具体的な変化を患者さんと共有できると、アドヒアランスの向上にも大きく貢献します。次回の面談では、点数の推移を眺めながら、ぜひ患者さんに「最近一番楽になったことは何ですか？」と投げかけてみてください。きっと、スコア以上の大切な情報が返ってくるはずです。
