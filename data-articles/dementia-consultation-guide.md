# 認知症が疑われるときは何科を受診すればいい？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  薬局の窓口で、70代の患者さんから「最近、財布をどこに置いたか忘れることが増えて不安なの。何科に行けばいいかしら？」と相談されたことがあります。咄嗟に「物忘れ外来ですかね…」と答えましたが、近隣にそうした看板を掲げるクリニックが少なく、次に何と助言すべきか言葉に詰まってしまいました。<br><br>
  後日、地域の包括支援センターと連携する中で、認知症の疑いといっても「隠れた内科的疾患」が原因であるケースや、精神的な症状が強いケースなど、原因によって適した受診先が異なることを学びました。ただ「何科へ行くか」を伝えるだけでなく、患者さんの背景に合わせて選択肢を提案することこそ、地域薬局の役割なのだと痛感した出来事でした。
</div>

「物忘れ」を主訴に来局される患者さんは少なくありません。しかし、一口に認知症といっても、その専門的なアプローチは診療科によって大きく異なります。今回は、患者さんからの相談にプロとして的確に答えるための知識を整理します。

## 🧠 1. 認知症を疑ったときの受診先の選び方

認知症専門医がいるからといって、必ずしもそこが最適解とは限りません。まずは「何が一番困っているか」を整理し、以下の診療科を案内するのが一般的です。

| 診療科 | 特徴・強み | こんな方におすすめ |
| :--- | :--- | :--- |
| <strong>もの忘れ外来</strong> | 認知症の診断に特化。画像診断や心理検査が充実 | 診断を確定させたい、家族が具体的なケアを知りたい |
| <strong>精神科・心療内科</strong> | 幻覚、妄想、興奮などの精神症状への対応が得意 | 徘徊や暴言、うつ症状など「周辺症状（BPSD）」が強い |
| <strong>神経内科（脳神経内科）</strong> | 脳の器質的疾患（脳梗塞、パーキンソン等）の鑑別 | 歩行障害や手足の震え、麻痺など身体症状を伴う場合 |
| <strong>かかりつけの内科</strong> | 全身疾患の管理、早期発見の窓口 | まずは他の病気でないか相談したい、気軽に受診したい |

## 🔍 2. 診療科選びの判断基準：薬剤師ができるトリアージ

患者さんの訴えを聞きながら、受診先をスムーズに提案するための「ヒント」をいくつか紹介します。

### ① 身体的なサインがある場合：神経内科へ
「歩き方が小刻みになった」「よく転ぶ」「手足が震える」といった身体症状がある場合は、パーキンソン病などの関連疾患を見逃さないため、<strong>脳神経内科</strong>を受診するのが合理的です。

### ② BPSD（周辺症状）が激しい場合：精神科へ
「家族を疑う」「深夜に騒ぐ」「攻撃的になる」といった症状は、家族の負担が非常に大きくなります。この場合は、精神症状の緩和に慣れている<strong>精神科</strong>の方が、薬剤調整や介護サポートの面で手厚いケアが受けられる可能性があります。

### ③ まずは何が原因か分からない場合：まずは「かかりつけ医」へ
「認知症かどうかすら怪しい」という段階では、まず普段診てもらっている<strong>内科</strong>に相談し、血液検査や画像診断が必要な専門機関を紹介してもらうルートが、患者さんにとっても心理的ハードルが低いです。

## 🛡️ 3. 地域連携のハブになるために

薬剤師ができる最大のサポートは、<strong>「地域連携のネットワークを把握しておくこと」</strong>です。

<div style="border: 2px solid #3b82f6; padding: 1.5rem; border-radius: 8px;">
  <h3 style="margin-top: 0;">✅ 薬剤師がやっておきたい「3つの準備」</h3>
  <ul style="line-height: 1.8;">
    <li><strong>地域の「認知症サポート医」を把握する</strong>：かかりつけ医が認知症サポート医であれば、連携がスムーズです。</li>
    <li><strong>「包括支援センター」と繋がっておく</strong>：受診先の案内だけでなく、介護保険の申請なども含めた相談先をリスト化しておきましょう。</li>
    <li><strong>お薬手帳から「サイン」を見逃さない</strong>：抗不安薬や睡眠薬の過剰な処方など、認知機能に悪影響を与えている薬がないかチェックし、医師へフィードバックすることも立派な認知症ケアです。</li>
  </ul>

## 🌿 4. まとめ：薬剤師の「言葉」が不安を和らげる

認知症の疑いに対して、「どこに行けばいいか」を提示することは、患者さんの抱える「得体の知れない不安」を、具体的な「医療への一歩」に変える作業です。

大切なのは、<strong>「認知症かもしれない」という不安を抱えた患者さんに寄り添い、一番相談しやすい窓口へ繋いでいくこと</strong>です。今回ご紹介した診療科の特徴を活かし、患者さんの症状や生活背景に合わせた提案ができるよう、地域の医療機関リストを更新しておきましょう。

「ここに行けば専門の先生が詳しく診てくれますよ」という薬剤師の一言が、患者さんやご家族にとって、どれほど心強い支えになるか分かりません。ぜひ、明日からの相談対応に役立ててください。
