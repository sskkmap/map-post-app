# Hugh-Jones分類とは？呼吸困難の重症度評価を解説

<div style="background-color: #f0f9ff; border-left: 4px solid #0ea5e9; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験談】</strong><br>
  ある日の服薬指導中、COPD（慢性閉塞性肺疾患）で吸入薬を使用されている高齢の患者さんから、「最近、前よりも息苦しさを感じることが増えた」というお話を伺いました。しかし、「具体的にどれくらいですか？」と聞いても、「なんとなくねぇ」と曖昧な返答しか返ってきません。<br><br>
  その際、私はHugh-Jones（ヒュー・ジョーンズ）分類の基準を頭に浮かべ、「歩く時はどうですか？」「家の中の移動は？」と段階的に質問を変えてみました。すると、患者さんの日常生活レベルでの息切れの状況が具体的に浮き彫りになり、医師へ「ADLの低下が見られるため、吸入薬の評価が必要かもしれません」と正確な情報提供ができたのです。
</div>

呼吸器疾患の患者さんと接する際、「息切れ」の程度をどう把握するかは、吸入薬や酸素療法の効果を判断する重要な指標となります。今回は、臨床現場で非常に役立つ「Hugh-Jones分類」について解説します。

---

## 📋 1. Hugh-Jones分類とは？

Hugh-Jones（ヒュー・ジョーンズ）分類は、<strong>呼吸困難の程度を日常生活動作（ADL）に基づき、Ⅰ度からⅤ度の5段階で分類したもの</strong>です。

患者さんの自覚症状を客観的な指標に変換できるため、呼吸器疾患の重症度把握や、治療による症状の改善度合いを確認する際によく用いられます。

### Hugh-Jones分類 一覧表

| 分類 | 症状の目安（息切れの程度） |
| :--- | :--- |
| <strong>Ⅰ度</strong> | 健康な人と同等に歩ける。労作時も息切れなし。 |
| <strong>Ⅱ度</strong> | 平地歩行では息切れしないが、坂道や階段で息切れする。 |
| <strong>Ⅲ度</strong> | 平地でも自分のペースで歩けるが、急ぎ足や長い距離で息切れする。 |
| <strong>Ⅳ度</strong> | 平地をゆっくり歩いても、途中で息切れして休みたくなる。 |
| <strong>Ⅴ度</strong> | 安静時でも息切れがあり、会話や着替えなどの動作でも苦しい。 |

---

## 🔍 2. 薬剤師が実務で活用するポイント

Hugh-Jones分類は、単にカルテに記載された数字を確認するだけでなく、以下のようなシーンで<strong>「患者さんの今の状態」を把握するツール</strong>として活用できます。

### ① 服薬指導でのヒアリング
「息苦しさはどうですか？」という抽象的な質問ではなく、分類に基づいた問いかけをしてみましょう。
*   <strong>「階段を上る時、途中で止まりたくなりますか？」（Ⅱ度〜Ⅲ度の確認）</strong>
*   <strong>「お散歩をしていて、どれくらい歩くと休憩が必要ですか？」（Ⅲ度〜Ⅳ度の確認）</strong>

この質問によって、患者さんの現在のADLを具体的に引き出せます。

### ② 吸入薬の効果判定
吸入薬を変更したり、開始したりした数週間後に「最近、歩くのは楽になりましたか？」と聞くことで、分類が「Ⅳ度→Ⅲ度」へと改善しているかを確認できます。これは患者さんの治療モチベーション維持にも繋がります。

## ⚠️ 3. 評価時の注意点

Hugh-Jones分類は非常に便利ですが、以下の点には注意が必要です。

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px;">
  <ul style="margin: 0; padding-left: 1.2rem;">
    <li style="margin-bottom: 0.5rem;"><strong>主観的な指標である</strong><br>あくまで患者さんの自覚症状に基づいています。加齢による筋力低下や、心不全など他疾患による息切れとの切り分けは医師の診断が必要です。</li>
    <li><strong>「安定した状態」を聞く</strong><br>風邪や肺炎などの急性増悪期ではなく、普段の状態（ベースライン）がどの程度かを聞き取ることが大切です。</li>
  </ul>
</div>

## 🩺 4. 医師への疑義照会・情報提供への応用

「Hugh-Jones分類が以前より悪化している」という情報は、医師にとって非常に価値のあるフィードバックになります。

*   <strong>「以前はⅡ度と言われていましたが、現在は平地歩行でも休みが必要なようで、Ⅳ度に悪化している印象を受けます。」</strong>

このように伝えると、医師は現在の治療方針（吸入薬のデバイス変更、併用薬の検討など）を再考するきっかけを得られます。

## 💡 まとめ：コミュニケーションの共通言語として

Hugh-Jones分類を理解しておくことで、患者さんと<strong>「今の息切れレベル」を共通言語で共有できるようになります。</strong>

吸入薬や酸素療法が必要な患者さんに対して、単に「薬を吸ってください」と伝えるだけでなく、「この薬で息切れが少しでも楽になって、階段が歩きやすくなるといいですね」と具体的な目標を共有する――。

そんな、患者さんに寄り添う服薬指導のために、ぜひこのHugh-Jones分類を毎日の業務に取り入れてみてください。
