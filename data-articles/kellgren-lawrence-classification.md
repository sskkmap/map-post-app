# Kellgren-Lawrence分類とは？変形性関節症のX線評価法

  <strong>【薬剤師の現場あるあるエピソード】</strong><br>

  当時の私は、KL分類という言葉を聞いたことはあっても、その数字が具体的にレントゲン写真の何を指しているのか、なぜそれが治療方針に影響するのかを即答できませんでした。「グレード3ってどの程度？人工関節レベルなの？」と焦った経験があります。薬の選択だけでなく、画像診断の基準を少し理解するだけで、患者さんの不安に寄り添った服薬指導ができるようになります。

今回は、変形性関節症（OA：Osteoarthritis）の評価で世界的に用いられる<strong>「Kellgren-Lawrence分類（KL分類）」</strong>について、臨床現場で役立つ視点で解説します。

---

## 🦴 1. Kellgren-Lawrence分類とは？

Kellgren-Lawrence分類（KL分類）は、変形性関節症（特に膝関節や股関節）の進行度を<strong>X線写真（レントゲン）の所見に基づいて0〜4の5段階で評価する手法</strong>です。

医師は「関節裂隙の狭小化（軟骨がすり減って隙間が狭くなる）」や「骨棘（こつきょく：骨のトゲ）」の有無を見て、このグレードを決定します。

| グレード | 重症度 | 主なX線所見 |
| :--- | :--- | :--- |
| <strong>Grade 0</strong> | 正常 | 所見なし |
| <strong>Grade 1</strong> | 疑い | 疑わしい骨棘形成、関節裂隙の変化なし |
| <strong>Grade 2</strong> | 軽度 | 明らかな骨棘形成、関節裂隙は保たれている |
| <strong>Grade 3</strong> | 中等度 | 明らかな関節裂隙の狭小化、複数の骨棘 |
| <strong>Grade 4</strong> | 高度 | 関節裂隙の消失、軟骨下骨の硬化、骨変形 |

<div style="background-color: #f0fdf4; border: 1px solid #bbf7d0; padding: 1rem; border-radius: 8px; margin: 1rem 0;">
  <strong>💡 薬剤師の視点：なぜこの分類を知るべきか？</strong><br>
  グレードが高い（3〜4）患者さんは、保存療法（薬物療法やリハビリ）の効果が出にくく、手術適応が検討されるフェーズです。NSAIDsの長期使用リスクを考慮しつつ、いつ頃から手術の話が出そうか、患者さんの状況を把握するヒントになります。
</div>

## 🔍 2. グレード別に見る「薬物療法のスタンス」

KL分類は、変形性関節症の治療において「どこまで薬で引っ張るか」という方針を決める目安にもなります。

### 🛡️ Grade 1〜2：保存療法の中心期
炎症を抑え、痛みを取り除きながら、リハビリや運動療法を並行する段階です。
- <strong>治療のポイント</strong>: 鎮痛薬の定期服用だけでなく、外用剤をうまく活用して全身性の副作用（消化管障害、腎機能低下など）を回避します。
- <strong>薬剤師のアドバイス</strong>: 「運動して筋肉をつけましょう」という医師の指示に対し、「痛みがあるから動けない」という患者さんへ、貼付剤の使い方や痛みのコントロール方法を具体的に提案します。

### ⚠️ Grade 3〜4：手術検討・保存療法の限界期
軟骨がほとんどなく、骨同士が接触しているため、鎮痛薬だけでは痛みの完全除去は困難です。
- <strong>治療のポイント</strong>: 保存療法を継続する場合、長期的なNSAIDs服用によるリスク管理（胃粘膜保護薬の併用や、eGFRのチェック）が必須です。
- <strong>薬剤師のアドバイス</strong>: 長期服用者の「最近、胃がもたれる」「むくみがひどい」といった訴えはサインかもしれません。医師への減量提案や、オピオイド系鎮痛薬への切り替えのタイミングを検討する意識を持ちましょう。

## 💊 3. 変形性関節症治療でよく登場する薬剤リスト

変形性関節症治療の基本薬を整理しました。KL分類の進行度に合わせてこれらの使い分けがなされます。

| 薬剤分類 | 代表的な薬剤 | 薬剤師のチェックポイント |
| :--- | :--- | :--- |
| <strong>外用NSAIDs</strong> | ロキソプロフェン、フェルビナク | 被れ（接触皮膚炎）の有無、貼付枚数 |
| <strong>経口NSAIDs</strong> | ロキソプロフェン、セレコキシブ | 高齢者の腎機能、消化性潰瘍の既往 |
| <strong>アセトアミノフェン</strong> | カロナール | 長期服用時の肝機能モニタリング |
| <strong>弱オピオイド</strong> | トラマドール、タリージェ等 | 便秘対策、ふらつき・眠気の注意喚起 |

## 🎯 まとめ：KL分類は患者さんの理解度を深めるツール

Kellgren-Lawrence分類はあくまで画像診断上の指標ですが、これを理解していると、患者さんの<strong>「病気に対する受け止め方」や「痛みの深刻さ」を推測する</strong>ことができます。

- <strong>「先生にグレード3と言われた」</strong>と言われたら、「骨の隙間が狭くなっている状態ですね。痛み止めだけでなく、リハビリで膝周りの筋肉を強化することも大切ですよ」と、骨格の変化に基づいた声掛けが可能になります。

単に薬を渡すだけでなく、レントゲン写真の背後にある「関節の状態」を少しだけ想像できるようになる。それだけで、整形外科領域の服薬指導はぐっと深みが増すはずです。今日から患者さんのカルテや診察記録にある「KL分類」の文字に、ぜひ注目してみてくださいね。
