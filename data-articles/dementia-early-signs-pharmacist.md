# 家族が気づきたい認知症の初期症状とは？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験：服薬管理の違和感】</strong><br>
  ある日の外来調剤で、いつもカレンダー形式の薬ケースを完璧に管理されていた80代の患者さんが、急に「薬が余っている」とおっしゃったことがありました。計算すると、ここ数週間で10日分以上の飲み忘れが発生していました。<br><br>
  詳しくお話を聞くと、お薬のことだけでなく、近所のコンビニでいつもの買い物ができなくなったというエピソードも。処方内容を調整する前に「一度、認知機能のスクリーニングを受けてみませんか」とご家族に提案し、結果として軽度認知障害（MCI）の段階で治療を開始することができました。
</div>

私たち薬剤師にとって、患者さんの「服薬状況の変化」は、認知症の予兆を捉える極めて重要なアラートです。今回は、ご家族が日常生活で気づくべき初期症状と、薬局での介入のヒントを整理します。

## 🧠 1. 「もの忘れ」と「認知症の健忘」の違い
単なる加齢による「もの忘れ」と、認知症に伴う「中核症状」は、ご家族が以下のような視点で観察すると違いが見えてきます。

| 特徴 | 加齢によるもの忘れ | 認知症によるもの忘れ |
| :--- | :--- | :--- |
| <strong>内容</strong> | 体験の一部を忘れる（夕飯のメニューなど） | 体験の全体を忘れる（夕飯を食べたこと自体） |
| <strong>自覚</strong> | 忘れている自覚がある | 忘れている自覚がない（病識の欠如） |
| <strong>ヒント</strong> | ヒントがあれば思い出せる | ヒントがあっても思い出せない |
| <strong>生活</strong> | 日常生活に支障はない | 日常生活や服薬管理に支障が出る |

## 🔍 2. ご家族が気づくべき「5つのサイン」
認知症の初期には、記憶障害以外にも「実行機能障害」や「見当識障害」が先行することがあります。

1. <strong>服薬管理の乱れ</strong>: 薬の飲み忘れ、あるいは逆に「飲んだことを忘れて」重複して服用する。
2. <strong>段取りの悪化</strong>: 料理の順序がわからなくなる、家電の使い方が急にわからなくなる。
3. <strong>金銭管理のトラブル</strong>: お金の計算間違いや、不要なものを買ってしまう。
4. <strong>性格の変化</strong>: 怒りっぽくなる、無気力になる（アパシー）、こだわりが強くなる。
5. <strong>衛生状態の低下</strong>: 入浴頻度が減る、同じ服を何日も着続ける。

  <strong>💡 薬剤師からのアドバイス</strong><br>
  ご家族が「少し様子がおかしいな」と感じたら、まずは「いつから」「どのような失敗が」「どのくらいの頻度で」起きているかを具体的にメモしてもらうようにお願いしてみてください。これが医師への橋渡しとして非常に役立ちます。
</div>

## 💊 3. 認知症の治療薬（抗認知症薬）の基本知識
早期発見され、診断がついた際に用いられる主な薬剤は以下の通りです。

| 薬剤名（成分名） | 作用機序 | 主な副作用 |
| :--- | :--- | :--- |
| <strong>ドネペジル</strong> | アセチルコリンエステラーゼ阻害 | 悪心、下痢、興奮、徐脈 |
| <strong>ガランタミン</strong> | 同左 ＋ ニコチン受容体刺激 | 悪心、嘔吐、食欲不振 |
| <strong>リバスチグミン</strong> | 同左（パッチ剤） | 皮膚かぶれ、悪心 |
| <strong>メマンチン</strong> | NMDA受容体拮抗薬 | めまい、眠気、便秘 |

## ⚠️ 4. 薬剤師が現場で行うべき「早期発見」への取り組み
私たちは、単に処方箋通りに薬を出すだけでなく、以下の対応で地域の認知症ケアの拠点となれます。

* <strong>服薬カレンダーのチェック</strong>: 
  飲み残しがある場合、「体調が悪かったのですか？」と優しく問いかけ、本人が答えられない場合はご家族へ確認の連絡を入れる。
* <strong>副作用か症状かの見極め</strong>: 
  服用開始後に混乱が増したように見える場合、副作用なのか、進行によるものなのかを医師に報告・相談する。
* <strong>生活習慣のヒアリング</strong>: 
  「最近、テレビ番組の内容についていけないことはないですか？」「趣味を楽しめていますか？」といった何気ない会話から、うつ状態や認知機能低下のサインを拾い上げる。

## 🌿 5. まとめ：かかりつけ薬剤師ができる「見守り」
認知症は早期に発見し、適切なケアや薬物療法を行うことで、その後の生活の質（QOL）を大きく維持できる可能性があります。

ご家族が「おかしいな」と感じたとき、一番身近な専門家である私たち薬剤師に相談してもらえる環境を作っておくことが重要です。<strong>「お薬の相談ついでに、最近のお父様の様子を教えていただけますか？」</strong>という一言が、早期発見への第一歩になるのです。

もし、服薬管理に不安が出てきたら、一包化やカレンダーの活用、あるいは訪問薬剤管理指導への切り替えなど、私たちは患者さんとご家族の負担を減らす提案をいつでも準備しています。認知症を「恥ずかしいこと」ではなく「一緒に支えるべきもの」として、薬局からサポートしていきましょう。
