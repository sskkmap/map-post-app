# CAGEとは？アルコール依存症スクリーニングテストを解説

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の実務エピソード】</strong><br>
  以前、血圧降下剤を処方されている患者さんの重複投薬を確認していた際、ふと「なぜか血圧が安定しないな」と気になりました。お薬手帳には他の薬局での受診歴もありません。<br><br>
  雑談の中で何気なく「晩酌は楽しんでいますか？」と聞いたところ、「仕事のストレスで毎日焼酎をボトル半分ほど飲まないと眠れない」との回答が。この時、もっと早くアルコール摂取量に注目していれば、血圧コントロールへの影響や睡眠薬の検討も早められたかもしれないと感じました。アルコール問題を切り出すのは気が引けることもありますが、適切なツールがあれば、より早期に介入できると学んだ経験です。
</div>

アルコール関連の問題は、患者さん自身が自覚していない、あるいは隠したがるケースも多いものです。そこで活用したいのが、わずか4つの質問で依存の可能性を探れる<strong>「CAGE」</strong>というスクリーニングテストです。

---

## 📋 1. CAGEテストとは？

CAGEは、アルコール依存症を早期発見するために開発された非常に簡便なスクリーニングテストです。以下の4つの質問項目からなり、<strong>「はい（Yes）」が2つ以上あれば、アルコール依存症の疑いが高い</strong>と判断されます。

| アルファベット | 質問項目（英語の頭文字） | 質問の内容 |
| :--- | :--- | :--- |
| <strong>C</strong> | <strong>C</strong>ut down | お酒を減らさなければいけないと感じたことがありますか？ |
| <strong>A</strong> | <strong>A</strong>nnoyed | 周囲の人から飲酒を注意されて、イラッとしたことがありますか？ |
| <strong>G</strong> | <strong>G</strong>uilty | 飲酒に対して罪悪感を感じたことがありますか？ |
| <strong>E</strong> | <strong>E</strong>ye-opener | 朝一番の迎え酒をしたことがありますか？ |

### 🔍 各質問の臨床的意義
- <strong>C（Cut down）</strong>: 自身でコントロールできない感覚がある（喪失感）。
- <strong>A（Annoyed）</strong>: アルコールに対する批判を防御的に捉える（否認のサイン）。
- <strong>G（Guilty）</strong>: 飲酒による社会的・倫理的な問題行動の自覚。
- <strong>E（Eye-opener）</strong>: 離脱症状を抑えるための飲酒（身体的依存の強さを示す）。

## 🛠 2. 薬剤師が実務で活用するためのポイント

CAGEは、必ずしも医師が診察室で行うだけのものではありません。私たち薬剤師も以下の場面で活用できます。

### 🚨 薬局での「疑義照会」や「受診勧奨」への繋げ方
「お酒を飲みすぎではないか？」とストレートに聞くのは関係性に悪影響を及ぼす可能性があります。CAGEを活用する際は、次のような工夫が有効です。

1. <strong>睡眠薬や抗不安薬の服薬指導時</strong>
   「このお薬はアルコールと一緒に飲むと効き目が強く出すぎて危険です。最近、眠るために寝酒をされることはありますか？」と、<strong>薬の安全性の観点から</strong>自然に質問を始めます。
2. <strong>血圧・血糖値のコントロール不良時</strong>
   「生活習慣の改善についてですが、食事以外で気にされていることはありますか？」と聞き出し、飲酒量が増えているようであれば、上記の質問を順番に投げかけます。

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 アドバイス：否定しない姿勢が重要</strong><br>
  もし「はい」と答えた場合でも、「依存症ですよ」と突き放すのではなく、「それはお辛いですね」「健康のために少し減らせるよう、お手伝いできることはありますか？」と、<strong>非難せず寄り添う姿勢</strong>を見せることで、患者さんの本音を引き出しやすくなります。
</div>

## ⚖️ 3. 他のスクリーニングテストとの違い

CAGEは「簡便さ」が武器ですが、アルコール関連問題の全体像を見るには他のツールも併用されます。

| ツール名 | 特徴 | 活用の目安 |
| :--- | :--- | :--- |
| <strong>CAGE</strong> | 4項目、短時間 | 依存の「スクリーニング」に最適 |
| <strong>AUDIT</strong> | 10項目、詳細 | リスクレベルの判定（軽〜重度） |
| <strong>MAST</strong> | 25項目、複雑 | 依存症の「診断」補助向け |

※AUDIT（Alcohol Use Disorders Identification Test）は、世界保健機関（WHO）が作成したもので、より広い飲酒問題を網羅しています。薬局でじっくり話ができる時間があれば、こちらを使うのも有効です。

## 🌿 4. 薬剤師としてできることのまとめ

アルコール依存症は「意志の弱さ」ではなく「疾患」です。CAGEを通じて患者さんの飲酒習慣に「気づき」を与えられたら、それは素晴らしい介入です。

*   <strong>2項目以上で「はい」の場合</strong>: アルコール依存のリスクが高いことを示唆します。
*   <strong>薬学的なリスクを伝える</strong>: アルコールと併用禁忌・併用注意の薬（睡眠薬、血糖降下薬、血圧薬など）があれば、具体例を挙げて注意喚起を行う。
*   <strong>専門医へ繋ぐ</strong>: 依存の疑いが強い場合は、心療内科や精神科、あるいはアルコール専門外来の受診を推奨するカードを渡すなど、具体的な道筋を示すことが大切です。

日々の服薬指導の「＋α」のコミュニケーションとして、ぜひCAGEの項目を頭の片隅に置いてみてくださいね。患者さんの健康寿命を守るための強力なツールになるはずです！
