# Alvaradoスコアとは？急性虫垂炎の診断に役立つ評価法

  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  薬局の窓口で、顔面蒼白で激しい腹痛を訴える患者さんから「いつもの胃腸薬が欲しい」と言われた経験はありませんか？<br><br>
  数年前、私の薬局にもそんな患者さんが来られました。一見ただの胃腸炎に見えましたが、右下腹部の痛みを強く訴えており、すぐ近医の受診を促しました。結果、急性虫垂炎（盲腸）による腹膜炎の初期段階だったのです。もし私が「お薬を出しておきますね」と安易に鎮痛剤や整腸剤を渡していたら……と思うと、今でも冷や汗が出ます。この出来事を機に、腹痛の緊急度を客観的に評価する重要性を痛感しました。

日常業務において、医師ではない私たちが診断を下すことはできません。しかし、患者さんの症状から<strong>「緊急受診が必要なサイン」</strong>を見抜くことは、薬剤師として非常に重要なスキルです。今回は、急性虫垂炎の診断を補助する評価ツール「Alvaradoスコア」について解説します。

## 🧐 Alvaradoスコアとは？

Alvaradoスコアは、腹痛を主訴とする患者さんが「急性虫垂炎」である可能性を、8つの項目（症状・徴候・検査値）を用いて数値化（最大10点）する評価法です。

このスコアを知っておくことで、患者さんの訴えが「ただの腹痛」なのか、それとも「すぐに応急処置が必要な虫垂炎の可能性が高いのか」を判断する指標になります。

### 【Alvaradoスコアの評価項目】

| 項目 | 内容 | 配点 |
| :--- | :--- | :--- |
| <strong>症状</strong> | 移動性右下腹部痛 | 1点 |
| | 食欲不振 | 1点 |
| | 悪心・嘔吐 | 1点 |
| <strong>身体所見</strong> | 右下腹部の圧痛（反跳痛など） | 2点 |
| | 反跳痛（腹膜刺激症状） | 1点 |
| | 発熱（37.3℃以上） | 1点 |
| <strong>検査値</strong> | 白血球増多（>10,000/μL） | 2点 |
| | 核左方移動（好中球の割合増加） | 1点 |
| <strong>合計</strong> | | <strong>10点</strong> |

## ⚠️ 薬剤師が知っておくべき注意点

Alvaradoスコアは非常に有用ですが、万能ではありません。以下の点に留意してください。

1. <strong>診断ツールではなく「補助ツール」</strong>: あくまで疑い度を測るものであり、最終診断は医師が血液検査やCT検査、エコー検査を組み合わせて行います。
2. <strong>高齢者や小児への適応</strong>: 高齢者は炎症反応が鈍い（熱が出ない、白血球が増えない）ケースがあり、また小児は症状が非典型的なことが多いです。スコアが低くても安心はできません。
3. <strong>「痛み止め」の安易な使用</strong>: 腹痛を訴える患者さんに対し、原因が特定される前に安易に鎮痛剤を販売・調剤することは禁物です。症状をマスクしてしまい、腹膜炎の発見を遅らせるリスクがあります。

### もし窓口で「腹痛がある」と言われたら…
患者さんの訴えを聞きながら、以下のステップを意識しましょう。

*   <strong>痛みの場所の確認</strong>: 「痛いのはお腹のどのあたりですか？」と聞き、右下腹部に限定しているか確認する。
*   <strong>症状の経緯</strong>: 「最初はみぞおちが痛く、だんだん右下に移ってきた」という典型的な経過を聞き出す。
*   <strong>バイタルの観察</strong>: 顔色が悪い、歩行時に響くような痛みがある、高熱がある場合は、薬の提案を止めて医師への受診を強く推奨する。

## 🌿 まとめ：患者さんのSOSを見逃さないために

私たち薬剤師の役割は、調剤して薬を渡すことだけではありません。患者さんの些細な訴えから「緊急性」を感じ取り、専門医への橋渡しをすることも重要な責務です。

Alvaradoスコアのような評価基準を頭の片隅に入れておくだけで、日々の窓口対応に「根拠のある自信」が生まれます。

「胃腸薬をください」と言われた時、その裏にある病態を推測する力。それこそが、ベテラン薬剤師への第一歩なのかもしれません。皆さんも、日々の業務の中でこうした視点をぜひ取り入れてみてくださいね。
