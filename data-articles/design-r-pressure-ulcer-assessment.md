# 🩺 DESIGN-Rとは？褥瘡評価スケールの見方と活用法

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  在宅訪問を始めたばかりの頃、褥瘡治療のために処方された外用薬を、特に確認もせず漫然と継続してしまった経験があります。ある日、往診医から「創の深さが変わったのに、まだその薬を出すの？」と指摘され、ハッとさせられました。<br><br>
  当時は「指示通りに調剤すること」が仕事だと思い込んでいましたが、褥瘡は日々変化するもの。患者さんの皮膚状態と評価スコアが連動していないと、適切な治療どころか治癒を遅らせてしまうリスクがあるのです。その後、褥瘡評価ツール「DESIGN-R」の重要性を学び、処方提案の幅がぐっと広がりました。
</div>

褥瘡治療に関わる中で、必ず一度は目にする<strong>「DESIGN-R」</strong>。今回は、薬剤師が知っておくべき評価スケールの見方と、外用薬の選択にどう活かすべきかを解説します。

---

## 🔍 1. DESIGN-R（デザイン・アール）とは？

DESIGN-Rは、日本褥瘡学会が作成した<strong>「褥瘡の重症度を評価するためのスケール」</strong>です。

単なる「深さ」だけでなく、創の大きさや炎症、壊死組織の状態などを点数化することで、<strong>「今の治療方針が適切か」「改善しているか」を客観的に評価</strong>できます。

### DESIGN-Rの評価項目
以下の7つの項目を0点〜数点で評価し、合計点数で重症度を判断します。

| 評価項目 | 評価内容のポイント |
| :--- | :--- |
| <strong>D</strong> (Depth/深さ) | 真皮までか、皮下組織までか、骨に達しているか |
| <strong>E</strong> (Exudate/滲出液) | 滲出液の量や性状（膿など） |
| <strong>S</strong> (Size/大きさ) | 長径×短径で面積を算出 |
| <strong>I</strong> (Inflammation/炎症) | 発赤、熱感、腫れ、疼痛の有無 |
| <strong>G</strong> (Granulation/肉芽) | 肉芽の状態（きれいか、不良か） |
| <strong>N</strong> (Necrotic tissue/壊死組織) | 壊死組織の有無と硬さ |
| <strong>P</strong> (Pocket/ポケット) | 皮膚の下に空洞ができているか |

## 💊 2. 評価結果と外用薬選択の考え方

薬剤師にとって最も重要なのは、<strong>「評価スコアを見て、外用薬が合っているか判断すること」</strong>です。

      滲出液を吸収する剤形（カデックスなど）や、創面を保護する被覆材との併用が考慮されます。逆に、乾燥させる薬が強すぎると創治癒を阻害することもあります。

    <p style="color: #92400e; font-size: 0.95em; margin: 0;">
      肉芽を保護し、適度な湿潤環境を保つことが大切です。刺激の少ないワセリン基剤や、肉芽形成を促進する外用薬（アズノールなど）が選択されます。

## 🎯 3. 薬剤師が現場で活用するための3つのステップ

医師に「この外用薬でいいですか？」と聞くのは勇気がいりますよね。以下のステップでデータを確認してみましょう。

### ステップ1：最新の「合計点数」を確認する
DESIGN-Rには合計点数で重症度を判断する「DESIGN-R 2020」があります。前回の訪問時と比較して、点数が減っているか（＝改善傾向か）を確認します。

### ステップ2：外用薬の「適応」と「現在の状態」を照らす
「この薬は壊死組織を溶かす薬だけど、今の創はもう肉芽ができているな…」というように、<strong>薬の効果と創の状態がマッチしているか</strong>をチェックリストとして活用してください。

### ステップ3：処方提案につなげる
もし変化が見られない、あるいは悪化している場合は、<strong>「創面が乾燥しているため、保湿力の高い基剤への変更はいかがですか？」</strong>といった、根拠に基づいた提案が可能になります。

---

## ⚠️ 4. 注意すべきポイント：感染の徴候

DESIGN-Rで「I（炎症）」の点数が高い場合、感染（蜂窩織炎など）を疑う必要があります。

- <strong>発赤・熱感・腫脹・疼痛の増強</strong>
- <strong>臭いの変化</strong>
- <strong>滲出液の色や量の急増</strong>

これらは外用薬だけで解決する問題ではありません。早期に医師へ情報提供を行い、抗菌薬の検討を含めた連携を取ることが、薬剤師の重要な役割です。

---

## 🌿 5. まとめ：データで医師と会話しよう

DESIGN-Rは、医師・看護師・薬剤師が<strong>「同じ目線で患者さんの状態を把握するための共通言語」</strong>です。

「なんとなくこの薬」から脱却し、「DESIGN-RのG項目の点数が良くなってきたので、刺激の強い薬から保護する薬へ切り替えませんか？」という提案ができるようになれば、あなたの訪問業務はさらに頼りがいのあるものになります。

ぜひ、次回の訪問診療や調剤の際、カルテや指示書に記載されているDESIGN-Rのスコアに注目してみてくださいね。
