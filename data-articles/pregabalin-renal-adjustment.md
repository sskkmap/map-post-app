# プレガバリンはなぜ腎機能で用量調整するのか？

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場あるある】</strong><br>
  ある日の門前薬局での出来事です。高齢の患者さんが持ってきた処方箋に「プレガバリンカプセル 75mg 1日2回」の記載がありました。問診で「最近、足がふらつくことがある」と言われ、ふと検査値を確認すると、eGFRが30mL/min/1.73m²台まで低下していました。<br><br>
  「プレガバリンは腎排泄だから減量が必要だった！」とすぐに気付けましたが、もし見逃していたら転倒などの重大なリスクに繋がっていたかもしれません。単に「腎臓で出るから減らす」と覚えるだけでなく、なぜそこまで注意が必要なのかを改めて復習しましょう。
</div>

プレガバリン（製品名：リリカなど）は、神経障害性疼痛などの治療に欠かせない薬です。しかし、<strong>腎排泄型薬剤</strong>であるため、腎機能に応じた緻密な用量調整が求められます。

## 📊 1. なぜ腎機能で用量調整が必須なのか？

最大の理由は、<strong>「プレガバリンのほぼ100%が未変化体のまま尿中に排泄される」</strong>という特徴にあります。

もし腎機能が低下していると、排泄が追いつかずに血中濃度が上昇し、副作用（めまい、眠気、ふらつき、意識障害など）が強く出現しやすくなります。特に高齢者では、腎機能低下を見落としたまま漫然と投与することで、転倒による骨折リスクが跳ね上がります。

### プレガバリンの主な薬物動態
| 項目 | 特徴 |
| :--- | :--- |
| <strong>代謝</strong> | ほとんど代謝されない |
| <strong>排泄</strong> | <strong>ほぼ100%未変化体が尿中へ</strong> |
| <strong>クリアランス</strong> | 腎クリアランスと全身クリアランスがほぼ一致 |
| <strong>主な副作用</strong> | めまい、傾眠、体重増加、ふらつき |

## 🛠 2. 腎機能別：プレガバリン投与量の目安

実務において、eGFR（またはCcr）を確認する癖をつけることは必須です。以下の表を頭に入れておきましょう。

<div style="overflow-x: auto;">

| 腎機能 (Ccr/eGFR) | 投与量の目安（1日最大用量） |
| :--- | :--- |
| <strong>60 mL/min 以上</strong> | 600 mg（常用量） |
| <strong>30 〜 60 mL/min 未満</strong> | 300 mg 以下 |
| <strong>15 〜 30 mL/min 未満</strong> | 150 mg 以下 |
| <strong>15 mL/min 未満 / 透析</strong> | 75 mg 以下（追加投与が必要） |

</div>

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 実務のポイント：透析患者へのアプローチ</strong><br>
  透析患者さんでは、プレガバリンは透析によって除去されます。そのため、透析後に「補足投与」を行う必要がある点に注意してください。添付文書の「補足投与量」の項目を必ず確認しましょう。
</div>

## ⚠️ 3. 薬剤師が現場でチェックすべき「注意点」

単に用量を守るだけでなく、以下の点も併せて確認することが「選ばれる薬剤師」への一歩です。

### ① 「めまい・ふらつき」のモニタリング
腎機能が保たれていても、高齢者は腎機能が過大評価されていることがあります（シスタチンCでの再評価が有効な場合もあります）。患者さんの様子を注意深く観察し、歩行時のふらつきがないかを聞き取ることが重要です。

### ② eGFRの推移を見る
処方時点のeGFRだけでなく、<strong>「前回値と比較して急激に低下していないか」</strong>を確認してください。特に脱水時や他の腎毒性のある薬剤（NSAIDsなど）を併用開始した際は、腎機能が一時的に悪化し、プレガバリンの血中濃度が急上昇する可能性があります。

### ③ 患者さんへの「服薬指導」
患者さんには、「飲み始めの数日間は、眠気やめまいが出やすいです」としっかり説明しましょう。特に自動車の運転については、重篤な事故を招く可能性があるため、医師と相談して慎重な判断を促す必要があります。

## 🌿 4. まとめ：プレガバリンの安全使用のために

プレガバリンの用量調整は、腎排泄型薬剤としての基本中の基本です。

1.  <strong>腎排泄100%</strong>という動態を理解し、必ず腎機能（eGFR/Ccr）を算出する。
2.  <strong>副作用の出現サイン</strong>（ふらつき、眠気）を早期に見つける。
3.  <strong>透析患者</strong>では補足投与の考え方を忘れない。

「腎臓が悪いから減らす」というルーチンワークではなく、<strong>「この患者さんの腎機能にとって、この用量は安全か？」</strong>と自問自答することで、より専門的な視点で薬物治療に貢献できるようになります。今日の処方箋から、ぜひ腎機能のチェックを習慣にしてみてくださいね！
