# NPUAP分類とは？褥瘡ステージ分類の基本を解説

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  在宅訪問を始めたばかりの頃、訪問看護師さんから「この褥瘡、ステージ2の評価だからアズノールでいいよね？」と尋ねられました。当時の私は「褥瘡＝とりあえずアズノール」という古い知識で「そうですね」と答えてしまいました。<br><br>
  しかし、後日その褥瘡は悪化し、壊死組織が見える状態に。実はその時、すでに深部組織損傷や壊死が進行していたにもかかわらず、分類の知識が曖昧だったために適切な薬剤提案ができず、患者さんに痛い思いをさせてしまったのです。創傷の深さを正確に把握することは、外用薬の選択において「必須のスキル」だと痛感した瞬間でした。
</div>

訪問診療や施設で働く薬剤師にとって、褥瘡の評価は避けて通れません。今回は、世界標準である「NPUAP分類」をベースに、薬剤師が臨床で活用すべき知識を整理します。

---

## 🔬 1. NPUAP分類とは？なぜ重要なのか

NPUAP（全米褥瘡諮問委員会）が作成したこの分類は、<strong>「褥瘡の深達度（どこまで組織が壊れているか）」</strong>をステージ1〜4および深部組織損傷、判定不能に分類したものです。

日本では日本褥瘡学会の「DESIGN-R®」が臨床現場の評価指標として有名ですが、そのベースにはこのNPUAP分類の考え方が根底にあります。創傷の状態を共通言語で評価することで、多職種連携がスムーズになり、<strong>「今、その創傷に何が必要か（洗浄、保湿、保護、壊死組織の除去）」</strong>を正しく判断できるようになります。

### 褥瘡ステージ分類一覧
以下の表で、各ステージの深さと薬剤選択の考え方を確認しましょう。

| ステージ | 状態の特徴 | 薬剤選択の考え方（例） |
| :--- | :--- | :--- |
| <strong>ステージ1</strong> | 発赤（圧迫しても消えない） | 保護・保湿（アズノールなど） |
| <strong>ステージ2</strong> | 真皮までの欠損（水疱含む） | 保護・滲出液管理（ゲーベン等） |
| <strong>ステージ3</strong> | 皮下組織までの欠損 | 壊死組織除去、感染対策 |
| <strong>ステージ4</strong> | 筋肉・骨・腱までの露出 | 専門的なデブリードマンが必要 |
| <strong>深部組織損傷</strong> | 紫・茶色の皮膚（内部の損傷） | 経過観察・除圧・保護 |

## 🛡️ 2. ステージ別・薬剤師が意識すべきポイント

薬剤師として特に意識したいのは、<strong>「ステージに応じた薬剤の使い分け」</strong>です。

### 🔹 ステージ1〜2（軽症期）：皮膚を「守る」
この段階では、皮膚のバリア機能が低下しています。摩擦や圧迫から皮膚を守り、湿潤環境を整えることが優先です。
*   <strong>アズノール軟膏</strong>: 炎症を抑え、皮膚を保護する基本薬。
*   <strong>プロペト</strong>: 物理的な刺激からの保護に最適。

### 🔹 ステージ3〜4（重症期）：組織を「治す」・「感染を防ぐ」
組織欠損が深い場合、感染予防や壊死組織の融解が重要です。
*   <strong>ゲーベンクリーム</strong>: 殺菌作用と壊死組織の融解作用がある。感染兆候がある場合によく使われます。
*   <strong>カデックス軟膏</strong>: 滲出液を吸収し、殺菌作用を発揮します。
*   <strong>イソジンゲル/シュアプラグ</strong>: 感染制御に有効ですが、健康な肉芽を傷つける可能性もあるため注意が必要です。

## ⚠️ 3. 「とりあえずアズノール」を卒業しよう

薬剤師が陥りやすい罠が、<strong>「何となくアズノールを出しておく」</strong>ことです。

<div style="background-color: #fff1f2; border: 1px solid #fecdd3; padding: 1rem; border-radius: 8px; color: #991b1b; margin-top: 1rem;">
  <strong>🚨 注意点：漫然投与は悪化を招く</strong><br>
  ステージ3以上の深い褥瘡に対し、保護目的のみの薬剤を使用し続けると、滲出液が溜まり、細菌感染を助長する恐れがあります。「創の状態が先週から変わっていない」「滲出液が増えている」と感じたら、必ずステージ評価の再検討（疑義照会）を行いましょう。
</div>

## 🎯 4. 実務で役立つ「観察のヒント」

訪問現場で褥瘡を観察する際、以下のポイントをチェックリストとして活用してみてください。

1.  <strong>色</strong>: 赤い（肉芽良好）のか、黒い（壊死）のか、黄色（フィブリン・膿）なのか。
2.  <strong>浸出液</strong>: 量は多いか？臭いはあるか？（感染のサイン）
3.  <strong>周囲皮膚</strong>: 浸軟（ふやけている）していないか？
4.  <strong>痛み</strong>: 患者さんの表情や訴えに変化はあるか？

## 🌿 5. まとめ：薬剤師も「創傷ケア」のチームの一員

NPUAP分類を理解することで、単に「薬を渡す」だけでなく、<strong>「この薬剤はこのステージの褥瘡に適切か？」</strong>という視点を持つことができます。

医師や看護師に「ステージ3の状態ですので、そろそろゲーベンなどの抗菌剤に変更しませんか？」と提案できるようになれば、患者さんの予後を大きく改善できるはずです。褥瘡ケアはチーム医療。ぜひ今日から、処方箋の中身だけでなく、その先の「皮膚の状態」にも目を向けてみてくださいね！
