# OHSS重症度分類とは？卵巣過剰刺激症候群の評価基準を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の薬局で、体外受精の誘発周期にある患者さんから「最近、お腹がパンパンに張って苦しいし、体重が急に増えた気がする…」と相談を受けました。私はその時、単なる便秘や消化不良かな？と軽く考え、「整腸剤を出しておきましょうか？」と答えてしまいました。<br><br>
  その後、その患者さんが緊急入院されたという報告を受け、愕然としました。診断名は「重症OHSS（卵巣過剰刺激症候群）」でした。「急激な体重増加」と「腹部膨満感」という、OHSSの重要なサインを見逃していたのです。もっと早くに重症度を意識し、主治医へ報告していればと深く反省しました。

不妊治療に携わる薬剤師にとって、<strong>OHSS（卵巣過剰刺激症候群）</strong>のリスク管理は非常に重要です。今回は、OHSSの評価基準と薬剤師がチェックすべきポイントを詳しく解説します。

---

## 📊 1. OHSS重症度分類：症状と評価の基準

OHSSは、卵巣刺激により卵巣が腫大し、血管透過性が亢進することで腹水や胸水が貯留する疾患です。重症度分類は以下のように整理されます。

| 重症度 | 主な臨床所見 |
| :--- | :--- |
| <strong>軽症</strong> | 下腹部膨満感、軽度の不快感、卵巣腫大（5cm程度） |
| <strong>中等症</strong> | 悪心・嘔吐、下痢、超音波での腹水貯留、卵巣腫大（5-10cm） |
| <strong>重症</strong> | 腹水・胸水による呼吸困難、乏尿、血液濃縮、血栓症リスク、卵巣腫大（10cm以上） |

<div style="background-color: #fef2f2; border: 1px solid #fecaca; padding: 1rem; border-radius: 8px; color: #991b1b; margin: 1.5rem 0;">
  <strong>⚠️ 薬剤師が注意すべき緊急サイン</strong><br>
  ・<strong>急激な体重増加</strong>（1日1kg以上、あるいは短期間の極端な増加）<br>
  ・<strong>腹囲の拡大</strong>（患者さんがベルトが閉まらないと訴えるなど）<br>
  ・<strong>尿量の減少</strong>（腎機能障害の兆候）<br>
  ・<strong>激しい腹痛・呼吸苦</strong>（即座に専門医への受診を促してください）
</div>

## 💊 2. 薬剤師が関わるOHSS治療薬の管理

OHSSの治療において薬剤師ができることは、症状のモニタリングと適切な服薬指導です。

### 主な治療薬一覧

<div style="overflow-x: auto;">

| 薬剤の種類 | 代表薬例 | 薬剤師の確認ポイント |
| :--- | :--- | :--- |
| <strong>抗凝固薬</strong> | ヘパリン、アスピリン | 重症化に伴う血栓症予防。内出血や紫斑の有無を確認。 |
| <strong>利尿薬</strong> | フロセミド等 | 尿量減少や乏尿時の管理。電解質バランスのモニタリング。 |
| <strong>ドーパミン受容体作動薬</strong> | カベルゴリン | 血管透過性を抑制。副作用のめまいや嘔気を確認。 |
| <strong>ヒト絨毛性性腺刺激ホルモン</strong> | hCG製剤（トリガー） | OHSSの主要な発症因子。投与後の経過観察を慎重に行う。 |

</div>

## 🎯 3. 服薬指導の現場で「聴く」べきこと

患者さんは不妊治療のストレスから、体調の不調を「我慢してしまう」傾向があります。薬局での窓口業務では、以下の点を意識的に確認してみてください。

1. <strong>「体重の増え方はどうですか？」</strong>
   - 「最近急に太った」という訴えは、脂肪ではなく腹水の可能性があります。
2. <strong>「おしっこはしっかり出ていますか？」</strong>
   - 尿量の減少は重症化の指標です。回数や色を確認しましょう。
3. <strong>「呼吸のしにくさや、足のむくみはありますか？」</strong>
   - 血栓症（DVT）や胸水の貯留を示唆するサインです。

## 💡 まとめ：薬剤師が防ぐ「重症化」

OHSSは進行が早く、軽症から重症へあっという間に変化することもあります。

*   <strong>知識のアップデート</strong>: 不妊治療薬（トリガーとなるhCGなど）の特性を理解する。
*   <strong>サインの見極め</strong>: 「なんとなく苦しい」という患者さんの声を、OHSSの初期症状と結びつける。
*   <strong>連携の強化</strong>: わずかでも異常を感じたら、迷わず不妊クリニックの主治医へ疑義照会・情報提供を行う。

患者さんの「なんとなく体調が悪い」という言葉の裏にあるサインを拾い上げることが、私たちの重要な責務です。明日からの服薬指導で、ぜひこれらを確認してみてくださいね。
