# 🧠 Brunnstrom stageとは？脳卒中後の運動機能回復段階を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験：服薬指導のヒント】</strong><br>
  以前、脳卒中後の患者さんの外来処方箋で「痙縮（けいしゅく）に対する薬」が増えた際のことです。患者さんに「手足の動かしやすさはどうですか？」と伺うと、ご家族が「最近、少しずつ指が動くようになってきて、リハビリの先生も喜んでいるんです」と仰いました。<br><br>
  その時、カルテにある「Brunnstrom stage（以下、Brs）がIIからIIIへ」という記録に気づきました。この評価指標の意味を知っていたことで、単なる「筋弛緩薬の処方」としてだけでなく、「回復の過程で出現した過剰な緊張を和らげる」という、より患者さんの背景に寄り添った服薬指導ができた経験があります。

脳卒中後の患者さんと接する際、リハビリの進捗を理解することは、服薬の必要性や副作用（特に運動機能に関連するもの）を評価するうえで非常に役立ちます。今回は、リハビリの指標として欠かせない<strong>Brunnstrom stage</strong>について深掘りしていきましょう。

---

## 📈 1. Brunnstrom stage（ブルンストロームステージ）とは？

Brunnstrom stageは、脳卒中後の片麻痺患者において、<strong>「運動機能がどの程度回復しているか」を6段階で評価する指標</strong>です。

このステージを知ることで、患者さんが現在「どの程度自分の意思で手足をコントロールできるか」「どの段階で痙縮が出現しやすいか」というリハビリの到達目標を把握することができます。

### 回復の6段階（概要）

| ステージ | 状態の特徴 | 回復の目安 |
| :--- | :--- | :--- |
| <strong>ステージ I</strong> | 弛緩（全く動かない） | 発症直後 |
| <strong>ステージ II</strong> | 共同運動の出現（わずかに動く） | 痙縮が始まる |
| <strong>ステージ III</strong> | 共同運動が顕著（硬く固まりやすい） | 痙縮が強まる |
| <strong>ステージ IV</strong> | 共同運動からの分離（少し自由に動く） | 痙縮が減り始める |
| <strong>ステージ V</strong> | 単独運動の増加（さらに滑らかに） | 痙縮がかなり減る |
| <strong>ステージ VI</strong> | 正常に近い機能 | 回復の最終段階 |

## 🔍 2. なぜ薬剤師がこの段階を知るべきなのか？

リハビリの専門家ではありませんが、薬剤師がBrsを意識すべき理由は、<strong>「痙縮治療薬（抗痙縮薬）」の処方意図を理解できるから</strong>です。

### 痙縮治療薬との関わり
脳卒中後の回復過程では、脳からの指令がうまく伝わらず、筋肉が過剰に緊張する「痙縮」が起こります。特に<strong>ステージIII</strong>では、無意識に手足が固まってしまう（共同運動）ため、痛みや拘縮（こうしゅく）のリスクが高まります。

## 💊 3. 痙縮治療によく使われる薬剤の視点

Brsを考慮しながら、臨床でよく目にする抗痙縮薬のポイントを整理しました。

| 薬剤名（一般名） | 特徴・使いどころ | 薬剤師の確認ポイント |
| :--- | :--- | :--- |
| <strong>ダントロレン</strong> | 筋肉に直接作用する | 肝機能障害に注意。初期の強い拘縮に。 |
| <strong>バクロフェン</strong> | 中枢神経に作用する | 眠気、ふらつきに注意。ステージIII頃の服薬。 |
| <strong>チザニジン</strong> | 脊髄への抑制作用 | 血圧低下に注意。リハビリ前の眠気に配慮。 |

## ⚠️ 4. 薬剤師としての「攻め」の服薬指導

患者さんやご家族への指導では、薬の知識だけでなく「回復の段階」に合わせた言葉選びを心がけましょう。

1. <strong>「頑張りすぎ」への配慮</strong>: 回復段階（IV〜V）にある患者さんに対し、「リハビリを頑張りたい」という意欲がある場合、抗痙縮薬の眠気がブレーキになっていないか確認する。
2. <strong>「麻痺の痛み」の確認</strong>: 「痛くて動かせない」という訴えは、ステージIII特有の緊張によるものかもしれません。医師へのフィードバック（「痙縮が強そうでリハビリの阻害因子になっている可能性があります」）を行うことで、多職種連携を深めることができます。
3. <strong>副作用の早期発見</strong>: 特にふらつきは転倒リスクに直結します。Brsのステージを把握していれば、「今、回復の途中で歩行訓練を頑張っている」という情報から、より注意深く歩行状態の変化を聞き出すことができます。

## 🌿 5. まとめ：回復過程を見守る伴走者として

Brunnstrom stageは、単なる医学的な分類ではなく、<strong>「患者さんが今、どのような体と心の状態でリハビリと向き合っているか」を示す道しるべ</strong>です。

薬剤師がこのステージを意識することで、処方薬が「ただの筋弛緩薬」から「患者さんの機能回復をサポートするためのツール」に変わります。

「最近、服薬後に少しリハビリがやりやすくなりましたか？」
この一言が、患者さんのモチベーションを引き出し、より深い信頼関係を築くきっかけになるはずです。ぜひ、次回の外来や調剤の際に、カルテのBrsチェックを習慣にしてみてくださいね。
