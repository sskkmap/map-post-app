# 災害時に薬がなくなったらどうする？

<div style="background-color: #fff7ed; border-left: 4px solid #f97316; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場からのエピソード：あの日、患者さんが持ってきたのは…】</strong><br>
  ある地方で震災が起きた際、避難所から徒歩で薬局へ駆け込んできた高齢の患者さんがいました。家は被災し、薬はすべて流されてしまったとのこと。「血圧の薬がないと不安で…」と震える手で渡されたのは、奇跡的にポケットに入っていた古い薬のシートの切れ端と、数年前の領収書だけでした。<br><br>
  当時、電子カルテのシステムも停電で動かず、患者さんの服用歴を照会する術がありませんでした。その時、私がとっさに行った対応と、その後の教訓を振り返り、皆さんに「今、患者さんに伝えておくべきこと」を整理したいと思います。
</div>

災害大国である日本において、薬局業務のBCP（事業継続計画）や災害時の服薬支援は避けて通れないテーマです。本稿では、災害発生時に患者さんから相談を受けた際の、薬剤師として取るべき実務的なアクションを解説します。

---

## 📋 1. 災害時に活用できる「特例措置」の知識

災害時、平時のような厳格な処方箋発行が困難な状況下では、厚生労働省から通知される「特例措置」が発動されます。

<div style="overflow-x: auto;">
| 項目 | 内容とポイント |
| :--- | :--- |
| <strong>お薬手帳の提示</strong> | お薬手帳が実質的な「処方箋代わり」になります。電子お薬手帳ならスマホで見せれば即座に把握可能。 |
| <strong>無処方箋交付</strong> | 処方箋なしでも、お薬手帳等で服用内容が確認できれば、薬剤師の判断で最小限の日数分を調剤・交付が可能。 |
| <strong>レセプトの省略</strong> | 緊急時はカルテ作成やレセプト請求が事後対応となるため、まずは「薬を渡すこと」が優先されます。 |
| <strong>公費負担</strong> | 災害救助法が適用される地域では、窓口負担が免除される場合があります。 |
</div>

※これらの措置は自治体からの通知によって運用が異なります。まずは<strong>「お薬手帳を確認する」</strong>ことが、薬剤師が最初に行う最大の貢献です。

---

## 🆘 2. 患者さんへ指導すべき「3つの備え」

患者さんから「地震が怖い」「薬がなくなったらどうしよう」と相談されたら、以下の3つを具体的にアドバイスしましょう。

### ① 「デジタルお薬手帳」の導入
紙の手帳は濡れたり紛失したりするリスクがあります。災害時こそ、スマホでデータが同期される「電子お薬手帳」が圧倒的に強いです。

### ② 常備薬プラスαの「災害セット」
普段飲んでいる薬の「プラス3日分」を常に予備として確保するよう指導します。
- 診察券のコピー
- お薬手帳のコピー（または最新ページのスマホ写真）
- 常用薬のメモ（成分名、規格、用法）

### ③ 「自分の薬を知る」ことの重要性
意外と多いのが「血圧の薬」としか認識しておらず、成分名を知らないケースです。「自分の病気は何か」「どの薬を飲んでいるか」を誰かに言えるようにしておくことが、避難所でのトリアージにおいて命を分けます。

システムが停止した際、あるいは患者さんの手帳がない場合、薬剤師は何を頼りにすべきでしょうか。

    <li><b>公的データベース：</b> 厚労省の災害時対応マニュアル（「被災時における薬局等の対応について」）をブックマークしておく。</li>
  </ul>

## 🌿 4. 薬剤師として、日常からできること

災害時に「あ、この人なら助けてくれる」と思われる薬局であるために、今日からできることを整理しました。

1. <strong>お薬手帳の積極活用：</strong> 毎回必ず手帳を見せてもらう習慣をつけ、アプリ利用を推奨する。
2. <strong>患者への防災意識の啓発：</strong> 「最近、防災バッグに薬は入れましたか？」という一声をカウンセリングに混ぜる。
3. <strong>地域薬局との連携：</strong> 災害時の連絡網や、BCPに基づいた役割分担を地域の薬剤師会で確認しておく。
