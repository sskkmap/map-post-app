## 🔍 J-HAQとはどのような指標か？

<strong>J-HAQ（Japanese Health Assessment Questionnaire）</strong>は、関節リウマチ患者さんの「身体機能」を評価するための質問票です。

簡単に言うと、<strong>「日常生活でどれくらい困りごとがあるか」を数値化したもの</strong>です。関節リウマチは関節の腫れや痛みだけでなく、日常生活動作（ADL）が制限されることが大きな問題となります。

### J-HAQの評価項目
J-HAQは主に以下の8つのカテゴリーで構成されています。

| カテゴリー | 具体的な評価内容 |
| :--- | :--- |
| <strong>更衣・整容</strong> | 服の着脱や髪をとかす動作 |
| <strong>立ち上がり</strong> | 椅子からの立ち上がり動作 |
| <strong>食事</strong> | お肉を切る、コップを持つなどの動作 |
| <strong>歩行</strong> | 平坦な場所での歩行動作 |
| <strong>衛生</strong> | お風呂に入る、トイレでの動作 |
| <strong>リーチ</strong> | 棚の上の物を取る動作 |
| <strong>グリップ</strong> | 瓶の蓋を開ける、ドアノブを回す動作 |
| <strong>その他</strong> | 買い物、家事、車の乗り降りなど |

## 📊 スコアの見方と薬剤師が知るべきポイント

### なぜ薬剤師がこのスコアを気にする必要があるのか？

1.  <strong>服薬アドヒアランスへの影響</strong>: 例えば「グリップ（瓶を開ける）」のスコアが悪ければ、PTPシートからの薬の取り出しや、ヒンジキャップの開封が困難である可能性があります。
2.  <strong>治療目標の再確認</strong>: 検査値が良くてもJ-HAQが改善しない場合、関節破壊が既に進行しているか、他の合併症が影響している可能性を疑うきっかけになります。
3.  <strong>多職種連携</strong>: スコアが悪化している場合、医師への報告だけでなく、理学療法士へのリハビリ依頼や、福祉用具導入の検討を促す架け橋になれます。

## 💡 実践！J-HAQスコアを活かした服薬指導

患者さんのスコアが高い（動きづらい）と分かっている場合、次のような具体的な提案が現場で活きます。

*   <strong>PTPシートの工夫</strong>: 「シートから薬を出しにくい」という訴えがある場合、一包化の提案だけでなく、シートの角をあらかじめ切っておく、または指の力がいらないピルカッターの導入を検討します。
*   <strong>薬剤の剤形変更</strong>: 指先のこわばりが強い患者さんには、大きな錠剤よりも、嚥下しやすく取り出しやすいOD錠への変更や、塗り薬であれば容器の蓋が開けやすいワンタッチキャップタイプへの切り替えを医師に提案します。
*   <strong>生活環境の確認</strong>: 「お風呂で困っている」とあれば、薬剤の影響（めまい等の副作用）だけでなく、関節の可動域制限による転倒リスクを視野に入れた指導が必要です。

## 📝 まとめ：検査値だけでなく「患者さんの生活」を診る

J-HAQは、関節リウマチ治療において<strong>「患者さんがどれくらい自分らしく生活できているか」</strong>を可視化する大切な指標です。

薬局のカウンターで検査値を眺めるだけではなく、「最近、ボタンを留めるのはどうですか？」「ペットボトルの蓋は開けられますか？」といった、J-HAQの項目を意識した会話を心がけてみてください。その何気ない一言が、患者さんのQOL（生活の質）を大きく引き上げる一助となるはずです。
