# Surgical Staging Systemとは？がんの外科的病期分類を解説

## 🔍 1. がんの「ステージ」を決める3つの評価タイミング

がんの病期（ステージ）は、いつ、どのような手段で評価したかによって大きく3つに分類されます。Surgical Staging Systemは、この中の「術後」の評価に該当します。

一般的に「Surgical Staging System」と言う場合、手術によって得られた知見（sStage）と、その後の病理検査結果（pStage）を統合して、<strong>最終的な治療方針を決定するための詳細な分類</strong>を指します。

---

## 🔬 2. TNM分類とSurgical Stagingの関係

世界的に最も使われているステージ分類法は「TNM分類」ですが、これも評価タイミングによって頭にアルファベットがつきます。

| 略称 | 正式名称 | 薬剤師が注目すべきポイント |
| :--- | :--- | :--- |
| <strong>cTNM</strong> | Clinical TNM | 術前化学療法（NAC）の適応判断に使用。 |
| <strong>pTNM</strong> | Pathological TNM | <strong>術後補助化学療法（Adjuvant）</strong>の必要性を判断する。 |
| <strong>ycTNM</strong> | Post-neoadjuvant | 術前化学療法「後」の評価。薬の効き具合を反映。 |

特に婦人科がん（子宮体がんや卵巣がんなど）では、<strong>FIGO分類</strong>という独自のSurgical Staging Systemが用いられます。これらは「手術進行期分類」とも呼ばれ、手術で見つかった腹水中の細胞診結果などがステージ決定に大きく関わります。

---

## 💊 3. なぜ薬剤師が「外科的病期」を把握すべきなのか？

「ステージが決まるのは医師の仕事でしょ？」と思うかもしれません。しかし、私たち薬剤師がSurgical Stagingの結果（特にpStage）を正しく把握することは、<strong>安全で適切な薬物療法（術後補助化学療法）</strong>を行う上で不可欠です。

### 🚨 理由①：術後補助化学療法の適応が変わる
例えば、大腸がんの場合、術前の画像診断（cStage）でStage IIであっても、手術でリンパ腺を詳しく調べた結果、転移が見つかりStage III（pStage）へ「アップステージ」することがあります。
この場合、再発リスクが高まるため、<strong>術後補助化学療法（XELOX療法や5-FU/LV療法など）</strong>が強く推奨されることになります。

### 🚨 理由②：レジメン選択と投与期間の監査
がん種によっては、ステージによって推奨されるレジメンや投与期間が異なります。
- <strong>Stage II（ハイリスク群のみ）</strong>: 単剤治療で様子を見る
- <strong>Stage III</strong>: 強力な多剤併用療法を行う
このように、カルテの「病理結果報告書」を確認し、現在のレジメンが最新のSurgical Stagingに基づいているかを確認するのは、薬剤師の高度な専門性が問われる部分です。

---

## 📊 4. 臨床病期(c)と外科的・病理学的病期(p)の比較

| 項目 | 臨床病期 (cStage) | 外科的・病理学的病期 (pStage) |
| :--- | :--- | :--- |
| <strong>評価手段</strong> | 画像診断（CT/MRI）、生検、触診 | 手術所見、切除組織の病理検査 |
| <strong>確定時期</strong> | 治療開始前（術前） | 手術終了後（通常2〜4週間後） |
| <strong>主な用途</strong> | 術式の決定、術前化学療法の検討 | <strong>術後補助化学療法の決定、予後予測</strong> |
| <strong>精度</strong> | 推定値（画像には限界がある） | <strong>確定値（ゴールドスタンダード）</strong> |

## 🌿 6. まとめ：正確なステージ把握が「個別化医療」の第一歩

Surgical Staging Systemは、単にがんの大きさを測るものではなく、<strong>「これから患者さんにどのような追加治療が必要か」を導き出すためのコンパス</strong>です。

1. <strong>cStage（術前）</strong>で大まかな方針を立てる
2. <strong>sStage/pStage（術後）</strong>で最終的なリスクを評価する
3. 薬剤師がその結果を基に、<strong>術後補助化学療法</strong>の内容を監査する

この流れを意識するだけで、カルテの見方がガラリと変わるはずです。

もし、化学療法の処方箋が回ってきたら、ぜひ一度術前のステージと術後の「確定ステージ」を比較してみてください。そこには、チーム医療の中で薬剤師が果たすべき「適正使用の推進」という重要な役割が隠されています！
