# 🔥 Artzの基準ってなに？薬剤師が知っておくべき熱傷重症度の評価法

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の「はっ」とした経験】</strong><br>
  ある日の夕方、近所の皮膚科から「ゲーベンクリーム」と「ソリタ-T3号」の処方箋を持った患者さんがいらっしゃいました。見ると、腕から胸にかけて広範囲に包帯が巻かれています。<br><br>
  「ひどいやけどだな…」とは思ったものの、その時の私は外用薬の在庫確認に必死で、患者さんの「全身状態」への意識が完全に抜け落ちていました。後で先輩から「その面積ならArtzの基準で『重症』かもしれないし、脱水や感染症のリスクを服薬指導に盛り込むべきだったね」と指摘され、自分のアセスメント不足を痛感したのです。
</div>

「やけど＝外用薬を塗るだけ」と考えてしまいがちですが、広範囲の熱傷は全身管理が必要な「急性疾患」です。今回は、医療現場で最も広く使われている<strong>熱傷重症度判定「Artz（アーツ）の基準」</strong>について、薬剤師の視点で分かりやすくまとめていきます！

---

## 🧐 1. 熱傷の重症度を左右する「3つの要素」

熱傷の重症度は、単に「痛そうかどうか」ではなく、以下の3つの要素を組み合わせて客観的に評価します。

  <div style="background-color: #fff1f2; padding: 1rem; border-radius: 8px; border: 1px solid #fecdd3;">
    <div style="font-weight: bold; color: #9f1239; margin-bottom: 0.5rem; border-bottom: 2px solid #fecdd3; padding-bottom: 0.25rem;">① 熱傷深度</div>
    <div style="color: #be123c; font-size: 0.85em;">どのくらいの深さまで組織が損傷しているか（I度〜III度）</div>
  </div>
  <div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe;">
    <div style="font-weight: bold; color: #1e3a8a; margin-bottom: 0.5rem; border-bottom: 2px solid #bfdbfe; padding-bottom: 0.25rem;">② 熱傷面積</div>
    <div style="color: #1e40af; font-size: 0.85em;">体表面積の何％に熱傷が及んでいるか（TBSA）</div>
  </div>
  <div style="background-color: #f0fdf4; padding: 1rem; border-radius: 8px; border: 1px solid #bbf7d0;">
    <div style="font-weight: bold; color: #166534; margin-bottom: 0.5rem; border-bottom: 2px solid #bbf7d0; padding-bottom: 0.25rem;">③ 合併症・部位</div>
    <div style="color: #14532d; font-size: 0.85em;">吸入熱傷の有無や、顔面・手足・会陰部などの特殊部位か</div>
  </div>
</div>

薬剤師として特に意識したいのが、<strong>「熱傷面積（TBSA）」</strong>です。面積が広ければ広いほど、皮膚からの水分喪失が激しくなり、輸液療法の必要性が高まるからです。

## 📏 2. 面積を算出するための「9の法則」と「5の法則」

Artzの基準を適用する前に、まずは熱傷面積を算出する必要があります。現場でよく使われる手法を整理しましょう。

### 🖐️ ① 9の法則（成人）
成人の体を「9%」の倍数で区切って計算する方法です。

| 部位 | 面積の割合（成人） |
| :--- | :--- |
| <strong>頭部・頸部</strong> | 9% |
| <strong>上肢（片腕全体）</strong> | 9% （両腕で18%） |
| <strong>下肢（片脚全体）</strong> | 18% （両脚で36%） |
| <strong>体幹（前面）</strong> | 18% |
| <strong>体幹（背面）</strong> | 18% |
| <strong>会陰部</strong> | 1% |

### 👶 ② 5の法則（小児）
小児は頭が大きく足が短いため、数値が変わります。

| 部位 | 面積の割合（乳幼児） |
| :--- | :--- |
| <strong>頭部・頸部</strong> | 20% |
| <strong>体幹（前面・背面各）</strong> | 20% （計40%） |
| <strong>上肢（片腕全体）</strong> | 10% （両腕で20%） |
| <strong>下肢（片脚全体）</strong> | 10% （両脚で20%） |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin-top: 1rem;">
  <strong>💡 豆知識：手掌法（しゅしょうほう）</strong><br>
  患者さんの「指を閉じた手のひら全体」の面積を、体表面積の<strong>約1%</strong>として計算する方法です。不整形な熱傷や、小さな熱傷が点在している場合に非常に便利です。
</div>

## 📊 3. 本題：Artzの基準（Artz's Criteria）

Artzの基準は、これら「深度」と「面積」を組み合わせて、<strong>軽症・中等症・重症（入院/転送が必要）</strong>を判定する指標です。

### 【Artzの基準 判定一覧】

| 判定 | 熱傷の状態（条件のいずれかに該当） | 対応の目安 |
| :--- | :--- | :--- |
| <strong>重症 (Major)</strong> | ・II度熱傷が30%以上<br>・III度熱傷が10%以上<br>・顔面、手、足、会陰部のIII度熱傷<br>・吸入熱傷、複雑な合併症（骨折等） | <strong>即入院・高度救急センター</strong><br>積極的な輸液と全身管理 |
| <strong>中等症 (Moderate)</strong> | ・II度熱傷が15〜30%<br>・III度熱傷が10%未満（顔面、手、足、会陰部を除く） | <strong>入院加療が望ましい</strong><br>感染予防と水分補給の徹底 |
| <strong>軽症 (Minor)</strong> | ・II度熱傷が15%未満<br>・III度熱傷が2%未満 | <strong>外来通院で可能</strong><br>適切な外用療法と経過観察 |

## 🔢 4. もう一つの重要指標「バーンインデックス (BI)」

日本の熱傷診療でArtzの基準と併せて頻用されるのが、<strong>バーンインデックス（Burn Index）</strong>です。これは「III度熱傷」の重みを2倍にして計算する指標です。

<div style="background-color: #f1f5f9; padding: 1.5rem; border-radius: 12px; border: 1px solid #cbd5e1; text-align: center; margin: 1.5rem 0;">
  <span style="font-size: 1.2rem; font-weight: bold; color: #334155;">バーンインデックス (BI) の計算式</span><br>
  <div style="font-size: 1.5rem; margin-top: 1rem; color: #1e293b;">
    <strong>BI ＝ (II度熱傷面積 × 1/2) ＋ (III度熱傷面積)</strong>
  </div>
</div>

*   <strong>BI 10〜15以上</strong>：重症と判定され、生命の危険性が高まります。
*   <strong>BI ＝ 年齢 ＋ BI</strong> が100を超える場合（年齢加算BI）、予後不良とされます。

## 💊 5. 薬剤師が実務で活かすポイント

処方箋から熱傷の重症度を推測できれば、服薬指導の質がグッと上がります。

### ① 外用薬の選択意図を読み取る
重症度や受傷からの経過（急性期か、肉芽形成期か）によって、選択される外用薬が変わります。

| 薬剤名 | 特徴と使用タイミング |
| :--- | :--- |
| <strong>ゲーベンクリーム</strong> | 強力な殺菌作用。水分含有量が多く、感染リスクの高い深い熱傷（II度〜III度）の初期によく使われます。※銀アレルギー注意 |
| <strong>アズノール軟膏</strong> | 消炎作用がメイン。比較的浅い熱傷（I度〜浅在性II度）や、治りかけの時期に。 |
| <strong>プロスタンディン軟膏</strong> | 血流改善作用。壊死組織の分離や肉芽形成を促進したい時期に。 |
| <strong>バラマイシン軟膏</strong> | 抗生物質配合。二次感染の予防・治療に。 |

### ② 全身状態への目配り
Artzの基準で「中等症」以上に該当しそうな広範囲熱傷の場合、外用薬だけでなく以下の点を確認しましょう。

*   <strong>脱水のサイン</strong>: 「尿はしっかり出ていますか？」「口の渇きはどうですか？」といった確認。
*   <strong>感染症の徴候</strong>: 患部の悪臭、膿、急な発熱がないか。
*   <strong>栄養状態</strong>: 皮膚の再生にはタンパク質やビタミンが必要です。食事が摂れているか確認を。

## 🌿 6. まとめ：基準を知れば「薬」の先が見える

熱傷の処方箋が来たとき、私たちはつい「どの軟膏をどれくらい混ぜるか」「在庫はあるか」に目を奪われがちです。

しかし、<strong>Artzの基準</strong>や<strong>バーンインデックス</strong>という物差しを持っておくことで、「この面積なら全身性の脱水に注意が必要だな」「この部位なら瘢痕拘縮（ひきつれ）のリスクがあるからリハビリの話が出るかも」といった、一歩進んだアセスメントが可能になります。

<div style="background-color: #fef2f2; border: 1px solid #fca5a5; padding: 1rem; border-radius: 8px; color: #991b1b; margin-top: 1rem;">
  <strong>🚨 薬剤師の注意点！</strong><br>
  もし外来で「明らかに広範囲（成人の手のひら15枚分以上など）」の熱傷患者さんが、外用薬のみの処方でふらつきながら来局された場合は、脱水によるショックの可能性も否定できません。速やかに疑義照会や受診勧奨を検討する勇気も必要です。
</div>

患者さんの皮膚だけでなく、「全身」を支えられる薬剤師を目指して、これらの基準をぜひ日々の業務に役立ててくださいね！
