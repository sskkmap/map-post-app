# Fisher分類とは？くも膜下出血後の血管攣縮リスク評価

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  脳神経外科病棟で働き始めた当初、ある患者さんのカルテに「Fisher 3」と書かれているのを見ました。当時の私は「脳出血の重症度かな？」程度にしか考えておらず、指示されていたオザグレルナトリウムの投与がなぜ数日間続くのか、その意味を深く理解していませんでした。<br><br>
  ある日、当直の医師から「Fisher分類が高いから血管攣縮に注意して」と言われましたが、正確なリスク評価と薬剤の関連を答えられず、非常に悔しい思いをしました。血管攣縮という「見えない敵」に対して、薬剤師がどう構えるべきか、その基本がこの「Fisher分類」に凝縮されていたのです。
</div>

くも膜下出血（SAH）後の患者さんを担当する際、必ず遭遇するのが<strong>「血管攣縮（けっかんれんしゅく）」</strong>への対策です。今回は、その予測指標であるFisher分類について、臨床現場で役立つエッセンスを整理します。

---

## 🧠 1. Fisher分類とは何か？

Fisher分類は、CT画像上で確認される<strong>「くも膜下出血の血液の広がり方」</strong>から、その後に発生する血管攣縮のリスクを評価するスコアリングシステムです。

出血が広範囲であるほど、周囲の血管に影響を与え、攣縮（血管が細くなって血流が滞る状態）を起こすリスクが高まります。

| 分類 | CT上の所見 | 血管攣縮リスク |
| :--- | :--- | :--- |
| <strong>Group 1</strong> | 出血を認めない | 低い |
| <strong>Group 2</strong> | 厚さ1mm未満の薄い出血 | 低い |
| <strong>Group 3</strong> | 厚さ1mm以上の濃い出血（局所） | <strong>非常に高い</strong> |
| <strong>Group 4</strong> | 脳室内出血や脳実質内血腫を伴う | <strong>高い</strong> |

※近年では、より詳細な評価が可能な「modified Fisher scale」も併用されますが、基本の考え方は「血腫の量が多い＝リスクが高い」という点に変わりありません。

---

## 💊 2. 薬剤師が知るべき血管攣縮予防の「基本戦略」

Fisher分類でリスクが高いと判断された場合、薬物療法による予防が強化されます。薬剤師としてチェックすべき主な薬剤は以下の通りです。

### 血管攣縮予防の主要薬剤一覧

| 薬剤名 | 作用機序 | 投与のポイント |
| :--- | :--- | :--- |
| <strong>オザグレルナトリウム</strong> | トロンボキサンA2合成酵素阻害（血小板凝集抑制・血管拡張） | 持続点滴が基本。血小板数に注意。 |
| <strong>ファスジル塩酸塩</strong> | Rhoキナーゼ阻害（血管平滑筋の収縮抑制） | 血管拡張作用による血圧低下に注意。 |
| <strong>ニカルジピン</strong> | カルシウム拮抗薬（脳血管選択的拡張） | 降圧を目的とせず、攣縮抑制目的に使用する場合がある。 |

<div style="background-color: #f0fdf4; padding: 1.5rem; border-radius: 8px; border: 1px solid #bbf7d0;">
  <strong>💡 薬剤師の実践アドバイス</strong><br>
  Fisher 3以上の患者さんでは、これらの薬剤が「予防」としてルーチンで開始されます。重要なのは、<strong>「これらの薬剤はあくまで予防であり、血管攣縮の発症そのものを完全に止める魔法ではない」</strong>という点です。<br><br>
  もし患者さんが「なんとなくぼんやりしている」「昨日までできていた会話がスムーズでない」といった変化を見せた場合、血管攣縮による脳虚血のサインかもしれません。カルテのFisher分類を再確認し、医師へ報告する意識を持つことが重要です。
</div>

## ⚠️ 3. 臨床現場での注意点：血圧管理とのバランス

血管攣縮のリスクが高い時期（発症後4〜14日目が多い）には、血流を維持するために、あえて血圧を少し高めに保つ<strong>「HHH療法（Hypervolemia, Hypertension, Hemodilution）」</strong>という考え方がとられることがあります。

*   <strong>ジレンマ</strong>: 降圧薬を漫然と投与しすぎると、脳への血流が減少し、虚血を悪化させるリスクがあります。
*   <strong>薬剤師の役割</strong>: 降圧薬のオーダーが入った際、単に「血圧が高いから下げる」のではなく、「今、血管攣縮のハイリスク期間ではないか？」「医師の目標血圧設定はどれくらいか？」を意識し、必要に応じて疑義照会を行うことが、患者さんの神経学的な予後を守ることにつながります。

## 🌿 4. まとめ：データと目の前の患者をつなぐ

Fisher分類は、単なる医学的なデータではなく、<strong>「今日、どの薬剤を優先して確認し、どんな患者変化に注意すべきか」</strong>を教えてくれるガイドマップです。

1.  <strong>カルテ確認</strong>: 脳外科の患者さんを見たら、まずFisher分類を確認する。
2.  <strong>薬剤評価</strong>: ハイリスク（Group 3, 4）なら、血管拡張薬の投与状況を注視する。
3.  <strong>観察</strong>: 神経学的所見の変化（意識レベル、麻痺の増悪）があれば、血管攣縮を疑いチームに共有する。

この視点を持つだけで、薬剤師としての病棟での存在感は大きく変わります。ぜひ、明日の臨床から意識してみてくださいね。
