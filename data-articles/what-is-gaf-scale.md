# 🧠 GAFとは？全般的機能評価尺度の点数の意味を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  精神科の外来調剤に従事していた頃のことです。ある患者さんの処方内容が、「症状は安定しているはずなのに、どんどん薬の量が増えている」というケースがありました。カルテには「GAF 45」との記載。<br><br>
  当時の私は「GAFって何かの検査値かな？」程度にしか考えていませんでした。しかし、後にGAFが「社会的機能の指標」であることを知り、点数が低いということは「薬が効いているようで、実は日常生活を送る能力（機能）が維持できていない」サインかもしれない、という視点を持つようになりました。薬の効き目だけでなく、患者さんの「生活の質」を測る視点が欠けていたことに気づかされた出来事です。
</div>

精神科領域の処方箋を扱う際、カルテに記載された「GAF」というアルファベットを目にしたことはありませんか？今回は、精神科における重要指標である<strong>GAF尺度</strong>について、薬剤師が知っておくべきポイントを解説します。

---

## 🔍 1. GAF尺度（全般的機能評価尺度）とは何か？

<strong>GAF（Global Assessment of Functioning：全般的機能評価尺度）</strong>とは、DSM-IV（精神疾患の診断・統計マニュアル第4版）で採用されていた、<strong>「精神症状の重さ」と「社会的・職業的機能の程度」を1〜100の点数で評価する指標</strong>です。

現在はDSM-5に移行し、WHODAS（世界保健機関障害評価尺度）などが使われる場面も増えましたが、依然として多くの医療現場で「患者さんの重症度を直感的に把握する指標」としてGAFスコアが活用されています。

### 📊 GAFスコアの目安（概要）

GAFは点数が高いほど機能が保たれており、低いほど日常生活に著しい支障があることを示します。

| 点数範囲 | 機能のレベル | 患者さんの状態イメージ |
| :--- | :--- | :--- |
| <strong>91 - 100</strong> | 非常に良好 | 多くの問題に対処でき、生活が充実している |
| <strong>71 - 80</strong> | 軽度の障害 | 一時的なストレスで低下するが、社会機能は維持 |
| <strong>61 - 70</strong> | 軽微な症状 | 軽度の社会的機能低下、仕事や対人に支障あり |
| <strong>51 - 60</strong> | 中等度の症状 | 対人関係のトラブルや社会生活にかなりの困難 |
| <strong>41 - 50</strong> | 重篤な症状 | 仕事を続けられない、対人関係が崩壊しがち |
| <strong>31 - 40</strong> | 思考・行動の障害 | 現実検討能力が低下し、生活に支援が必要 |
| <strong>1 - 10</strong> | 持続的な危険 | 自傷・他害の恐れ、自己管理が全くできない |

## 💊 2. 薬剤師がGAFを知っておくべき理由

「GAFは医師がつけるもので、薬剤師には関係ないのでは？」と思われるかもしれません。しかし、薬物療法を最適化する上で、この点数は強力なヒントになります。

### ① 薬の増量が必要か、それとも「機能の回復」が必要か
GAFが低い患者さんに対し、漫然と抗精神病薬を増量しても、GAFスコア（機能）が改善するとは限りません。むしろ、過鎮静（ふらつき、眠気）によってさらにGAFが低下するという悪循環も考えられます。
<strong>「症状は抑えられているが、社会機能が戻らない」</strong>という場合、薬の減量や、生活支援の切り替えが必要なサインかもしれません。

### ② 在宅医療での生活環境の把握
精神科の訪問薬剤管理指導を行う際、GAFが40以下の患者さんは、服薬管理そのものが困難である可能性が高いです。
- <strong>GAF 40台以下</strong>: 自己管理は極めて困難。お薬カレンダーや一包化だけでは不十分で、訪問看護やヘルパーとの連携、配薬回数の検討が必要です。
- <strong>GAF 60以上</strong>: ある程度の自己管理が可能。服薬状況を確認しつつ、自立を促す介入が適しています。

## ⚠️ 3. 評価の際の注意点：GAFは万能ではない

GAFを読み解く際に、薬剤師が忘れてはならない注意点があります。

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px;">
  <strong>💡 薬剤師へのアドバイス</strong><br>
  GAFは、あくまで「その瞬間の評価」です。精神症状には波があります。<br>
  <ul style="margin: 0.5rem 0 0 1.2rem; padding: 0;">
    <li>点数だけで患者さんを決めつけないこと。</li>
    <li>「前回の点数より低いから悪化している」ではなく、「何が生活を阻害しているのか（副作用？症状？環境？）」を患者さんと話すことが大切です。</li>
  </ul>
</div>

## 🌿 4. まとめ：点数の向こう側にいる「人」を見る

GAFスコアは、精神疾患を数値化して理解するための非常に便利なツールですが、あくまで指標の一つです。

私たち薬剤師にとっての最大の武器は、<strong>「服薬という日常行動を通じて、患者さんの生活の変化にいち早く気づけること」</strong>です。カルテのGAFが40点台であっても、薬の飲み忘れが減り、表情が少し明るくなったのであれば、それは立派な「機能の回復」です。

点数という数字だけに振り回されず、患者さん一人ひとりの生活の質（QOL）向上を目指す指標として、GAFを頭の片隅に置いておいてくださいね。明日からの外来や訪問調剤の視点が、きっと少し変わるはずです！
