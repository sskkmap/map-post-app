# 修正アシュワーススケールとは？痙縮評価の基本

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  以前、脳卒中後の痙縮で「バクロフェン錠」が処方されている患者さんの在宅訪問に行った際のことです。ご家族から「最近、足のつっぱりがひどくて歩行訓練が辛そう」という相談を受けました。<br><br>
  私は「薬の副作用かな？」と反射的に疑ってしまいましたが、先輩薬剤師から「まずは主治医と一緒に『修正アシュワーススケール』で今の筋緊張の状態を客観的に評価してみよう」と助言されました。感覚的に「ひどい」「良くなった」と判断するのではなく、共通言語としての「スケール」を持つことの重要性を痛感した出来事でした。
</div>

薬剤師として抗痙縮薬を管理する際、患者さんの「なんとなく」の訴えだけでなく、客観的な評価指標を知っておくことは非常に重要です。今回は、痙縮評価のゴールドスタンダードである<strong>「修正アシュワーススケール（Modified Ashworth Scale: MAS）」</strong>について整理します。

---

## 🔬 1. そもそも「痙縮（けいしゅく）」ってなに？

痙縮とは、中枢神経障害（脳卒中、脊髄損傷、脳性麻痺など）の後遺症でみられる<strong>「筋肉の過度な緊張（筋緊張の亢進）」</strong>のことです。

- <strong>状態</strong>: 筋肉が常に緊張して硬くなり、自分の意思とは無関係に関節が固まってしまいます。
- <strong>弊害</strong>: 可動域制限、歩行困難、痛み、介護負担の増大（着替えや清拭が困難になる）。
- <strong>薬剤師の関わり</strong>: 抗痙縮薬（バクロフェン、チザニジンなど）の適正使用と、効果判定が重要な役割となります。

---

## 📊 2. 修正アシュワーススケール（MAS）の評価基準

MASは、他動的に関節を動かした際の「抵抗感」を0〜4の6段階で評価する指標です。

| スケール | 評価の内容 |
| :--- | :--- |
| <strong>0</strong> | 筋緊張の増加なし（正常） |
| <strong>1</strong> | わずかな筋緊張の増加（関節を動かした終末端で「ひっかかり」やわずかな抵抗を感じる） |
| <strong>1+</strong> | わずかな筋緊張の増加（関節可動域の半分以下に抵抗を感じる） |
| <strong>2</strong> | 全体的に筋緊張の増加（可動域の大半で抵抗を感じるが、他動運動は容易） |
| <strong>3</strong> | 顕著な筋緊張の増加（他動運動が困難） |
| <strong>4</strong> | 関節が強剛（屈曲または伸展位で固まり、他動運動が不可能） |

<div style="background-color: #fffbeb; padding: 1rem; border-radius: 8px; border: 1px solid #fde68a; margin-top: 1rem;">
  <strong>💡 評価のコツ</strong><br>
  関節を動かすときは、<strong>「ゆっくりと（1秒かけて）」</strong>動かすのがルールです。速く動かすと「伸張反射（つっぱる反射）」が誘発され、正確な評価ができません。
</div>

---

## 💊 3. 抗痙縮薬のモニタリングと薬剤師の役割

MASを指標にすることで、抗痙縮薬の用量調節がより論理的になります。

### 代表的な抗痙縮薬の特徴

| 薬剤名 | 主な作用機序 | 留意点 |
| :--- | :--- | :--- |
| <strong>バクロフェン</strong> | GABAB受容体刺激（脊髄抑制） | 眠気、ふらつき、急な減量による離脱症状 |
| <strong>チザニジン</strong> | α2受容体刺激（脊髄抑制） | 血圧低下、眠気、肝機能障害に注意 |
| <strong>ダントロレン</strong> | 筋小胞体からのCa2+放出抑制 | 肝機能検査が必須、全身倦怠感 |

### 薬剤師のアクションプラン
1. <strong>ベースラインの確認</strong>: 服薬開始前のMASスコアを把握する。
2. <strong>定期的評価</strong>: 服薬後のMASを観察し、数値を比較する。
3. <strong>ADL（日常生活動作）との照らし合わせ</strong>: 「数値が1下がった」ことだけでなく、「靴下が履きやすくなった」「排泄介助が楽になった」といった<strong>生活の変化（QOL）</strong>を聴取する。

---

## ⚠️ 4. 注意点：痙縮は「悪」ではない？

臨床現場で非常に重要な視点があります。それは、<strong>「痙縮を完全に取り除くと、かえって動けなくなることがある」</strong>という事実です。

<div style="background-color: #fef2f2; border: 1px solid #fecdd3; padding: 1rem; border-radius: 8px; color: #991b1b;">
  <strong>🚨 臨床現場の落とし穴</strong><br>
  患者さんの中には、<strong>「足の突っ張りを利用して立位を保っている（支えにしている）」</strong>方がいらっしゃいます。抗痙縮薬で筋緊張を緩めすぎると、「膝折れ」を起こして転倒のリスクが高まる場合があります。
</div>

「MASの数値が下がった＝治療成功」と単純化せず、必ず<strong>「今の生活において、その筋肉の緊張がプラスに働いているか、マイナスに働いているか」</strong>をリハビリ職や医師と共有することが、私たち薬剤師に求められる高度な視点です。

---

## 🌿 5. まとめ

修正アシュワーススケール（MAS）は、痙縮の程度を言語化し、多職種と共有するための共通言語です。

- <strong>評価時はゆっくりと</strong>: 伸張反射を出さないように行う。
- <strong>生活の質を指標に</strong>: 数値の変化だけでなく、患者さんのADLの変化をセットで記録する。
- <strong>バランスを見極める</strong>: 痙縮の軽減が、立位や歩行の妨げにならないか注意深くモニタリングする。

最初は慣れない評価かもしれませんが、まずは「手足の硬さ」に注目することから始めてみてください。それが患者さんのより良い薬物療法につながる第一歩となります！
