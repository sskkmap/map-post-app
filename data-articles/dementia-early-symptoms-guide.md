# 家族が気づきたい認知症の初期症状10選

<div style="background-color: #f0f9ff; border-left: 4px solid #0ea5e9; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  ある日の薬局、普段は血圧の薬を淡々と受け取られている70代の患者さんが、その日は「あれ、今日のお薬まだもらっていなかったかしら？」と、全く同じ内容の確認を3回繰り返されました。ご本人にその自覚はなく、ただ少し疲れているだけのように見えました。<br><br>
  数ヶ月後、その方の奥様から「最近、料理の味が変わったと言われる」「同じ物を何度も買ってきてしまう」という相談を受けました。あの日の些細な記憶の揺らぎが、実は認知症の初期兆候だったのではないかと、当時もっと踏み込んでご家族にお声がけすべきだったと痛感した経験があります。

薬局は、患者さんの「小さな変化」に気づける貴重な場所です。今回は、ご家族が相談に来られた際や、日頃の服薬指導で注意深く観察すべき「認知症の初期症状10選」を解説します。

---

## 🔍 なぜ「早期発見」が重要なのか？
認知症は進行性の疾患ですが、アルツハイマー型などでは早期からの薬物療法（抗認知症薬の活用など）や生活環境の調整によって、生活の質を長く維持できる可能性があります。我々薬剤師が「年齢によるもの」とスルーせず、専門医受診への橋渡しをすることが、患者さんとご家族の未来を大きく変えるきっかけになります。

---

## ⚠️ 認知症を疑うべき初期症状10選

「単なる加齢による物忘れ」と「認知症による記憶障害」を分けるポイントは、<strong>「体験そのものを忘れているか否か」</strong>です。

| 分類 | 初期症状のチェックポイント | 薬剤師の視点 |
| :--- | :--- | :--- |
| <strong>記憶</strong> | ① 同じことを何度も尋ねる | 服薬回数や残薬の異常を確認 |
| <strong>記憶</strong> | ② 財布や鍵など物の置き忘れ | 紛失トラブルの有無を聞き取る |
| <strong>記憶</strong> | ③ 約束を忘れる・間違える | 受診予約やイベントの失念 |
| <strong>生活</strong> | ④ 料理の味が極端に変わる | 調味の加減やレシピの変化 |
| <strong>生活</strong> | ⑤ 以前はできた家事ができない | 薬のシートを剥がせない等の器用さ |
| <strong>思考</strong> | ⑥ 計画を立てるのが困難になる | 薬の飲み方やスケジュールの混乱 |
| <strong>思考</strong> | ⑦ 些細なことに怒りっぽくなる | 薬の服用を拒否するなどの行動変化 |
| <strong>思考</strong> | ⑧ 趣味や関心が薄れる | 表情の硬さや意欲の低下 |
| <strong>空間</strong> | ⑨ いつもの道で迷う | 薬局への来局ルートの変化 |
| <strong>思考</strong> | ⑩ 薬の管理ができなくなる | 著しい残薬や飲み残しの急増 |

## 💊 薬剤師が現場で活用できる「気づき」の質問術

患者さん本人やご家族に、「認知症ですか？」と直接聞くことはできません。以下のような、服薬指導の流れで自然に確認できるスクリーニング質問が有効です。

### 🚨 特に注意すべき「服薬状況」の変化
- <strong>残薬のパターンが突然変わった</strong>: 毎週きれいに飲めていた方が、急に半分残るようになった。
- <strong>薬のシートがバラバラ</strong>: 錠剤を取り出す際にシートがボロボロになっていたり、異様な開封のされ方をしている。
- <strong>受診間隔の混乱</strong>: 「まだ薬があるのに病院に行こうとする」あるいは「薬がないのに気づかない」。

## 🤝 専門家としての連携のステップ

もし、これらの症状から「認知症の疑い」を強く感じた場合、薬剤師として取るべきアクションは以下の通りです。

1. <strong>記録を残す</strong>: 薬歴に「記憶障害の疑い」「家事等の変調についての家族の相談」と事実を客観的に記載する。
2. <strong>多職種連携</strong>: かかりつけ医に対して「認知症の可能性について受診時に評価をお願いしたい」と申し送る。
3. <strong>ご家族への支援</strong>: 「加齢による物忘れか、精査が必要なものかを確認するために、一度脳神経内科や物忘れ外来へ相談してみませんか？」と、早期受診のメリットを伝え背中を押す。

## 🌿 まとめ：薬局は「地域のゲートキーパー」

認知症の初期症状を見抜くことは、単なる病気の発見ではありません。<strong>「生活の混乱」を未然に防ぎ、患者さん本人の尊厳を守るための最初の一歩</strong>です。

「また同じことを聞いているな」と感じたとき、その背後には認知症というサインが隠れているかもしれません。我々薬剤師が日々の服薬指導の中で、患者さんの変化に対して誠実に向き合い、必要であれば専門医へ繋ぐ「ゲートキーパー」としての役割を果たすことが、地域医療において非常に重要です。

ぜひ明日からの服薬指導で、患者さんの「いつもと違う」を見逃さないようにしてみてください。
