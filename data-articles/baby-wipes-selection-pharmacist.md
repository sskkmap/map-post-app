# 👶 おしりふきの選び方を薬剤師が解説

<div style="background-color: #fff7ed; border-left: 4px solid #fb923c; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード】</strong><br>
  ある日、外来で「何度軟膏を塗っても子どものおむつかぶれが繰り返す」というお母さんが相談に来られました。患部を見ると、炎症がかなり広がっています。<br><br>
  薬の塗り方や回数を確認しても特に問題はありません。ふと、「今使っているおしりふきは何ですか？」と尋ねてみると、「特売の時にまとめ買いした、香料が強いタイプです」という答えが。その場で成分表を見せてもらうと、肌への刺激になり得る成分が含まれていました。軟膏の効能ばかりに目が行きがちですが、<strong>「汚れを落とす時の刺激」</strong>という物理的要因を見落としていた自分に気づかされた出来事でした。

おむつかぶれ（おむつ皮膚炎）の患者さんに対して、私たちはどうしても「どのステロイドを使うか」「亜鉛華軟膏をどう塗るか」という治療面に集中しがちです。しかし、実は毎日のケアである「おしりふき」の選択こそが、皮膚バリアを守る鍵となります。

今回は、薬剤師として患者さんに指導できる<strong>「おしりふきの選び方」</strong>を徹底解説します。

おしりふきを選ぶ際、単に「価格」や「厚手かどうか」だけで選んでいませんか？皮膚バリアが未熟な乳幼児にとって、おしりふきは「洗浄」と同時に「摩擦」を生むリスクでもあります。

### 🧪 ① 成分：何が入っているかを理解する
おしりふきの成分表示は意外と複雑です。皮膚炎を起こしている患者さんには、以下の成分が含まれていないかを確認するようアドバイスしましょう。

*   <strong>PG（プロピレングリコール）など</strong>: 保湿剤ですが、敏感肌には刺激になる可能性があります。
*   <strong>香料・着色料</strong>: 不要な添加物は、皮膚感作を助長するリスクがあります。
*   <strong>アルコール（エタノール）</strong>: 清涼感はありますが、皮膚の水分を奪い、乾燥を悪化させます。

### ☁️ ② 素材：厚手と薄手の使い分け
「薄手」は経済的ですが、何度も拭く必要があり、結果として摩擦回数が増えます。炎症があるときは「厚手」で、<strong>「こすらず、吸着させる」</strong>ケアが推奨されます。

### 🤲 ③ 物理的摩擦：拭き取るというより「押さえる」
薬剤師として指導すべきは「成分」だけではありません。おしりふきを使う際の「動作」です。ゴシゴシ拭くのは、バリア機能を破壊するのと同じです。

## 📋 2. おしりふき・トラブル別選択ガイド

患者さんに指導する際の目安として、以下の表を参考にしてください。

| 状態・ニーズ | 推奨されるおしりふきの特徴 | 注意点・アドバイス |
| :--- | :--- | :--- |
| <strong>肌が荒れている時</strong> | 水分量が多く、添加物フリーのもの | こすらず、ポンポンと押さえて拭く |
| <strong>乾燥肌・敏感肌</strong> | ノンアルコール・無香料・保湿成分配合 | 拭いた後の乾燥を防ぐため保湿剤を併用 |
| <strong>新生児・低月齢</strong> | 超純水使用、極薄手でも高密度 | 頻繁な排便に対応できるようコスパも考慮 |
| <strong>外出時・携帯用</strong> | 密閉性の高い容器入りのもの | 乾燥して摩擦が強くなる前に交換 |

## 🌿 まとめ：スキンケアの第一歩は「摩擦を減らすこと」

おしりふきは、毎日何回も肌に触れるものです。だからこそ、その小さな刺激の蓄積が皮膚トラブルを大きく左右します。

軟膏を渡して終わりにするのではなく、<strong>「今使っているおしりふきで、赤くなったりしていませんか？」</strong>と一言添えるだけで、患者さんの安心感は大きく変わります。ぜひ、次の服薬指導から活用してみてくださいね。
