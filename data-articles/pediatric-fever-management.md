# 子どもが38℃の熱でも元気なら受診は必要？

<div style="background-color: #f0f9ff; border-left: 4px solid #0284c7; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の夕方、小さなお子さんを連れたお母さんが駆け込んできました。「保育園から呼び出されて熱が38.5℃あるんです。すぐに座薬を使ってもいいですか？それとも夜間救急に行くべきですか？」とひどく動揺されていました。<br><br>
  お子さんの顔色は良く、待合室でアンパンマンの絵本を見て笑っています。当時の私は「38.5℃あれば解熱剤を使って、念のため明日受診してください」とだけ伝えました。しかし、後から「元気なら無理に下げず、様子を見るという選択肢もあったのでは？」と自問自答したことがあります。親の不安を払拭しつつ、医学的な正解を伝えることの難しさを痛感した出来事でした。
</div>

保護者にとって「子どもの発熱」は一大事です。しかし、薬剤師として、数値だけで判断せず「子どもの全身状態」を見る大切さを伝えていく必要があります。

---

## 🌡️ 1. 「熱」と「全身状態」の優先順位

実は、小児科医の多くは<strong>「体温の高さ」よりも「本人の機嫌や活動性」</strong>を重視します。

- <strong>熱があっても元気な場合</strong>: 免疫系がウイルスと戦っている証拠です。水分が取れていて、笑顔があり、いつものように遊べていれば、急いで救急外来を受診する必要性は低いです。
- <strong>熱が低くても注意が必要な場合</strong>: 37.5℃程度でも、ぐったりしている、呼びかけに反応が鈍い、呼吸が苦しそう、水分が全く取れないという場合は危険信号です。

### 受診すべき「危険なサイン」リスト
以下の項目に当てはまる場合は、時間外であっても医療機関への相談が必要です。

| 症状 | 判断のポイント |
| :--- | :--- |
| <strong>意識状態</strong> | 呼びかけに反応しない、目線が合わない、ぐったりして動かない |
| <strong>呼吸</strong> | 呼吸が速い、ゼーゼー・ヒューヒューする、肩で息をしている |
| <strong>水分摂取</strong> | 半日以上おしっこが出ていない、水分を全く受け付けない |
| <strong>けいれん</strong> | 初めてのけいれん、5分以上続くけいれん |
| <strong>発疹</strong> | 数分〜数時間で全身に広がる赤い発疹がある |

## 💊 2. 薬剤師が伝える「解熱剤」との正しい付き合い方

「熱が高い＝すぐに座薬」という考え方は、必ずしも正解ではありません。保護者の方には以下のガイドラインを伝えるのがおすすめです。

<div style="display: grid; grid-template-columns: repeat(2, 1fr); gap: 1rem; margin: 1.5rem 0;">
  <div style="background-color: #fff7ed; padding: 1rem; border-radius: 8px; border: 1px solid #fed7aa;">
    <div style="font-weight: bold; color: #9a3412; margin-bottom: 0.5rem;">使用を推奨するケース</div>
    <ul style="font-size: 0.9em; padding-left: 1.2rem;">
      <li>高熱で本人が眠れず、辛そうにしている</li>
      <li>水分が取れず、脱水が心配なとき</li>
      <li>熱による不快感で泣き止まない</li>
    </ul>
  </div>
  <div style="background-color: #f3f4f6; padding: 1rem; border-radius: 8px; border: 1px solid #e5e7eb;">
    <div style="font-weight: bold; color: #374151; margin-bottom: 0.5rem;">様子見でも良いケース</div>
    <ul style="font-size: 0.9em; padding-left: 1.2rem;">
      <li>熱はあっても、遊んだり笑ったりしている</li>
      <li>水分が取れていて、機嫌が安定している</li>
      <li>夜間、ぐっすりと眠れている</li>
    </ul>
  </div>
</div>

<strong>薬剤師からのアドバイス:</strong>
「解熱剤は熱を下げる薬ではなく、<strong>『辛い症状を和らげてあげるためのサポーター』</strong>です。熱はウイルスを退治するための免疫反応なので、無理に下げすぎないのがコツですよ」と伝えると、保護者も納得されやすいです。

## 🎯 3. 服薬指導の現場で使える「魔法のフレーズ」

保護者の不安を和らげ、かつ過剰な受診を控えてもらうために、以下のフレーズをカルテにメモしておくと便利です。

> 「今夜はこれを使って、朝までゆっくり眠れたら安心ですね。もし明日になっても熱が下がらず、機嫌が悪いようであれば、明日改めて小児科を受診しましょう。」

これにより、<strong>「今日すぐ受診すべきか」という不安を「明日まで様子を見て良い」という安心感</strong>にシフトさせることができます。

---

## 📝 4. まとめ：親の「心配」に寄り添うカウンセリングを

私たち薬剤師にとって大切なのは、薬を渡すだけでなく、<strong>「受診の判断基準」を明確に提示してあげること</strong>です。

1.  <strong>熱の高さよりも「全身状態」を見るように伝える。</strong>
2.  <strong>解熱剤は「熱を下げるもの」ではなく「辛さを取るもの」と定義する。</strong>
3.  <strong>「何があったら受診すべきか」という具体的なアウトラインを示す。</strong>

今回のケースのように、もし保護者がパニックになっている場合は、まず「よくお話ししてくださいましたね、今の様子なら…」と一度受け止めてあげてください。それが、結果として無駄な救急受診を減らし、本当にお子さんの体調を守るための薬剤師の役目です。
