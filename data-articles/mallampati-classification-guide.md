# Mallampati分類とは？気道確保困難の予測に役立つ分類

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の手術前カンファレンスでの出来事です。鎮静下での処置を予定している患者さんのカルテを見ていると、麻酔科医の記録に「Mallampati Class Ⅳ」という記載がありました。当時の私は「Mallampati？ 何かの略語かな？」と深く気に留めていませんでした。<br><br>
  その後、処置中に気道確保が難航し、緊急で補助器具が導入される事態に。その時、先輩から「Mallampati分類を確認していれば、予期せぬ気道トラブルに対する準備や、鎮静薬の投与量調節についてもっと慎重になれたはずだよ」と指摘されました。「ただ薬を準備するだけ」ではなく、患者さんの身体的なリスクを把握することの重要性を痛感した瞬間でした。
</div>

麻酔科医や救急医が日常的に行う「Mallampati（マランパチ）分類」。病棟業務や周術期管理に関わる薬剤師にとっても、患者さんの安全を担保するための大切な知識です。今回は、気道確保困難を予測するこの分類について詳しく解説します。

## 🔍 1. Mallampati分類とは？
Mallampati分類は、<strong>「座位で口を大きく開けて舌を出した際、喉の奥がどの程度見えるか」</strong>によって、気管挿管やマスク換気などの気道確保の難易度を予測する指標です。

口の中の軟口蓋、口蓋垂（のどちんこ）、扁桃柱がどのくらい見えるかを確認し、Class Ⅰ〜Ⅳの4段階に分類します。

| クラス | 視認できる範囲 | 気道確保のリスク |
| :--- | :--- | :--- |
| <strong>Class Ⅰ</strong> | 軟口蓋、口蓋垂、扁桃、咽頭弓がすべて見える | 比較的容易 |
| <strong>Class Ⅱ</strong> | 軟口蓋、口蓋垂、扁桃まで見える | 比較的容易 |
| <strong>Class Ⅲ</strong> | 軟口蓋、口蓋垂の基部までが見える | <strong>困難の可能性あり</strong> |
| <strong>Class Ⅳ</strong> | 軟口蓋のみ見える | <strong>非常に困難な可能性が高い</strong> |

## ⚠️ 2. なぜ薬剤師がこの分類を知るべきなのか？

「気道管理は医師の仕事ではないか？」と思うかもしれませんが、薬剤師がこの情報を知っておくことには大きなメリットがあります。

### 鎮静薬・筋弛緩薬の投与リスクを評価できる
鎮静薬（プロポフォール、ミダゾラム等）や筋弛緩薬を使用する際、Mallampati分類がⅢやⅣの患者さんは、<strong>呼吸抑制や気道閉塞が起こった際に人工換気が困難になるリスク</strong>が高いと予測できます。
* <strong>事前の薬剤準備</strong>: 挿管困難が予想される場合、迅速導入のための薬剤セットを揃える。
* <strong>投与量の調整</strong>: 気道確保に懸念がある場合、鎮静深度を深くしすぎないよう提案する。

### 予期せぬイベントに対する準備
術前訪問などで「この患者さんは気道確保が難しい可能性がある」という認識を薬剤師が持っていれば、緊急時の薬剤（ボスミンや救急セット）の配置場所や、使用薬剤の準備において、より的確なサポートが可能になります。

## 🌿 4. まとめ：薬剤師の視点で安全な医療を

Mallampati分類は、麻酔科医にとってはルーチンワークですが、薬剤師にとっては<strong>「患者さんの安全管理をより一段高めるための指標」</strong>となります。

* <strong>Class Ⅰ〜Ⅱ</strong>: 通常通り慎重に薬剤を管理。
* <strong>Class Ⅲ〜Ⅳ</strong>: 気道確保が難しいリスクを念頭に置き、鎮静時の呼吸状態をより一層注意深く観察する（あるいは医師への事前の情報共有・確認を行う）。

このように、臨床現場で得られる小さな情報を「薬」と結びつけて考えることが、ミスを防ぎ、患者さんの命を守るプロフェッショナルな薬剤師への第一歩です。次回のカルテ閲覧時や術前カンファレンスでは、ぜひMallampati分類にも注目してみてくださいね！
