# SANSとは？統合失調症の陰性症状評価尺度を解説

<div style="background-color: #f8fafc; border-left: 4px solid #64748b; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の臨床エピソード】</strong><br>
  ある日の精神科門前薬局でのこと。長期服用中の患者さんが、「薬を飲んでいるのに、最近なんだかやる気が出ないんだよね」と仰いました。診察室では主治医に「調子は変わりないです」と伝えているようでしたが、服薬指導の様子を見ていると、以前よりも明らかに視線が合わず、会話の反応も鈍くなっているように感じました。<br><br>
  当時は「陽性症状（幻覚・妄想）が落ち着いているから大丈夫」と安易に考えていましたが、後から振り返ると、それが治療のターゲットにすべき「陰性症状」の増悪だったのではないかと反省しました。陽性症状の改善だけでなく、社会機能に直結する陰性症状をどう捉えるか。その際の一つの指標となるのが、今回ご紹介する「SANS」です。
</div>

統合失調症の治療において、薬物療法は欠かせません。しかし、陽性症状が改善しても「意欲が湧かない」「人と関わろうとしない」といった<strong>陰性症状</strong>が残存し、患者さんの社会復帰を阻むケースは非常に多いです。今回は、陰性症状を評価する代表的な尺度「SANS」について解説します。

---

## 🔬 1. SANS（Scale for the Assessment of Negative Symptoms）とは？

SANS（サンズ：陰性症状評価尺度）は、統合失調症の陰性症状を評価するために開発された構造化面接形式の尺度です。

精神科の臨床現場や治験などで、患者さんの精神状態を客観的に評価するために用いられます。私たち薬剤師が直接SANSを使ってスコアリングを行う機会は少ないかもしれませんが、<strong>「どのような視点で症状が評価されているのか」</strong>を知ることは、患者さんの状態変化に気づくための強力な武器になります。

---

## 📊 2. SANSが評価する5つのドメイン

SANSでは、陰性症状を主に以下の5つのカテゴリー（ドメイン）に分けて評価します。これらは、薬剤師が服薬指導の際に観察すべきポイントとも重なります。

| ドメイン | 概要と評価の視点 |
| :--- | :--- |
| <strong>感情の平板化</strong> | 表情が乏しい、声の抑揚がない、感情表現が単調など |
| <strong>意欲・関心の欠如</strong> | 仕事や趣味への意欲低下、身だしなみへの無頓着など |
| <strong>注意力の障害</strong> | 会話の集中力の維持困難、周囲への関心の低下 |
| <strong>失語（無口）</strong> | 会話の量や質が低下する。自発的な発言がない |
| <strong>無快楽症・社会的興味の欠如</strong> | 喜びを感じる能力の低下、人との交流を避けるなど |

## 🔄 4. 陰性症状と陽性症状：治療の捉え方

統合失調症の治療目標は、陽性症状（幻覚・妄想）を抑えることだけではありません。

*   <strong>陽性症状:</strong> 薬物療法（抗精神病薬）で比較的速やかに改善しやすい。
*   <strong>陰性症状:</strong> 薬物療法に加え、精神療法的アプローチや生活支援（デイケアなど）による介入が必要な場合が多い。

もし患者さんが「薬を飲んでも何も変わらない」と訴える場合、それは陽性症状ではなく、評価が難しい「陰性症状」で苦しんでいる可能性があります。

## 🌿 5. まとめ：患者さんの「生活の質（QOL）」を見守る

SANSはあくまで評価尺度ですが、その背景にある「患者さんのつらさ」に寄り添うことが薬剤師には求められています。

1.  <strong>SANSの構成要素を知る:</strong> 感情の平板化や意欲低下が統合失調症の一部であることを理解する。
2.  <strong>日常の観察を大切に:</strong> 服薬指導時のわずかな表情の変化や発言の減りをサインとして捉える。
3.  <strong>副作用との鑑別:</strong> 薬剤性パーキンソニズムによる無表情ではないか注意する。

「最近、少し元気がないように見えるのですが、何かお困りのことはありませんか？」
この一言が、患者さんの孤立を防ぎ、治療継続を支える大きな一歩になります。日々の業務の中で、数値化された評価尺度を意識しつつ、患者さん一人ひとりの「変化」を丁寧にフォローしていきましょう。
