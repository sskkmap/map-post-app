# FASTとは？認知症の進行度を評価する機能評価尺度

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ノート：見落としがちなADLの変化】</strong><br>
  在宅訪問をしているある患者さん。長年、ドネペジルを服用中で認知機能は「横ばい」と記録されていました。しかし、ある時ご家族から「最近、服のボタンをかけるのに時間がかかるようになった」という相談を受けました。<br><br>
  私は「認知症の進行」と思い込んでいましたが、実はその背景には関節痛の影響があったのです。その時、単なる認知機能テストの点数だけでなく、<strong>「日常生活機能（ADL）」がどう変化しているか</strong>という視点で評価する重要性を痛感しました。薬剤師がFAST（Functional Assessment Staging）の概念を知っていれば、もっと早く患者さんのQOL低下のサインに気づけたかもしれません。
</div>

認知症治療において、MMSEやHDS-Rといった知能検査は重要ですが、それだけで患者さんの「暮らし」を把握することはできません。今回は、日常生活動作（ADL）の変化から認知症の進行度を評価する<strong>「FAST」</strong>について、薬剤師視点で深掘りします。

---

## 📋 1. FAST（機能評価尺度）とは何か？

FAST（Functional Assessment Staging）は、アルツハイマー型認知症の進行段階を評価するために作成された尺度です。<strong>「生活の自立度が、どの程度まで保たれているか」</strong>というADLの変化に焦点を当てています。

最大の特徴は、<strong>「失われていく能力」を逆順に追うことで、ステージを判断する</strong>という点です。

### FASTの評価ステージ一覧

FASTはステージ1〜7に分類されます。以下にその概要をまとめました。

| ステージ | 状態の目安 | 薬剤師が着目すべきポイント |
| :--- | :--- | :--- |
| <strong>1-3</strong> | 正常〜軽度認知障害 | 服薬自己管理は概ね可能 |
| <strong>4</strong> | 軽度認知症 | 複雑なIADL（金銭管理など）の困難 |
| <strong>5</strong> | 中等度認知症 | 適切な衣類選択の困難（季節に合わない服） |
| <strong>6a</strong> | 中等度〜重度 | 入浴の介助が必要になる |
| <strong>6b-d</strong> | 重度 | 更衣、排泄、入浴の完全介助が必要 |
| <strong>7</strong> | 末期 | 歩行困難、発語の喪失、摂食困難 |

## 🔍 2. 薬剤師が実務でFASTを活用するメリット

なぜ、医師ではなく薬剤師がFASTの概念を知っておくべきなのでしょうか？それは、<strong>「服薬管理の難易度」を予測できるから</strong>です。

### ① お薬カレンダーの導入判断
ステージ4までは本人の自己管理が可能ですが、ステージ5に入ると「薬を飲んだかどうか忘れる」といったことが頻発します。FASTのステージを知っていれば、「次はカレンダーや一包化を提案しよう」といった先回りのケアが可能になります。

### ② 副作用モニタリングの精度向上
例えば、コリンエステラーゼ阻害薬を服用中の患者さんがステージ6へ移行し、急激に歩行が不安定になった場合、「認知症の進行」と判断する前に、「薬剤によるパーキンソニズムや過鎮静」を疑う視点が持てます。

## ⚠️ 4. 注意点：FASTは「魔法のツール」ではない

FASTは非常に有用ですが、以下の点には注意が必要です。

*   <strong>身体疾患の除外</strong>: 膝が悪くて歩けないのか、認知症の影響で歩かないのか。FASTはあくまで認知症による機能低下を評価するものなので、身体的な疾患の影響は切り分けて考える必要があります。
*   <strong>多職種連携のツールとして使う</strong>: FASTの評価結果を「薬剤師の独断」で処方変更に繋げるのではなく、ケアマネジャーや医師との情報共有ツールとして活用しましょう。「最近、FASTのステージ6aに近い状況が見受けられますが、入浴介助の状況はいかがですか？」というように、他職種との共通言語として用いるのがベストです。

---

## 🌿 5. まとめ：認知症ケアは「暮らし」の評価から

認知症治療において、薬剤師は「薬の専門家」であると同時に、患者さんの<strong>生活リズムの番人</strong>です。

FASTという指標を持つことで、「単に薬を渡す」だけでなく、「今の生活スタイルに最適な服薬支援方法は何か？」を論理的に提案できるようになります。ぜひ、明日からの服薬指導で、患者さんの「日々の暮らしぶり」を観察する視点に取り入れてみてください。
