# 高齢者の体重減少が続くときに考えられる原因とは？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ヒヤリハット】</strong><br>
  薬局で、ずっと「高血圧」で通院されている80代の患者さんが、久しぶりに来店された際、驚くほど痩せていて顔色が優れないことに気づきました。「最近、食欲がなくて…」とおっしゃるものの、特に新しい病気の診断は受けていないとのこと。<br><br>
  「加齢のせいかしら」と流しかけましたが、お薬手帳を確認すると、数ヶ月前から便秘薬が追加され、利尿薬の用量も増えていました。実は、その利尿薬による食欲不振と、便秘薬の飲み過ぎによる消化管への負担が積み重なっていたのです。もっと早く気づいていれば、と胸が痛んだ経験でした。
</div>

高齢者の体重減少は、単なる「加齢」と片付けられがちですが、実際には<strong>「見逃してはいけない重大なサイン」</strong>であることが多々あります。今回は、薬剤師として高齢者の体重減少に直面した際、どのような視点で介入すべきか、その考え方を整理します。

---

## 🔍 1. 体重減少の原因を探る「MEALS」の考え方

高齢者の原因不明の体重減少を考える際、医師やケアマネジャーと連携するために役立つのが<strong>「MEALS」</strong>というフレームワークです。

| 頭文字 | 項目 | 薬剤師の確認ポイント |
| :--- | :--- | :--- |
| <strong>M</strong> | Medication（薬剤） | 食欲低下や嘔気をもたらす薬はないか？利尿薬やNSAIDsなど |
| <strong>E</strong> | Emotional（感情） | うつ状態はないか？独居による孤食の影響は？ |
| <strong>A</strong> | Anorexia/Alcohol | 食欲不振や飲酒習慣、または口腔内のトラブルはないか？ |
| <strong>L</strong> | Late-life paranoia | 認知症による食事摂取能力の低下や、妄想等の影響はないか？ |
| <strong>S</strong> | Swallowing/Social | 嚥下障害はないか？経済的な困窮や孤立（社会的要因）はないか？ |

特に私たちが真っ先にチェックすべきは、<strong>「Medication（薬剤）」</strong>です。

---

## 💊 2. 薬剤性が疑われる「体重減少」のチェックリスト

高齢者は多剤服用（ポリファーマシー）が多いため、薬が直接的・間接的に食欲や体重に影響を与えているケースが少なくありません。

## ⚠️ 3. 疾患・身体的要因のサイン（レッドフラッグ）

薬剤調整だけで解決しない場合、背後に隠れた疾患を見抜く必要があります。

<div style="background: #fff5f5; border: 1px solid #feb2b2; padding: 1.5rem; border-radius: 8px;">
  <h3 style="margin-top: 0; color: #c53030;">こんな時はすぐに受診勧奨を！</h3>
  <ul style="color: #9b2c2c;">
    <li><strong>急速な体重減少</strong>：短期間で5%以上の減少がある。</li>
    <li><strong>嚥下障害・口腔内異常</strong>：入れ歯が合わない、飲み込みに時間がかかる。</li>
    <li><strong>全身的な炎症・悪性腫瘍の疑い</strong>：微熱が続く、寝汗をかく。</li>
    <li><strong>内分泌異常</strong>：甲状腺機能亢進症など、食事量は変わらないのに痩せるケース。</li>
  </ul>
</div>

## 🛠 4. 薬剤師にできる「明日からの実務」

1. <strong>お薬手帳の履歴を追う</strong>: 過去半年〜1年で「新しい薬が増えていないか」「増量されていないか」を必ず確認します。
2. <strong>「食事量」よりも「活動量」と「体重」</strong>: 患者さんに「最近、お洋服が緩くなっていませんか？」「歩くスピードは落ちていませんか？」と具体的に尋ねます。
3. <strong>多職種連携へのつなぎ役</strong>: 薬剤性の疑いが強い場合は、疑義照会を通じて処方見直しを提案します。一方で、栄養不足が疑われる場合は、かかりつけ医や訪問看護師へ栄養指導の依頼を促しましょう。
