# 🩺 MUSTとは？低栄養リスクをスクリーニングする評価法

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【現場の薬剤師からの一言】</strong><br>
  在宅訪問で担当している高齢の患者さん。最近、食欲が落ちている様子でしたが、ご家族からは「歳だから食べる量が減るのは仕方ないわよね」と聞き流されていました。しかし、ある時訪問すると以前よりあきらかに服が緩くなっており、顔色も優れません。<br><br>
  「なんとなく痩せた」という主観的な判断ではなく、もっと客観的に介入の必要性を判断できる指標はないか……そんな時に出会ったのが「MUST（マスト）」というツールです。これを使って評価したところ、重症度の高さが数値として可視化され、医師への栄養剤提案が非常にスムーズになりました。

今回は、臨床の現場や在宅医療で即戦力となる<strong>「MUST（Malnutrition Universal Screening Tool）」</strong>について、その評価手順と薬剤師としての関わり方をまとめます。

---

## 🔍 1. MUST（低栄養ユニバーサルスクリーニングツール）とは？

MUSTは、英国栄養学会（BAPEN）が開発した、低栄養リスクを評価するための簡便なスクリーニングツールです。病院や介護施設だけでなく、地域薬局の現場でも活用しやすいのが最大の特徴です。

以下の<strong>3つのステップ</strong>で点数を計算するだけで、患者さんの「低栄養リスクの程度」と「治療方針」を導き出せます。

### 📊 MUSTの評価ステップ（表）

| ステップ | 項目 | 内容 | スコア |
| :--- | :--- | :--- | :--- |
| <strong>1</strong> | <strong>BMIの算出</strong> | BMI 20以上 | 0 |
| | | BMI 18.5〜19.9 | 1 |
| | | BMI 18.5未満 | 2 |
| <strong>2</strong> | <strong>体重減少</strong> | 過去3〜6ヶ月の減少率 5%未満 | 0 |
| | | 過去3〜6ヶ月の減少率 5〜10% | 1 |
| | | 過去3〜6ヶ月の減少率 10%超 | 2 |
| <strong>3</strong> | <strong>急性疾患</strong> | 急性疾患による食欲低下が5日以上見込まれる場合 | 2 |

## 💊 3. 薬剤師ができる「低栄養介入」のポイント

MUSTを用いて低栄養リスクが高いと判断された場合、薬剤師として以下のような提案が可能になります。

### ① 薬剤による食欲不振の確認
服用している薬の中に、副作用として食欲不振や味覚障害を引き起こすものがないか確認します。
*   <strong>チェック項目</strong>: 副作用（消化管症状）、多剤併用による飲み込みづらさ、口腔内の乾燥、薬の苦味など。

### ② 栄養剤の処方提案
経口摂取が十分にできない患者さんに対し、医師へ栄養補助食品（経腸栄養剤など）の提案を行います。
*   <strong>提案例</strong>: 「BMIが低下しており低栄養スコアが2点以上です。食欲不振も強いため、まずは半固形タイプの栄養剤を試してみませんか？」

### ③ 服薬方法の工夫
*   「食事と一緒に飲まなければならない」という思い込みが、逆に食事量を減らしている場合があります。服薬タイミングの調整や、飲み込みやすい剤形変更（散剤・ゼリー化）を医師に提案するのも薬剤師の重要な役割です。

## 💡 まとめ：客観的な指標で「守れる命」がある

薬剤師の実務において、MUSTのような客観的評価ツールを持つことは、<strong>「医師への適切な疑義照会」</strong>や<strong>「多職種連携」</strong>において強力な武器になります。

「なんとなく調子が悪そう」で終わらせず、MUSTで数値を可視化することで、早期の介入ができれば、患者さんのADL維持やフレイル予防に大きく貢献できます。ぜひ明日からの業務で、患者さんのBMIや体重変化を意識的にチェックしてみてくださいね！
