# Killip分類とは？急性心筋梗塞の重症度評価を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ヒヤリハット】</strong><br>
  かつて、病棟担当になったばかりの頃、急性心筋梗塞（AMI）の患者さんの処方を見て「なぜこの人はACE阻害薬が導入されず、利尿薬が増量されているんだろう？」と不思議に思っていました。<br><br>
  先輩薬剤師に尋ねると「患者さんのKillip分類（キリップ分類）を確認した？」と一言。恥ずかしながら当時はこの分類を知らず、カルテの記載を読み飛ばしていたのです。心不全症状の重症度を把握していなければ、適切な薬学的介入や副作用のモニタリングなど到底できないのだと痛感した出来事でした。

循環器系の疾患は、一歩間違えると命に関わるため、薬剤師にも「心臓の今の状態」を正しく把握するスキルが求められます。今回は、急性心筋梗塞（AMI）の評価指標として汎用される<strong>Killip（キリップ）分類</strong>について深掘りしていきましょう。

## 🔬 1. Killip分類とは？心不全の重症度を測るものさし

Killip分類は、<strong>急性心筋梗塞（AMI）を発症した患者さんに対して、臨床症状から「どれくらい心不全が進んでいるか」を評価する指標</strong>です。

心筋梗塞によって心臓のポンプ機能が低下すると、肺に水が溜まったり（肺うっ血）、血圧が維持できなくなったりします。この状態を分類することで、<strong>「予後がどれくらい悪いか」</strong>を予測し、治療方針を決定する重要な道しるべとなります。

### Killip分類の定義一覧

| 分類 | 臨床所見 | 心不全の状態 |
| :--- | :--- | :--- |
| <strong>Class I</strong> | 肺ラ音なし、心音異常なし | 心不全なし |
| <strong>Class II</strong> | 肺野の下方にラ音（湿性ラ音）、S3（奔馬調律）あり | 軽症〜中等症 |
| <strong>Class III</strong> | 急性肺水腫（肺全体にラ音） | 重症 |
| <strong>Class IV</strong> | 心原性ショック（血圧低下、冷汗、乏尿） | 最重症 |

<br>

## ⚠️ 2. 各クラス別の薬剤管理・注意点

Killip分類は、単なるラベルではなく<strong>「今の薬物療法の適応」</strong>と密接にリンクしています。

### Class I：全身管理期
特別な心不全治療薬は不要ですが、再発予防に向けた抗血小板薬やスタチン、適応があればACE阻害薬・β遮断薬の導入が検討されます。

### Class II〜III：利尿薬の出番
肺うっ血があるため、ループ利尿薬（ラシックス®など）が第一選択になります。
- <strong>モニタリングポイント</strong>: 低カリウム血症、脱水、血清クレアチニンの上昇。

### Class IV：心原性ショック（緊急対応）
血管収縮薬や強心薬（ドブタミンなど）による循環補助がメインとなります。この段階で「普段飲んでいる血圧降下薬はどうするか？」を即座に判断・確認するのが薬剤師の腕の見せ所です。

## 🎯 4. 薬剤師が現場で役立てるための3ステップ

明日からの臨床でKillip分類を活かすためのポイントをまとめました。

1.  <strong>カルテの「経過記録」を確認する</strong>
    *   回診時に「ラ音の有無」や「Killip分類」が記載されているか探しましょう。
2.  <strong>「ラ音」の記載があったら要注意！</strong>
    *   聴診で「湿性ラ音（Crackles）」が聞こえる＝心不全が進行中。利尿薬の変更や追加がないか要チェックです。
3.  <strong>心不全薬の「導入・増量・中止」に根拠を持つ</strong>
    *   なぜその薬が必要なのかをKillip分類と関連づけて考える習慣をつけると、医師とのコミュニケーションが格段にスムーズになります。

## 🌿 まとめ：心臓の声を薬学的知見で読み解こう

Killip分類は、急性心筋梗塞という危機的な状況下にある患者さんの「重症度を測る共通言語」です。

- <strong>Class I</strong>：予後は比較的良好。再発予防がメイン。
- <strong>Class II〜III</strong>：肺うっ血へのアプローチ（利尿薬の管理）が必須。
- <strong>Class IV</strong>：生命維持が最優先。全薬剤の再評価が必要。

「なんとなく処方されている薬」を「理由があって処方されている薬」として解釈できるようになると、薬剤師としての臨床力は一段階上がります。ぜひ、次の病棟業務や外来服薬指導の際には、患者さんのKillip分類を意識してみてくださいね！
