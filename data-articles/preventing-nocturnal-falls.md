# 「夜間の転倒を防ぎたい」薬剤師が考える安全対策アイテム

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ヒヤリハット】</strong><br>
  ある夜、在宅訪問している患者さんのご家族から緊急の連絡がありました。「昨夜、トイレに起きた際にベッドから転落して骨折してしまった」と。ご本人もショックを受けていましたが、ご家族の「夜中にお薬を飲んだ後、フラフラしていたのかも…」という言葉に、私は言葉を失いました。<br><br>
  その患者さんには不眠のために睡眠導入剤が処方されていましたが、服用後の離床に関する指導が十分にできていなかったのです。「薬を渡して終わり」ではなく、生活の動線まで含めて考える必要性を痛感した出来事でした。
</div>

薬剤師にとって、患者さんのQOLを守ることは非常に重要です。特に高齢者の転倒は、ADL（日常生活動作）の低下に直結します。今回は、薬剤師が現場で提案できる「安全対策」と、お薬以外からアプローチする「環境づくり」について深掘りします。

## 🔍 1. なぜ「薬」が転倒リスクになるのか？

夜間の転倒リスクを考える際、薬剤師がまず確認すべきは「<strong>睡眠薬の副作用</strong>」です。

| 薬剤の種類 | 転倒リスク要因 | 薬剤師の確認ポイント |
| :--- | :--- | :--- |
| ベンゾジアゼピン系 | 筋弛緩作用によるふらつき | 服用後の離床タイミングの確認 |
| 非ベンゾジアゼピン系 | 持ち越し効果（翌朝の眠気） | 常用量内か、減量の余地があるか |
| 抗ヒスタミン薬（市販薬含む） | 鎮静作用・口渇 | 併用による過度な鎮静がないか |
| 抗高血圧薬 | 起立性低血圧によるめまい | 血圧日誌での夜間低血圧チェック |

特に、半減期の長いお薬が翌朝まで残っている場合、あるいは夜中に追加服用してしまうケースでは、転倒リスクが飛躍的に高まります。

---

## 🛠️ 2. 薬剤師が現場で提案したい「安全対策アイテム」

お薬の調整に加え、患者さんの「動線」を物理的にサポートするアイテムを提案することは、薬局薬剤師の大切な役割です。

### 💡 視認性を高めるアイテム
夜間の転倒の多くは「足元が暗い」ことが原因です。
* <strong>人感センサー付き足元ライト</strong>: 足元を照らすことで、ふらつきがあっても障害物に気づきやすくなります。
* <strong>蓄光テープ</strong>: 階段の端やベッドのフレームに貼ることで、暗所でも距離感が掴みやすくなります。

### 🛌 離床を安全にするアイテム
* <strong>ベッドサイドの手すり</strong>: ベッドから起き上がる際の支点を作り、ふらつきを軽減します。
* <strong>滑り止めマット</strong>: トイレの入り口や洗面所など、特に転びやすい場所に配置します。

## 🚀 4. まとめ：トータルケアとしての安全対策

薬剤師にできることは、お薬の整理（処方提案）だけではありません。患者さんの生活環境を想像し、<strong>「どんな時に転びやすいか？」を具体的にシミュレーションして提案すること</strong>が、信頼される薬剤師の第一歩です。

1. <strong>お薬の副作用確認</strong>（減薬、服薬時間の見直し）
2. <strong>夜間動線の環境チェック</strong>（ライト、手すりの活用）
3. <strong>継続的な声かけ</strong>（体調変化やヒヤリハットの共有）

「夜間にお薬を飲んでフラフラする」という訴えは、重大なサインです。ぜひ、次回の訪問や服薬指導の際、ご家庭の様子を少しだけ詳しく聞いてみてください。その些細な気づきが、患者さんの骨折を防ぐ一番の「お守り」になるはずです。
