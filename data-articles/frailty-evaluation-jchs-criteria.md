# フレイル評価（J-CHS基準）とは？高齢者医療で押さえたいポイント

<div style="background-color: #f8fafc; border-left: 4px solid #94a3b8; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の実務エピソード】</strong><br>
  訪問診療の現場で、「先生、この患者さん、最近少しふらつきが多くて……」と相談を受けた時のことです。処方内容を確認すると、長年服用されている抗不安薬や降圧薬が並んでいました。<br><br>
  当時、私は「薬の副作用」ばかりに目がいっており、「フレイル」という視点が欠けていました。先輩薬剤師から「ただ減薬するだけでなく、その方のフレイルの状態（身体的・精神的予備能の低下）を評価して、生活機能の維持を最優先に考えよう」と教えられ、J-CHS基準を用いた評価の重要性を痛感しました。薬剤師として薬を管理するだけではなく、患者さんの生活そのものを評価する大切さを学んだ瞬間でした。
</div>

高齢化社会において、薬剤師が患者さんの「フレイル（虚弱）」を見抜き、適切な服薬支援を行うことは、QOL維持に直結します。今回は、臨床現場で最も活用されている<strong>「J-CHS基準」</strong>について、わかりやすく解説します。

---

## 🔬 1. J-CHS基準とは？フレイルを測る「5つのものさし」

J-CHS基準（日本版CHS基準）は、日本人の特性を考慮して改訂されたフレイルの評価基準です。以下の5つの項目から判断します。

| 評価項目 | 内容（簡易版） |
| :--- | :--- |
| <strong>① 体重減少</strong> | 半年で2〜3kg以上の意図しない減少 |
| <strong>② 筋力低下</strong> | 握力の低下（男性28kg未満、女性18kg未満） |
| <strong>③ 疲労感</strong> | わけもなく疲れたような感じがする（過去2週間） |
| <strong>④ 歩行速度</strong> | 1.0m/秒未満の低下 |
| <strong>⑤ 身体活動</strong> | 軽い運動や体操をしていない、または定期的なスポーツをしていない |

※これら<strong>5項目のうち3項目以上該当で「フレイル」</strong>、<strong>1〜2項目該当で「プレフレイル」</strong>と判定します。

---

## ⚠️ 2. 薬剤師が押さえるべきフレイルとポリファーマシーの関係

フレイル状態にある高齢者は、薬の代謝・排泄能力が低下しており、<strong>「ポリファーマシー（多剤併用）」</strong>による副作用のリスクが非常に高まります。

## 🎯 3. 実務で役立つ「フレイルチェック」の聞き取り術

全ての項目を厳密に検査するのは困難ですが、服薬指導の会話の中に自然に組み込むことができます。

*   <strong>「最近、重い荷物を持つのがしんどくなっていませんか？」</strong> → 握力低下の確認
*   <strong>「散歩や買い物に出るペース、少しゆっくりになったと感じますか？」</strong> → 歩行速度の確認
*   <strong>「急に体重が減ったなんてことはありませんか？」</strong> → 栄養状態（低栄養）の確認

### 💡 アドバイスの切り口
もし「プレフレイル」を疑うサインがあれば、単に薬を減らす提案だけでなく、<strong>「たんぱく質の摂取」や「簡単なスクワット」を医師と共有</strong>し、生活習慣への介入を提案しましょう。

---

## 🚀 4. まとめ：薬以外の視点を持つ薬剤師へ

フレイル（J-CHS基準）を意識することは、<strong>「薬を渡す専門家」から「患者さんの生活を支える専門家」へステップアップするための重要なツール</strong>です。

1.  <strong>5つの項目（体重、握力、疲労感、速度、活動）を頭に入れる。</strong>
2.  <strong>処方カスケードに注意し、副作用の兆候を早期に察知する。</strong>
3.  <strong>多職種連携を通じ、栄養・運動・薬の三方向からアプローチする。</strong>

日々の処方箋チェックに加え、「この薬は本当に今の患者さんの活動レベルに合っているか？」という視点を一秒だけプラスしてみてください。それが、患者さんの健康寿命を延ばす大きな一歩になります。
