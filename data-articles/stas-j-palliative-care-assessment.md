# STAS-Jとは？日本版緩和ケア評価尺度の使い方

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場エピソード：数値化することの難しさ】</strong><br>
  在宅医療に携わり始めた頃、がん患者さんの訪問薬剤管理指導で悩んだことがあります。ご家族から「最近、本人がなんだか辛そうで……」と相談された際、自分なりに傾聴はしていましたが、その苦痛がどれほど改善しているのかを、客観的に評価する術を持っていませんでした。<br><br>
  多職種カンファレンスで「患者さんの苦痛が軽減しています」と伝えても、「どの程度？」と聞かれると言葉に詰まる日々。そんな時に出会ったのが、緩和ケアの質を評価する<strong>「STAS-J」</strong>でした。これを導入してからは、チーム内での共通言語が増え、薬の調整根拠を明確に示せるようになりました。
</div>

緩和ケアに関わる薬剤師にとって、患者さんのQOL（生活の質）向上は最大のミッションです。しかし、患者さんの主観的な「辛さ」をどう記録し、他職種と共有するかが課題になりがちです。今回は、客観的指標であるSTAS-Jについて解説します。

## 🔍 1. STAS-J（日本版緩和ケア評価尺度）とは？

STAS-J（Support Team Assessment Schedule - Japanese version）は、緩和ケアを受けている患者さんの身体的、心理的、社会的、霊的（スピリチュアル）な苦痛を、<strong>「0（問題なし）〜 4（極めて重篤）」</strong>の5段階で評価する指標です。

もともと英国で開発されたSTASを日本向けに調整したもので、以下の特徴があります。

*   <strong>多職種で共有しやすい</strong>: 誰が評価しても判断基準が一定になるよう定義されています。
*   <strong>経時的変化がわかる</strong>: 1週間ごとの点数をつけることで、介入の効果（薬の調整など）がグラフで見える化できます。
*   <strong>緩和ケアの質を評価できる</strong>: ケアプランの改善や、薬物療法の見直しの指標になります。

---

## 📊 2. STAS-Jの評価項目（8つの視点）

STAS-Jでは、以下の8つの項目を評価します。薬剤師として特に注目すべき項目を整理しました。

| 評価項目 | 薬剤師の関わりポイント |
| :--- | :--- |
| <strong>① 身体的症状</strong> | 疼痛、悪心、便秘、呼吸困難などの制御状況 |
| <strong>② 心理的症状</strong> | 不安、抑うつ状態と、抗不安薬・抗うつ薬の効果 |
| <strong>③ 患者の自立度</strong> | 倦怠感や筋力低下によるADLの低下度合い |
| <strong>④ 家族の苦痛</strong> | 介護負担感、ご家族への薬の説明や精神的サポート |
| <strong>⑤ 病状の理解</strong> | 疾患や今後の予後に関する認識のズレ |
| <strong>⑥ 治療への同意</strong> | 服薬アドヒアランスや治療方針に対する納得感 |
| <strong>⑦ 介護者との関係</strong> | 在宅環境での良好なケア体制の確認 |
| <strong>⑧ スピリチュアルな苦痛</strong> | 存在意義や死への不安（傾聴と受容） |

<div style="background-color: #eff6ff; padding: 1.5rem; border-radius: 8px; border: 1px solid #bfdbfe; margin: 1.5rem 0;">
  <strong>💡 薬剤師のヒント</strong><br>
  全ての項目を完璧に埋める必要はありません。まずは<strong>「身体的症状」</strong>と<strong>「治療への同意（服薬状況）」</strong>から評価を始め、多職種連携ツールとしてカルテに記載していくことが大切です。
</div>

## 🚀 3. 在宅現場での活用・ステップアップ

STAS-Jを導入する際は、以下のステップで進めるとスムーズです。

### ステップ1：現状の苦痛を「点数化」する
まずは、現在の患者さんの症状をSTAS-Jの基準（0〜4）に当てはめてカルテに残します。「なんとなく辛そう」を「身体的症状：3点」と記録することで、医師へ「オピオイドの増量を検討しませんか？」と提案する強力な根拠になります。

### ステップ2：多職種と共有する
訪問看護師やケアマネジャーと「STAS-Jのスコア」を共有します。「今週は身体が3点から2点に下がりましたが、家族の不安が2点から3点に上がっています」といった会話が可能になり、チーム全体でどのケアを優先すべきか明確になります。

### ステップ3：振り返り（カンファレンス）
STAS-Jのスコア推移を見ながら、週単位で薬の評価を行います。
*   点数が改善：現在の薬物療法が奏功している。
*   点数が改善しない：原因（薬剤の副作用、心理的背景、社会的要因など）を再検討する。

---

## ⚠️ 4. 注意点：STAS-Jは「目的」ではなく「手段」

大切なのは、STAS-Jの点数を取ること自体ではなく、<strong>「その点数から、患者さんの苦痛をどう取り除くか」</strong>を考えることです。

*   <strong>機械的にならない</strong>: スコアが高いからと言って機械的に薬を増やすのではなく、必ず患者さん自身との対話を重視してください。
*   <strong>スピリチュアルな側面</strong>: 薬剤師は「薬」だけでなく、患者さんの「語り」を受け止めることで、心理的・スピリチュアルな点数を下げられる可能性も持っています。

## 🌿 まとめ：客観的指標を味方に、一歩先行く緩和ケアを

STAS-Jを活用することで、曖昧だった「緩和ケアの質」が可視化され、私たち薬剤師の専門性もチームに伝わりやすくなります。

「この患者さん、少し楽になった気がするけど、本当にこれでいいのかな？」と迷ったときは、ぜひSTAS-Jというモノサシを使ってみてください。客観的なデータは、医師や看護師との信頼関係を深め、何より<strong>「苦しんでいる患者さんへの最適な医療」</strong>を届けるための強力な武器になります。
