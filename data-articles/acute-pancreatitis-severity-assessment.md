# 急性膵炎重症度判定基準とは？重症度評価のポイント

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の臨床現場エピソード】</strong><br>
  在宅医療で患者様ご自宅に訪問した際に、患者さんが背部痛を訴えており発熱や嘔吐もあるとのことでこれは緊急で医師にもてもらわないとダメな症状だとすぐに分かりました。在宅医師へ緊急に連絡緊急往診ののち大きな病院へと搬送されました。その後急性膵炎と診断されました。<br><br>
 その話を先輩薬剤師に話したときに「その後病院でどんな処置が起こると思う？」と尋ねられました。<br>  
  消化器内科の医師は、急性膵炎重症度判定基準ってゆうので重症度を判断する。
  それによって治療薬も変わるよと教えてもらいました<br><br>
  当時は、患者さんの全身状態と重症度がどうリンクしているのかを理解していませんでした。
</div>

急性膵炎は、病勢が急激に変化することが特徴です。今回は、臨床現場で今なお活用されている<strong>「急性膵炎重症度判定基準」</strong>について、薬剤師として押さえておくべき評価のポイントを解説します。

---

## 🔍 なぜ「重症度判定」が重要なの？

急性膵炎の予後は、いかに早期に重症度を把握し、適切な輸液や集中治療を開始できるかにかかっています。この判定基準を理解しておくことで、<strong>「なぜ点滴を使わないで経腸管栄養が行われているのか」「なぜ抗生剤が使われているのか？」</strong>といった、医師の処方の意図をより深く理解できるようになります。

---

## 💡 まずは知っておきたい！急性膵炎の代表的な症状

急性膵炎の症状は、とても激しい痛みが特徴です。大きく分けて「主な痛み」「その他の症状」「危険なサイン」の3つがあります。

### どんな症状が出るの？（症状のまとめ）

| 症状の種類 | 具体的な様子（どんな感じ？） | 特徴・ポイント |
| :--- | :--- | :--- |
| **🚨 激しいお腹と背中の痛み** | **みぞおちの激痛**: 突然、または時間をかけてひどく痛みます。じっとしていても治まりません。<br>**背中への広がり**: 痛みが背中や腰まで突き抜けるように広がります。 | 胃潰瘍（いかいよう）や胆石（たんせき）の痛みと間違われることがあります。<br>※前かがみで膝を抱える姿勢（エビのような姿勢）をとると、少し痛みが和らぐことがあります。 |
| **⚠️ その他の症状** | **吐き気・嘔吐**: 激しい痛みのせいで、何度も吐いてしまうことがあります。<br>**発熱**: 炎症のせいで熱が出ます。<br>**お腹の張り**: お腹がパンパンに張って、食欲がなくなります。 | 痛みに伴って、いろいろな不調が体全体に現れます。 |
| **🆘 重症化のサイン<br>（すぐに救急車！）** | **意識がぼんやりする**<br>**冷や汗・強いめまい**（血圧が急に下がるため）<br>**皮膚や白目が黄色くなる**（黄疸：おうだん） | 状態が急激に悪くなっている証拠です。命に関わるため、一刻も早い治療が必要です。 |

**💡 要注意な人**
お酒をたくさん飲む人や、過去に「胆石（たんせき）がある」と言われたことがある人は、急性膵炎になりやすいので特に注意が必要です。

---

## 📊 重症度判定の基本

この基準では、以下の2つの軸で重症度を分類します。

| 分類 | 評価指標 | 薬剤師の着目ポイント |
| :--- | :--- | :--- |
| <strong>重症度（段階）</strong> | 予後因子（9項目）の数 | 3点以上で「重症」。輸液量や介入の緊急度 |
| <strong>重症度（病態）</strong> | 造影CTによる膵壊死・浸出液 | Grade2以上で「重症」。感染予防のための抗生剤投与や外科的介入の必要性 |

### 1. 予後因子（9項目）の評価
以下の項目のうち、3点以上あれば「重症」と判定されます。臨床検査値がこの基準をクリアしているか、電子カルテで確認する習慣をつけましょう。

<div style="background-color: #f0f9ff; padding: 1.5rem; border-radius: 8px; border: 1px solid #bae6fd;">
  <ul style="margin: 0; padding-left: 1.2rem;">
    <li><strong>SIRS（全身性炎症反応症候群）の診断基準3項目以上</strong></li>
    <li><strong>Base Excess（BE）≦ -3 mEq/L、またはショック（収縮期血圧≦80mmHg）</strong></li>
    <li><strong>BUN ≧ 40mg/dl（もしくはCr ≧ 2mg/dl）、または乏尿（輸液後も１日尿量が400ml 以下であるもの）</strong></li>
    <li><strong>総Ca 値 ≦ 7.5 mg/dl</strong></li>
    <li><strong>PaO2 ≦ 60mm Hg (room air)、または呼吸不全（人工呼吸器管理を必要とするもの）</strong></li>
    <li><strong>LDHが基準値上限の２倍以上（約450〜500 U/L以上）</strong></li>
    <li><strong>CRP ≧ 15 mg/dL</strong></li>
    <li><strong>血小板数 ≦ 10万/mm3</strong></li>
    <li><strong>SIRS 診断基準における陽性項目数 ≧ 3</strong></li>
    <div style="background-color: #f0f9ff; padding: 1.5rem; border-radius: 8px; border: 1px solid #bae6fd;">
    SIRS 診断基準項目<br>
    （１）体温＞38℃または＜36℃<br>
    （２）脈拍＞90 回/分<br>
    （３）呼吸数＞20 回/分またはPaCO2＜ 32 mm Hg<br>
    （４）白血球数＞12,000 /mm3もしくは＜4,000 /mm3、または10%超の幼若球の出現</strong></li></div><br>
    <li><strong>年齢 ≧ 70 歳</strong></li>
  </ul>
</div>


### 2. 造影CTの評価（CT画像でのお腹のダメージチェック）

CT検査（体の断面を見る検査）の画像を使って、「炎症がどこまで広がっているか」と「膵臓のダメージ（血流が悪くなっている部分）」の2つを点数で評価します。

#### ① 炎症の広がり（どこまで腫れているか？）
| 広がりの範囲（カッコ内は専門用語） | 点数 |
| :--- | :--- |
| **膵臓のすぐ近くにとどまっている**（前腎傍腔） | 0点 |
| **少し離れたところまで広がっている**（結腸間膜根部） | 1点 |
| **腎臓の下の方まで広く広がっている**（腎下極以遠） | 2点 |

#### ② 膵臓のダメージの広がり（血流が悪い部分はどれくらいか？）
※膵臓を「右側（頭）」「真ん中（体）」「左側（しっぽ）」の3つのブロックに分けて判定します。
| ダメージの範囲 | 点数 |
| :--- | :--- |
| **1つのブロックだけ**、または膵臓の周りだけ | 0点 |
| **2つのブロック**にまたがっている | 1点 |
| **2つのブロック全体**、またはそれ以上 | 2点 |

💡 **ここがポイント！**
①と②の点数を合計したものが「CT Grade（グレード）」になります。これが**Grade 2以上になると「重症」**と判断され、より注意深い治療が必要になります。

## 💡 薬剤師が現場でチェックするポイント

薬剤師は、患者さんの治療が安全で効果的に進むように、主に「痛み止め」「ばい菌をやっつける薬（抗菌薬）」「栄養のとり方」の3つのポイントを確認します。

### 薬剤師の3つのチェックポイント

| 分野 | 薬剤師が確認・提案すること | その理由（なぜそうするの？） |
| :--- | :--- | :--- |
| **① 痛み止め（鎮痛薬）** | • 点滴で使える「アセトアミノフェン」を基本にする。<br>• 痛みの強さに合わせて「NSAIDs（痛み止め）」を追加・変更。<br>• 痛みがひどい場合は「医療用麻薬（オピオイド）」を使うことも。 | 急性膵炎の痛みは非常に激しく、ずっと続きます。痛みをしっかり取ることが、患者さんの体力を守るためにとても重要だからです。 |
| **② 抗菌薬（抗生剤）** | • **軽症**: 原則、使いません。<br>• **重症**: 使っても効果がはっきりしないため、予防としては使いません。<br>• **例外**: 胆石（たんせき）が原因で、胆管炎などを合併している場合は積極的に使います。 | 膵炎は「自分の酵素で自分が溶ける」ことで起こるため、最初はばい菌がいません。重症でも「念のための抗菌薬」は効果がないと分かってきたからです。 |
| **③ 栄養のとり方** | • 栄養は「点滴」よりも、できるだけ早く「腸」からとる（経腸栄養）ようにサポートする。<br>• 発症から72時間（3日）以内には経腸栄養を始めることが推奨されている。 | 腸を使わないと、腸が弱ってしまい、腸の中のばい菌が体の中に入りやすくなってしまいます。腸を使うことで、腸のバリア機能を守ることができます。 |

**📌 薬剤師の目線**
「痛みがしっかり取れているか」「不必要な抗菌薬が使われていないか」「いつから腸を使った栄養補給が始まるか」を医師や看護師と相談しながら、患者さんをサポートします！

## ⚠️ まとめ

急性膵炎重症度判定基準（2008）は、単なる医学的な分類ではありません。<strong>患者さんが今、どのステージの危機に直面しており、どのような治療（輸液、抗菌薬、栄養管理）を優先すべきかを判断するためのツール</strong>です。

日々の業務の中で、「この患者さんのLDHが高いな」「CRPが下がってきたな」といった小さな変化を、重症度判定基準と照らし合わせて評価してみてください。それが、薬剤師としての信頼性を高める第一歩となります。
