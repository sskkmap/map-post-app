# 親が同じ話を何度もする…認知症の始まり？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験談】</strong><br>
  ある日、薬局で定期処方の患者さんの娘さんからこんな相談を受けました。「最近、母が同じ話を1日に何回もしてくるんです。これって認知症の始まりでしょうか？」。<br><br>
  私はつい「年齢を重ねれば誰でも少しずつ……」と、その場を収めるような回答をしてしまいました。しかし、後日その患者さんが受診した際、実は「レビー小体型認知症」の初期症状だったことが発覚。もう少し早く専門医への受診を促せていれば、と深く反省した経験があります。
</div>

患者さんやご家族と接する機会の多い私たち薬剤師にとって、このような「サイン」に気づくことは、早期治療への架け橋になります。今回は、加齢による「良性のもの忘れ」と「認知症の兆候」の見分け方、そして実務で役立つ対応のポイントをまとめました。

## 🧠 1. 単なる「もの忘れ」と「認知症」の違い

「同じ話を繰り返す」という症状は認知症の代表的な兆候の一つですが、判断には注意が必要です。以下の表で違いを確認してみましょう。

| 特徴 | 加齢によるもの忘れ | 認知症（軽度認知障害含む） |
| :--- | :--- | :--- |
| <strong>体験の想起</strong> | 一部を忘れる | 体験全体を忘れる |
| <strong>ヒント</strong> | ヒントがあれば思い出す | ヒントがあっても思い出せない |
| <strong>自覚症状</strong> | 忘れている自覚がある | 忘れている自覚がない |
| <strong>日常生活</strong> | 支障はない | 支障が出始める |
| <strong>進行</strong> | 進行しない（急激ではない） | 時間とともに確実に悪化する |

## 🔍 2. 薬剤師がチェックすべき「4つのレッドフラッグ」

服薬指導の際、以下の兆候が併発していないかを確認してみてください。これらが重なる場合は、専門医への受診をお勧めするサインになります。

1.  <strong>「話の内容」ではなく「体験自体」が消えている</strong>: さっき話したことではなく「話したという事実」が抜け落ちている。
2.  <strong>以前と性格が変わった</strong>: 怒りっぽくなった、逆に無気力になった（アパシー）。
3.  <strong>時刻や場所の認識が曖昧</strong>: 「今が朝か夜かわからない」「季節感がない」。
4.  <strong>薬の管理ができなくなった</strong>: 飲んだかどうかわからない、何度も重複して服用しようとする。

## 💊 3. 認知症治療薬の基本知識：薬剤師が知っておくべきこと

現在、認知症治療において頻用される主な薬剤は以下の通りです。

| 薬剤名（一般名） | 特徴 | 主な注意点 |
| :--- | :--- | :--- |
| <strong>ドネペジル</strong> | アセチルコリンエステラーゼ阻害薬（定番） | 吐き気、下痢、徐脈に注意 |
| <strong>ガランタミン</strong> | 認知機能改善に加え、行動・心理症状にも期待 | 食欲低下に注意 |
| <strong>リバスチグミン</strong> | パッチ製剤（貼り薬）で安定した効果 | 皮膚かぶれ、貼り忘れに注意 |
| <strong>メマンチン</strong> | NMDA受容体拮抗薬（興奮を抑える） | 眠気、めまいに注意 |

<div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe;">
  <strong>💡 服薬指導のヒント</strong><br>
  認知症の患者さんにとって「薬を飲み忘れる」ことは日常茶飯事です。家族に「カレンダーに印をつける」「一包化して日付を印字する」など、視覚的にわかりやすい工夫を提案するだけで、服薬遵守率は大きく向上します。
</div>

## 🤝 4. 家族の不安にどう寄り添うか

「同じ話を何度もしてくる」という家族のストレスは非常に大きいです。薬剤師として、以下の姿勢で対話することが大切です。

*   <strong>否定しない</strong>: 「さっきも聞きましたよ」はNG。一度は受け止めてあげましょう。
*   <strong>怒らない</strong>: 患者さん自身も「何かおかしい」という不安の中にいます。
*   <strong>専門家へ繋ぐ</strong>: 「一度、もの忘れ外来などで相談してみると、お母様も安心されるかもしれませんよ」と、"受診＝認知症と確定させるもの"ではなく"安心を得るためのもの"として提案しましょう。

## 🌿 5. 終わりに：早期発見の「きっかけ」になるために

「同じ話を繰り返す」という訴えは、家族からのSOS信号です。その背景に、抑うつ症状や、あるいは特定の薬剤の副作用（ベンゾジアゼピン系など）が隠れていることもあります。

私たち薬剤師は、薬のプロであると同時に、患者さんの生活の変化をいち早くキャッチできる「身近なサポーター」です。この記事が、皆さんの日々の服薬指導のヒントになれば幸いです。

まずは今日、投薬口で患者さんやご家族の話を、いつもよりほんの少しだけ長く聞いてみることから始めてみませんか？
