# 認知症と物忘れの違いをわかりやすく解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場体験談】</strong><br>
  ある日の午後、長年降圧薬を取りに来られている患者さん（70代）が、「最近、薬を飲んだかどうか思い出せなくて困っている」と相談されました。以前は自己管理がしっかりできていた方だけに、ご家族も心配されています。<br><br>
  単なる「加齢による物忘れ」なのか、それとも「認知症の始まり」なのか。窓口での短い会話だけで判断するのは非常に勇気がいることです。しかし、漫然と「年齢のせいですよ」と流してしまい、早期発見の機会を逃すことは避けなければなりません。この記事では、私たち薬剤師が服薬指導の現場で「どこに注目すべきか」を整理しました。
</div>

患者さんから「物忘れがひどくて……」と訴えられたとき、まずは「それが生活にどの程度支障をきたしているか」を確認することが、鑑別の第一歩となります。

## 🧠 1. 「加齢による物忘れ」と「認知症」の決定的な違い

両者の大きな違いは、<strong>「体験した出来事のすべてを忘れるか、一部を忘れるか」</strong>という点です。

| 特徴 | 加齢による物忘れ | 認知症の症状 |
| :--- | :--- | :--- |
| <strong>忘れる内容</strong> | 体験の一部（晩ごはんのメニューなど） | 体験の全体（食事をしたこと自体） |
| <strong>自覚症状</strong> | 「忘れた」という自覚がある | 「忘れた」という自覚がない（病識の欠如） |
| <strong>生活への支障</strong> | ほとんどない | 日常生活に支障が出る |
| <strong>進行スピード</strong> | とても緩やか | 徐々に悪化する |
| <strong>ヒントの提示</strong> | ヒントがあれば思い出せる | ヒントがあっても思い出せない |

### 💡 見極めのチェックポイント
*   <strong>「ヒント」で思い出せるか？</strong>
    「昨日の朝ごはんは何を食べましたか？」と聞いたとき、「えーっと何だったかな……あ、そういえばパンだった！」と戻る場合は「加齢性健忘」の可能性が高いです。一方で、そもそも食べたこと自体を認識していない場合は注意が必要です。
*   <strong>「生活の質（QOL）」は保たれているか？</strong>
    薬の管理、買い物の支払い、近所の付き合いなど、日常生活が自立できているかを観察してください。

## ⚠️ 3. 認知症の背景に「薬剤性」がないか確認を

「認知機能が低下している」と思ったとき、意外と見落としがちなのが<strong>「薬剤の影響」</strong>です。特に高齢者では、以下の薬が認知機能を一時的に低下させることがあります。

*   <strong>抗コリン作用の強い薬</strong>: 第1世代抗ヒスタミン薬、一部の抗うつ薬、過活動膀胱治療薬など
*   <strong>ベンゾジアゼピン系睡眠薬・抗不安薬</strong>: ふらつきや転倒、記憶障害の原因になります。

「最近ぼーっとしている気がする」という訴えがあれば、処方内容を精査し、医師へ減薬や切り替えの提案（疑義照会）を行うことも、薬剤師の大切な役割です。

## 🌿 4. まとめ：私たちは「気づきのゲートキーパー」

認知症と物忘れの鑑別を最終的に行うのは医師ですが、<strong>薬局でのちょっとした変化に気づけるのは薬剤師です。</strong>

*   <strong>加齢によるもの</strong>: 体験の一部を忘れる、自覚がある、日常生活に支障がない。
*   <strong>認知症の疑い</strong>: 体験の全体を忘れる、自覚がない、生活に支障がある。

患者さんの「忘れっぽい」という言葉を鵜呑みにせず、背景にある「生活の様子」を丁寧に聞くことで、適切な介入につなげていきましょう。もし「おかしいな」と感じたら、次回の処方箋受付時に医師へ「ご家族が飲み忘れを心配されているようです」と情報を共有するだけでも、大きな一歩になります。

日常の服薬指導の中で、患者さんの「いつもと違う」を見逃さないプロフェッショナルであり続けたいですね。
