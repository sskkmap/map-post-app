# Braunwald分類とは？不安定狭心症の重症度評価を解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の臨床現場エピソード】</strong><br>
  循環器内科の門前薬局で働いていた頃、「昨日から胸が苦しい感じがして……」と来局された患者さんがいました。心電図検査の結果、すぐに急性冠症候群（ACS）の疑いで緊急入院となったのですが、その時のカルテを見て衝撃を受けたのが「Braunwald分類」の記載でした。<br><br>
  当時の私は、「不安定狭心症＝すぐ手術」といった大雑把なイメージしかなく、なぜ医師が細かく重症度を分類し、抗血小板薬や抗凝固薬の強弱を調整しているのかを深く理解できていませんでした。この分類を理解して初めて、医師がなぜその薬を選んだのか、その裏にある「リスク評価」の論理が見えてきたのです。
</div>

循環器疾患の管理において、薬剤師が「どの程度のリスクがあるのか」を把握することは、服薬指導の質を大きく向上させます。今回は、不安定狭心症の評価指標として広く用いられる「Braunwald分類」について、実務的な視点から深掘りします。

---

## 🔍 1. Braunwald分類とは？

Braunwald分類（ブラウンワルド分類）は、<strong>不安定狭心症の重症度を評価するための臨床的な分類指標</strong>です。心筋梗塞や心臓突然死のリスクを予測し、適切な治療介入のタイミングを決定するために非常に重要です。

この分類は、以下の3つの要素を組み合わせて評価します。

1.  <strong>重症度（Severity）</strong>：どの程度の労作で胸痛が出るか。
2.  <strong>臨床状況（Clinical Circumstances）</strong>：心外性か、発作後か、入院中か。
3.  <strong>心電図変化（ECG）</strong>：発作時のST-T変化の有無。

これらを組み合わせることで、低リスクから高リスクまでを層別化し、治療の緊急度を判断します。

---

## 📊 2. Braunwald分類の評価基準一覧

不安定狭心症の評価は、以下の基準で行われます。

| 分類 | 項目 | 内容の目安 |
| :--- | :--- | :--- |
| <strong>重症度（Class）</strong> | Class I | 新規または増悪する狭心症（安静時痛なし） |
| | Class II | 亜急性安静時狭心症（1ヶ月以内） |
| | Class III | 急性安静時狭心症（48時間以内） |
| <strong>臨床状況（Type）</strong> | Type A | 二次性（貧血、感染、頻脈等が原因） |
| | Type B | 原発性（特発的） |
| | Type C | 心筋梗塞後（発症2週間以内） |

<div style="background-color: #fffbeb; border: 1px solid #fde68a; padding: 1rem; border-radius: 8px; margin: 1.5rem 0;">
  <strong>💡 薬剤師のための補足：</strong><br>
  治療方針を決める際は、上記の分類に加えて「ST変化の有無」が重要です。ST低下やT波の陰転化が伴う場合、より予後が不良と判断され、ヘパリンや強力な抗血小板薬（P2Y12阻害薬など）の使用が積極的に検討されます。
</div>

## 💊 3. リスク分類と薬剤師の実務ポイント

Braunwald分類を用いてリスクが高いと判断された場合、薬物療法には以下のような特徴が出てきます。

### 高リスク群（Class III, Type Cなど）への薬物療法
*   <strong>抗血小板薬の二剤併用（DAPT）</strong>: アスピリン＋P2Y12阻害薬（クロピドグレルやプラスグレルなど）の早期開始。
*   <strong>抗凝固療法の強化</strong>: ヘパリン持続点滴等による血栓形成抑制。
*   <strong>薬剤師の視点</strong>: 出血リスクのモニタリングが最優先。皮下出血、歯肉出血、血尿などのサインを服薬指導時に確認し、必要に応じて患者さんに注意喚起を行います。

### 低〜中リスク群へのアプローチ
*   <strong>内服薬の最適化</strong>: β遮断薬、カルシウム拮抗薬、硝酸薬による心筋酸素消費量の抑制。
*   <strong>薬剤師の視点</strong>: アドヒアランスの向上と、副作用（特にβ遮断薬による徐脈や、硝酸薬による頭痛）への対応を丁寧にフォローします。

---

## ⚠️ 4. 薬剤師が知っておくべき「予後のサイン」

臨床現場では、医師がBraunwald分類で「高リスク」と判断している患者さんに対し、薬剤師は以下の情報を収集・報告することが推奨されます。

1.  <strong>胸痛の頻度と持続時間</strong>: 「以前より長く続くようになった」「安静時にも出るようになった」という変化は、分類を悪化させる（重症化）サインです。
2.  <strong>併用薬の確認</strong>: 他科から処方されている薬や、市販薬（NSAIDsなど）の中には、血圧上昇や血栓リスクに影響を与えるものがあります。
3.  <strong>バイタルサイン</strong>: 狭心症の増悪時には血圧や心拍数の変動が起こりやすいため、お薬手帳の履歴と現在の症状の乖離を確認しましょう。

---

## 🌿 5. まとめ：分類を治療の「地図」にする

Braunwald分類は、単なる医学的な分類ではありません。<strong>「患者さんの心臓が今、どの程度危険な状態にあり、医師がどのような治療（カテーテル治療や薬物療法）を優先しているか」</strong>を示すための地図です。

*   <strong>重症度（Class）が高い</strong> → 積極的な侵襲的治療（PCIなど）や強力な薬物療法が行われる可能性大。
*   <strong>心筋梗塞後（Type C）である</strong> → 再梗塞予防のための厳格な服薬管理が必要。

私たちが「この薬はなぜ処方されたのか？」という疑問を、Braunwald分類という視点から読み解くことで、より精度の高い薬学的管理を提供できます。

日々の業務の中で、「この処方は重症度が高いから、副作用のモニタリングを強化しよう」と判断できる薬剤師を目指していきましょう。
