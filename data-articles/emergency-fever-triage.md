# 夜中に熱が出たら救急へ行くべき？

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師のヒヤリハット体験】</strong><br>
  ある日の夜間当番中、薬局に「子どもが39度の熱を出して泣き止まない。今すぐ救急車を呼ぶか、夜間診療所に行くか迷っている」という緊急の電話が入りました。焦る親御さんの声を聞き、私もつい「すぐに病院へ行った方がいいかもしれません」と不安を煽るような回答をしてしまいました。<br><br>
  しかし、冷静に状況を聞くと水分は取れており、意識もはっきりしている様子。結局、その後すぐに診察を受けましたが、医師からは「帰宅して様子見で十分」という判断。無用な深夜の受診は、患者さんにも医療現場にも負担をかけます。この一件で、<strong>「ただ不安に寄り添うだけでなく、医学的なトリアージの視点を持って助言する重要性」</strong>を痛感しました。
</div>

薬剤師の元には、夜間や休日に「熱が出たけどどうしたらいい？」という相談が寄せられます。その際、的確な判断基準を伝えることは、医療リソースを守ることにもつながります。今回は、現場で使える「受診判断のガイド」をまとめました。

---

## 🛑 1. 救急受診が必要な「レッドフラッグサイン」

まずは、夜間であっても迷わず救急外来や＃8000（小児救急電話相談）へ連絡すべきサインを把握しておきましょう。

| チェック項目 | 具体的な症状・状態 |
| :--- | :--- |
| <strong>意識状態</strong> | 呼びかけへの反応が鈍い、ぐったりしている、呼びかけても目が合わない |
| <strong>呼吸</strong> | ゼーゼーと苦しそう、呼吸数が異常に多い、鼻翼を広げて呼吸している |
| <strong>水分摂取</strong> | 水分が全く取れず、おしっこが半日以上出ていない（脱水症状） |
| <strong>けいれん</strong> | 初めてのけいれん、5分以上続くけいれん、繰り返すけいれん |
| <strong>外観</strong> | 顔色が明らかに悪い（蒼白・チアノーゼ）、生後3ヶ月未満の赤ちゃんの発熱 |

<div style="background-color: #fff1f2; border: 1px solid #fecdd3; padding: 1rem; border-radius: 8px; color: #9f1239; margin: 1.5rem 0;">
  <strong>⚠️ 注意：生後3ヶ月未満の乳児</strong><br>
  生後3ヶ月未満の赤ちゃんが発熱した場合、感染症が重症化しやすいため、<strong>「夜間でも迷わず受診」</strong>が原則です。迷わず病院へ誘導してください。
</div>

## 🩺 2. 自宅で様子を見て良いケースとは？

一方で、以下のような場合は、翌朝の診療時間まで自宅で安静にして様子を見ることが推奨されます。

- 水分がしっかり摂れていて、おしっこも出ている。
- 熱があっても、機嫌よく遊んだり眠ったりできている。
- 呼吸が落ち着いていて、顔色も悪くない。

### 💡 薬剤師からのアドバイス：自宅ケアの鉄則
1. <strong>水分補給</strong>: OS-1などの経口補水液を少量ずつ頻回に。
2. <strong>環境調整</strong>: 室温を適切に保ち、布団で着込みすぎない（解熱時は汗をかくので着替えをこまめに）。
3. <strong>睡眠優先</strong>: 無理に熱を下げるために起こす必要はありません。

## 🎯 4. まとめ：薬剤師が果たすべき役割

「救急へ行くべき？」という相談は、患者さんにとって<strong>「安心感の提供」</strong>を求めるサインでもあります。

1. <strong>まずは傾聴</strong>: 不安な気持ちを否定せず受け止める。
2. <strong>レッドフラッグ確認</strong>: 緊急度を客観的に評価する（上記表を参考に）。
3. <strong>受診のハードルを下げる/上げる</strong>: 不必要な受診には自宅ケアを提案し、必要な場合は迷わず受診を促す。
4. <strong>電話相談の紹介</strong>: 地域の「＃8000（小児救急電話相談）」を必ず案内する。

夜間の電話対応一つで、患者さんの不安は大きく変わります。私たち薬剤師が適切なトリアージの知識を持つことで、医療現場全体の適正化に貢献していきましょう。
