# MADRSとは？うつ病の重症度を評価する尺度をわかりやすく解説

<div style="background-color: #f8fafc; border-left: 4px solid #3b82f6; padding: 1rem 1.5rem; margin-bottom: 1.5rem; border-radius: 0 8px 8px 0;">
  <strong>【薬剤師の現場ノート】</strong><br>
  ある日、外来で「最近、薬を飲んでもなかなか気分が晴れない」と話される患者さんがいらっしゃいました。私は「副作用かな？それとも効果不十分？」と悩み、お薬手帳の履歴を追うことしかできませんでした。<br><br>
  後日、医師から「この患者さんはMADRSの点数が横ばいで、今は治療の正念場なんだ」と教わりました。薬剤師の私が「薬の飲み忘れの有無」だけでなく、<strong>「治療の指標（スケール）」</strong>という視点を持つことで、患者さんの苦しみに、より深く寄り添えるようになると気づかされた瞬間でした。
</div>

臨床現場で耳にする機会が増えた「MADRS（Montgomery-Åsberg Depression Rating Scale：マッドス）」。この記事では、うつ病の評価尺度として重要視されているMADRSについて、薬剤師が知っておくべきポイントを解説します。

---

## 🧐 1. MADRS（マッドス）とは？

MADRSは、<strong>うつ病の重症度を測定するために開発された臨床評価尺度</strong>です。医師が患者さんの状態を「10個の項目」で点数化し、うつ症状の変化を客観的に把握するために使われます。

### MADRSで評価される10項目
それぞれ0〜6点で評価され、合計点数（0〜60点）で重症度を判断します。

| 評価項目 | 内容の概要 |
| :--- | :--- |
| <strong>憂うつな気分</strong> | 悲しみや絶望感など |
| <strong>悲観的な思考</strong> | 将来への不安や後悔など |
| <strong>自殺の考え</strong> | 自殺願望の有無 |
| <strong>眠りの障害</strong> | 睡眠の質や持続時間 |
| <strong>食欲の低下</strong> | 食事量の減少 |
| <strong>集中困難</strong> | 思考や作業への集中力 |
| <strong>無気力</strong> | 物事への意欲や関心 |
| <strong>感情の欠如</strong> | 感情的な反応の乏しさ |
| <strong>緊張感</strong> | 内面的な張り詰めや焦燥感 |
| <strong>悲嘆</strong> | 感情を外に出す力 |

---

## 📈 2. HAM-D（ハミルトンうつ病評価尺度）との違い

これまで長く使われてきた「HAM-D」と、近年好まれる「MADRS」には明確な違いがあります。

<div style="overflow-x: auto;">

| 特徴 | HAM-D（ハミルトン） | MADRS（マッドス） |
| :--- | :--- | :--- |
| <strong>主な対象</strong> | 身体症状を含めて包括的に評価 | <strong>精神症状の変化に敏感</strong> |
| <strong>項目数</strong> | 17〜21項目（やや多い） | <strong>10項目（シンプル）</strong> |
| <strong>メリット</strong> | 長年のデータ蓄積がある | <strong>短期間の治療反応性が高い</strong> |
| <strong>現場の傾向</strong> | 複雑な症例向け | <strong>薬の効果判定・経過観察向け</strong> |

</div>

<div style="background-color: #eff6ff; padding: 1rem; border-radius: 8px; border: 1px solid #bfdbfe; margin-top: 1.5rem;">
  <strong>💡 薬剤師の視点</strong><br>
  MADRSは「精神症状の変化」を鋭敏に拾い上げます。そのため、<strong>SSRIやSNRIなどの抗うつ薬を開始・増量した際の「治療反応性（薬が効いているか）」を評価するのに非常に適している</strong>と言えます。
</div>

## 💊 3. 薬剤師が実務で活かす「MADRS」の知識

「医師がつける点数だから関係ない」と思っていませんか？実は、服薬指導やモニタリングにおいて以下の点が非常に有用です。

### ① 「睡眠」と「食欲」の点数変化に注目する
MADRSの項目には「睡眠」と「食欲」が含まれています。患者さんの主訴が「眠れない」「食欲がない」という場合、それが「薬の副作用」なのか、「うつ病自体の症状（MADRSのスコアが高い状態）」なのかを見極めるヒントになります。

### ② 治療の「フェーズ」を理解する
点数が高い状態から少しずつ下がっていく過程は、患者さんにとっても精神的な支えになります。「今はまだ治療の調整期間（スコアが高め）ですから、焦らずに飲み続けましょう」という言葉は、MADRSという客観的指標の裏付けがあることで、より説得力を増します。

### ③ 「自殺の考え」という項目があることの認識
MADRSには「自殺の考え」という項目があります。患者さんが「死にたい」と口にするのは、MADRSのスコアが非常に高い状態を示唆している場合があります。早期に疑義照会や医師へのフィードバックを行うべきサインとして、この項目が評価されているという事実は知っておくべきです。

---

## ⚠️ 4. 注意点：あくまで「目安」であることを忘れない

MADRSは非常に有用なツールですが、以下の点には注意が必要です。

*   <strong>スコアは絶対ではない</strong>: 患者さんの主観的な苦しさと、スコアが必ずしも一致するとは限りません。
*   <strong>身体疾患の除外が必要</strong>: 甲状腺疾患など、うつ症状を呈する身体疾患がある場合、単純にMADRSだけでうつ病と判断することはできません。

<div style="background-color: #fef2f2; border: 1px solid #fca5a5; padding: 1rem; border-radius: 8px; color: #991b1b; margin-top: 1rem;">
  <strong>🚨 薬剤師へのアドバイス</strong><br>
  点数だけで患者さんを判断せず、<strong>「患者さん自身の言葉」と「MADRSの傾向」を掛け合わせて考える</strong>ことが大切です。「最近、少しは眠れるようになりましたか？」といった、MADRSの評価項目に関連した具体的な質問を投げかけることで、治療の進捗をより的確に把握できます。
</div>

---

## 🌿 5. まとめ

MADRSは、うつ病治療の「現在地」を教えてくれる地図のようなものです。

1.  <strong>MADRSはうつ病の「重症度」を評価する指標である。</strong>
2.  <strong>HAM-Dよりも精神症状の変化を捉えやすく、薬の効果判定に適している。</strong>
3.  <strong>薬剤師は「睡眠」や「食欲」などの項目を意識することで、服薬指導の質が向上する。</strong>

日々、患者さんと向き合う中で「この患者さんは今、どのあたりにいるのかな？」と想像する。その想像力を補完してくれるのが、こうした評価尺度です。ぜひ、次回の服薬指導から、少しだけ視点を広げてみてくださいね！
