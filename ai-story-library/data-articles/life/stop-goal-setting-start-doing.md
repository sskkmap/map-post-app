# 「計画倒れ」が止まらない人へ：あえて今日から「目標を捨てる」技術

「今年こそは英語をマスターする」「今月中にこのプロジェクトを完遂させる」。
私たちは、新しい季節や週の始まりに、いつも立派な目標を立てます。しかし、その結末は決まって「三日坊主」か「自己嫌悪」。なぜ私たちは、計画を立てるたびに自滅してしまうのでしょうか。

結論から言います。それは、あなたの「意志が弱い」からではありません。単に「目標」という名の「巨大なモンスター」を一度に食べようとしているからです。

### 壮大な目標は「脳の敵」である

脳は、あまりに大きな抽象的な目標を与えられると、フリーズするようにできています。「英語をマスターする」という言葉は、脳にとってはあまりに広大で、どこから手をつけていいのかわかりません。結果として脳は「今はまだ準備が整っていない」と言い訳を作り、先延ばしという名の防御壁を築きます。

目標を立てることの最大の弊害は、達成感の先取りです。私たちは「目標を立てた」という時点で、脳内でわずかな報酬系が働き、少しだけ満足してしまいます。その結果、肝心の「行動」がおろそかになるのです。

もしあなたが計画倒れを繰り返しているなら、今日から**「目標」を捨てる**ことをお勧めします。

### 「タスクの細分化」という名の外科手術

目標を捨てる代わりに、やるべきは「タスクの超細分化」です。

「本を1冊読む」という目標は捨ててください。代わりに「1ページだけ開く」というタスクに変えます。「英語をマスターする」という目標は捨て、「単語を1つだけ調べる」に変えるのです。

ここで重要なのは、**「達成可能な最小単位」まで分解する**ことです。
目標が「山を登ること」だとしたら、必要なのは山頂を眺めることではなく、今の自分の足元にある「一歩分の地面」だけを見ることです。

### 即時完了が「ドーパミン」を呼ぶ

なぜ小さな行動が重要なのか。それは、人間が「完了」によってのみ、脳に快感物質であるドーパミンを供給できるからです。

大きな目標を立てると、完了するまでに時間がかかりすぎます。しかし、タスクを数分で終わる単位まで分解すれば、1時間に何度も「完了」という成功体験を味わうことができます。

「ああ、また今日も何もしなかった」ではなく、「今日は5回も小さなタスクを完了させた」という実績が、あなたに自信を与え、次の行動を促すガソリンになります。

### 今すぐできる「たった一つの小さな行動」

人生を動かすのは、壮大な計画ではありません。今、この瞬間にできる「1分以内の動作」の積み重ねだけです。

*   ブログを書くのが面倒なら、「PCを開く」だけをする。
*   ジムに行くのが辛いなら、「ウェアを着る」だけをする。
*   企画書が進まないなら、「タイトルを1行書く」だけをする。

結果は後からついてきます。まずは「完了」という心地よい感覚を、自分に毎日プレゼントしてください。

今日から、高すぎる目標という重荷を下ろしましょう。そして、足元にある小さな一歩に集中してください。遠くの頂上を見つめて立ち尽くすよりも、まずは目の前の地面を掘り進める人だけが、気づいたときには絶景の中に立っているのです。

さあ、スマホを置いて、最初のタスクを1分で終わらせに行きませんか。
